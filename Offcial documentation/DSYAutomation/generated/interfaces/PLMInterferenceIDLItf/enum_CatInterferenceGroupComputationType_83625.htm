CatInterferenceGroupComputationType (Enumeration)

enum CatInterferenceGroupComputationType {
  catInterferenceGroupComputationTypeAllAgainstAllInGroup1,
  catInterferenceGroupComputationTypeGroup1AgainstGroup2
}
Interference Specification groups.

The interference may have followings types of groups:
Values:
CatInterferenceGroupComputationTypeAllAgainstAllInGroup1,
The computation takes into account all entities of first group versus all others.
CatInterferenceGroupComputationTypeGroup1AgainstGroup2,
The computation takes into account all entities of first group versus all entities of second group.
