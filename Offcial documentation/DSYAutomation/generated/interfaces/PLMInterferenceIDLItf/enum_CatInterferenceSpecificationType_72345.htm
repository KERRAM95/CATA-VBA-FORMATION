CatInterferenceSpecificationType (Enumeration)

enum CatInterferenceSpecificationType {
  catInterferenceSpecificationTypeNone,
  catInterferenceSpecificationTypeClash,
  catInterferenceSpecificationTypeClearance,
  catInterferenceSpecificationTypeClashWithoutContact
}
Interference Simulation Specification type of computation for the standard specification.

Values:
catInterferenceSpecificationTypeNone
No use of standard specification.
It means that the interference simulation uses:
- engineering connection specification: see method with the option "No Check" unchecked (see: catInterferenceSpecificationTypeEngCnxNoCheck in ).
- or knowledge rules specification (see method: ).
catInterferenceSpecificationTypeClash
The standard specification is used and it is a clash + contact computation.
catInterferenceSpecificationTypeClearance
The standard specification is used and it is a clearance + contact + clash computation.
catInterferenceSpecificationTypeClashWithoutContact
The standard specification is used and it is a clash + contact computation and only interferences of type "Clash" are kept.

By default, Interference Simulation is created with the type: catInterferenceSpecificationTypeClash
