DPCOperation (Object)

Represents a user operation.
Role: A User Operation is a set of tasks performed on a technological object. It can be V6 command, CAA command, VBA command, external application or operation defined into the DPC product. An operation contains a set of variables: the inputs entered by the user and the outputs produced by the operation.
