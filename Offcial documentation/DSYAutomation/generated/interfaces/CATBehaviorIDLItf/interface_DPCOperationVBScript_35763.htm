DPCOperationVBScript (Object)

Represents the VBScript DPC operation.
Role: The VB script operation is designed to run a VBScript (catvbs or CATScript) or a VBA macro from a VBA project. This interface derives from . It enables the manipulation of the variables of the VBScript operation.
Method Index
GetInput Returns the value of one available input of the operation.
GetInternal Returns one available io of the operation.
GetOutput Returns the value of one available output of the operation.
PutInternal provide output of one available io of the operation.
PutOutput Valuates an available output of the operation.
TestInput Tests if the operation's input is set or not.
TestInternal Test for one available output of the Operation.
TestOutput Tests if the operation's output is set or not.

Returns:
