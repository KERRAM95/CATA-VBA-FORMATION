StrMaterialMngt (Object)

Object to manage Structure Functional Modeler functions material.
Role: To access material of structure object.
Method Index
GetMaterial Returns the material and grade as a string on this function.
SetMaterial Sets the material and grade as a string on this function.

Methods


Func GetMaterial() As
Returns the material and grade as a string on this function.
Example: 

 This example retrieves the material of object.
 
 Dim ObjStrMaterialMngt As StrMaterialMngt
 Set ObjStrMaterialMngt = ObjSfdStiffenerOnFreeEdge.StrMaterialMngt
 StrMaterial = ObjStrMaterialMngt.GetMaterial
Sub SetMaterial( iMaterialName)
Sets the material and grade as a string on this function.
Parameters:
iMaterialName
Material name.
Example: 

 This example sets the material of the object
 
 ObjStrMaterialMngt.SetMaterial "Steel A42"
