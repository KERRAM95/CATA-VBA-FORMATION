StrProfilePtLimit (Object)

Object to manage Profile created with a point and a limit.
Property Index
Direction Returns or Sets the Profile's Direction.
StartPoint Returns or Sets the Profile's Start point.
UpToLimit Returns or Sets the Profile's UpToLimit.
Method Index
GetOrientation Orientation Returns the orientation associated to the direction.
ReverseDirection Reverses Direction

Properties


Property Direction() As
Returns or Sets the Profile's Direction.
Example: 
 

 This example retrieves the profile's direction
 
 Dim ObjStrProfilePtLimit As StrProfilePtLimit
 Set ObjStrProfilePtLimit = ObjSfdMember.StrProfilePtLimit
 Set RefDirection = ObjStrProfilePtLimit.Direction
Property StartPoint() As
Returns or Sets the Profile's Start point.
Property UpToLimit() As
Returns or Sets the Profile's UpToLimit.

Methods


Func GetOrientation() As
Orientation Returns the orientation associated to the direction.
Legal values for Orientation:
-1 = InvertOrientation
0 = UnknownOrientation
1 = SameOrientation
2 = X+ ( X+ vector = ( 1, 0, 0) )
3 = X- ( X- vector = (-1, 0, 0) )
4 = Y+ ( Y+ vector = ( 0, 1, 0) )
5 = Y- ( Y- vector = ( 0,-1, 0) )
6 = Z+ ( Z+ vector = ( 0, 0, 1) )
7 = Z- ( Z- vector = ( 0, 0,-1) )
8 = Inside
9 = Outside
10 = Inboard ( Toward center line )
11 = Outboard ( Opposite of previous one )
12 = Inboard ( Toward midship )
13 = Outboard ( Opposite of previous one )
Example: 
 

 This example retrieves the orientation associated to the direction
 
 Orientation = ObjStrProfilePtLimit.GetOrientation
Sub ReverseDirection()
Reverses Direction
