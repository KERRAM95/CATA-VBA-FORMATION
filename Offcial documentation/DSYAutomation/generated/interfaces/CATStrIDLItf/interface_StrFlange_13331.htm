StrFlange (Object)

Object to manage Flange of a Plate.
Role: Allows accessing of Flange's data.
See also:
Property Index
BendingAngle Sets the Bending Angle.
BendingRadius Sets the Bending Radius.
Edge Gets the Edge of Plate.
EndEndCutAngle Sets End EndCut Angle.
EndEndCutDistance Sets End EndCut Distance.
EndEndCutOffset Sets End EndCut Offset.
EndEndCutRadius Sets End EndCut Radius.
FlangeEndLimit
FlangeStartLimit
FlangeWidth Sets the Flange Width.
OperatedPlate Returns the reference of Plate object.
StartEndCutAngle Sets Start EndCut Angle.
StartEndCutDistance Sets Start EndCut Distance.
StartEndCutOffset Sets Start EndCut Offset.
StartEndCutRadius Sets Start EndCut Radius.
Type The types of Flange:
- 1 : Centered
- 2 : Tangent
WidthMeasurementType The types of Width Measurement:
- 1 : FlangeWidthToInnerFace
- 2 : FlangeWidthToOuterFace
- 3 : FlangeWidthToNeutralFibre
