CATStrProfileMode (Enumeration)

enum CATStrProfileMode {
  catStrProfileModeUndefined,
  catStrProfileModePtLength,
  catStrProfileModePtLimit,
  catStrProfileModePts,
  catStrProfileModeCrv,
  catStrProfileModeSurf2Crvs,
  catStrProfileModeSurfSurf,
  catStrProfileModeOnOpening,
  catStrProfileModeOnLimits
}
Creation mode for profile.
It can be:
Values:
catStrProfileModePtLength
- ProfilePtLength: profile defined by a point, a direction and a length.
catStrProfileModePtLimit
- ProfilePtLimit: profile defined by a point, a direction and a limit.
catStrProfileModePts
- ProfilePts: straight profile between two points.
catStrProfileModeCrv
- ProfileCrv: profile on a curve and optionally a reference surface. a spec point type or a point on curve.
catStrProfileModeSurf2Crvs
- ProfileSurf2Crvs: profile by the intersection of a surface and two curves.
catStrProfileModeSurfSurf
- ProfileSurfSurf: profile define by the intersection of two surfaces.
catStrProfileModeOnOpening
- ProfileOnOpening: profile on the edges of a Panel Opening.
catStrProfileModeOnLimits
- ProfileOnLimits: profile on the edges of Panel limits.
