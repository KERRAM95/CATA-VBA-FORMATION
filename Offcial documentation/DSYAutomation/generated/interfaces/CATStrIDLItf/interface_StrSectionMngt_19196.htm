StrSectionMngt (Object)

Object to manage Structure Functional Modeler section applied to a Profile.
Role: To access section parameters of Structure profile.
Property Index
AnchorPoint Returns or Sets the anchor point of this Profile.
AngleMode Returns or Sets the angle mode of this stiffener function.
FlangeOrientation Returns or Sets the Flange orientation of the positioned section of this Profile.
RefElementForAngle Returns or Sets the reference element used in "AlongPlane" mode for this Profile.
WebOrientation Returns or Sets the Web orientation of the positioned section of this Profile.
Method Index
GetAngle Returns the section's angle for this Profile.
GetFlangeAnchorPointOffset Returns the Flange Anchor Point Offset parameter.
GetParameter Returns a KWE parameter in the section document.
GetPlacedSectionAxis Computes the axis used to place the section on the guide curve.
GetSectionName Returns the section name of the section used by this Profile.
GetWebAnchorPointOffset Returns the Web Anchor Point offset parameter.
InvertFlangeOrientation Inverts the Flange orientation of the positioned section of this Profile.
InvertWebOrientation Inverts the Web orientation of the positioned section of this Profile.
SetSectionName Sets the section name on this Profile.

Properties


Property AnchorPoint() As
Returns or Sets the anchor point of this Profile.
Legal Values:
catStrBottomCenter
catStrBottomLeft
catStrBottomRight
catStrGravity
catStrTopCenter
catStrTopLeft
catStrTopRight
catStrCenterCenter
catStrCenterLeft
catStrCenterRight
Example: 

 This example retrieves  the anchor point of the Profile.
 
  Dim ObjStrSectionMngt As StrSectionMngt
  Set ObjStrSectionMngt = ObjSfdStiffener.StrSectionMngt
  AnchorPoint = ObjStrSectionMngt.AnchorPoint
Property AngleMode() As
Returns or Sets the angle mode of this stiffener function.
Legal values :
-1 : undefined mode
1 : normal to plate
0 : along plane
Example: 

 This example retrieves  the angle mode of this Profile.
 
  AngleMode = ObjStrSectionMngt.AngleMode
Property FlangeOrientation() As
Returns or Sets the Flange orientation of the positioned section of this Profile.
Legal values :
-1 : Invert
1 : Normal
0 : Unknown orientation
Example: 

 This example retrieves flange orientation of the section of the profile.
 
  FlangeOrientation = ObjStrSectionMngt.FlangeOrientation
Property RefElementForAngle() As
Returns or Sets the reference element used in "AlongPlane" mode for this Profile.
Example: 

 This example retrieves reference element for the profile.
 
  RefElement = ObjStrSectionMngt.RefElementForAngle
Property WebOrientation() As
Returns or Sets the Web orientation of the positioned section of this Profile.
Legal values :
-1 : Invert
1 : Normal
0 : Unknown orientation
Example: 

 This example retrieves web orientation of the section of the profile.

Computes the axis used to place the section on the guide curve.
