SddPlate (Object)

Object for representing Structure Detail Design Plate.
Role: To manage the Plate structure object.
Example: 

 This example retrieves in SddPlate.
 
 Dim ObjSddPlate As SddPlate
 Set ObjSddPlate = ObjSddProductPlate.SddPlate
