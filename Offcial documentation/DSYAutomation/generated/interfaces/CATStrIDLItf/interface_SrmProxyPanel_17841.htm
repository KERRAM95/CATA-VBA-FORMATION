SrmProxyPanel (Object)

Object to manage the Proxy Panel.
Planning breaks are treated similar like limits.
Method Index
GetLocation Returns the proxy panel location under the proxy tool.
GetPanel Returns the panel pointed by the proxy.
GetPlanningBreaks Create the planning break to the Proxy Panel.
GetProfileProxies Returns the List of Profile Proxies
