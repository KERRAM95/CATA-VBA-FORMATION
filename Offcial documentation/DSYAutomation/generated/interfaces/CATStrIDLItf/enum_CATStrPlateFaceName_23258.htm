CATStrPlateFaceName (Enumeration)

enum CATStrPlateFaceName {
  catStrPlateFaceNameUndefined,
  catStrPlateFaceBottom,
  catStrPlateFaceTop
}
Define the face plate name.
This parameter indicates the faces of Plate.
Values:
catStrPlateFaceNameUndefined
If face of plate is undefined (Including lateral faces).
catStrPlateFaceBottom
Bottom face of the Plate.
catStrPlateFaceTop
Top face of the Plate.
