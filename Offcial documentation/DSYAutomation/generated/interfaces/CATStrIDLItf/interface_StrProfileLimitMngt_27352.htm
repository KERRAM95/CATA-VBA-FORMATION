StrProfileLimitMngt (Object)

Object to manage limits of the Profile.
Role: To manage Profile's limits.
Method Index
ExplicitLimits In case the profile support is a Panel/Plate(s), it will set the limit ancestor of the DelimitedMoldedSurface edge stopping the CanonicTrace as a limit.
GetLimitType Returns the type of the limit applied on the Profile.
GetLimitingObject Returns the feature set by the user on the extremity
GetOffset Returns the limit offset
InvertProfile Invert the definition of the start and end for the selected profile.
RemoveLimit Removes the limit associated to the extremity.
SetLimitType Sets the type of the limit applied on the Profile.
SetLimitingObject Sets a feature selected by the user on the extremity

Methods


Sub ExplicitLimits( iExtr)
In case the profile support is a Panel/Plate(s), it will set the limit ancestor of the DelimitedMoldedSurface edge stopping the CanonicTrace as a limit.
Parameters:
iExtr
Extremity of the Profile that will be set with the Plate limit. If iExtr=-1, we reset both extremities.
If iExtr == 1 : Start extremity.
If iExtr == 2 : End extremity.
Func GetLimitType( iExtr) As
Returns the type of the limit applied on the Profile. This info is just captured at the support level and then realized at the Profile level in SDD.
limit type can have following values
-1 = Undefined limit
0 = Short Point
1 = Long Point
2 = Weld
3 = Miter Start
4 = Miter End
5 = Metal To Metal
Parameters:
iExtr
Extremity of the Profile.
If iExtr == 1 : Start extremity.
If iExtr == 2 : End extremity.
Example: 

 This example retrieves the limit type.
 
  Type = ObjStrProfileLimitMngt.GetLimitType
Func GetLimitingObject( iExtr) As
Returns the feature set by the user on the extremity
Parameters:
iExtr
Extremity of the Profile where to Set the Limiting Object.
If iExtr == 1 : Start extremity.
If iExtr == 2 : End extremity.
Example: 

 This example retrieves in Start limiting object of the profile.
 
 Set RefLimitingObject = ObjStrProfileLimitMngt.GetLimitingObject(1)

Sub RemoveLimit( iExtr)
Removes the limit associated to the extremity. Remove the Limit + EndCut.

Extremity of the Profile.
If iExtr == 1 : Start extremity.

Sub SetLimitType( iExtr, iType)
Sets the type of the limit applied on the Profile.
Parameters:
iExtr
Extremity of the Profile.
If iExtr == 1 : Start extremity.
If iExtr == 2 : End extremity.
iType
limit type
It can have following values
-1 = Undefined limit

Sub SetLimitingObject( iExtr, iLimitingObject)
Sets a feature selected by the user on the extremity
Parameters:
iExtr
Extremity of the Profile where to Set the Limiting Object.
If iExtr == 1 : Start extremity.
If iExtr == 2 : End extremity.
iLimitingObject
the feature to be set on the extremity.
Example: 

 This example sets in Start limiting object of the profile.
 
  Set ObjLimit1 = Manager.GetReferencePlane(ObjPart, 1, "DECK.8")
  Set Limit1 = ObjPart.CreateReferenceFromObject(ObjLimit1)
  Dim ObjStrProfileLimitMngt As StrProfileLimitMngt
  Set ObjStrProfileLimitMngt = ObjSfdMember.StrProfileLimitMngt
  ObjStrProfileLimitMngt.SetLimitingObject 1, Limit1
