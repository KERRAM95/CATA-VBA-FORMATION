StrService (Object)

Object services related to import and retrieval of real structure objects from its reference.
Method Index
ConvertReferenceToObject Returns the actual structure object from its corresponding reference.
Import Returns the result of Import of feature (source) into another document (target).

Methods


Func ConvertReferenceToObject( iReference) As
Returns the actual structure object from its corresponding reference.
Parameters:
iReference
The reference of structure object.
oBase
The real structure object.
Example: 

 This example retrieves the plate object on which flange is present.
 
 Dim RefSfdPlate As Reference
 Set RefSfdPlate = ObjStrFlange.OperatedPlate
 Dim ObjBase As AnyObject
 Set ObjBase = ObjStrService.ConvertReferenceToObject(RefSfdPlate)
