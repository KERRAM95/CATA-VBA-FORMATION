SrmProxyProfile (Object)

Object to manage the Proxy Profile.
Planning breaks are treated similar like limits.
Method Index
GetLocation Returns the proxy profile location.
GetPlanningBreaks Returns the planning breaks under the Proxy Profile.
GetProfile Returns the Profile pointed by the proxy.

Methods


Func GetLocation() As
Returns the proxy profile location. Return Values
1 : IN.
2 : OUT.
Example: 

 This example retrieves the location of SrmProxyProfile.
 
  Dim ObjSrmProxyProfile As SrmProxyProfile
  Set ObjSrmProxyProfile = ObjSrmProxyProfiles.Item(1)
  Location = ObjSrmProxyProfile.GetLocation
