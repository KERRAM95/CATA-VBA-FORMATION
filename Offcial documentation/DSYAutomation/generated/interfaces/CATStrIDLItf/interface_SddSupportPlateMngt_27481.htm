SddSupportPlateMngt (Object)

Object managing the Plates on which this Stiffener/StiffenerOnFreeEdge is attached to.
Role: Manages the plates on which this Stiffener/StiffenerOnFreeEdge is attached to.
Property Index
OtherSupportPlates Returns or Sets the other support Plates on which this is attached to.
ReferenceSupportPlate Returns the Source ReferencePlate on which this is attached to.
Method Index
AddOtherSupportPlate Adds a support Plate.
PromoteToReferencePlate Promotes a support Plate to a ReferencePlate and move the previous ReferencePlate to the list of the other support Plates.
RemoveOtherSupportPlate Removes a support Plate.
