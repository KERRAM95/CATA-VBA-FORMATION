StrDetailFeature (Object)

Object to manage the Endcut / Slot.
There are two ways to initialize the attributes of a detailing feature: - Automatic way: use InitByName() to automatically set the type and parameters from the PRMTable - Manual way: use Type, FeatureName and SetParameters() to explicitly set the values.
Role: To Manage Endcut / Slot.
Property Index
FeatureName Returns or Sets the name of this endcut/slot.
Type Returns or Sets the type of this endcut / slot.
Method Index
GetCoordinates Retrieves the location where the connection is
GetParameters Returns the list of parameters of this endcut / slot.
InitByName This method is for automatic initialization of Slot / Endcut parameters The name of the Slot / Endcut will be used to find the type and list of parameters in the PRMTable.
SetParameters Sets the list of parameters of this endcut / slot.
Update Update the connection coordinate and visulization for endcut / slot.

Properties


Property FeatureName() As
Returns or Sets the name of this endcut/slot.
Example: 

 This example sets FeatureName of the Slot/Endcut.
 
 ObjStrDetailFeature.FeatureName = "RECT_UNI_WT"
Property Type() As
Returns or Sets the type of this endcut / slot. - Build In => EndcutName(ShortPoint/LongPoint/Weld/Miter/MetalToMetal) - Generic => EndcutType(Weld/Trim/Snip/...) EndcutName(Generic) - Slot => SlotName(Slot_Buld1/...) SlotType(Rect/Boot/Gate/Bulb/Flar/...)
Example: 

 This example retrieves Type of the Slot/Endcut.
 
 Dim ObjStrDetailFeature As StrDetailFeature
 Set ObjStrDetailFeature = ObjStrSlot.StrDetailFeature
 Type = ObjStrDetailFeature.Type

Sub InitByName( iName)
This method is for automatic initialization of Slot / Endcut parameters The name of the Slot / Endcut will be used to find the type and list of parameters in the PRMTable.
Parameters:
iName
Name.
Sub SetParameters( iParamters, iParamRoleList)
Sets the list of parameters of this endcut / slot. Those parameters are copied and aggregated by this endcut / slot.
Parameters:
iParamters
List of parameters.
iParamRoleList
List of names of parameters.
Example: 

 This example sets parameters of the Slot.
 
  'This will retrieve part parameters
  Set ObjPartParameters = ObjPart.Parameters
  'Aggregate parameters list under iObjSfdPanel 
  Dim HB As AnyObject
  Set HB = iObjSfdPanel
  Set HBParameters = ObjPartParameters.SubList(HB, True)
  'Create parameters  
  Dim DBB As Dimension
  Set DBB = HBParameters.CreateDimension("DBB", "LENGTH", "300")
  Dim DR As Dimension
  Set DR = HBParameters.CreateDimension("DR", "LENGTH", "200")
  Dim DL As Dimension
  Set DL = HBParameters.CreateDimension("DL", "LENGTH", "200")
  Dim DTT As Dimension
  Set DTT = HBParameters.CreateDimension("DTT", "LENGTH", "50")
  'Create an array which will contain the role of each parameter
  Dim ParamRoles(3) As Variant
  ParamRoles(1) = "DBB"
  ParamRoles(2) = "DR"
  ParamRoles(3) = "DL"
  ParamRoles(0) = "DTT"
  'Get reference of element which will be impacted by this Slot/EndCut
  Set RefSfdPanel = ObjPart.CreateReferenceFromObject(iObjSfdPanel)
  'Set parameters
  ObjStrDetailFeature.SetParameters HBParameters, ParamRoles

Update the connection coordinate and visulization for endcut / slot.
