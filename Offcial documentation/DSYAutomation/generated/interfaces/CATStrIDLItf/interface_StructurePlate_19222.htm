StructurePlate (Object)

Object to manage the Structure Plate object.
Role: Allows accessing and setting of Plate's data.
Property Index
PanelType Returns the Panel's type.
StrAdvPanel Returns the StrAdvPanel object.
StrBreak Returns the StrBreak object.
StrCategoryMngt Returns the StrCategoryMngt object.
StrOpeningsMgr Returns the StrOpeningsMgr object.
StrPanelLimitMngt Returns the StrPanelLimitMngt object.
StrPanelSurf Returns the StrPanelSurf object.
StrPlateExtrusionMngt Returns the StrPlateExtrusionMngt object.
StrSlots Returns the Slots that are inside this object.
Method Index
GetCanonicSupport Returns the Canonical MoldedSurface of this plate.
GetDelimitedSupport Returns the delimited MoldedSurface of this Plate.
GetOpenings Returns the list of Openings that are inside this object.

Func GetOpenings( iType) As
Returns the list of Openings that are inside this object.
Parameters:
iType
The type of Opening you want to retreive:
- 0: All
- 1: Openings piercing the Plates inside the OpeningPlateSet
- 2: Openings piercing the Plates and interrupting the Profiles inside the OpeningPlateProfileSet
- 3: Openings piercing the Profile inside the OpeningProfileSet
Example: 
 

 This example retrieves all type of Openings
 
 Dim ObjStrOpenings As StrOpenings
 Set ObjStrOpenings = ObjSddPlate.GetOpenings(0)
