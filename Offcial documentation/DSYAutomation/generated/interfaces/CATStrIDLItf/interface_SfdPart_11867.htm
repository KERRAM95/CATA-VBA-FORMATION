SfdPart (Object)

Object to manage attributes on the Part.
Role: To manage attributes on the Part.
Method Index
GetConnectionsSet Returns the ConnectionSet under the part.
GetProjectData Returns the ProjectData tool.
IsSFDSystem Verifies that the reference has the correct V_Discipline value.
SetAsSFDSystem Initiliazes the Reference as a SFD system by setting the V_Discipline.

Methods


Func GetConnectionsSet() As
Returns the ConnectionSet under the part.
Example: 

 This example returns the ConnectionSet under the part.
 
  Dim ObjSfdConnectionsSet As SfdConnectionSet
  Set ObjSfdConnectionsSet = ObjSfdPart.GetConnectionsSet
Func GetProjectData() As
Returns the ProjectData tool.
Example: 

 This example returns the project data of this SFD part.
 
  Dim ProjectData As SfdProjectData
  Set ProjectData = ObjSfdPart.GetProjectData
Sub IsSFDSystem( oDiagnostic, oIncorrectDisciplineName)
Verifies that the reference has the correct V_Discipline value.
Parameters:
oDiagnostic
Project data tool.
oIncorrectDisciplineName
Returns the V_Discipline if there is already one.
Example: 

 This example checks whether this system is SFD or not.
 
  Dim Diagnosis As Long
  Dim oIncorrectDiscName As String
  ObjSfdPart.IsSFDSystem Diagnosis, oIncorrectDiscName
Sub SetAsSFDSystem( oDiagnostic, oIncorrectDisciplineName)
Initiliazes the Reference as a SFD system by setting the V_Discipline.
Parameters:
oDiagnostic

0 - V_discipline was successfully set for SFD.
1 - V_discipline is already set for SFD.
2 - V_discipline set not succeeded.
3 - V_discipline set not succeeded.
4 - V_discipline set not succeeded.
5 - V_discipline set not succeeded.
oIncorrectDisciplineName
Returns the V_Discipline if there is already one.
Example: 

 This example sets this system to SFD system.
 
  Dim ObjSfdPart As SfdPart
  ObjSelection.Add ObjPart
  Set ObjSfdPart = ObjSelection.FindObject("CATIASfdPart")
  Dim Diagnosis As Long
  Dim oIncorrectDiscName As String
  ObjSfdPart.SetAsSFDSystem Diagnosis, oIncorrectDiscName
