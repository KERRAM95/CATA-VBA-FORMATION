ManufacturingWireEDMOperation (Object)

Interface representing Wire EDM manufacturing operation.
Role: This interface offers services to manage Wire EDM manufacturing operation.
Method Index
GetOffsetParameter Returns the Offset associated with the operation set on iParameterName.
GetTechnoSetParameter Returns the Techno Set associated with the operation set on iParameterName.
GetWireEDMFeatures Returns the WEDM features associated with the operation.
SetTechnoSetParameter Sets the Techno Set on operation associated with the iParameterName.

Methods


Func GetOffsetParameter( iParameterName, PassIndex) As
Returns the Offset associated with the operation set on iParameterName.
Parameters:
iParameterName
Name of the Parameter from which Offset parameter has to be retrieved.
PassIndex
index of the Pass from which Offset parameter has to be retrieved. Only for Finishing and Seperation Pass.
Returns:
oOffsetParm Retrieved Offset parameter.
Func GetTechnoSetParameter( iParameterName, PassIndex) As
Returns the Techno Set associated with the operation set on iParameterName.
Parameters:
iParameterName
Name of the Parameter from which Techno Set has to be retrieved.
PassIndex
index of the Pass from which Techno Set has to be retrieved. Only for Finishing and Seperation Pass.
Returns:
oTechnoSetParm Retrieved Techno Set.
Func GetWireEDMFeatures() As
Returns the WEDM features associated with the operation.
Returns:
oListWEDMFeatures Retrieved list of Wire EDM Features set on the Operation.
Sub SetTechnoSetParameter( iParameterName, PassIndex, iTechnoSet)
Sets the Techno Set on operation associated with the iParameterName.
Parameters:
iParameterName
Name of the Parameter on which iTechnoSet has to be set.
PassIndex
index of the Pass on which iTechnoSet has to be set. Only for Finishing and Seperation Pass.
iTechnoSet
Techno Set to be set.
Returns:
Status of the method.
