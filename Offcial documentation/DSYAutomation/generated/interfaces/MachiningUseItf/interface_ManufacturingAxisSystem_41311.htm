ManufacturingAxisSystem (Object)

ManufacturingMachiningAxis defines a set of methods to manage a Machining Axis System or an Origin.
Method Index
GetAxisType Gets the machining axis type.
GetIsOrigin Gets if the machining axis system is an origin.
GetOriginGroup Gets the origin group in case of an origin usage.
GetOriginNumber Gets the origin number in case of an origin usage.
GetOriginPoint Gets the origin point of the machining axis system.
GetXDirection Gets the X direction of the machining axis system.
GetYDirection Gets the Y direction of the machining axis system.
GetZDirection Gets the Z direction of the machining axis system.
SetAxisType Sets the machining axis type.
