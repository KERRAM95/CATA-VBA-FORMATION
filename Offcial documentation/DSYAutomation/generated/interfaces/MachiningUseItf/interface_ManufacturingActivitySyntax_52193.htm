ManufacturingActivitySyntax (Object)

Interface dedicated to activity object managing PP words.
Role: This interface offers services to manage PP words of activity.
Method Index

GetPPWORDSyntax Retrieves the PP words syntax.
ResetPPWORDSyntax Resets the PP words syntax.
