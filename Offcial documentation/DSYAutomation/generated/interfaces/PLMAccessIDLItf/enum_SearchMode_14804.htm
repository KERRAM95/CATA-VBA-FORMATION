SearchMode (Enumeration)

enum SearchMode {
  SearchMode_Easy,
  SearchMode_Extended,
  SearchMode_Expert,
  SearchMode_Predefined
}
Type of search to launch.
Values:
SearchMode_Easy
Easy Search, uses a pedefined operator(depending upon the type of the attribute) and AND condition among the attribute criteia.
SearchMode_Extended
Extended Search, can use AND or OR condition among the attributes and any operator applicable to the type of attribute.
SearchMode_Expert
Extended Search, can use a complex experssion containing combination of both AND and OR condition and operators.
SearchMode_Predefined
Predefined Query Search, can be used to launch a query with just a PQ abbreviaton and given criteria.
