SearchOperator (Enumeration)

enum SearchOperator {
  SearchOperator_EQ,
  SearchOperator_NOT_EQ,
  SearchOperator_LIKE,
  SearchOperator_NOT_LIKE,
  SearchOperator_LT,
  SearchOperator_LT_EQ,
  SearchOperator_GT,
  SearchOperator_GT_EQ,
  SearchOperator_BETWEEN,
  SearchOperator_NOT_BETWEEN,
  SearchOperator_NULL,
  SearchOperator_NOT_NULL
}
Operator for attribute criteria
Values:
SearchOperator_EQ.
e.g. 'Physical Product'."Created On" == "Today".
SearchOperator_NOT_EQ
e.g. 'Physical Product'."Created On" != "Yesterday".
SearchOperator_LIKE
e.g. * e.g. 'Physical Product'."Title" LIKE "MyProduct".
SearchOperator_NOT_LIKE
e.g. 'Physical Product'."Title" NOT_LIKE "YourProduct".
SearchOperator_LT
e.g. 'Physical Product'."Number of Sub-Products" < "10".
SearchOperator_LT_EQ
e.g. 'Physical Product'."Number of Sub-Products" <= "10".
SearchOperator_GT
e.g. 'Physical Product'."Number of Sub-Products" > "10".
SearchOperator_GT_EQ
e.g. 'Physical Product'."Number of Sub-Products" >= "10".
SearchOperator_BETWEEN
e.g. 'Physical Product'."Created On" BETWEEN "Today" AND "Yesterday".
SearchOperator_NOT_BETWEEN
e.g. 'Physical Product'."Created On" NOT_BETWEEN "02/01/2013" AND "02/28/2013".
SearchOperator_NULL
e.g. This in general means that attribute has an empty value.
SearchOperator_NOT_NULL
e.g. This in general means that attribute has non-empty value.
