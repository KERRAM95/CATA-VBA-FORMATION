SearchService (Object)

This Interface is starting point to automate the search.

Example of how to retrieve such an object using 

  Dim aSearchSrv as SearchService
  Set aSearchSrv = CATIA.GetSessionService("Search")

Property Index
DatabaseSearch Returns a Search object that can prepare search with the database.
Title Sets the title of the Search Browser.
Method Index
Search Launch the Search and display the results in the Search Browser.

Properties


Property DatabaseSearch() As (Read Only)
Returns a Search object that can prepare search with the database. This provides APIs to simulate the "Advanced Search" provided in the UI.
Property Title( iTitle) (Write Only)
Sets the title of the Search Browser.

Methods


Sub Search()
Launch the Search and display the results in the Search Browser.
