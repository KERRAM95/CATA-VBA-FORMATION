SimUserDefinedPhysics (Enumeration)

enum SimUserDefinedPhysics {
  SimUserDefinedMechanical,
  SimUserDefinedThermal,
  SimUserDefinedThermoMechanical
}
User defined physics of the material.

The user defined physics will be:

For User Defined Material
Values:
SimUserDefinedMechanical
The mechanical behavior of the material. Use this if the constants are used in defining the mechanical behavior of the material.
SimUserDefinedThermal
The thermal behavior of the material. Use this if the constants are used in defining the thermal behavior of the material.
SimUserDefinedThermoMechanical
The mechanical and thermal behavior of the material. Use this if the constants are used in defining both mechanical and thermal behavior of the material.
