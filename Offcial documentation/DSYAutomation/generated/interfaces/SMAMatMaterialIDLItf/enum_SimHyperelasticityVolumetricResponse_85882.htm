SimHyperelasticityVolumetricResponse (Enumeration)

enum SimHyperelasticityVolumetricResponse {
  SimHyperelasticityIgnore_Test_Data,
  SimHyperelasticityVolumetric_Test_Data,
  SimHyperelasticityLateral_Nominal_Strain,
  SimHyperelasticityPoissons_Ratio
}
Defines the volumetric response.
Values:
SimHyperelasticityIgnore_Test_Data
Leaves test data out of the calculation of volumetric response.
SimHyperelasticityVolumetric_Test_Data
Calculates volumetric response using test data and specifies test data behavior.
SimHyperelasticityLateral_Nominal_Strain
Specifies the lateral strains together with the uniaxial test data.
SimHyperelasticityPoissons_Ratio
Specifies the Poisson's ratio value directly.
