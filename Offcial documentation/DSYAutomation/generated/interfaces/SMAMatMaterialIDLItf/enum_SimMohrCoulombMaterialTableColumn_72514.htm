SimMohrCoulombMaterialTableColumn (Enumeration)

enum SimMohrCoulombMaterialTableColumn {
  SimMohrCoulombFrictionAngle,
  SimMohrCoulombDilationAngle,
  SimMohrCoulombCohesionYieldStress,
  SimMohrCoulombPlasticStrain,
  SimMohrCoulombTemperature
}
Columns in material table.

The columns in the material table will be:

For Mohr Coulumb material table
Values:
SimMohrCoulombFrictionAngle
Friction angle, at high confining pressure in the P-Rq plane. Quantity: ANGLE, Units: degrees
SimMohrCoulombDilationAngle
Dilation angle, at high confining pressure in the P-Rq plane. Quantity: ANGLE, Units: degrees
SimMohrCoulombTemperature
Temperature (Optional). Quantity: TEMPRTRE, Units: Kdeg

For Mohr Coulumb hardening table
SimMohrCoulombCohesionYieldStress
Cohesion yield stress. Quantity: STRESS, Units: N_m2
SimMohrCoulombPlasticStrain
Absolute value of the corresponding plastic strain. Quntity: DIMENSIONLESS, Units: None
SimMohrCoulombTemperature
Temperature (Optional). Quantity: TEMPRTRE, Units: Kdeg
