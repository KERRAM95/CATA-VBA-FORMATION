SimNonlinearViscoelastic (Object)

Represents the Nonlinear Viscoelastic object.
Example:
Given a SimMaterialOptions object, you can create a SimNonlinearViscoelastic as following: 
 Dim MyMaterialOptions As SimMaterialOptions
 ...
 Dim MyNonlinearViscoelastic As SimNonlinearViscoelastic
 Set MyNonlinearViscoelastic = MyMaterialOptions.Add("SimNonlinearViscoelastic")
Given a SimMaterialOptions object, you can retrieve a SimNonlinearViscoelastic named "Nonlinear Viscoelastic.1" as following: 
 Dim MyMaterialOptions As SimMaterialOptions
 ...
 Dim MyNonlinearViscoelastic As SimNonlinearViscoelastic
 Set MyNonlinearViscoelastic = MyMaterialOptions.Item("Nonlinear Viscoelastic.1")
Example in Python:
Given a SimMaterialOptions object myMaterialOptions, you can create a SimNonlinearViscoelastic as following: 
 ...
 myNonlinearViscoelastic = myMaterialOptions.Add("SimNonlinearViscoelastic")
Given a SimMaterialOptions object myMaterialOptions, you can retrieve a SimNonlinearViscoelastic named "Nonlinear Viscoelastic.1" as following: 
 ...
 myNonlinearViscoelastic = myMaterialOptions.Item("Nonlinear Viscoelastic.1")
