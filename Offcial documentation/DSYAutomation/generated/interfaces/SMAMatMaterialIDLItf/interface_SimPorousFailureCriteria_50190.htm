SimPorousFailureCriteria (Object)

Represents the Porous Failure Criteria object.
Example:
Given a SimMaterialOptions object, you can create a SimPorousFailureCriteria as following: 
 Dim MyMaterialOptions As SimMaterialOptions
 ...
 Dim MyPorousFailureCriteria As SimPorousFailureCriteria
 Set MyPorousFailureCriteria = MyMaterialOptions.Add("SimPorousFailureCriteria")
Given a SimMaterialOptions object, you can retrieve a SimPorousFailureCriteria named "Porous Failure Criteria.1" as following: 
 Dim MyMaterialOptions As SimMaterialOptions
 ...
 Dim MyPorousFailureCriteria As SimPorousFailureCriteria
 Set MyPorousFailureCriteria = MyMaterialOptions.Item("Porous Failure Criteria.1")
Example in Python:
Given a SimMaterialOptions object myMaterialOptions, you can create a SimPorousFailureCriteria as following: 
 ...
 myPorousFailureCriteria = myMaterialOptions.Add("SimPorousFailureCriteria")
Given a SimMaterialOptions object myMaterialOptions, you can retrieve a SimPorousFailureCriteria named "Porous Failure Criteria.1" as following: 
 ...
 myPorousFailureCriteria = myMaterialOptions.Item("Porous Failure Criteria.1")
