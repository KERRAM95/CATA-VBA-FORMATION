SimLowDensityFoam (Object)

Represents the Low Density Foam object.
Example:
Given a SimMaterialOptions object, you can create a SimLowDensityFoam as following: 
 Dim MyMaterialOptions As SimMaterialOptions
 ...
 Dim MyLowDensityFoam As SimLowDensityFoam
 Set MyLowDensityFoam = MyMaterialOptions.Add("SimLowDensityFoam")
Given a SimMaterialOptions object, you can retrieve a SimLowDensityFoam named "Low Density Foam.1" as following: 
 Dim MyMaterialOptions As SimMaterialOptions
 ...
 Dim MyLowDensityFoam As SimLowDensityFoam
 Set MyLowDensityFoam = MyMaterialOptions.Item("Low Density Foam.1")
Example in Python:
Given a SimMaterialOptions object myMaterialOptions, you can create a SimLowDensityFoam as following: 
 ...
 myLowDensityFoam = myMaterialOptions.Add("SimLowDensityFoam")
Given a SimMaterialOptions object myMaterialOptions, you can retrieve a SimLowDensityFoam named "Low Density Foam.1" as following: 
 ...
 myLowDensityFoam = myMaterialOptions.Item("Low Density Foam.1")

Properties


Property FailFlag() As
Returns or sets the failure enable flag that enables element deletion when tension cutoff is specified and the value reached.
TRUE: The failure or element deletion enabled.

FALSE: The failure or element deletion disabled.
Property LateralStrainDataType() As
Returns or sets the lateral strain data type.
Property OptionalLDFoamRelaxCoeffDataFlag() As
Returns or sets the flag for optional low density foam rate data with relaxation coefficients.
TRUE: The relaxation coefficients is used.

FALSE: The relaxation coefficients is not used.
Property RateExtrapolationFlag() As
Returns or sets the strain rate extrapolation enable flag that enables extrapolation strain rate dependent stress-strain data beyond maximum specified strain rate.
TRUE: The strain rate extrapolation enabled.

FALSE: The strain rate extrapolation disabled.
Property RelaxCoefficientAlpha() As
Returns or sets the relax coefficient alpha. Quantity: DIMENSIONLESS, units: None.
Property RelaxCoefficientMu0() As
Returns or sets the relax coefficient mu0. Quantity: TIME, units: s.
Property RelaxCoefficientMu1() As
Returns or sets the relax coefficient mu1. Quantity: TIME, units: s.
Property StrainRateType() As
Returns or sets the strain rate type.
Property TensionCutOff() As
Returns or sets the tension cut off. Quantity: STRESS, units: N_m2.
Property TensionCutOffFlag() As
Returns or sets the tension cutoff flag that allows to specify maximum principal stress the low-density foam material can sustain in tension.
TRUE: The tension cutoff is used.

FALSE: The tension cutoff is not used.
