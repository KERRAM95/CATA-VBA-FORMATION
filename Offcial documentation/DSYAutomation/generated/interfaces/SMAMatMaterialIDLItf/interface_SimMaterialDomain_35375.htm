SimMaterialDomain (Object)

Represents the Simulation Material Domain object.
Example:
Given a MaterialDomains object, you can create and retrieve a SimMaterialDomain object as following: 
 Dim MyMaterialDomains As MaterialDomains
 ...
 Dim MyMaterialDomain As MaterialDomain
 MyMaterialDomains.Add "dsc_matref_rep_SmaOptions", MyMaterialDomain

 Dim MySimulationDomain As SimMaterialDomain
 Set MySimulationDomain = MyMaterialDomain.MaterialDomainContent
Property Index
Behaviors Retrieves a collection of all behaviors in the material domain.
DefaultBehavior Default material behavior on a simulation domain.
Material Retrieves material reference of simulation domain.
Options Retrieves a collection of all material options in the material domain.
Method Index
DuplicateAndAddMaterialOption Duplicates and adds a material option in a simulation domain.

Sub SyncAllBehaviors()
Synchronises all material behaviors. This should be done after modifications to individual material options and behaviors within a material domain are completed and before the material domain is used elsewhere. It is not necessary to sync the material domain after each modification to a material domain or behavior
