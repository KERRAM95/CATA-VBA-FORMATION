SimNonlinearViscoelasticNetworkLaw (Enumeration)

enum SimNonlinearViscoelasticNetworkLaw {
  SimNonlinearViscoelasticNetworkHyperbolicSine,
  SimNonlinearViscoelasticNetworkPowerLaw,
  SimNonlinearViscoelasticNetworkUser,
  SimNonlinearViscoelasticNetworkBergstromBoycePowerLaw,
  SimNonlinearViscoelasticNetworkDarveauxLaw,
  SimNonlinearViscoelasticNetworkModifiedDarveauxLaw
}
Defines the law for nonlinear viscoelasticity.
Values:
SimNonlinearViscoelasticNetworkHyperbolicSine
Defines nonlinear viscoelasticity with hyperbolic-sine law.
SimNonlinearViscoelasticNetworkPowerLaw
Defines nonlinear viscoelasticity with power law.
SimNonlinearViscoelasticNetworkUser
Defines nonlinear viscoelasticity with user.
SimNonlinearViscoelasticNetworkBergstromBoycePowerLaw
Defines nonlinear viscoelasticity with Bergstrom Boyce power law.
SimNonlinearViscoelasticNetworkDarveauxLaw
Defines nonlinear viscoelasticity with Darveaux law.
SimNonlinearViscoelasticNetworkModifiedDarveauxLaw
Defines nonlinear viscoelasticity with Modified Darveaux Law.
