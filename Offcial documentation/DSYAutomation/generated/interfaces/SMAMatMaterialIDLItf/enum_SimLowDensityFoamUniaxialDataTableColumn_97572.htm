SimLowDensityFoamUniaxialDataTableColumn (Enumeration)

enum SimLowDensityFoamUniaxialDataTableColumn {
  SimLowDensityFoamNominalStress,
  SimLowDensityFoamNominalStrain,
  SimLowDensityFoamNominalLatStrain,
  SimLowDensityFoamNominalStrainRate,
  SimLowDensityFoamTemperature
}
Columns in material table.

The columns in the material table will be:

For Uniaxial compression or tension table
Values:
SimLowDensityFoamNominalStress
The Nominal Stress. Quantity: STRESS, Units: N_m2
SimLowDensityFoamNominalStrain
The Nominal Strain. Quantity: DIMENSIONLESS, Units: None
SimLowDensityFoamNominalLatStrain
The Nominal Lateral Strain (optional), when lateral strain data is present. Quantity: DIMENSIONLESS, Units: None
SimLowDensityFoamNominalStrainRate
The Nominal Strain Rate (optional). Quantity: STRAIN_RATE, Units: _s
SimLowDensityFoamTemperature
Temperature (optional). Quantity: TEMPRTRE, Units: Kdeg
