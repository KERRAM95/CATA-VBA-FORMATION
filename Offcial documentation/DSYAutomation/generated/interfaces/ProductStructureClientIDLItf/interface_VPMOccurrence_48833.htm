VPMOccurrence (Object)

Represents a PLM Product Occurrence.
Property Index
InstanceOccurrenceOf Returns its associated PLM Product Instance.
Occurrences Returns its PLM Product Occurrences collection.
Position Returns the PLM Product Occurrence position.
RepOccurrences Returns its PLM Product Rep Occurrences Collection.

Properties


Property InstanceOccurrenceOf() As (Read Only)
Returns its associated PLM Product Instance.
Example: 
   This example shows you how retrieve the instance associated with a given occurrence.
   
   Dim  oProdInstToRetrieve  As VPMInstance
   Dim  oProdOccurrence  As VPMOccurrence
   ...
   Set oProdInstToRetrieve = oProdOccurrence.InstanceOccurrenceOf
Property Occurrences() As (Read Only)
Returns its PLM Product Occurrences collection.
Role: This method retrieves the PLM Product Occurrences collection object directly aggregated within this object.
Example: 
   This example shows you how retrieve the collection of occurrences just beneath a given occurrence.
   
   Dim oProdOccurrenceToScan As VPMOccurrence
   ...
   Dim cListOccurrences As VPMOccurrences
   Set cListOccurrences = oProdOccurrenceToScan.Occurrences
Property Position() As (Read Only)
Returns the PLM Product Occurrence position.
Example: 
    This example shows you how to get the PLM Product Occurence position.
   
 
   Dim  oProdOccurrence As VPMOccurrence
   ...
   Dim  oProdOccurrencePosition  As Position
   Set oProdOccurrencePosition = oProdOccurrence.Position
Property RepOccurrences() As (Read Only)
Returns its PLM Product Rep Occurrences Collection.
Role : This method retrieves the PLM Product Rep Occurrences collection object directly aggregated within this object.
Parameters:
oRepOccurrences
The collection of PLM Product Occurrences
Example: 
   This example shows you how to retrieve the collection of rep occurrences
   aggregated under a PLM Product Occurrence
   
   Dim oProdOccurrenceToScan As VPMOccurrence 
   ...
   Dim cListRepOccurrences As VPMRepOccurrences
   Set cListRepOccurrences = oProdOccurrenceToScan.RepOccurrences
