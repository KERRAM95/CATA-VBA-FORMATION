SimAxis (Object)

Represents the Axis object.
Property Index
Axis Returns of set the axis.
AxisType Returns of set the axis type.
Method Index
GetAxis Retrieves the axis point and direction.
SetAxis Sets the axis data using point and direction.

Properties


Property Axis() As (Read Only)
Returns of set the axis.
Property AxisType() As
Returns of set the axis type. The graphical user interface editor for this SimAxis only supports geometric axis types. See SimAxisAxisType for additional information.

Methods


Sub GetAxis( oOrigin, oVector)
Retrieves the axis point and direction.
Parameters:
oOrigin[out]
is the point
oVector[out]
is the direction of the line
Sub SetAxis( iOrigin, iVector)
Sets the axis data using point and direction. If this method is used to set the Axis then GetAxis( CATMathPoint & oOrigin, CATMathVector & oVector ) must be used to retrieve the axis and not GetAxis(CATISimLinkAccess_var & ispSupport ).
Parameters:
iOrigin[in]
is the point
iVector[in]
is the direction of the line
