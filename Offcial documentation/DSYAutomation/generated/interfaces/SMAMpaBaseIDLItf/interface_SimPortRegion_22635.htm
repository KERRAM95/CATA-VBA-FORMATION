SimPortRegion (Object)

Represents the Port Region object.
Example:
Given a SimFeatures object, you can create a SimPortRegion as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyPortRegion As SimPortRegion
 Set MyPortRegion = MyFeatures.Add("SimPortRegion")
Given a SimFeatures object, you can retrieve a SimPortRegion named "Port Region.1" as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyPortRegion As SimPortRegion
 Set MyPortRegion = MyFeatures.Item("Port Region.1")
Example in Python:
Given a SimFeatures object myFeatures, you can create a SimPortRegion as following: 
 ...
 myPortRegion = myFeatures.Add("SimPortRegion")
Given a SimFeatures object myFeatures, you can retrieve a SimPortRegion named "Port Region.1" as following: 
 ...
 myPortRegion = myFeatures.Item("Port Region.1")
