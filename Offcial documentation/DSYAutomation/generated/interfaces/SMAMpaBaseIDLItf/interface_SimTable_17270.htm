SimTable (Object)

Represents the Table object.
Example:
Given a SimTabularAmplitude object with an attribute Table of type SimTable, you can retrieve SimTable object as following: 
 Dim MyTabularAmplitude As SimTabularAmplitude
 ...
 Dim MyTable As SimTable
 Set MyTable = MyTabularAmplitude.Table
Example in Python:
Given a SimTabularAmplitude object with an attribute Table of type SimTable, you can retrieve SimTable object as following: 
 ...
 myTable = myTabularAmplitude.Table
See also:
Property Index
EmptyCellsInTableFlag Retrieves the flag that determines whether table has empty cells.
NumberOfColumns Retrieves the number of columns in the table.

Method Index
DeleteRow Deletes the specified row.
InsertRow Inserts a row at the specified location.

Properties


Property EmptyCellsInTableFlag() As (Read Only)
Retrieves the flag that determines whether table has empty cells. This boolean will be TRUE if some cells are empty, FALSE otherwise.
Property NumberOfColumns() As (Read Only)
Retrieves the number of columns in the table.
Property NumberOfRows() As (Read Only)
Retrieves the number of rows in the table.
Property TableColumns() As (Read Only)
Retrieves the table columns. Each item of the returned list adheres to the CATISimTableColumn interface.

Methods


Sub DeleteRow( iRow)
Deletes the specified row.
Parameters:
iRow[in]
The row number. To delete the first row iRow should be 1.
Sub InsertRow( iRow)
Inserts a row at the specified location.
Newly created cells will have an illegal value. They will return True using the GetEmptyCellFlag method. This method will fail if multiple rows are not allowed for the specific table.
Parameters:
iRow[in]
The row number at which a new row is inserted. To insert the row at the top of the table iRow should be 1.
