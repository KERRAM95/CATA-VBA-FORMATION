SimMathVector (Object)

Represents the Math Vector object.
Example:
Given a SimMathAxis object with an attribute FirstDirection of type SimMathVector, you can retrieve SimMathVector object as following: 
 Dim MyMathAxis As SimMathAxis
 ...
 Dim MyMathVector As SimMathVector
 Set MyMathVector = MyMathAxis.FirstDirection
Example in Python:
Given a SimMathAxis object with an attribute FirstDirection of type SimMathVector, you can retrieve SimMathVector object as following: 
 ...
 myMathVector = myMathAxis.FirstDirection
