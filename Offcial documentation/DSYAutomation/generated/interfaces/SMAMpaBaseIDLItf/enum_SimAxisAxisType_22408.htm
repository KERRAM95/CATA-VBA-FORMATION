SimAxisAxisType (Enumeration)

enum SimAxisAxisType {
  SimAxisGeometric,
  SimAxisExplicit
}
The mode used to define the axis.
SimAxisExplicit is a convenience for cases when the geometry does not exist. For features using SimAxis, the graphical user interface editor does not support the SimAxisExplicit option. Editing one of these features in the graphical user interface will reset the axis definition to SimAxisGeometric.
Values:
SimAxisGeometric
The geometrical line or axis or edge of the geometry.
SimAxisExplicit
The Explicit is defined by the user using origin and direction.
