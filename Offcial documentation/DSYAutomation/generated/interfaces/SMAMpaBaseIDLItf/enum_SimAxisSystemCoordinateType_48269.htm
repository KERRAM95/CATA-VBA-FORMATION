SimAxisSystemCoordinateType (Enumeration)

enum SimAxisSystemCoordinateType {
  SimAxisSystemNoAxis,
  SimAxisSystemCartesian,
  SimAxisSystemCylindrical,
  SimAxisSystemSpherical
}
The coordinate type of the FEA axis system.
Values:
SimAxisSystemNoAxis
No axis definition.
SimAxisSystemCartesian
A cartesian axis definition.
SimAxisSystemCylindrical
A cylindrical axis definition.
SimAxisSystemSpherical
A spherical axis definition.
