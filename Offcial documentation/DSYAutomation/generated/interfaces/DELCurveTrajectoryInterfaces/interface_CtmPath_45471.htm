Sub GetPosition( iIndex, oMpoint, oVone, oVtwo, oVthree)
Get the specified position relative to the aggregating product (eg. the station).
Parameters:
iIndex
The index of the position to retrieve. The index of the first position is 1.
oMpoint
The MathPoint which consists of three coordinates.
oVone
The first vector which consists of three coordinates.
oVtwo
The second vector which consists of three coordinates.
oVthree
The third vector which consists of three coordinates.
Returns:
An HRESULT value.
Legal values:
S_OK
The position was returned correctly.
E_INVALIDARG

E_UNEXPECTED
