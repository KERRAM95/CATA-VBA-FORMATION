CtmCurveOrienter (Object)

Interface representing the orientation for a curve
Role: This interface is used to get and set global YPR and rake angles.
Method Index
GetBaseAxisMode Retrieves the value of the BaseAxis mode.
GetGlobalBRR Retrieves the value of the GLOBAL angles.
GetGlobalYPR Retrieves the value of the GLOBAL angles.
GetRakeAngle Retrieves the value Rake on either start or end tag.
SetBaseAxisMode Sets the value of the BaseAxis mode.
SetGlobalBRR Set the value of the GLOBAL angles.
SetGlobalYPR Set the value of the GLOBAL angles.
SetRakeAngle Set the value Rake on either start or end tag.
