CtmCurveSampler (Object)

Interface representing the curve sampler.

Role: This interface is used to get and set start and end offsets for Curve trajectory.
Property Index
EndOffset Returns or sets the end offset for the curve.
StartOffset Returns or sets the start offset for the curve.

Properties


Property EndOffset() As
Returns or sets the end offset for the curve. The offset indicates the distance from the end of the curve at which sampling will end, and a tag will be placed at the end.
Example: 
 Dim objSampler As CtmCurveSampler
       ........
 Dim oEndOffset As Double
 objSampler.EndOffset = 12
 oEndOffset = objSampler.EndOffset
Property StartOffset() As
Returns or sets the start offset for the curve. The offset indicates the distance along the curve at which sampling will start, after a tag has been placed at the start of the curve.
Example: 
 Dim objSampler As CtmCurveSampler
       ........
 Dim oStartOffset As Double
 objSampler.StartOffset = 10
 oStartOffset = objSampler.StartOffset
