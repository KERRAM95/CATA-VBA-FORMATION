KinSimulationChannel (Object)

The interface to access a result channel.
It corresponds to a result column as it appears in the View Scenario Results window in an interactive session. The KinSimulationChannel as AnyObject offers the Name property giving the name of the channel as it appears in the View Scenario Results window, page Specifications.
Property Index
ChannelSize Returns the number of numerical values inside the channel.
ChannelType Returns the KinSimulationChannel, type of the channel.
ChannelUnit Returns the unit of the numerical values.
Method Index
GetChannelValue Returns a double as the value of the input index.
