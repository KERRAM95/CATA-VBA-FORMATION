VALConcerns (Object)

A collection of Concerns.
Method Index
Add Creates a Concern and adds it to the Concern collection.
Item Returns a Concern using its index from the Concerns collection.
Remove Removes a Concern from the Concerns collection.

Methods


Func Add() As
Creates a Concern and adds it to the Concern collection.
Returns:
The created Concern
Example: 
    This example creates a new Concern in the cConcerns collection.
    
    Dim oNewConcernText As Concern
    Set oNewConcernText = cConcerns.Add
Func Item( iIndex) As
Returns a Concern using its index from the Concerns collection.
Parameters:
iIndex
The index or the name of the Concern to retrieve from the collection of Concerns. As a numerics, this index is the rank of the Concern in the collection. The index of the first Concern in the collection is 1, and the index of the last Concern is Count. As a string, it is the name you assigned to the Concern.
Returns:
The retrieved Concern
Example: 
    This example retrieves in oConcern1 the ninth Concern,
    and in oConcern2 the Concern named
    Concern3 from the cConcerns collection. 
    
    Dim oConcern1 As Concern
    Set oConcern1 = cConcerns.Item(9)
    Dim oConcern2 As Concern
    Set oConcern2 = cConcerns.Item("Concern3")
Sub Remove( iIndex)
Removes a Concern from the Concerns collection.
Parameters:
iIndex
The index or the name of the Concern to retrieve from the collection of Concerns. As a numerics, this index is the rank of the Concern in the collection. The index of the first Concern in the collection is 1, and the index of the last Concern is Count. As a string, it is the name you assigned to the Concern.
Example: 
    The following example removes the tenth Concern and the Concern named
    Concern2 from the cConcerns collection.
    
    cConcerns.Remove(10)
    cConcerns.Remove("Concern2")
