Marker (Object)

Represents a marker.
Property Index
Angle Returns or sets the Angle of the marker.
DocumentRef Sets a path of the document for Picture, Hyperlink and Audio.
Fill Returns or sets the Fill of the Marker.
Frame Returns or sets the Frame of the Marker.
Type Returns the type of the Marker.
Method Index
GetPointing Returns the pointing info related to a marker.
GetPositions Returns the coordinates of the positions of the Marker.
IsPointing Returns the check if pointing info exists.
SetPositions Sets the coordinates of the positions of the Marker.
Update Updates the marker for account of modification if any.

Func GetPositions() As
Returns the coordinates of the positions of the Marker.

These positions depend on the type of the Marker. 2D Line: 2 positions (two end points). 2D Arrow: 2 positions (head and tail points). 2D Rectangle: 2 positions (bottom left and top right corner). 2D Circle: 2 positions (center and point through which it passes). 2D Freehand: 2 positions (series of points). 2D & 3D Text, Picture, Hyperlink, Audio: 1 anchor position (top left point).

Parameters:
oCoordinates
2D: oCoordinates (0) is the X coordinate of the first point oCoordinates (1) is the Y coordinate of the first point oCoordinates (2) is the X coordinate of the second point oCoordinates (3) is the Y coordinate of the second point oCoordinates (n*2-2) is the X coordinate of the n-th point oCoordinates (n*2-1) is the X coordinate of the n-th point 3D: oCoordinates (0) is the X coordinate of the anchor point oCoordinates (1) is the Y coordinate of the anchor point oCoordinates (2) is the Z coordinate of the anchor point
Example: 
 This example returns the 3D Text anchor position
 
 Dim cMarker As Marker
 Dim Pos (2)
 Pos = cMarker.GetPositions

Sub SetPositions( iCoordinates)
Sets the coordinates of the positions of the Marker.

These positions depend on the type of the Marker. 2D Line: 2 positions (two end points). 2D Arrow: 2 positions (head and tail points). 2D Rectangle: 2 positions (bottom left and top right corner). 2D Circle: 2 positions (center and point through which it passes). 2D Freehand: 2 positions (series of points). 2D & 3D Text, Picture, Hyperlink, Audio: 1 anchor position (top left point).

Parameters:
iCoordinates
2D: iCoordinates (0) is the X coordinate of the first point iCoordinates (1) is the Y coordinate of the first point iCoordinates (2) is the X coordinate of the second point iCoordinates (3) is the Y coordinate of the second point iCoordinates (n*2-2) is the X coordinate of the n-th point iCoordinates (n*2-1) is the X coordinate of the n-th point 3D: iCoordinates (0) is the X coordinate of the anchor point iCoordinates (1) is the Y coordinate of the anchor point iCoordinates (2) is the Z coordinate of the anchor point
Example: 
 The following example set the 3D Text anchor position
 
 Dim Pos (2)
 Dim cMarker As Marker
 Pos (0) = 10, Pos (1) = 10, Pos (2) = 10
 cMarker.SetPositions Pos
