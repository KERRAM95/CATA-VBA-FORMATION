RscFor (Object)

Interface representing a Resource For Loop Instruction.

Role: This interface represents a RscFor Instruction in a Resource Task
Property Index
ForIndex Get/Set index identifier name (Default identifier is i).
From Get/Set the FROM expression (Default is 1).
To Get/Set the TO expression (Default is 10).
Type Get/Set for-loop type (Default is DELRscForType_Up).
Method Index
GetIndex Get index as RscDataEntity
