ResourceTask2 (Object)

Interface representing a Resource Task.

Role: This interface is used to create Task-specific objects.
Property Index
ExecutionType The Type of Task execution
Instructions Retreives the list of instructions from the task.
MainRscSequence Retreives the main RscSequence of the task.
RscConstList List all the constants in the current ResourceTask.
RscLocalVarList List all the local variables in the current ResourceTask.
Method Index
CreateRscConst Create a constant in the current ResourceTask.
CreateRscLocalVar Create a local variable in the current ResourceTask.
CreateSPMOperation Creates an SPMOperation under a DeviceTask (not supported for RobotTask)
DeleteRscConst Deletes a RscConst.
DeleteRscLocalVar Deletes a RscLocalVar.
MoveAfter Move a ResourceTask after another one inside the same father.
MoveBefore Move a ResourceTask before another one inside the same father.

Properties


Property ExecutionType() As (Read Only)
The Type of Task execution
Returns:
oTaskExecutionType The Type of execution of the current Task. Legal values:
  • DELRscTaskExecutionType_Internal : Internal Task
  • DELRscTaskExecutionType_Service : Service Task
Example: 
   
   Dim oResourceTask As ResourceTask2
   ......
   'Display DELRscTaskExecutionType
   If (DELRscTaskExecutionType_Internal = oResourceTask.RscTaskExecutionType) Then
   MsgBox "Execution type =  Internal"
   Else
   MsgBox "Execution type =  Service"
   End If
See also:
Property Instructions() As (Read Only)
Retreives the list of instructions from the task.
Returns:
oInstructions The list of instructions.
Example: 
   
   Dim oResourceTask As ResourceTask2
   ......
   Dim oInstructions
   oInstructions = oResourceTask.Instructions
   
   Dim oInstr As RscInstruction
   Set oInstr = oInstructions(0)
   oInstr.Name = "ResourceInstructionVB"
   
   Dim oCustomInstr As RscCustomInstruction
   Set oCustomInstr = oInstructions(1)
   oCustomInstr.Name = "ResourceCustomInstructionVB"
   
   Dim oMotionAct As RobotMotion
   Set oMotionAct = oInstructions(2)
   oMotionAct.Name = "RobotMotionVB"

Methods


Func CreateRscConst( iName, iType, iVal) As
Create a constant in the current ResourceTask.
Parameters:
iName
The name of the constant to create.
iType
The type of the constant to create.
iVal
The value of the constant to create.
Returns:
oConst The created constant.
Example: 
   
   Dim oResourceTask As ResourceTask2
   ......
   Dim iName As String
   iName = "MyConst"
   Dim iVal As String 
   iVal = "MyVal"
   Dim oConst As RscConst
   Dim iType As DELRscDataEntityType
   iType = DELRscDataEntityType_Integer
   Set oConst = oResourceTask.CreateRscConst(iName, iType, iVal)
See also:

Func CreateSPMOperation( iBefore, iReferenceInstruction) As
Creates an SPMOperation under a DeviceTask (not supported for RobotTask)
Returns:
oCreatedSPMOp The created SPMOperation.
Parameters:
iBefore
Create before the reference instruction or not.
iReferenceInstruction
The reference instruction before/after which the SPMOperation has to be created.
Example: 
   
   Dim objResTask As ResourceTask2
          ......
   Dim  objNewSPMOp As SPMOperation
   Dim  objRefAct As DeviceMotion
   Dim  CreateBefore As Boolean
   CreateBefore = FALSE
          ......
   Call objResTask.CreateSPMOperation(CreateBefore, objRefAct, objNewSPMOp)
