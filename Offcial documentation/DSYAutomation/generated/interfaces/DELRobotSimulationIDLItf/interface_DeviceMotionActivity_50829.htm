DeviceMotionActivity (Object)

Interface representing a DeviceMotion.

Role: This interface is used to get and set motion attributes on a device motion
Property Index
AccuracyProfile This property returns and sets Accuracy profile used by the device Motion.
HomeTarget This property returns and sets the Home target of the device motion.
JointTarget This property returns and sets the Joint target of the device motion.
KeepAccuracyProfile This property determines whether the Accuracy profile stored in the device motion is to used during simulation.
KeepMotionProfile This property determines whether the Motion profile stored in the device motion is to used during simulation.
MotionGroups Retreives the list of motion groups managed by the device motion.
MotionProfile This property returns and sets Motion profile used by the device motion.
TargetList This property retrieves the Target List of the device motion.
TargetType This property retrieves the Target Type of the device motion.
UserProfileList This property returns and sets the user profiles used by the device motion.
Method Index
GetAuxillaryAxisHome Retrieves the underlying Auxillary device Home.
GetAuxillaryAxisValues Retrieves the Auxillary device Joint values.
GetTargetListForMotionGroup This property retrieves the Target List for a specific motion group.
SetAuxillaryAxisHome Sets the underlying Auxillary device Home.
SetAuxillaryAxisValues Sets the Auxillary device Joint values.

Property KeepAccuracyProfile() As
This property determines whether the Accuracy profile stored in the device motion is to used during simulation. Keep Accuracy Profile TRUE is profile stored is not to be used. FALSE is profile stored is to be used.
Returns:
oKeepAccuracyProfile The Keep Accuracy Profile mode.
Parameters:
iKeepAccuracyProfile
The Keep Accuracy Profile mode.
Example: 
   
   Dim objDeviceMotion As DeviceMotionActivity
          ......
	Dim iKeepAccuracyProfile As Boolean
	iKeepAccuracyProfile = False
	objDeviceMotion.KeepAccuracyProfile = iKeepAccuracyProfile
