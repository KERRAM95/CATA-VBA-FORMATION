RobotMotionActivity (Object)

Interface representing a RobotMotion.

Role: This interface is used to get and set motion attributes on a robot motion
Property Index
AccuracyProfile This property returns and sets Accuracy profile used by the robot Motion.
CartesianTarget This property returns and sets the cartesian target of the robot motion.
Config This property returns and sets the robot posture used by the robot motion.
ElbowAngle This property returns and sets the Elbow angle used by the robot motion.
HomeTarget This property returns and sets the Home target of the robot motion.
InterpolationMode This property returns and sets the interpolation mode of the robot motion.
JointTarget This property returns and sets the Joint target of the robot motion.
KeepAccuracyProfile This property determines whether the Accuracy profile stored in the robot motion is to used during simulation.
KeepMotionProfile This property determines whether the Motion profile stored in the robot motion is to used during simulation.
KeepObjectProfile This property determines whether the Object profile stored in the robot motion is to used during simulation.
KeepToolProfile This property determines whether the tool profile stored in the robot motion is to used during simulation.
MotionGroups Retreives the list of motion groups managed by the robot motion.
MotionProfile This property returns and sets Motion profile used by the robot motion.
MotionType This property returns and sets the motion type of the robot motion.
ObjectProfile This property returns and sets Object profile used by the robot Motion.
OrientationMode This property returns and sets the Orientation Mode of the robot motion.
TagTarget This property returns the Tag target of the robot motion.
TargetList This property retrieves the Target List of the robot motion.
TargetType This property retrieves the Target Type of the robot motion.
ToolProfile This property returns and sets tool profile used by the robot motion.
TurnNumbers This property returns and sets the turn numbers of the robot motion.
TurnSigns This property returns and sets the turn signs of the robot motion.
UserProfileList This property returns and sets the user profiles used by the robot motion.

Property InterpolationMode() As
This property returns and sets the interpolation mode of the robot motion. The Interpolation Mode. 0 for NEARMODE, // Shortest Angle Strategy 1 for KEEPCFG_KEEPTURN // The posture and the turn values are unchanged at the end of move 2 for KEEPCFG_SETTURN, // The turn values in the robot motion is applied at the end of move 3 for SETCFG_KEEPTURN, // The posture set in the robot motion is applied at the end of move 4 for SETCFG_SETTURN // The posture and the turn values in the robot motion is applied at the end of move
Returns:
oInterpolationMode The interpolation mode used.
Parameters:
iInterpolationMode
The interpolation mode to use.
Example: 
   
   Dim objRobotMotion As RobotMotionActivity
          ......
	Dim oInterpolationMode
	oInterpolationMode = objRobotMotion.InterpolationMode

Property MotionType() As
This property returns and sets the motion type of the robot motion.
