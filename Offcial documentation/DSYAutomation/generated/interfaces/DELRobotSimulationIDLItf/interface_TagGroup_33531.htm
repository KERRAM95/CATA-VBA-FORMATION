TagGroup (Object)

Interface representing a Tag Group.

Role: This interface is used to retieve/assign the attributes from the tag group.
Property Index
Owner This property returns the owner of the tag group
TagCount This property returns the number of tags under the tag group
TagList This property returns the tag list under the tag group
Method Index
DeleteTag Deletes a Tag
Empty Empties a Tag Group
GetTag Returns a Tag by Index.

Properties


Property Owner() As (Read Only)
This property returns the owner of the tag group
Returns:
oOwner The first occurence of the owner.
Example:

          ......
	Dim oOwner As AnyObject
	Set oOwner = objTagGroup.Owner
