RscInstruction (Object)

Interface representing a Resource Instruction.

Role: This interface represents a Resource Instruction in a Resource Task
Property Index
FatherInstruction Retrieve the Father Sequence.
GotoInstructions Get all Goto instruction pointing to this label.
Index Retrieve the index in the father Sequence.
Label Get/Set the label associated to this instruction.
Method Index
GetDataEntityFromName Retrieve a data entity from its name.
Move Move the current instruction in the parent sequence.
MoveToOtherSequence Move the current instruction in a target sequence.
RemoveLabel Remove the label associated to this instruction.

Properties


Property FatherInstruction() As (Read Only)
Retrieve the Father Sequence.
Returns:
oFatherInstr The returned Father Sequence.
Example: 
   
   Dim oResourceTask As ResourceTask2
   ......
   Dim ResourceSequence As RscSequence
   Set ResourceSequence = oResourceTask.MainRscSequence
   ......
   Dim oInstructions
   oInstructions = ResourceSequence.Instructions
   
   Dim oInstr As RscInstruction
   Set oInstr = oInstructions(0)
   Dim FatherInstruction As RscInstruction
   Set FatherInstruction = oInstr.FatherInstruction
