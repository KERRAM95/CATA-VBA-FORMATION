TagPoint (Object)

Interface representing a Tag.

Role: This interface is used to retieve/assign the attributes from the tag.
Property Index
Axis This property returns and sets the relative location of the tag w.
Type This property returns and sets the Type of the tag.
Method Index
GetAbsoluteAxis Retrieves the underlying axis of the Tag.
GetName Retrieves the name of the tag.
SetAbsoluteAxis Sets axis of the Tag.
SetName Sets the name of the tag.

Methods


Func GetAbsoluteAxis( iContext, iTagOwner) As
Retrieves the underlying axis of the Tag. This will return the Tag's location wrt the specified context If no context is specified, this will return the Tag's location wrt the global context
Parameters:
iContext
The context in which Tag's location is required
iTagOwner
Occurrence of the Tag owner product in the root context.
Returns:
oAxis The underlying Axis.
Example: 
   
   Dim objTag As TagPoint
          ......
	Dim iContext, iTagOwner
          ......
	Dim oAbsAxis(0 To 5)
	Call objTag.GetAbsoluteAxis(iContext, iTagOwner, oAbsAxis)
