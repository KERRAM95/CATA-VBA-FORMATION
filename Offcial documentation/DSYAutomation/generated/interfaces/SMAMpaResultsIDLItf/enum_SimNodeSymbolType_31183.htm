SimNodeSymbolType (Enumeration)

enum SimNodeSymbolType {
  SimNodeSymbolSphere,
  SimNodeSymbolCross,
  SimNodeSymbolPlus,
  SimNodeSymbolConcentric,
  SimNodeSymbolCoincident,
  SimNodeSymbolFullCircle,
  SimNodeSymbolFullSquare,
  SimNodeSymbolStar,
  SimNodeSymbolDot,
  SimNodeSymbolSmallDot,
  SimNodeSymbolFullCircle2,
  SimNodeSymbolFullSquare2
}
The available types of symbol nodes.
Values:
SimNodeSymbolSphere
Displays the symbols as a sphere.
SimNodeSymbolCross
Displays the symbols as a cross.
SimNodeSymbolPlus
Displays the symbols as a plus sign.
SimNodeSymbolConcentric
Displays the symbols as concentric.
SimNodeSymbolCoincident
Displays the symbols as coincident.
SimNodeSymbolFullCircle
Displays the symbols as a filled circle.
SimNodeSymbolFullSquare
Displays the symbols as a filled square.
SimNodeSymbolStar
Displays the symbols as a star.
SimNodeSymbolDot
Displays the symbols as a dot.
SimNodeSymbolSmallDot
Displays the symbols as a small dot.
SimNodeSymbolFullCircle2
Displays the symbols as a full circle of bigger size.
SimNodeSymbolFullSquare2
Displays the symbols as a full square of bigger size.
