SimSymbolicFrameRangesEnum (Enumeration)

enum SimSymbolicFrameRangesEnum {
  SimUndefined,
  SimAllFrames,
  SimLastFrameOfEachStep,
  SimAllFramesInStep,
  SimTimeBasedFrames,
  SimFrequencyFrames
}
The predefined symbolic frame range types for the frame selection.
Values:
SimUndefined
Specifies that no values is to be defined.
SimAllFrames
Considers all the frames from all the steps.
SimLastFrameOfEachStep
Considers the last frame of each step.
SimAllFramesInStep
Considers all the frames of a particular step. User needs to specify the Step ID through the SMAIAMpaFramesSelection::put_StepList method. For retrieving the step ID, one needs to use SMAIAMpaResultsStep::GetIdentifier method.
SimTimeBasedFrames
Considers all the frames present in all the time based steps.
SimFrequencyFrames
Considers all frames present in all the frequency steps.
