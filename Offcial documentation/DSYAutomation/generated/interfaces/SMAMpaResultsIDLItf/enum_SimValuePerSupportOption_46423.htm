SimValuePerSupportOption (Enumeration)

enum SimValuePerSupportOption {
  SimValuePerSupportNone,
  SimValuePerSupportMax,
  SimValuePerSupportMin,
  SimValuePerSupportAbsoluteMax,
  SimValuePerSupportAverage,
  SimValuePerSupportSum,
  SimValuePerSupportLast
}
Represents the available parameter options.
Values:
SimValuePerSupportNone
If specified, the Global calulcation will be based on the parameters set in SetParameters().
SimMax
The maximum value will be used for the computation.
SimMin
The minimum value will be used for the computation.
SimAbsoluteMax
The absolute maximum value will be used for the computation.
SimAverage
The average value will be used for the computation.
SimSum
The sum value will be used for the computation.
SimLast
The last value will be used for the computation. It will create the paramter for the last region/subregion. Its useful only for History sensors.
