SimStressLine (Object)

Represents the Stress Line class.
Role:After creating the Stress line, one needs set the different API's provided in the Interface.
Then update the Stress line using and the method.
Example:
 Given a SimResultsAnalysisCase object, you can create a Stress line object as following.
 
 Dim oStressLine As SimStressLine
 Set oStressLine = oResultsAnalysisCase.CreateStressLine
 oStressLine.SetStartPoint .04, .03, .02
 oStressLine.SetEndPoint .04, -.03, .02
 oStressLine.SetMembraneComponentForXYdata True, False, False, False, False, False
 oStressLine.SetBendingComponentForXYdata True, False, False, False, False, False
 oStressLine.SetBendingComponentForInvariant True, False, False, False, False, False
 oStressLine.UseCurvatureCorrection = False
 oStressLine.Update True

Properties


Property EndPointSpecification() As
Gets/Sets the type of End point specification needed. See the enum SimStressLineSpecification. If not specified, the default will be SimSpecificationPoints,
Property NumberOfIntervals() As
Gets/Sets the number of intervals needed for the stress line.
Property PathDefinition() As
Sets the Path plot to create the stress line. This method is to be used when SimSpecificationPath option is used. The ospPath will be an Path Plot Interface. Only the path containing 2 points are valid.

Gets/Sets the use of curvature correction option.

Sub SetInplaneCurvatureRadius( ibSpecify, idValue)
Specifies a value for the In-plane radius of curvature of the midplane of your model at the location of the stress line.
Parameters:
ibSpecify
Set true to specify a value. If set false, infinite value will be considered, which indicates a lack of curvature. The unit must be MKS.
idValue
Curvature radius value. It will be neglected if ibSpecify is set false.

Sub SetStepAndFrame( icsPersistentStepID, inFrameIndex, inLoadCaseIndex)
Sets the step, frame and load case information needed for which the Stress Line will be created. If not set, the last Frame of the first Step will be considered.
Parameters:
icsPersistentStepID
The persistent ID of the step.
inFrameIndex
The frame index. The index starts from 1.
inLoadCaseIndex
The load case index. The index starts from 1. Set 0 if there is no load case. SimResultsStep::GetLoadCases should be used to retrieve the required LoadCase ID for that step.
