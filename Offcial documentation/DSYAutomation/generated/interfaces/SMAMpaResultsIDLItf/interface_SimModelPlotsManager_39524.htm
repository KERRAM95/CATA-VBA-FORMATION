SimModelPlotsManager (Object)

Represents the FEM Rep manager and can be used to create feature used in Model Plots.
 
Example:

 Given a CATFmtRoot object, you can create a SimResultsManager object as following.
 
 
 Set MyRef0 = oRootOcc.ReferenceRootOccurrenceOf
 Dim oResultsManager As SimModelPlotsManager
 Set oResultsManager = oSimulationReference.GetItem("SimResultsManager")

Func GetSet( ieSetType) As
Gets a list of result features created within this Manager.
Parameters:
ieSetType
The type of Set to access such as field plot, sensors, frequency sensors and resultant sensors. The keyword for various sets are as follows: 1. Field plot set: "ModelPlotSet". 2. Sensor set: "DisplayGroupSet".
Returns:
The set of specified model plots feature.
Example:
The following example returns in ModelPlotSet the field plot set created in MyResultsAnalysisCase : 
 Dim oFmtResultsRep As SimFemRoot
 Set oFmtResultsRep = ...
 Set oFieldPlotSet = oFmtResultsRep.GetSet("FieldPlotSet")
 Dim oFieldPlot As SimFieldPlot
 Set oFieldPlot = oFieldPlotSet.Item(1)
