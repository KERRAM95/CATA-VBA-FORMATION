SimShellStyle (Enumeration)

enum SimShellStyle {
  SimShellNone,
  SimThick,
  SimOffset,
  SimElevation
}
The shell styles.
Values:
SimNone
The shell is rendered with no thickness.
SimThick
The shell is rendered with thickness at the actual location. The thickness is determined by the value selected for the Thickness Scale Factor and thickness data specified in the analysis element property definitions.
SimOffset
The shell is rendered with no thickness at the mid-plane of the thick shell. The displacement of the shell from the reference surface is determined by the Thickness Scale Factor and thickness (Offset) data specified in the analysis element property definitions.
SimElevation
The edges of the shell are rendered with thickness at the actual location. The face of the shell is rendered at appropriately the location of the result data within that thickness.
