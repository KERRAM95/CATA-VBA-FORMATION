SimSensorBase (Object)

Represents the Base Sensor.
All the sensors are derived from this sensor.
Property Index
FrameSelector Sets the frame selector.
SimSupport Sets the support for orphan results.
Support Sets the support for native results.
Method Index
Update Updates the sensor.

Properties


Property FrameSelector( ispFrameSelector) (Write Only)
Sets the frame selector.
Property SimSupport( icusSupport) (Write Only)
Sets the support for orphan results. The name of the support such as node set.
Property Support( ilsSupport) (Write Only)
Sets the support for native results. Support can be features such as loads, restraints.

Methods


Sub Update()
Updates the sensor. It should be called at the end after setting all the sensor properties and the frame selector.
