SimModelPlotsExperienceContent (Object)

Represents the Experience Content class for Model Plots.
Role:After creating the Experience Content, one needs set the different API's provided in the Interface.
Then call the generate method of SimExperienceContent to create the contents.
Example:
 Given a SimFEMRoot object, you can create an Experience Content object as following.
 
 Dim oExpContent
 Set oExpContent = oFEMRoot.CreateExperienceContent
 Dim eSupportType As SimExpContentSupportTypes
 eSupportType = SimExpContentDisplayGroups
 Dim oModelPlots(1)
 Set oModelPlots(0) = MyDispatch1
 Set oModelPlots(1) = MyDispatch2
 oExpContent.AddModelPlots oFieldPlot, eSupportType, oModelPlots
 oExpContent.Generate
Method Index
AddModelPlots Specifies the plots to be added in the experience content.

Methods


Sub AddModelPlots( iPlot, ieSupportType, ilsDisplayGroups)
Specifies the plots to be added in the experience content.
Parameters:
iPlot
The plot features to be added as CATBaseDispatch. The existing plots can be retieved using SimModelPlotsManager::GetSet() mehtod.
ieSupportType
The support types. Refer SimExpContentSupportTypes for all the options.
ilsDisplayGroups
The display group features to be set for each plot. The existing display groups can be retieved using SimModelPlotsManager::GetSet() mehtod. It must be specified only if ieSupportType is SimExpContentDisplayGroups, else it can be empty.
Returns:
S_OK if successful.
