SimSamplingFilterTypes (Enumeration)

enum SimSamplingFilterTypes {
  SimTimeInterval,
  SimFrameInterval,
  SimTotalFrames
}
The sampling filter types.
Values:
SimTimeInterval
Specifies the time interval.
SimFrameInterval
Specifies the frame interval.
SimTotalFrames
Total number of frame to be extracted from overall frames, these will be equally spaced frames.
