SimCalculationBetweenSupports (Enumeration)

enum SimCalculationBetweenSupports {
  SimDiffSupport1AndSupport2,
  SimDiffSupport2AndSupport1,
  SimSumSupport1AndSupport2
}
Represents the available types calculation between two supports.
Values:
SimDiffSupport1AndSupport2
Difference between Support1 and Support2.
SimDiffSupport2AndSupport1
Difference between Support2 and Support1.
SimSumSupport1AndSupport2
Sum of the two supports.
