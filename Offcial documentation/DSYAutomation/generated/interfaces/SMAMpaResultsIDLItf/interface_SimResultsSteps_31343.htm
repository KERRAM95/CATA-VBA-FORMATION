SimResultsSteps (Object)

Represents the collection of results analysis cases and can be used to retrieve each results case.
Example:
 Given a ResultsAnalysisCase object, you can create a Results Steps object as following.
 
 Dim oResultsSteps As SimResultsSteps
 Set oResultsSteps = oResultsAnalysisCase.ResultsSteps
