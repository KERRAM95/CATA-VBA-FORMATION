SimThresholdType (Enumeration)

enum SimThresholdType {
  SimThresholdNone,
  SimThresholdLowerLimit,
  SimThresholdUpperLimit,
  SimThresholdPercentLowerLimit,
  SimThresholdPercentUpperLimit
}
The Threshold types.
Values:
SimNone
No threshold value is used (Default) .
SimLowerLimit
Establishes a lower limit for the assignment of color to results. Results below the Lower limit are assigned the out of range color. Extrema can be displayed for each contiguous region with results values greater than the Lower limit.
SimUpperLimit
Establishes an upper limit for the assignment of color to results. Results above the Upper limit are assigned the out of range color. Extrema can be displayed for each contiguous region with results values less than the Upper limit.
SimPercentLowerLimit
Establishes a lower limit as a percentage of the entire range for the assignment of color to results.
SimPercentUpperLimit
Establishes an upper limit as a percentage of the entire range for the assignment of color to results.
