SimExpContentSupportTypes (Enumeration)

enum SimExpContentSupportTypes {
  SimExpContentWholeModelSingleBlock,
  SimExpContentWholeModelByPart,
  SimExpContentAsCurrent,
  SimExpContentDisplayGroups
}
Represents the available types to supports which will be used to generate the contents.
Values:
SimWholeModelSingleBlock
Generates experience content for the whole model. If your model has a large number of parts, this option increases performance. However, you cannot then select the individual parts.
SimWholeModelByPart
Generates experience content for the model extracted by part. This option enables you to retain the ability to manipulate the individual parts.
SimAsCurrent
Generates experience content using the current display group support. * @param SimDisplayGroups Generates experience content for the model extracted by display groups. This option enables you to select one or more display groups from which to generate experience content for the lightweight viewer apps.
