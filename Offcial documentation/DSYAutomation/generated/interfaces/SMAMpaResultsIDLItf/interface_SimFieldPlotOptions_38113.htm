SimFieldPlotOptions (Object)

Represents the field plot rendering options.
Role:After creating the field plot, one can set the various options provided in this interface.
The Update method needs to be called at the end after setting all the options.
Example:
 Given a field plot object, you can set the various options.
 
 Dim oFieldPlotOptions As SimFieldPlotOptions
 Set oFieldPlotOptions = oFieldPlot.GetItem("SimFieldPlotOptions")
 oFieldPlotOptions.Transparency = False
 oFieldPlotOptions.SetShowLabels False, False
 oFieldPlotOptions.Update
Property Index
ExteriorRenderStyle Sets the exterior render style for the model.
ExteriorSymbols Sets the exterior symbols to the field plot.
HideModel Hides the model and only shows the symbols.
IsocontourRenderStyle Sets the iso-contour render style.
ModelRenderStyle Sets the render style for the model.
SectionPointsLocation Specifies the section point location.
ShowIsocontourEdges Displays the isocontour edges.
SymbolDensity Sets the symbol density to the field plot.
Transparency Sets the transparency to the field plot.
TransparentIsocontours Sets the transparency of the iso-contours.
Method Index
SetNodeSymbolAttributes Sets the node symbol attributes.
SetShellStyle Specifies the shell styles options.
SetShowLabels Displays the node and element and labels.
SetShowSPHDisplay Specifies the SPH display options.
SetThreshold Specifies the threshold type to set the measurement criteria.
SetVisibleEdges Specifies the way in which the edges will be displayed.
Update Updates the field plot.

Properties


Property ExteriorRenderStyle( ieStyle) (Write Only)
Sets the exterior render style for the model. The SimRenderStyle enum can be found in SMAIAMpaFieldPlotEnums file
Property ExteriorSymbols( ibExtSymbol) (Write Only)
Sets the exterior symbols to the field plot. Set True to show the exterior symbol.
Property HideModel( ibHideModel) (Write Only)
Hides the model and only shows the symbols.
Property IsocontourRenderStyle( ieStyle) (Write Only)
Sets the iso-contour render style. The SimRenderStyle enum can be found in SMAIAMpaFieldPlotEnums file
Property ModelRenderStyle( ieStyle) (Write Only)
Sets the render style for the model. The SimRenderStyle enum can be found in SMAIAMpaFieldPlotEnums file
Property SectionPointsLocation( ieLocation) (Write Only)
Specifies the section point location. The SimSectionPointLocation enum can be found in SMAIAMpaFieldPlotEnums file.
Property ShowIsocontourEdges( ibShowEdges) (Write Only)
Displays the isocontour edges.
Property SymbolDensity( idValue) (Write Only)
Sets the symbol density to the field plot. The value ranges from 1 to 100.
Property Transparency( ibTransperency) (Write Only)
Sets the transparency to the field plot. Set True to make it transparent.
Property TransparentIsocontours( ibTransparency) (Write Only)
Sets the transparency of the iso-contours.

Methods


Sub SetNodeSymbolAttributes( ieSymbolType, idValue, ibUseDepthEffect)
Sets the node symbol attributes. The value ranges from 1 to 100.
Parameters:
SimNodeSymbolType
Specifies the symbol type. See SimNodeSymbolType for more options
ibUseDepthEffect
Set True to show the depth effect.
Sub SetShellStyle( ieShellStyle, ieSourceType, idScaleFactor)
Specifies the shell styles options.
Parameters:
ieShellStyle
Specifies the shell style. The SimShellStyles enum can be found in SMAIAMpaFieldPlotEnums file.
ieSourceType
Specify the thickness data source for the rendering thickness. The SimThicknessDataSourceType enum can be found in SMAIAMpaFieldPlotEnums file.
idScaleFactor
Specifies the Thickness Scale Factor, expressed as a multiplier, to enhance visibility.
Sub SetShowLabels( ibNodeLabels, ibElementLabels)
Displays the node and element and labels.
Parameters:
ibNodeLabels
Set true to show the node labels.
ibElementLabels
Set true to show the element labels.
Sub SetShowSPHDisplay( ibShowParticles, ibShowSurface)
Specifies the SPH display options. If both the options are set True, the SPH particle as well as SPH surface will be shown.
Parameters:
ibShowParticles
Set True to show the SPH particles.
ibShowSurface
Set True to show the SPH surface.
Sub SetThreshold( ieThresholdType, idValue, ibShowExtrema)
Specifies the threshold type to set the measurement criteria. It also displays the minimum and maximum values in the plot.
Parameters:
ieThresholdType
Specifies the threshold type. The SimThresholdType enum can be found in SMAIAMpaFieldPlotEnums file.
idValue
Specifies the upper or the lower limit depending on the threshold type.
ibShowExtrema
True to display the extrema on the plot.
Sub SetVisibleEdges( ieEdgeType, idEdgeAngle)
Specifies the way in which the edges will be displayed.
Parameters:
ieEdgeType
Set the type of edge option. The SimVisibleEdges enum can be found in SMAIAMpaFieldPlotEnums file.
idEdgeAngle
This option is valid only for SimOutline option only. The edge angle is the minimum angle between adjoining element faces in order for an edge to be shown in the display. The edge angle must be greater than zero and less than or equal to 90°.
Sub Update()
Updates the field plot. This method should be called at the end after setting all the above attributes.
