MOTExecutionServices (Object)

Automation interface for the MOTExecutionServices object.

Interface representing services for the execution of a motion simulation.
Example:
Given a CATIAEditor object, you can retrieve a MOTExecutionServices object as following: 
  Dim oEditor As Editor
  Set oEditor = CATIA.ActiveEditor
  
  Dim oMOTExecutionServices As MOTExecutionServices
  Set oMOTExecutionServices = oEditor.GetService("MOTExecutionServices")
