MOTAnalysisCaseExecutionInfos (Object)

Automation interface for the MOTAnalysisCaseExecutionInfos object.

Interface to access the following execution information after the run of a motion analysis case:
  • The name of the motion analysis case
  • The status of the run analysis case: enum MOTExecutionStatus
  • The execution log
Examples:
Given a MOTExecutionServices object, you can run a specific motion analysis case and retrieve a MOTAnalysisCaseExecutionInfos object as following: 
  Dim oAnalysisCaseExecutionInfos As MOTAnalysisCaseExecutionInfos
  Call oMOTExecutionServices.RunAnalysisCase(oSimulationReference, "Motion Analys Case.1", oAnalysisCaseExecutionInfos)
Given a MOTExecutionServices object, you can run all the motion analysis cases and retrieve a list of MOTAnalysisCaseExecutionInfos objects as following: 
  Dim listOfAnalysisCaseExecutionInfos()
  Call oMOTExecutionServices.RunAllAnalysisCases(oSimulationReference, listOfAnalysisCaseExecutionInfos)
Property Index
AnalysisCaseName Returns the name of the motion analysis case that has been run.
ExecutionLog Returns the execution log of the run analysis case.
ExecutionStatus Returns the execution status of the run analysis case.
