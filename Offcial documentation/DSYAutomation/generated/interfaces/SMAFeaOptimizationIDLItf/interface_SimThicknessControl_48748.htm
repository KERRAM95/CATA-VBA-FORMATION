SimThicknessControl (Object)

Represents the thickness control object.
Example:
Given a SimDesignImprovementFeatures object, you can create a SimThicknessControl as following: 
 Dim MyFeatures As SimDesignImprovementFeatures
 ...
 Dim MyThicknessControl As SimThicknessControl
 Set MyThicknessControl = MyFeatures.Add("SimThicknessControl")
Given a SimDesignImprovementFeatures object, you can retrieve a SimThicknessControl named "Thickness Control.1" as following: 
 Dim MyFeatures As SimDesignImprovementFeatures
 ...
 Dim MyThicknessControl As SimThicknessControl
 Set MyThicknessControl = MyFeatures.Item("Thickness Control.1")
Example in Python:
