SimCenterOfGravityConstraint (Object)

Represents the Center of gravity constraint object.
Example:
Given a SimDesignImprovementFeatures object, you can create a SimCenterOfGravityConstraint as following: 
 Dim MyFeatures As SimDesignImprovementFeatures
 ...
 Dim MyCenterOfGravityConstraint As SimCenterOfGravityConstraint
 Set MyCenterOfGravityConstraint = MyFeatures.Add("SimCenterOfGravityConstraint")
Given a SimDesignImprovementFeatures object, you can retrieve a SimCenterOfGravityConstraint named "Center Of Gravity Constraint.1" as following: 
 Dim MyFeatures As SimDesignImprovementFeatures
 ...
 Dim MyCenterOfGravityConstraint As SimCenterOfGravityConstraint
 Set MyCenterOfGravityConstraint = MyFeatures.Item("Center Of Gravity Constraint.1")
Example in Python:
Given a SimDesignImprovementFeatures object, you can create a SimCenterOfGravityConstraint as following:
