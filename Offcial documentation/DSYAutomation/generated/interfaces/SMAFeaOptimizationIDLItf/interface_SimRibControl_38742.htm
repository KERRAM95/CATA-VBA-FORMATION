SimRibControl (Object)

Represents the rib control object.
Example:
Given a SimDesignImprovementFeatures object, you can create a SimRibControl as following: 
 Dim MyFeatures As SimDesignImprovementFeatures
 ...
 Dim MyRibControl As SimRibControl
 Set MyRibControl = MyFeatures.Add("SimRibControl")
Given a SimDesignImprovementFeatures object, you can retrieve a SimRibControl named "Rib Control.1" as following: 
 Dim MyFeatures As SimDesignImprovementFeatures
 ...
 Dim MyRibControl As SimRibControl
 Set MyRibControl = MyFeatures.Item("Rib Control.1")
Example in Python:
Given a SimDesignImprovementFeatures object, you can create a SimRibControl as following: 
 ...
