SimStrainResponseVariable (Object)

Represents the strain response variable object.
Example:
Given a SimDesignImprovementFeatures object, you can create a SimStrainResponseVariable as following: 
 Dim MyFeatures As SimDesignImprovementFeatures
 ...
 Dim MyStrainResponseVariable As SimStrainResponseVariable
 Set MyStrainResponseVariable = MyFeatures.Add("SimStrainResponseVariable")
Given a SimDesignImprovementFeatures object, you can retrieve a SimStrainResponseVariable named "Strain Response Variable.1" as following: 
 Dim MyFeatures As SimDesignImprovementFeatures
 ...
 Dim MyStrainResponseVariable As SimStrainResponseVariable
 Set MyStrainResponseVariable = MyFeatures.Item("Strain Response Variable.1")
Example in Python:
Given a SimDesignImprovementFeatures object, you can create a SimStrainResponseVariable as following: 
 ...

Given a SimDesignImprovementFeatures object MyFeatures, you can retrieve a SimStrainResponseVariable named "Strain Response Variable.1" as following:
