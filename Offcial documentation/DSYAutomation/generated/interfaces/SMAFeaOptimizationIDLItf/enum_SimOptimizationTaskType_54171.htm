SimOptimizationTaskType (Enumeration)

enum SimOptimizationTaskType {
  SimMaximizeStiffness,
  SimMinimizeMass,
  SimMaximizeLowestFrequency,
  SimMinimizeMaximumStress,
  SimMaximizeResponseVariableValues,
  SimMinimizeResponseVariableValues,
  SimMinimizeTheMaximumResponseVariableValues
}
The objective type.
Values:
SimMaximizeStiffness
The optimization target is to maximize the stiffness for a given target mass
SimMaximizeLowestFrequency
The optimization target is to maximize the first five EigenFrequencies for a given target mass
SimMinimizeMass
The optimization target is to minimize the mass
SimMaximizeResponseVariableValues,
The optimization target is to maximize response variable
SimMinimizeResponseVariableValues,
The optimization target is to minimize response variable
SimMinimizeTheMaximumResponseVariableValues
The optimization target is to minimize the maximum response variable
