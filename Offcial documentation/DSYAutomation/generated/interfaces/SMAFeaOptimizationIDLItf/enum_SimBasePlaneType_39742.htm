SimBasePlaneType (Enumeration)

enum SimBasePlaneType {
  SimPrintingBed,
  SimSupportStructures
}
The overhang control type.
Values:
SimPrintingBed
Support structures must be directly connected to the bed.
SimSupportStructures
Presumes the boundaries are supported by features that are not in the design space
