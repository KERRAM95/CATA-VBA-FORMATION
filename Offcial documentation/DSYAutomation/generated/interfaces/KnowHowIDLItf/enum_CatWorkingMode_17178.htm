CatWorkingMode (Enumeration)

enum CatWorkingMode {
  WholeObjects,
  OccurenceObjects,
  PLMObjects,
  AllOccurenceObjects
}
Protocole to retrieve the facts for the rulebase.

Values:
WholeObjects
0 : the rule base works on all the objects that are in session (even the objects inside the representation).
OccurenceObjects
1 : the rule base works on occurrences (leaf positionned instances) for clash rules for example.
PLMObjects
2 : the rule base only works on PLM Objects and their attributes.
AllOccurenceObjects
3 : the rule base only works on occurrences for implement link into RFLP.
