DNBInterpolater (Enumeration)

enum DNBInterpolater {
  FitLINEAR,
  FitSPLINE,
  FitCOMPOSITE
}
Interpolater Types.


Role: This defines the 3 types of Interpolations.

Values:
FitLINEAR
Linear Interpolation.
FitSPLINE
Spline Interpolation.
FitCOMPOSITE
Composite Spline Interpolation.
