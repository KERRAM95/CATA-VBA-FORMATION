CATPPRTreeItemType (Enumeration)

enum CATPPRTreeItemType {
  CATProcessList,
  CATProductList,
  CATResourcesList
}
Item of the PPR tree to which a PathElement object corresponds, when the Editor which is the parent of the Selection object edits a ProcessDocument.
Role: : This enum is used by the Selection.SelectElement method.
Values:
CATProcessList
The LeafNode property of the PathElement object belongs to the ProcessList sub-tree of the ProcessDocument
CATProductList
The LeafNode property of the PathElement object belongs to the ProductList sub-tree of the ProcessDocument
CATResourcesList
The LeafNode property of the PathElement object belongs to the ResourcesList sub-tree of the ProcessDocument
