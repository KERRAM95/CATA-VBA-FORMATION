CATInteractionType (Enumeration)

enum CATInteractionType {
  CATSelection,
  CATIndication,
  CATMouseMove,
  CATInputUndo,
  CATInputRedo,
  CATEscape,
  CATOtherEditor,
  CATExclusiveCommand
}
Type of the interaction the user actually implemented.
Role: This enum is used by interactive methods such as .
Values:
CATSelection
The user selected a feature
CATIndication
The user indicated a location in a Window of the active Editor
CATMouseMove
The user moved the mouse pointer
CATInputUndo
The user selected the Undo item of the Edit menu
CATInputRedo
The user selected the Redo item of the Edit menu
CATEscape
The user pressed the Escape key. The user wants to cancel the script execution. The script must exit immediately
CATOtherEditor
The user selected the Window (for example the title bar) of an Editor which is not the active Editor. The user wants to cancel the script execution. The script must exit immediately
CATExclusiveCommand
The user selected an exclusive command. The user wants to cancel the script execution, the script must exit immediately.
CAUTION: All scripts should, when an interactive method returns the CATExclusiveCommand value, sequentially:
  • do the necessary cleanings
  • call CSO.ActivateExclusiveCommand
  • exit the script
