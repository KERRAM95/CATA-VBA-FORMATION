SimGroup (Object)

Represents a simulation group.
Property Index
Activated Returns or sets the activation status of the group.
EntityType Returns or set the simulation entities type captured by the group.
GroupType Retrieves the type of group.
NumberOfEntities Retrieves the number of entities that have been captured by the group.
Method Index
AddBoundary Creates a new boundary and add it into the boundary description of the simulation group.
GetAttributeValue Retrieves the value corresponding to the given attribute.
GetContentCoordinates Get the coordinates of the content of the group (double).
GetNumberingLabels Retrieves all the numbering labels of current object
RemoveBoundary Removes a boundary to the boundary description of the simulation group.
RemoveNumbering Removes numbering specification.
SetAttributeValue Sets the value corresponding to the given attribute.
SetGroupType

Properties


Property Activated() As
Returns or sets the activation status of the group.
Property EntityType() As
Returns or set the simulation entities type captured by the group.
The group entity must be initialized before any update of feature.
Property GroupType() As (Read Only)
Retrieves the type of group.
Property NumberOfEntities() As (Read Only)
Retrieves the number of entities that have been captured by the group.

Methods


Sub AddBoundary( iBoundary)
Creates a new boundary and add it into the boundary description of the simulation group.
Only available for simulation group by boundary.
Parameters:
iBoundary:
The object that represents the boundary geometry to add.
Sub GetAttributeValue( iAttribute, oValue)
Retrieves the value corresponding to the given attribute.
Parameters:
iAttribute:
The name of attribute.
Returns:
The value of the attribute.
Sub GetContentCoordinates( oX, oY, oZ)
Get the coordinates of the content of the group (double). If the group contains nodes, it will return the coordinates of the nodes. If the group contains elements, it will return the coordinates of the elements centroids.
Func GetNumberingLabels( iType) As
Retrieves all the numbering labels of current object
Parameters:
iType
Type of mesh entity: simMeshNodeEntity or simMeshElementEntity.
Sub RemoveBoundary( iBoundary)
Removes a boundary to the boundary description of the simulation group.
Only available for simulation group by boundary.
Parameters:
iBoundary:
The object that represents the boundary geometry to remove.
Sub RemoveNumbering()
Removes numbering specification.
Sub SetAttributeValue( iAttributeName, iValue)
Sets the value corresponding to the given attribute.
Parameters:
iAttribute:
The name of attribute.
iValue:
The value of the attribute.
Sub SetGroupType( iGroupType)
Deprecated:
R427 : Use EntityType property instead. Sets the type of the simulation group.
This method must initialize the group content type before any update of feature.
Parameters:
iGroupType:
The type of the group.
