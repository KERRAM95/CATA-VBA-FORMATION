SimSpecializedFEMFactory (Object)

Represents the service to create specialized FEM representations
Example:
Given a VPMReference object, you can retrieve a SimSpecializedFEMFactory as following: 
 Dim MyReference As VPMReference
 Set MyReference = …
 Dim MyFactory As SimSpecializedFEMFactory
 Set MyFactory = MyReference.GetItem("SimSpecializedFEMFactory")

Func CreateAxisymmetricFEM( iParameters) As
Creates an axisymmetric FEM representation.
Parameters:
iParameters
[in] The factory parameters.
Legal values:
Name Type Description
AxiOrigin Object Mandatory : the model origin.
AxiAxis Object Mandatory : the model axis.
AxiPlane Variant Mandatory : the model plane.
Returns:

Func CreatePreloadedFEM( iParameters) As
Creates a preloaded FEM representation.
Parameters:
iParameters
[in] The factory parameters.
Legal values:
Name Type Description
Sequence Object Mandatory : The original analysis case.
Prefix string Optional : The preloaded FEM prefix name.
ModelType Integer Optional : the kind of model to generate.
Legal values:
  • 1 (default) : Creates a standard preloaded FEM.
  • 2 : Creates a preloaded FEM using symmetric model generation.
StepIndex Integer Optional : the analysis case step index to import.
Increment Integer Optional : Specifies the increment number within the step from which to generate the preloaded FEM.
Legal values:
  • -1 (default) : Last Increment
  • 1 to Increment Count : the increment number
IncludeConnections Integer Optional : Copies the connections from the original model along with the preloaded FEM.
Legal values:
  • 0 (default) : no.
  • 1 : yes.
ResetReferenceConfig Integer Optional : Uses the deformed mesh as the baseline for downstream simulations.
Legal values:
  • 0 (default) : no.
  • 1 : yes.
ImportMaterialState Integer Optional : Uses the material state from the selected step and increment as the initial material state for downstream simulations.
Legal values:
  • 0 (default) : no.
  • 1 : yes.
SegmentAngles Double (list) Optional : Defines the segment angles for the mesh segments of the generated model.
Only for symmetric model generation. Size of SegmentAngles list must be equal to size of Divisions and BiasRatio lists.
Divisions Integer (list) Optional : The number of divisions for each segment.
Only for symmetric model generation.
BiasRatios Double (list) Optional : to set a nonuniform distribution of nodes along the segment.
Only for symmetric model generation.
AxisOfSpin Integer Optional : to defines the axis about which the generated mesh revolves.
Only for symmetric model generation.
Legal values:
  • 1 : Uses the x-axis.
  • 2 : Uses the y-axis.
  • 3 : Uses the z-axis.
  • 4 (default) : Uses the axis from the upstream axisymmetric FEM.
Returns:
The created preloaded FEM representation.
