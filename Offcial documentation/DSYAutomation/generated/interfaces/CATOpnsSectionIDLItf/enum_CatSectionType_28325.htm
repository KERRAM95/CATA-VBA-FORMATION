CatSectionType (Enumeration)

enum CatSectionType {
  catSectionTypePlane,
  catSectionTypeSlice,
  catSectionTypeBox
}

This enum CatSectionType defines the 3 types of section plane.

  • catSectionTypePlane : Simple Plane Type
  • catSectionTypeSlice : Slice Plane Type
  • catSectionTypeBox : Box Plane Type
