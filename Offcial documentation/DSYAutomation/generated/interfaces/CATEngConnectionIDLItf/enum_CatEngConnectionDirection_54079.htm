CatEngConnectionDirection (Enumeration)

enum CatEngConnectionDirection {
  catDirectionOut,
  catDirectionIn
}
Engineering Connection direction.
Role: Define the Engineering Connection direction.
Values:
catDirection_Out
means that the Impacted can move. this is the default value.
catDirection_In
means that the Impacted cannot move.
