SimMCXDuplicateMode (Enumeration)

enum SimMCXDuplicateMode {
  simNoMCX,
  simIncludedMCX,
  simAllMCX
}
The MCX duplication mode.
Values:
simNoMCX
No MCX are duplicated.
simIncludedMCX
Only the included MCX are duplicated.
simAllMCX
All the MCX are duplicated.
