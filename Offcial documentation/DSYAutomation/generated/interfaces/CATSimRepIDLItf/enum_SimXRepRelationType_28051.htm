SimXRepRelationType (Enumeration)

enum SimXRepRelationType {
  SimXRepTo3DShapeRelation,
  SimXRepToXRepRelation,
  SimXRepToDocSpecRelation,
  SimXRepToDocResultRelation
}
The XRep relation type.
Values:
SimXRepTo3DShapeRelation
The relation is between the XRep and a 3D Shape.
SimXRepToXRepRelation
The relation is between the XRep and another XRep.
SimXRepToDocSpecRelation
The relation is between the XRep and a specification document.
SimXRepToDocResultRelation
The relation is between the XRep and a result document.
