SimXRep (Object)

Represents a XRep.
Property Index
FileName Sets or retrieves the data file name associated to the XRep.
NavRepName Sets or retrieves the nav rep file name associated to the XRep.
Method Index
AddRelation Adds a relation to the XRep.
ExportFile Exports the data of the XRep to a data file.
ExportNavRep Exports the data of the XRep to a data file.
GenerateNavRep Generates a navrep file from a data file and imports it into the X representation.
ImportFile Imports a data file in the XRep.
ImportNavRep Imports a navrep file in the XRep.
RemoveRelation Removes a relation from the XRep.

Properties


Property FileName() As
Sets or retrieves the data file name associated to the XRep.
Property NavRepName() As
Sets or retrieves the nav rep file name associated to the XRep.

Methods


Sub AddRelation( iPointedObject, iRelationType)
Adds a relation to the XRep.
Parameters:
iPointedObject:
The link object that represent the pointed object.
iRelationType:
The kind of relation, depending on the linked objects.
Sub ExportFile( iFolderPath)
Exports the data of the XRep to a data file.
Parameters:
iFolderPath:
The path of the folder in which the data file will be exported.
Sub ExportNavRep( iFolderPath)
Exports the data of the XRep to a data file.
Parameters:
iFolderPath:
The path of the folder in which the navrep file will be exported.
Sub GenerateNavRep( iFilePath, iLengthUnit)
Generates a navrep file from a data file and imports it into the X representation.
Parameters:
iFilePath
The path of the data file from which the navrep file will be generated.
iLengthUnit
The length unit of the data file.
Legal values:
METER
INCH
FOOT
YARD
...
Sub ImportFile( iFolderPath, iKeepNavRep)
Imports a data file in the XRep.
Parameters:
iFolderPath:
The path of the folder from which the data file will be imported.
iKeepNavRep:
Tells whether the existing navrep (if any) will be kept or replaced by the default one.
Sub ImportNavRep( iFolderPath)
Imports a navrep file in the XRep.
Parameters:
iFolderPath:
The path of the folder from which the navrep file will be imported.
Sub RemoveRelation( iRelationPointedObject)
Removes a relation from the XRep.
Parameters:
iRelationPointedObject:
The link object that represent the pointed object, stored into the relation.
