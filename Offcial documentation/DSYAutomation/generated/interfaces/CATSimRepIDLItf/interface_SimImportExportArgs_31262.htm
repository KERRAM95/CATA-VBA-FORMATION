Sub SetUnits( iMagnitudes, iUnits)
Sets the units of the base magnitudes (length, mass, time, ...) of the imported/exported data.
Note: by default, MKS units are assumed and only those of the provided magnitudes are overloaded.
Parameters:
iMagnitudes:
Imported/exported data base magnitudes.
iUnits:
Imported/exported data base units.
