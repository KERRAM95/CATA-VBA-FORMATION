SimSequenceResults (Object)

Interface representing Simulation Sequence Results.
Property Index
IsCurrent Retrieves the current status of the sequence results.
IsSuccessfullyComputed Retrieves the compute status of the sequence results.
ScenarioResults Retrieves the collection of all scenario results.
ScenarioToCompute Sets or retrieves the scenario to compute.
Sequence Retrieves the sequence associated to this sequence results.
