SimGenerativeSpecifications (Object)

Represents the collection of generative specification.
See also:

Methods


Func Item( iIndex) As
Returns a generative specification using its index from the genrative specifications collection.
Parameters:
iIndex
The index of the generative specification to retrieve from the collection of generative specifications.
This index is the rank of the generative specification in the collection. The index of the first generative specification in the collection is 1, and the index of the last generative specification is Count.
Returns:
The retrieved generative specification.
