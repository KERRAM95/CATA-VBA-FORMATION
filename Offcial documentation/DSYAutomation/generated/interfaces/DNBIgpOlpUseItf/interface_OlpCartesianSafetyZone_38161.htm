OlpCartesianSafetyZone (Object)

A Cartesian safety zone.

This interface can only be used by a translator within the Robotics Off-line Programming (OLP) Download or Upload command.
This object can be retrieved using
Property Index
Active Get/Set whether the zone is active.
Index Get the zone index.
IndexSet Get whether an index is set for the zone.
SafeInside Get/Set whether the safe state is when the robot is inside or outside the zone.
ShapeType Get/Set the zone shape type.
Method Index
GetShape Get the zone shape.

Properties


Property Active() As
Get/Set whether the zone is active.
Property Index() As
Get the zone index.
In some robot languages an index is used to identify the zone. If the Index property is not set (IndexIsSet returns FALSE), then getting the Index will fail.
Property IndexSet() As
Get whether an index is set for the zone.
If not set, then getting the Index property will fail.
Property SafeInside() As
Get/Set whether the safe state is when the robot is inside or outside the zone.
Property ShapeType() As
Get/Set the zone shape type.
Can be one of "Prism" or "Cylinder". When set, the existing shape (if any) is deleted and a the new shape of this type is created.

Methods


Func GetShape() As
Get the zone shape.
Shape is NULL during upload until ShapeType is set.
