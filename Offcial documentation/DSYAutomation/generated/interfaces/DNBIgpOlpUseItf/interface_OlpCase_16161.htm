OlpCase (Object)

A case block for a test case instruction.

This interface can only be used by a translator within the Robotics Off-line Programming (OLP) Download or Upload command.
Property Index
Instructions The list of instructions executed if the case is matched.
Values The expression that determines when this case is executed.

Properties


Property Instructions() As (Read Only)
The list of instructions executed if the case is matched.
Property Values() As
The expression that determines when this case is executed.
Multiple values can be specified if they are separted by comma operators (,). When set, the of the AST tree is used for the expression. When retrieved, the DELMIA expression is parsed and is returned in the format expected by . You can find more information on the OLP Expression AST format in the documentation under Automation | Robotics | Robotics Offline Programming | Offline Programming Expression Translation.
