DELOlpTimeLinearAngularBasis (Enumeration)

enum DELOlpTimeLinearAngularBasis {
  delOlpTimeBasis,
  delOlpLinearBasis,
  delOlpAngularBasis
}
Defines the mode used to specify the motion speed.
It is used for languages where both linear and angular speeds cannot both be specified for a single motion. see
delOlpTimeBasis
Speed specified as a time.
delOlpLinearBasis
Speed specified as a linear TCP speed.
delOlpAngularBasis
Speed specified as an angular TCP speed.
