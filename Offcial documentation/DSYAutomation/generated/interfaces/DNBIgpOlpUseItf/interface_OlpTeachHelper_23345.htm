OlpTeachHelper (Object)

Native Robot Language(NRL) Teach customization.

This interface can only be used by a translator within the Robotics Off-line Programming (OLP) NRL Teach command.
This interface is used to customize the commands available in NRL teach and to retrieve information about the state of NRL teach when one of those commands is executed.

CurrentAstProcedure The AST node for the current task selected in NRL teach.

CurrentProcedure The task displayed in Teach.
EditedAstNodesNew The value of the AST node(s) after they were edited by the user.
EditedAstNodesPrevious The value of the AST node(s) before they were edited by the user.

LogicalKeywords Set the list logical keywords to be highlighted in NRL teach.
NewInstruction The newly created instruction in Teach.
PositionAccessMode Set the type of position access mode Must be set during MacroGetTranslatorInfo.

SupportsNRLView Set whether the translator supports the NRL view feature of NRL teach.

Property IsTeachActive() As (Read Only)
Returns TRUE when teach is active. For example when the current download is performed for display in NRL Teach. Used in MacroDownload.
Property LogicalKeywords( iKeywords) (Write Only)
Set the list logical keywords to be highlighted in NRL teach.
Each value must be a string. Normally this is keywords like IF, WHILE, FOR, etc. which are colored in blue. These words are only colored if the is delAstKEYWORD. Used in MacroGetTranslatorInfo.
Property NewInstruction() As (Read Only)
The newly created instruction in Teach.
This will return NULL if no new instruction was created.
Property PositionAccessMode() As
Set the type of position access mode Must be set during MacroGetTranslatorInfo. Enabling Get or GetAndSet may cause slightly slower performance in NRL teach. Valid values are:
delOlpNoAccess
Translators cannot use any method or property related to robot motion positions in NRL teach. This is the default value.
delOlpGetAccess
Translators can get position related property values or call methos that return position related values but cannot call methods or properties that modify positions or related values.
delOlpGetAndSetAccess
Translator can call any methods, including ones that modify robot motion positions and related properties.
Property RefreshMode( iRefreshMode) (Write Only)
Set the type of refresh needed on the task after an an edit operation. Valid values are:
- Download : Full download of task is performed. This will ensure everything is up to date but takes the most time. This is the default value if RefreshMode is not set.
- AstInstruction : Only the AST nodes for the modified instruction have been changed. This just refreshes the UI for the current AST instruction. The AST must already have been updated by the transaltor.
- AstTask : Some AST nodes throughout the task have been changed. This refreshed the entire task display UI. The AST must have already been updated by the translator.
- None : No refresh is needed.
Property SupportsNRLView( iSupportsNRLView) (Write Only)
Set whether the translator supports the NRL view feature of NRL teach.
Used in MacroGetTranslatorInfo.

Methods

The the list of values the user can pick from when editing a combo or editablecombo with the given editor ID. All objects in the list are of type .
Sub AddCustomColumn( iEditorID, iColumnTitle, iApplicableApplications)
Declare a NRL teach editor as available as a teach table column.
Parameters:
iEditorID
The editor ID to enable as a custom column.
iColumnTitle
Title to display for the column.
iApplicableApplications
Applications to show this column for. This is a list of strings. Each value should correspond to a value in the Robot Applicactions preference. If the list is blank the column will be avaiable to be added for all applications.
Func AddCustomCommand( iGroup, iCommandName) As
Add a new command to Teach.
Parameters:
iGroup
The name of the group to place the command in.
This is the name displayed for the group and also the identifier for the group. The group is created automatically when a new ID is specified here.
iCommandName
The name of the command to add.
The command's Name and properties are both initialized to this value.
Returns:
The created command.
Sub DeclareSetAndWaitEditors( iSetIOEditor, iWaitIOEditor)
Declare which columns are used for IO Set and Wait.
These columns will then be displayed in the interference zone command. The iEditorID must already be declared as a custom column using . The title displayed for the column in the interference zone will the same as teach and will be the one set in . The editor must be available on the AST node for a robot motion to be dipslayed in the interference zone command. The IO is the one associated with the robot motion for entering/exiting the interference zone. The API must be called during MacroGetTranslatorInfo.
Parameters:
iSetIOEditor
The editor ID to use for the Set IO column in the inteference zone table.
iWaitIOEditor
The editor ID to use for the Set IO column in the inteference zone table.
