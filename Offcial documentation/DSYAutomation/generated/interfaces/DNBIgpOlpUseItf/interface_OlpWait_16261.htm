OlpWait (Object)

A wait instruction.

This interface can only be used by a translator within the Robotics Off-line Programming (OLP) Download or Upload command.
This is an instruction that waits for the condition expression to evaluate to true. If the timeout is enabled and the specified amount of time elapses before the the condition is satisfied, then execution proceeds to the next instruction. This instruction can also be used as a pure delay.
Property Index
Condition The expression that must evaluate to true before execution continues.
ConveyorTrackingProfile Get/Set the conveyor tracking profile associated with the wait instruction if this is a conveyor tracking wait.
IsConveyorWait Get/Set whether the wait instruction is waiting for a conveyor to reach a certain position.
IsPureDelay Indicates if this instruction is used as a pure delay.
IsSimpleConveyorWait Get/Set whether the wait instruction is in the simple format waiting for the conveyor to reach the inbound offset of the conveyor tracking profile.
TimeOutExpress Get/Set the timeout value in expression.
Timeout Get/Set the timeout value in seconds.
UseTimeout Get/Set whether the wait instruction uses a timeout.
Method Index
MakePureDelay Sets the condition such that this instruction is a pure delay.
