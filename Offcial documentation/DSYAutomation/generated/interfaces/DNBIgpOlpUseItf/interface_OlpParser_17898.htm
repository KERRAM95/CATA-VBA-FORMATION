OlpParser (Object)

Parser for compiling a Native Robot Language program using Flex and Bison.

This interface can only be used by a translator within the Robotics Off-line Programming (OLP) Download or Upload command.
Output of the parsed NRL program is the Abstract Syntax Tree (AST). This object can be retrieved using
Property Index
AstRoot Used to set the root of the Abstract Syntax Tree (AST).
ErrorReporter The object used to log any parse errors.
Method Index
Parse Used to parse the absolute file name sent.
ParseString Used to parse a string rather than a file.

Properties


Property AstRoot( iAST) (Write Only)
Used to set the root of the Abstract Syntax Tree (AST). All AST branches and leaves created during parse will be descendents of this root.
Parameters:
iAST
The AST root node.
Property ErrorReporter( iReporter) (Write Only)
The object used to log any parse errors.
All warnings and errors will be displayed using this reporter.
Parameters:
iReporter
The error reporter to use.

Methods


Sub Parse( iFile)
Used to parse the absolute file name sent. The parser generates the Abstract Syntax Tree (AST) starting from the root set in AstRoot. Errors and warnings are displayed using the ErrorReporter.
Parameters:
iFile
The file to parse.
Sub ParseString( iText)
Used to parse a string rather than a file.
Parameters:
iText
The text to parse.
