OlpSyncTask (Object)

An instruction that can sync with another instruction from another task.

This interface can only be used by a translator within the Robotics Off-line Programming (OLP) Download or Upload command.
Property Index
Lineup The lineup object that contains all the resources to synchronize.
SyncID The sync ID.
WaitTime Amount of time to wait for synchronization before continuing.
WaitedOn The number of resources to wait for to synchronize.

Properties


Property Lineup() As
The lineup object that contains all the resources to synchronize.
Property SyncID() As
The sync ID. The same ID is set on all instructions to be synchronized.
Property WaitTime() As
Amount of time to wait for synchronization before continuing. 0 means no wait, NULL value or blank expression means wait forever. The value can be an expression that is evaluated when the instruction is executed.
Property WaitedOn() As
The number of resources to wait for to synchronize. It must be greater than 2 and less than or equal to the number of resources in the lineup.
