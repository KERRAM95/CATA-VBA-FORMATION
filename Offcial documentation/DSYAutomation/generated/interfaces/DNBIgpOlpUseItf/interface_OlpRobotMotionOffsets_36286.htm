OlpRobotMotionOffsets (Object)

Object to manage the choreography events for a robot motion.

see This interface can only be used by a translator within the Robotics Off-line Programming (OLP) Download or Upload command.
Method Index
AddOffset Create an offset.
DeleteOffset Delete an offset.
Item Get a choreography event.
RemoveAllOffsets Remove all offsets.

Methods


Func AddOffset( iIndex) As
Create an offset.
The new offset is returned and initialized as a delOlpCartesianToolOffset with a 0 offset value.
Parameters:
iIndex
The location in the list to insert the new array. Specify -1 to append.
Returns:
oOffset The offset.
Sub DeleteOffset( iOffset)
Delete an offset.
Func Item( iIndex) As
Get a choreography event.
You can also use the For ... Each syntax to iterate through the collection. Index starts at 1.
Parameters:
iIndex
The index of the offset to retreive.
oOffset
The offset.
Sub RemoveAllOffsets()
Remove all offsets.
After this call the motion will be a standard move without any offsets.
