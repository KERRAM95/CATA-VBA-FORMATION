Classifies the instruction type.

see
delOlpCustom
Custom instruction is a generic instruction that can contain any custom NRL text.
delOlpRobotMotion
A robot motion.
delOlpSpotOperation
A spot operation: This is a specialized type of robot motion.
delOlpArcOperation
An arc operation: This is a specialized type of robot motion.
delOlpSeamSearchOperation
A seam search operation: This is a specialized type of robot motion.
delOlpGrab
A grab activity.
delOlpRelease
A release activity.
delOlpAssign
An instruction that sets or assigns a value to a variable or IO.
delOlpWait
An instruction that waits for an expression to be true.
delOlpRun
An instruction that calls a procedure.
delOlpIf
A conditional if-then-else instruction.
delOlpLoop
A while or do...until loop instruction.
delOlpFor
A counting loop instruction.
delOlpGoto
An instruction that jumps to another instruction.
delOlpBreak
An instruction that breaks a loop.
delOlpReturn
An instruction that returns from a procedure.
delOlpPointOperation-deprecated
Please use delOlpRivetOperation, delOlpBoltOperation, delOlpClinchOperation, or delOlpPointAdhesiveOperation instead.
delOlpRivetOperation
This is a specialized type of robot motion used for riveting. The target is a tag in a point trajectory of type rivet.
delOlpBoltOperation
This is a specialized type of robot motion used for fastening bolts. The target is a tag in a point trajectory of type bolt.
delOlpClinchOperation
This is a specialized type of robot motion used for clinching. The target is a tag in a point trajectory of type clinch.
delOlpPointAdhesiveOperation
This is a specialized type of robot motion used for applying a point of adhesive. The target is a tag in a point trajectory of type adhesive.
delOlpSealantOperation
A path operation: This is a specialized type of robot motion for sealant processes that happen along a path. The target is a tag in a curve trajectory of type Sealant.
delOlpPathAdhesiveOperation
A path operation: This is a specialized type of robot motion for adhesive processes that happen along a path. The target is a tag in a curve trajectory of type Adhesive.
delOlpPulse
An assignment instruction that pulses the output and/or sets the output with a delay.
delOlpTestCase
A select or case instruction that tests a variable and matches it against one of several values.
delOlpTimer
An instruction for starting or stopping a stopwatch timer that measures the simulation time.
delOlpPaintOperation
A surface operation: This is a specialized type of robot motion for paint processes that happen over a surface. The target is a tag in a surface trajectory of type Paint.
delOlpSealantSurfaceOperation
A surface operation: This is a specialized type of robot motion for sealing processes that happen over a surface. The target is a tag in a surface trajectory of type Sealant.
delOlpGeneralSurfaceOperation
A surface operation: This is a specialized type of robot motion for general processes that happen over a surface. The target is a tag in a surface trajectory of type General.
delOlpTrigger
An assignment instruction that pulses the output and/or sets the output with a delay.
delOlpTemplate
A template instruction that contains other instructions.
delOlpDrillOperation
This is a specialized type of robot motion used for drilling. The target is a manufacturing fastener in a drill rivet manufactruing pattern.
delOlpPRun
An instruction that calls another procedure to be run in parallel.
delOlpWaypointOperation
An instruction that contains 1 or more waypont motions. A waypoint is a via point to move between 2 process points.
delOlpWaypointMotion
A waypoint motion instruction. A waypoint is a via point to move between 2 process points. Waypoint motions can only be inserted under a waypoint operation or in a dedicated waypoint task that has only waypoint motions in it.
delOlpRunByString
A run by string instruction. This is a run instruction that calls another procedure by specifying the name as a string which can be computed during the simulation.
delOlpAssignByString
A assign variable by string instruction. This is an assign instruction that assigns a variable by specifying the name of the variable as a string.
