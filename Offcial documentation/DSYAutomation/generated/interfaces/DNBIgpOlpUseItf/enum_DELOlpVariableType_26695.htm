DELOlpVariableType (Enumeration)

enum DELOlpVariableType {
  delOlpExternalIO,
  delOlpProcedureIO,
  delOlpLocalVariable,
  delOlpConstant,
  delOlpAllVariables,
  delOlpAlias
}
Defines the types of target locations in DELMIA.
see
delOlpExternalIO
An external IO variable (input, output).
delOlpProcedureIO
An argument to a procedure.
delOlpLocalVariable
A normal variable.
delOlpConstant
A constant.
delOlpAllVariables
Used for retrieving variables of all types.
