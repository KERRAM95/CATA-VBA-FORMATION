OlpTimer (Object)

An instruction to start, stop or reset a timer.

This interface can only be used by a translator within the Robotics Off-line Programming (OLP) Download or Upload command.
Property Index
Action The action to perform on the timer.
Destination The timer which is being started, stopped or reset.
DestinationExpress The variable which is being assigned as an expression.
