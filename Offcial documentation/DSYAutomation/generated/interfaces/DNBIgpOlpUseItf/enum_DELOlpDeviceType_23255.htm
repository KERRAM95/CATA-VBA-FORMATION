DELOlpDeviceType (Enumeration)

enum DELOlpDeviceType {
  delOlpRobotDevice,
  delOlpRailDevice,
  delOlpToolDevice,
  delOlpWorkpiecePositionerDevice,
  delOlpAllDevices,
  delOlpConveyorDevice
}
Defines the types devices.
see
delOlpRobotDevice
A robot.
delOlpRailDevice
A rail or track that the robot base is attached to.
delOlpToolDevice
A tooling devices mounted on the end of the robot.
delOlpWorkpiecePositionerDevice
A device that moves the workpiece.
delOlpConveyorDevice
A conveyor.
delOlpAllDevices
Any device. Used with to retrieve all devices in the group.
