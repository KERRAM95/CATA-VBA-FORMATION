DELOlpOrientationMode (Enumeration)

enum DELOlpOrientationMode {
  delOlpOrient1Axis,
  delOlpOrient2Axis,
  delOlpOrient3Axis,
  delOlpOrientWrist
}
The orientation mode used by the motion planner to create the robot trajectory for a robot motion activity.

see
delOlpOrient1Axis
1 axis of rotation for the entire motion.
delOlpOrient2Axis
2 axes
delOlpOrient3Axis
3 axes
delOlpOrientWrist
The tcp orientation is not considered. Joints 4,5,6 are moved using joint interpolation.
