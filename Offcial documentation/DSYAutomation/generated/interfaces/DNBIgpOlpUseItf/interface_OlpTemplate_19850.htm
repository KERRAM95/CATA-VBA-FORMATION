OlpTemplate (Object)

A template instruction.

This interface can only be used by a translator within the Robotics Off-line Programming (OLP) Download or Upload command.
Property Index
Instructions Returns the list of instructions in the template.
LibraryName Returns library name of the template.
SubType Returns subtype of the template.
TemplateName Returns name of the template.
TemplateType Returns type of the template.
WaypointID Get/Set the waypoint ID.
