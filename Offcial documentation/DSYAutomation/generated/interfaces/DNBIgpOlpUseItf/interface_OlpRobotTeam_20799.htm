Func Item( iIndex) As
Retrieve a device by its name or index.
Parameters:
iIndex
If iIndex is a string, then a device with that name is returned. If the index is a number, it is the index of the device in the collection. The index of the first parameter in the collection is 1, and the index of the last parameter is Count.
Returns:
The device retrieved. If the device name was not found, Nothing is returned but the function succeeds. If the index is out of bounds, the function fails.
