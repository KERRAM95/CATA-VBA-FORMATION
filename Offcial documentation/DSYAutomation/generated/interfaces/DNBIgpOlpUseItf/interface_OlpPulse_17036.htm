OlpPulse (Object)

An instruction used to asynchronously change the value of an external output.

This interface can only be used by a translator within the Robotics Off-line Programming (OLP) Download or Upload command.
Property Index

Property SourceType() As
Get/Set the type of Pulse value.
This is a simple way to access the most common types of pulsed values. If this property is set, the expression is automatically generated.
Property UseDuration() As
Get/Set whether the duration is used.
If False, then the duration is effectively infinity and the property should not be retrieved. If True, then the Duration value is used.
