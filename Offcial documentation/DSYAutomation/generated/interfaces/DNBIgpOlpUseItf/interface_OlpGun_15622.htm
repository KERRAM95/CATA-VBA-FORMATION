OlpGun (Object)

Interface for accessing a gun's properties.

This interface can only be used by a translator within the Robotics Off-line Programming (OLP) Download or Upload command.
This object can be retrieved from a .

Property Index() As (Read Only)
Get/Set the gun index.
If no gun index was assigned by the user, a unique index will automatically be generated by the infrastructure.
Property JointIndex() As (Read Only)
The joint in the Controller that is moved for this gun.
Property LowerTipAbrasion() As
The abrasion (maximum tip wear) value for clash analysis of the lower tip of a spot gun.
The lower tip is usually the stationary tip from the spot profile but depends on how it was configured by the user in the Tool Part Grouping command. If no value is set on the gun in the tool part grouping command the value from the preferences is returned during get. If no tool grouping has been done, the set will not work and a warning will be displayed for the user in the messages. The method fails if it the is not "SpotWeld". The values are not set during upload if the user has unchecked the option to update device parameters from the controller backup.

Property SubType() As
Get the gun technology.
This is the gun technology used for the application. Each application uses its own sub types. For Paint the options are : "Air" and "Electrostatic". For SpotWeld the options are : "Servo" and "Air".
Property Type() As (Read Only)
Get the gun application.
Currently only "Paint" and "SpotWeld" are supported, but more may be added in the future (e.g. "Rivet", etc).
