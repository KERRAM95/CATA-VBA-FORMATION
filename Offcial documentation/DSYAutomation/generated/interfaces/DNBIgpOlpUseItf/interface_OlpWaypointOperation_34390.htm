OlpWaypointOperation (Object)

An instruction that contains the path of waypoint motions.

This interface can only be used by a translator within the Robotics Off-line Programming (OLP) Download or Upload command.
Property Index
Instructions The list of instructions in the waypoint operation.
WaypointTask The waypoint task used for this waypoint operation.

Properties


Property Instructions() As (Read Only)
The list of instructions in the waypoint operation. All instructions must be waypoint motions.
Property WaypointTask() As
The waypoint task used for this waypoint operation. The same waypoint task must be used for all waypoint operations. This must be set during upload. It may not be available during download in the case of a template task being used.
