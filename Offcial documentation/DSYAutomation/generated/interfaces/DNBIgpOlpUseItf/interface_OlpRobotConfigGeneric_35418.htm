OlpRobotConfigGeneric (Object)

A robot configuration.

This interface can only be used by a translator within the Robotics Off-line Programming (OLP) Download or Upload command.
A configuration is the robot posture and the turn numbers or turn signs used to define a unique inverse kinematics solution for a Cartesian position.
