DELOlpProcessType (Enumeration)

enum DELOlpProcessType {
  delOlpUndefinedProcess,
  delOlpApproach,
  delOlpDepart,
  delOlpViaPoint,
  delOlpStartWeld,
  delOlpWeld,
  delOlpEndWeld,
  delOlpOscillationPoint,
  delOlpTouchPoint,
  delOlpStartProcess,
  delOlpMidProcess,
  delOlpEndProcess
}
Defines the purpose of a target location in a process.
see
delOlpUndefinedProcess
Use is undefined or unknown.
delOlpApproach
An approach to the position where work is done.
delOlpDepart
A departure from the position where work was done.
delOlpViaPoint
A position used to move between positions where work is done.
delOlpStartWeld
The first welding point in an arc weld sequence. This is where the arc is turned on.
delOlpWeld
A position where welding is done.
delOlpEndWeld
The last welding point in an arc weld sequence. This is where the arc is turned off.
delOlpOscillationPoint
A point that defines the oscillation pattern for arc welding.
delOlpTouchPoint
One of the touch points in a seam search.
delOlpStartProcess
The first point in a process that takes place along a path. This is where the process equiptment is turned on.
delOlpMidProcess
A process position in between the start and then end.
delOlpEndProcess
The last point in a process that takes place along a path. This is where the process equiptment is turned off.
