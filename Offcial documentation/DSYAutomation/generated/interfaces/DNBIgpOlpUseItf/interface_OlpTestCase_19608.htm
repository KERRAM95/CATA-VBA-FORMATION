OlpTestCase (Object)

A test case instruction (TEST, SELECT, SWITCH, CASE).

This interface can only be used by a translator within the Robotics Off-line Programming (OLP) Download or Upload command.
Property Index
Cases The list of cases.
DefaultCaseInstructions The default case if no other case is matched.
Expression The expression that determines which branch is executed.
HasDefaultCase Get/Set whether the test case instruction has a default case.

Properties


Property Cases() As (Read Only)
The list of cases.
Property DefaultCaseInstructions() As (Read Only)
The default case if no other case is matched.
This method will return Nothing if there is no default case.
Property Expression() As
The expression that determines which branch is executed.
When set, the of the AST tree is used for the expression. When retrieved, the DELMIA expression is parsed and is returned in the format expected by . You can find more information on the OLP Expression AST format in the documentation under Automation | Robotics | Robotics Offline Programming | Offline Programming Expression Translation.
Property HasDefaultCase() As
Get/Set whether the test case instruction has a default case.
The default case is executed if no other case matches.
