OlpRunByString (Object)

An instruction that calls another procedure by name.

This interface can only be used by a translator within the Robotics Off-line Programming (OLP) Download or Upload command.
Property Index
Arguments The arguments passed to the procedure.
ProcedureExpression The expression that evaluates to a string for the name of the procedure that is run from this instruction.
RunnableProcedurePrefix The prefix to identify runnable procedures.
RunnableProcedureSuffix The suffix to identify runnable procedures.
RunnableProcedures The list of runnable procedures.

Properties


Property Arguments() As
The arguments passed to the procedure.
The order of the arguments corresponds to the order of the procedure inputs retrieved from . The AST format is an "Arguments" node which contains a list of "Expression" and "Operator" nodes. Each "Expression" node has the id "Argument" and is in the proper format to be used with . The operator nodes are just commas.
When set, each child node with id="Argument" will be taken as an argument, in order. You can find more information on the OLP Expression AST format in the documentation under Automation | Robotics | Robotics Offline Programming | Offline Programming Expression Translation. The AST Arguments format used in function expressions as described in that document is the same as the AST Arguments format used here.
Property ProcedureExpression() As
The expression that evaluates to a string for the name of the procedure that is run from this instruction.
Property RunnableProcedurePrefix() As
The prefix to identify runnable procedures. During upload, the translator can set this to filter the list of runnable procedures. If not set we will try to identify the prefix from the procedure expression.
Property RunnableProcedureSuffix() As
The suffix to identify runnable procedures. During upload, the translator can set this to filter the list of runnable procedures. If not set we will try to identify the suffix from the procedure expression.
Property RunnableProcedures() As
The list of runnable procedures. The procedure to call must be one of the ones in this list. Each item in the list is a . During download, all tasks in this list will have been added to the list of tasks to download if the user reqeusted a download of all called tasks. During upload, setting this list is optional. If not set, we will identify all tasks that are valid to be called (don't result in recursive called) and then filter the list based on a constant prefix or suffix identified in the procedure expression.
