DELOlpPulseType (Enumeration)

enum DELOlpPulseType {
  delOlpPulseOn,
  delOlpPulseOff,
  delOlpPulseInvert,
  delOlpPulseLiteralInteger,
  delOlpPulseExpression
}
Defines the types of pulses.
see
delOlpPulseOn
Pulse output on.
delOlpPulseOff
Pulse output off.
delOlpPulseInvert
Invert the signal for pulse.
delOlpPulseLiteralInteger
Pulse an integer value specified as a literal number, not as an expression or variable.
delOlpPulseExpression
Pulse value is an expression or variable. (not yet supported)
