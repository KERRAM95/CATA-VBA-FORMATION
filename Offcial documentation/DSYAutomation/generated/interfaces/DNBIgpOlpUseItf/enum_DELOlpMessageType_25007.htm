DELOlpMessageType (Enumeration)

enum DELOlpMessageType {
  delOlpError,
  delOlpWarning,
  delOlpNotice
}
List of the possible OLP translation message types.
delOlpError
An error that caused the translation to completely fail. If any entities were uploaded or text generated on download is it invalid and should be deleted.
delOlpWarning
A problem happened during translation which could be recovered from. There is a good chance that the downloaded program or uploaded task is not correct but the incorrect information is not critical.
delOlpNotice
Something happened in translation which the user may want to correct or should be notified about. However the result of the translation does not have any errors. For example the name of the task was changed on download because the name contained invalid characters.
