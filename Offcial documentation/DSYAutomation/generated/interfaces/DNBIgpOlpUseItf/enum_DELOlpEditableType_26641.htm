DELOlpEditableType (Enumeration)

enum DELOlpEditableType {
  delOlpNoEditor,
  delOlpStringEditor,
  delOlpIntEditor,
  delOlpDoubleEditor,
  delOlpComboEditor,
  delOlpEditableComboEditor
}
Possible type of editors in NRL teach.
see
delOlpNoEditor
No editor - not an editable value.
delOlpStringEditor
Editable as a string. User can type any value.
delOlpIntegerEditor
Editable as an integer. User can type any positive or negative whole number.
delOlpDoubleEditor
Editable as a double. User can type any number.
delOlpComboEditor
Editable as a drop down menu. User can pick a value from the list. For this editor type you must also call with the editor ID to specify the list of valid values the user can pick from.
delOlpEditableComboEditor
Editable as a text field with a drop down menu. Same as delOlpCombo but additionally the user can type a new value that is not in the list.
