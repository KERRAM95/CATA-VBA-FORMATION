DELOlpTeachCommand (Enumeration)

enum DELOlpTeachCommand {
  delOlpCmdNone,
  delOlpCmdModify,
  delOlpCmdDelete,
  delOlpCmdRobotMotionTag,
  delOlpCmdRobotMotionCartesian,
  delOlpCmdRobotMotionJoint,
  delOlpCmdSpotOperation,
  delOlpCmdArcOperation
}
Possible built-in teach commands.

see and
delOlpCmdNone
No command.
delOlpCmdModify
Modify the position of a motion.
This command must be used by itself, RunMacroCommand cannot also be called.
delOlpCmdDelete
Delete the current instruction.
delOlpCmdRobotMotionTag
Insert a new tag target robot motion.
delOlpCmdRobotMotionCartesian
Insert a new Cartesian target robot motion.
delOlpCmdRobotMotionJoint
Insert a new joint target robot motion. This will create a home target if the robot is at a home position.
delOlpCmdSpotOperation
Insert a spot operation.
delOlpCmdArcOperation
Insert an arc operation.
