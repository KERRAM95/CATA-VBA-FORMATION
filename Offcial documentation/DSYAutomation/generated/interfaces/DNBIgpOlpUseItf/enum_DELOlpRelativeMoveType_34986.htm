DELOlpRelativeMoveType (Enumeration)

enum DELOlpRelativeMoveType {
  delOlpNoRelative,
  delOlpJointRelative,
  delOlpCartesianToolRelative
}
Defines the types of relatives moves in DELMIA.
see
delOlpNoRelative
No relative motion is used.
delOlpJointRelative
An relative motion that shifts the joint values of the robot.
delOlpCartesianToolRelative
An offset that shifts the target in Cartesian space using the TCP orientation at the target as the frame of reference.
