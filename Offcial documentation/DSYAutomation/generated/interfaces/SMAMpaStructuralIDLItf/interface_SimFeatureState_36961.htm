SimFeatureState (Object)

Represents the Feature State object.
Property Index
Amplitude Returns or sets the amplitude reference.
AmplitudeFlag Returns or sets the flag that determines if the feature has an amplitude.
BeginningIncrement Returns or sets the Beginning Increment.
BeginningStep Returns or sets the Beginning Step.
EndingIncrement Returns or sets the Ending Increment.
EndingStep Returns or sets the Ending Step.
Feature Returns the feature referenced by the feature state.
FieldFlag Returns if the feature state has a field.
PhaseAngle Returns or sets the phase angle value.
PhaseAngleFlag Returns or sets the flag that determines if the feature has a phase angle.
PropagationState Returns the propagation state of the feature referenced by the feature state.
ScaleFactor Returns or sets the scale factor value.
ScaleFactorFlag Returns or sets the flag that determines if the feature has a scale factor.

Properties


Property Amplitude() As
Returns or sets the amplitude reference. It will adhere to SMAIMpaTabularAmplitude interface if the amplitude reference points to a tabular amplitude.
Property AmplitudeFlag() As (Read Only)
Returns or sets the flag that determines if the feature has an amplitude.
TRUE: the feature has an amplitude.
FALSE: the feature has no amplitude.
Property BeginningIncrement() As
Returns or sets the Beginning Increment. Use 1 for the first increment. Use 999999 for the last increment.
Property BeginningStep() As
Returns or sets the Beginning Step.
Property EndingIncrement() As
Returns or sets the Ending Increment. Use 1 for the first increment. Use 999999 for the last increment.
Property EndingStep() As
Returns or sets the Ending Step.
Property Feature() As (Read Only)
Returns the feature referenced by the feature state.
Property FieldFlag() As (Read Only)
Returns if the feature state has a field. This example demonstrates how to use SimFeatureState from a field. 
 Dim MyFeatureState As SimFeatureState
 Dim MySteadyStateHeatTransferStep As SimSteadyStateHeatTransferStep
 ...
 If MyFeatureState.FieldFlag = True Then
     MyFeatureState.BeginningStep = MySteadyStateHeatTransferStep
     MyFeatureState.BeginningIncrement = 1
     MyFeatureState.EndingStep = MySteadyStateHeatTransferStep
     MyFeatureState.EndingIncrement = 999999
 End If
Property PhaseAngle() As
Returns or sets the phase angle value.
Property PhaseAngleFlag() As (Read Only)
Returns or sets the flag that determines if the feature has a phase angle.
TRUE: the feature has a phase angle.
FALSE: the feature has no phase angle.
Property PropagationState() As (Read Only)
Returns the propagation state of the feature referenced by the feature state. Possible values:
Created
Created in this step.
Propagated
Propagated from a preceding General step in this General step.
Modified
Propagated and modified in this General step.
Deactivated
Propagated and deactivated in this General step.
Inactive
Propagated and inactive in this step.
Reactivated
Deactivated in preceding General step is activated in this General step.
PropagatedfromBaseState
Propagated from a preceding General step into this Perturbation step.
ModifiedfromBaseState
Propagated and modified from a preceding General step in this Perturbation step.
DeactivatedfromBaseState
Propagated and deactivated from a preceding General step in this Perturbation step.
BuiltintoBaseState
Inherited state from the preceding General step in this Perturbation step. Applies only to boundary conditions.
BuiltintoMode
Inherited modes of the preceding Frequency step in this Linear Dynamic step. Applies only to boundary conditions.
NotApplicable
Does not apply in this step.
Property ScaleFactor() As
Returns or sets the scale factor value.
Property ScaleFactorFlag() As (Read Only)
Returns or sets the flag that determines if the feature has a scale factor.
TRUE: the feature has a scale factor.
FALSE: the feature has no scale factor.
