SimAppliedAcceleration (Object)

Represents the Applied Acceleration object.
Example:
Given a SimFeatures object, you can create a SimAppliedAcceleration as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyAppliedAcceleration As SimAppliedAcceleration
 Set MyAppliedAcceleration = MyFeatures.Add("SimAppliedAcceleration")
Given a SimFeatures object, you can retrieve a SimAppliedAcceleration named "Applied Acceleration.1" as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyAppliedAcceleration As SimAppliedAcceleration
 Set MyAppliedAcceleration = MyFeatures.Item("Applied Acceleration.1")
Example in Python:
Given a SimFeatures object myFeatures, you can create a SimAppliedAcceleration as following: 
 ...
 myAppliedAcceleration = myFeatures.Add("SimAppliedAcceleration")
Given a SimFeatures object myFeatures, you can retrieve a SimAppliedAcceleration named "Applied Acceleration.1" as following: 
 ...
 myAppliedAcceleration = myFeatures.Item("Applied Acceleration.1")

Property XAcceleration() As
Returns or sets the X acceleration magnitude. Quantity: ACCELRTN, units: m_s2.
Setting the magnitude will also set the corresponding flag to TRUE. Retrieving the magnitude without setting it first will result in an error.
Property XAccelerationSet() As
Returns or sets the flag that determines if a acceleration in the X direction is specified.
TRUE: acceleration is specified.
FALSE: acceleration is not specified.
Set the flag to FALSE to unset the acceleration. Setting to TRUE is not supported and will result in an error. The flag will automatically be set to TRUE when you set a acceleration.
