SimSolutionControlsDefinition (Enumeration)

enum SimSolutionControlsDefinition {
  SimSolutionControlsPropagate,
  SimSolutionControlsReset,
  SimSolutionControlsSpecify
}
The definition of the feature.
Values:
SimSolutionControlsPropagate
Use the same solution controls for the current step as those in the preceding step.
SimSolutionControlsReset
Reset all solution controls for the current step to the solver defaults.
SimSolutionControlsSpecify
Enable modification of the solution controls.
