SimSurfaceTraction (Object)

Represents the Surface Traction object.
Example:
Given a SimFeatures object, you can create a SimSurfaceTraction as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MySurfaceTraction As SimSurfaceTraction
 Set MySurfaceTraction = MyFeatures.Add("SimSurfaceTraction")
Given a SimFeatures object, you can retrieve a SimSurfaceTraction named "Surface Traction.1" as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MySurfaceTraction As SimSurfaceTraction
 Set MySurfaceTraction = MyFeatures.Item("Surface Traction.1")
Example in Python:
Given a SimFeatures object myFeatures, you can create a SimSurfaceTraction as following: 
 ...
 mySurfaceTraction = myFeatures.Add("SimSurfaceTraction")
Given a SimFeatures object myFeatures, you can retrieve a SimSurfaceTraction named "Surface Traction.1" as following: 
 ...
 mySurfaceTraction = myFeatures.Item("Surface Traction.1")

Properties


Property Activated() As
Returns or sets the activation status.
Property AxisSystem() As (Read Only)
Returns the axis system used for the surface traction.
Property ConstantResultantFlag() As
Returns or sets the flag that determines if the surface traction is integrated over the surface in the reference configuration.
Only applicable if the uniform traction flag is TRUE.
TRUE: the surface traction is integrated over the surface in the reference configuration.
FALSE: the surface traction is integrated over the surface in the current configuration.
Property FeatureHistory() As (Read Only)
Returns the feature history.
Property FollowerLoadFlag() As
Returns or sets the flag that determines if the surface traction should follow the deformation of the support.
Only applicable if the uniform traction flag is TRUE.
TRUE: the surface traction remains normal to the support as it deforms.
FALSE: the surface traction remains in its original orientation as the support deforms.
Property SpecTreeCategory() As (Read Only)
Returns a string representing the specification tree category of the feature. See for usage.
Property UniformTractionFlag() As (Read Only)
Returns the flag that determines if the surface traction is uniform.
TRUE: the surface traction is uniform.
FALSE: the surface traction is not uniform.
Property VectorField() As (Read Only)
Returns the vector field.
