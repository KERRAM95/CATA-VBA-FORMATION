SimConnectorDamping (Object)

Represents the Connector Damping object.
Property Index
Damping Returns or sets the damping value.
DampingOrder Returns or sets the damping order.
Direction Returns the DOF on which the elasticity connector is defined.
Table Returns the table that determines the damping behavior.
Method Index
GetTableColumn Retrieves a specific column of the table that determines the elasticity behavior.

Properties


Property Damping() As
Returns or sets the damping value. Quantity: FORCE_OVER_VELOCITY, units: kg_s
Property DampingOrder() As
Returns or sets the damping order.
Property Direction() As (Read Only)
Returns the DOF on which the elasticity connector is defined.
Property Table() As (Read Only)
Returns the table that determines the damping behavior.

Methods


Func GetTableColumn( iTableColumnName) As
Retrieves a specific column of the table that determines the elasticity behavior. This method is only applicable if the Damping Order is NonLinear.
Parameters:
iTableColumnName[in]
The table column name.
Returns:
The table column.
