SimDirectHarmonicResponseStepTableColumn (Enumeration)

enum SimDirectHarmonicResponseStepTableColumn {
  SimDirectHarmonicResponseStepLower,
  SimDirectHarmonicResponseStepUpper,
  SimDirectHarmonicResponseStepIncrement,
  SimDirectHarmonicResponseStepNumberOfPoints,
  SimDirectHarmonicResponseStepBias,
  SimDirectHarmonicResponseStepScaleFactor,
  SimDirectHarmonicResponseStepSpread
}
The columns of the table that determines the direct harmonic response step behavior.

The columns in the connector direct harmonic response step table will be:
For FrequencyIncrement interval type:
  1. Lower. Quantity: FREQUENCY, Units: Hz
  2. Upper. Quantity: FREQUENCY, Units: Hz
  3. Increment. Quantity: FREQUENCY, Units: Hz

For EigenFrequency interval type:
  1. Lower. Quantity: FREQUENCY, Units: Hz
  2. Upper. Quantity: FREQUENCY, Units: Hz
  3. NumberOfPoints. Quantity: DIMENSIONLESS, Units: None
  4. Bias. Quantity: DIMENSIONLESS, Units: None
  5. ScaleFactor. Quantity: DIMENSIONLESS, Units: None

For DirectRange interval type:
  1. Lower. Quantity: FREQUENCY, Units: Hz
  2. Upper. Quantity: FREQUENCY, Units: Hz
  3. NumberOfPoints. Quantity: DIMENSIONLESS, Units: None
  4. Bias. Quantity: DIMENSIONLESS, Units: None

For FrequencySpread interval type:
  1. Lower. Quantity: FREQUENCY, Units: Hz
  2. Upper. Quantity: FREQUENCY, Units: Hz
  3. NumberOfPoints. Quantity: DIMENSIONLESS, Units: None
  4. ScaleFactor. Quantity: DIMENSIONLESS, Units: None
  5. Spread. Quantity: DIMENSIONLESS, Units: None
