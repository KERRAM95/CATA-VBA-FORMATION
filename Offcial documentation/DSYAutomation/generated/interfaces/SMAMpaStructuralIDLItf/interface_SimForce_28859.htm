SimForce (Object)

Represents the Force object.
Deprecated:
R428. Use SimConcentratedLoad or SimDistributedLoad as a replacement.
Example:
Given a SimFeatures object, you can create a SimForce as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyForce As SimForce
 Set MyForce = MyFeatures.Add("SimForce")
Given a SimFeatures object, you can retrieve a SimForce named "Force.1" as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyForce As SimForce
 Set MyForce = MyFeatures.Item("Force.1")
Example in Python:
Given a SimFeatures object myFeatures, you can create a SimForce as following: 
 ...
 myForce = myFeatures.Add("SimForce")
Given a SimFeatures object myFeatures, you can retrieve a SimForce named "Force.1" as following: 
 ...
 myForce = myFeatures.Item("Force.1")

Property FollowerFlag() As
Returns or sets the flag that determines if the force remains normal to the support as it deforms.

TRUE: the force remains normal to the support as it deforms.

FALSE: the force remains in its original orientation as the support deforms.
Property SpecTreeCategory() As (Read Only)
Returns a string representing the specification tree category of the feature. See for usage.
