SimAcousticAbsorber (Object)

Represents the Acoustic Absorber object.
Example:
Given a SimAbstractions object, you can create a SimAcousticAbsorber as following: 
 Dim MyAbstractions As SimAbstractions
 ...
 Dim MyAcousticAbsorber As SimAcousticAbsorber
 Set MyAcousticAbsorber = MyAbstractions.Add("SimAcousticAbsorber")
Given a SimAbstractions object, you can retrieve a SimAcousticAbsorber named "Acoustic Absorber.1" as following: 
 Dim MyAbstractions As SimAbstractions
 ...
 Dim MyAcousticAbsorber As SimAcousticAbsorber
 Set MyAcousticAbsorber = MyAbstractions.Item("Acoustic Absorber.1")
Example in Python:
Given a SimAbstractions object myAbstractions, you can create a SimAcousticAbsorber as following: 
 ...
 myAcousticAbsorber = myAbstractions.Add("SimAcousticAbsorber")
Given a SimAbstractions object myAbstractions, you can retrieve a SimAcousticAbsorber named "Acoustic Absorber.1" as following: 
 ...
 myAcousticAbsorber = myAbstractions.Item("Acoustic Absorber.1")
