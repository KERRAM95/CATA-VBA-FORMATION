SimStabilization (Object)

Represents the Stabilization object.
Property Index
AdaptiveFlag Returns or sets the flag that determines if the adaptive automatic stabilization is applied.
DampingFactor Returns or sets the stabilization damping factor.
EnergyFraction Returns or sets the stabilization energy fraction.
EnergyRatioTolerance Returns or sets the stabilization energy ratio tolerance.
StabilizationType Returns or sets the stabilization type.

Properties


Property AdaptiveFlag() As
Returns or sets the flag that determines if the adaptive automatic stabilization is applied.

TRUE: the adaptive automatic stabilization is applied.

FALSE: the adaptive automatic stabilization is not applied.
Property DampingFactor() As
Returns or sets the stabilization damping factor. Quantity: Real, units: None.
Property EnergyFraction() As
Returns or sets the stabilization energy fraction. Quantity: Real, units: None.
Property EnergyRatioTolerance() As
Returns or sets the stabilization energy ratio tolerance. Quantity: Real, units: None.
Property StabilizationType() As
Returns or sets the stabilization type.
