SimConnectorRotationVelocity (Object)

Represents the Connector Rotation Velocity object.
Deprecated:
R428. Use SimConnectorVelocity as a replacement.
Example:
Given a SimFeatures object, you can create a SimConnectorRotationVelocity as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyConnectorRotationVelocity As SimConnectorRotationVelocity
 Set MyConnectorRotationVelocity = MyFeatures.Add("SimConnectorRotationVelocity")
Given a SimFeatures object, you can retrieve a SimConnectorRotationVelocity named "Connector Rotation Velocity.1" as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyConnectorRotationVelocity As SimConnectorRotationVelocity
 Set MyConnectorRotationVelocity = MyFeatures.Item("Connector Rotation Velocity.1")
Example in Python:
Given a SimFeatures object myFeatures, you can create a SimConnectorRotationVelocity as following: 
 ...
 myConnectorRotationVelocity = myFeatures.Add("SimConnectorRotationVelocity")
Given a SimFeatures object myFeatures, you can retrieve a SimConnectorRotationVelocity named "Connector Rotation Velocity.1" as following: 
 ...
 myConnectorRotationVelocity = myFeatures.Item("Connector Rotation Velocity.1")

Properties


Property Activated() As
Returns or sets the activation status.
Property AvailableConnectorComponents() As (Read Only)
Returns the list of available connector components of relative motion.
The list can contains the following values:
1 : the first rotation DOF is available.
2 : the second rotation DOF is available.
3 : the third rotation DOF is available.

Func GetRotationVelocity( iDegreeOfFreedom) As
Retrieves the velocity value for a degree of freedom.
Parameters:

The degree of freedom.
Valid input values are : SimRotation1, SimRotation2 and SimRotation3
oRotationVelocity[out]
The velocity value. Quantity: ANGULAR_VELOCITY, Units: turns_s.
Func GetRotationVelocityActivationFlag( iDegreeOfFreedom) As
Retrieves the flag which determines if a rotation velocity is active on a degree of freedom.
Parameters:
iDegreeOfFreedom[in]
The degree of freedom.
Valid input values are : SimRotation1, SimRotation2 and SimRotation3
oRotationVelocityActivationFlag[out]
The activation flag.
TRUE : the rotation velocity is active for the specified DOF.
FALSE : the rotation velocity is not active for the specified DOF.
