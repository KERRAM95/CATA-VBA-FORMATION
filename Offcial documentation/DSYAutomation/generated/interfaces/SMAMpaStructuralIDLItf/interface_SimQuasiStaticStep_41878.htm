SimQuasiStaticStep (Object)

Represents the Quasi Static Step object.
Example:
Given a SimSteps object, you can create a SimQuasiStaticStep as following: 
 Dim MySteps As SimSteps
 ...
 Dim MyQuasiStaticStep As SimQuasiStaticStep
 Set MyQuasiStaticStep = MySteps.Add("SimQuasiStaticStep")
Given a SimSteps object, you can retrieve a SimQuasiStaticStep named "Quasi-Static Step.1" as following: 
 Dim MySteps As SimSteps
 ...
 Dim MyQuasiStaticStep As SimQuasiStaticStep
 Set MyQuasiStaticStep = MySteps.Item("Quasi-Static Step.1")
Example in Python:
Given a SimSteps object mySteps, you can create a SimQuasiStaticStep as following: 
 ...
 MyQuasiStaticStep = mySteps.Add("SimQuasiStaticStep")
Given a SimSteps object mySteps, you can retrieve a SimQuasiStaticStep named "Quasi-Static Step.1" as following: 
 ...
 MyQuasiStaticStep = mySteps.Item("Quasi-Static Step.1")

FALSE: the step does not use explicit integration for creep behavior.
