SimOutputElementLocation (Enumeration)

enum SimOutputElementLocation {
  SimOutputAtIntegrationPoints,
  SimOutputAtNodes,
  SimOutputAtCentroid,
  SimOutputAtNodesAveraged
}
The element location of the output request.
Values:
SimOutputAtIntegrationPoints
Element output at the element integration points.
SimOutputAtNodes
Element output at the element nodes.
SimOutputAtCentroid
Element output at the element centroid.
SimOutputAtNodesAveraged
Element output at the element nodes, averaged with the elements in the set that contribute to each node.
