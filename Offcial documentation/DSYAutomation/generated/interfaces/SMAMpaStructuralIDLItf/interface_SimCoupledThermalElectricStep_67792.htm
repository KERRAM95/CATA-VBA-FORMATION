SimCoupledThermalElectricStep (Object)

Represents the Coupled Thermal Electric Step object.
Example:
Given a SimSteps object, you can create a SimCoupledThermalElectricStep as following: 
 Dim MySteps As SimSteps
 ...
 Dim MyCoupledThermalElectricStep As SimCoupledThermalElectricStep
 Set MyCoupledThermalElectricStep = MySteps.Add("SimCoupledThermalElectricStep")
Given a SimSteps object, you can retrieve a SimCoupledThermalElectricStep named "Coupled Thermal Electric Step.1" as following: 
 Dim MySteps As SimSteps
 ...
 Dim MyCoupledThermalElectricStep As SimCoupledThermalElectricStep
 Set MyCoupledThermalElectricStep = MySteps.Item("Coupled Thermal Electric Step.1")
Example in Python:
Given a SimSteps object mySteps, you can create a SimCoupledThermalElectricStep as following: 
 ...
 myCoupledThermalElectricStep = mySteps.Add("SimCoupledThermalElectricStep")
Given a SimSteps object mySteps, you can retrieve a SimCoupledThermalElectricStep named "Coupled Thermal Electric Step.1" as following: 
 ...
 myCoupledThermalElectricStep = mySteps.Item("Coupled Thermal Electric Step.1")
