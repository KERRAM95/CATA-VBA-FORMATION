SimAMSEigensolver (Object)

Represents the A M S Eigensolver object.
Property Index
AcousticCouplingType Returns or sets the acoustic coupling type.
DampingProjectionFlag Returns or sets the flag that determines if the damping projection option is used.
MaximumFrequency Returns or sets the maximum frequency.
MinimumFrequency Returns or sets the minimum frequency.
MinimumFrequencyFlag Returns or sets the flag that determines if the minimum frequency is specified.
NumberOfModes Returns or sets the number of modes.
ResidualModesFlag Returns or sets the flag that determines if the residual modes option is used.
SpecifiedModesFlag Returns or sets the flag that determines if the number of eigenvalues to be calculated is specified.

Properties


Property AcousticCouplingType() As
Returns or sets the acoustic coupling type.
Property DampingProjectionFlag() As
Returns or sets the flag that determines if the damping projection option is used.
TRUE : the damping projection option is used.
FALSE : the damping projection option is not used.
Property MaximumFrequency() As
Returns or sets the maximum frequency. Quantity: FREQUENCY, units: Hz.
Property MinimumFrequency() As
Returns or sets the minimum frequency. Quantity: FREQUENCY, units: Hz.
Property MinimumFrequencyFlag() As
Returns or sets the flag that determines if the minimum frequency is specified.
TRUE : the minimum frequency is specified.
FALSE : the minimum frequency is not specified.
Property NumberOfModes() As
Returns or sets the number of modes.
Property ResidualModesFlag() As
Returns or sets the flag that determines if the residual modes option is used.
TRUE : the residual modes option is used.
FALSE : the residual modes option is not used.
Property SpecifiedModesFlag() As
Returns or sets the flag that determines if the number of eigenvalues to be calculated is specified.
TRUE : the number of eigenvalues to be calculated is specified.
FALSE : all the eigenvalues from the minimum frequency of interest up to the maximum frequency of interest will be calculated.
