SimMatrixStorageScheme (Enumeration)

enum SimMatrixStorageScheme {
  SimMatrixStorage_Default,
  SimMatrixStorage_Symmetric,
  SimMatrixStorage_Unsymmetric
}
The matrix storage scheme.
Values:
SimMatrixStorage_Default
The application will determine which scheme is used.
SimMatrixStorage_Symmetric
A symmetric matrix is used.
SimMatrixStorage_Unsymmetric
An unsymmetric matrix is used.
