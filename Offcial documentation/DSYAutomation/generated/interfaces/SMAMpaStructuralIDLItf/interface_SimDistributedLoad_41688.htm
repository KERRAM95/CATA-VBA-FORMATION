SimDistributedLoad (Object)

Represents the Distributed Load object.
Example:
Given a SimFeatures object, you can create a SimDistributedLoad as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyDistributedLoad As SimDistributedLoad
 Set MyDistributedLoad = MyFeatures.Add("SimDistributedLoad")
Given a SimFeatures object, you can retrieve a SimDistributedLoad named "Distributed Load.1" as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyDistributedLoad As SimDistributedLoad
 Set MyDistributedLoad = MyFeatures.Item("Distributed Load.1")
Example in Python:
Given a SimFeatures object myFeatures, you can create a SimDistributedLoad as following: 
 ...
 myDistributedLoad = myFeatures.Add("SimDistributedLoad")
Given a SimFeatures object myFeatures, you can retrieve a SimDistributedLoad named "Distributed Load.1" as following: 
 ...
 myDistributedLoad = myFeatures.Item("Distributed Load.1")

Property FollowerFlag() As
Returns or sets the flag that determines if the load remains normal to the support as it deforms.

TRUE: the load remains normal to the support as it deforms.

FALSE: the load remains in its original orientation as the support deforms.

Property XForce() As
Returns or sets the X force magnitude. Quantity: FORCE, units: N.
Setting the magnitude will also set the corresponding flag to TRUE. Retrieving the magnitude without setting it first will result in an error.
Property XForceSet() As
Returns or sets the flag that determines if a force in the X direction is specified.
TRUE: force is specified.
FALSE: force is not specified.
Set the flag to FALSE to unset the force. Setting to TRUE is not supported and will result in an error. The flag will automatically be set to TRUE when you set a force.
