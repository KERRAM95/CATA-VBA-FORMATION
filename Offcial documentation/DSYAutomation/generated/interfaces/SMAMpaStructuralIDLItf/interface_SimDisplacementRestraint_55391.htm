SimDisplacementRestraint (Object)

Represents the Displacement Restraint object.
Example:
Given a SimFeatures object, you can create a SimDisplacementRestraint as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyDisplacementRestraint As SimDisplacementRestraint
 Set MyDisplacementRestraint = MyFeatures.Add("SimDisplacementRestraint")
Given a SimFeatures object, you can retrieve a SimDisplacementRestraint named "Displacement Restraint.1" as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyDisplacementRestraint As SimDisplacementRestraint
 Set MyDisplacementRestraint = MyFeatures.Item("Displacement Restraint.1")
Example in Python:
Given a SimFeatures object myFeatures, you can create a SimDisplacementRestraint as following: 
 ...
 myDisplacementRestraint = myFeatures.Add("SimDisplacementRestraint")
Given a SimFeatures object myFeatures, you can retrieve a SimDisplacementRestraint named "Displacement Restraint.1" as following: 
 ...
 myDisplacementRestraint = myFeatures.Item("Displacement Restraint.1")
