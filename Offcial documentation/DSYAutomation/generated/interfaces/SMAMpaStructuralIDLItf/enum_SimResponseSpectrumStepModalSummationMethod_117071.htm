SimResponseSpectrumStepModalSummationMethod (Enumeration)

enum SimResponseSpectrumStepModalSummationMethod {
  SimResponseSpectrumStepAbsoluteValues,
  SimResponseSpectrumStepSquareRootOfSumOfSquaresMethod,
  SimResponseSpectrumStepNavalResearchLaboratory,
  SimResponseSpectrumStepTenPercentMethod,
  SimResponseSpectrumStepCompleteQuadraticCombination,
  SimResponseSpectrumStepGroupingMethod,
  SimResponseSpectrumStepDoubleSumCombination
}
Method for combining the peak modal responses to estimate the peak physical response.
Values:
SimResponseSpectrumStepAbsoluteValues
Sums the absolute value resulting from each mode.
SimResponseSpectrumStepSquareRootOfSumOfSquaresMethod
Sums the modal response by taking the square root of the sum of the squares of those responses.
SimResponseSpectrumStepNavalResearchLaboratory
Sums the absolute value of the mode associated with the maximum response and the square root of the sum of the squares of the peak responses in all other modes.
SimResponseSpectrumStepTenPercentMethod
Sums all pairs of modes alpha and beta that are within 10% of each other in a modified square-root-of-the-sum-of-squares method.
SimResponseSpectrumStepCompleteQuadraticCombination
Sums all pairs of modes with closely spaced eigenvalues in a manner similar to the ten-percent rule, but uses cross-correlation coefficients based on the ratio of the frequency and modal damping between the two modes.
SimResponseSpectrumStepGroupingMethod
Sums the modal responses absolutely within groups before performing a square root of the sum of the squares on that combination.
SimResponseSpectrumStepDoubleSumCombination
Sums the modal responses based on random vibration theory.
