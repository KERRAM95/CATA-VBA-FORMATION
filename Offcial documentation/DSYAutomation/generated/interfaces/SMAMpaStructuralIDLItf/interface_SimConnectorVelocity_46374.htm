SimConnectorVelocity (Object)

Represents the Connector Velocity object.
Example:
Given a SimFeatures object, you can create a SimConnectorVelocity as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyConnectorVelocity As SimConnectorVelocity
 Set MyConnectorVelocity = MyFeatures.Add("SimConnectorVelocity")
Given a SimFeatures object, you can retrieve a SimConnectorVelocity named "Connector Velocity.1" as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyConnectorVelocity As SimConnectorVelocity
 Set MyConnectorVelocity = MyFeatures.Item("Connector Velocity.1")
Example in Python:
Given a SimFeatures object myFeatures, you can create a SimConnectorVelocity as following: 
 ...
 myConnectorVelocity = myFeatures.Add("SimConnectorVelocity")
Given a SimFeatures object myFeatures, you can retrieve a SimConnectorVelocity named "Connector Velocity.1" as following: 
 ...
 myConnectorVelocity = myFeatures.Item("Connector Velocity.1")

Returns or sets the X angular velocity magnitude. Quantity: ANGULAR_VELOCITY, units: turn_mn.
Setting the magnitude will also set the corresponding flag to TRUE. Setting or retrieving the magnitude of an unavailable component is not supported and will result in an error. Retrieving the magnitude without setting it first will also result in an error.
Property XAngularVelocityAvailable() As (Read Only)
Returns the flag that determines if the connector angular velocity component in the X direction is available.
TRUE: component is available.
FALSE: component is not available.
Property XAngularVelocitySet() As
Returns or sets the flag that determines if a angular velocity in the X direction is specified.
TRUE: angular velocity is specified.
FALSE: angular velocity is not specified.
Set the flag to FALSE to unset the angular velocity. Setting to TRUE is not supported and will result in an error. The flag will automatically be set to TRUE when you set a angular velocity. Setting an unavailable component to FALSE will also result in an error.
