SimInitialVelocity (Object)

Represents the Initial Velocity object.
Example:
Given a SimFeatures object, you can create a SimInitialVelocity as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyInitialVelocity As SimInitialVelocity
 Set MyInitialVelocity = MyFeatures.Add("SimInitialVelocity")
Given a SimFeatures object, you can retrieve a SimInitialVelocity named "Initial Velocity.1" as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyInitialVelocity As SimInitialVelocity
 Set MyInitialVelocity = MyFeatures.Item("Initial Velocity.1")
Example in Python:
Given a SimFeatures object myFeatures, you can create a SimInitialVelocity as following: 
 ...
 myInitialVelocity = myFeatures.Add("SimInitialVelocity")
Given a SimFeatures object myFeatures, you can retrieve a SimInitialVelocity named "Initial Velocity.1" as following: 
 ...
 myInitialVelocity = myFeatures.Item("Initial Velocity.1")
