SimApplicationType (Enumeration)

enum SimApplicationType {
  SimApplicationType_Solver_Default,
  SimApplicationType_Transient_Fidelity,
  SimApplicationType_Moderate_Dissipation,
  SimApplicationType_Quasi_Static
}
The simulation step time application type.
Values:
SimApplicationType_Solver_Default
Default application is used for the step.
SimApplicationType_Transient_Fidelity
Transient fidelity application is used for the step.
SimApplicationType_Moderate_Dissipation
Moderate_dissipation application is used for the step.
SimApplicationType_Quasi_Static
Transient fidelity application is used for the step.
