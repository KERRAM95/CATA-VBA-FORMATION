SimRotaryAccelerationForce (Object)

Represents the Rotary Acceleration Force object.
Example:
Given a SimFeatures object, you can create a SimRotaryAccelerationForce as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyRotaryAccelerationForce As SimRotaryAccelerationForce
 Set MyRotaryAccelerationForce = MyFeatures.Add("SimRotaryAccelerationForce")
Given a SimFeatures object, you can retrieve a SimRotaryAccelerationForce named "Rotary Acceleration Force.1" as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyRotaryAccelerationForce As SimRotaryAccelerationForce
 Set MyRotaryAccelerationForce = MyFeatures.Item("Rotary Acceleration Force.1")
Example in Python:
Given a SimFeatures object myFeatures, you can create a SimRotaryAccelerationForce as following: 
 ...
 myRotaryAccelerationForce = myFeatures.Add("SimRotaryAccelerationForce")
Given a SimFeatures object myFeatures, you can retrieve a SimRotaryAccelerationForce named "Rotary Acceleration Force.1" as following: 
 ...
 myRotaryAccelerationForce = myFeatures.Item("Rotary Acceleration Force.1")
