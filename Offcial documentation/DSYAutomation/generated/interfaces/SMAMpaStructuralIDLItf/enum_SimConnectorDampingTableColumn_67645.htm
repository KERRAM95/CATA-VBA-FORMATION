SimConnectorDampingTableColumn (Enumeration)

enum SimConnectorDampingTableColumn {
  SimConnectorDampingCoefficient,
  SimConnectorDampingTemperature,
  SimConnectorDampingVelocity
}
The columns of the table that determines the damping behavior.

The columns in the connector damping table will be:
For Linear damping:
  1. Coefficient.
    • For translational DOF, Quantity: FORCE_OVER_VELOCITY, Units: kg_s
    • For rotational DOF, Quantity: MOMENT_OVER_PULSATION, Units: m2kg_s_rad
  2. Temperature (optional). Quantity: TEMPRTRE, Units: Kdeg

For NonLinear damping:
  1. Coefficient.
    • For translational DOF, Quantity: FORCE, Units: N
    • For rotational DOF, Quantity: MOMENT, Units: N_m
  2. Velocity.
    • For translational DOF, Quantity: SPEED, Units: m_s
    • For rotational DOF, Quantity: PULSATION, Units: rad_s
  3. Temperature (optional). Quantity: TEMPRTRE, Units: Kdeg
