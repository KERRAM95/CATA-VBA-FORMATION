SimInitialStress (Object)

Represents the Initial Stress object.
Example:
Given a SimFeatures object, you can create a SimInitialStress as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyInitialStress As SimInitialStress
 Set MyInitialStress = MyFeatures.Add("SimInitialStress")
Given a SimFeatures object, you can retrieve a SimInitialStress named "Initial Stress.1" as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyInitialStress As SimInitialStress
 Set MyInitialStress = MyFeatures.Item("Initial Stress.1")
Example in Python:
Given a SimFeatures object myFeatures, you can create a SimInitialStress as following: 
 ...
 myInitialStress = myFeatures.Add("SimInitialStress")
Given a SimFeatures object myFeatures, you can retrieve a SimInitialStress named "Initial Stress.1" as following: 
 ...
 myInitialStress = myFeatures.Item("Initial Stress.1")
