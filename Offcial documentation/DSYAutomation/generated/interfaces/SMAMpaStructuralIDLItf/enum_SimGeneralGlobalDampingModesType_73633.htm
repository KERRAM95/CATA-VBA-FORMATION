SimGeneralGlobalDampingModesType (Enumeration)

enum SimGeneralGlobalDampingModesType {
  SimGeneralGlobalDampingNone,
  SimGeneralGlobalDampingStructuralAndAcoustic,
  SimGeneralGlobalDampingStructural,
  SimGeneralGlobalDampingAcoustic
}
The type of modes.
Values:
SimGeneralGlobalDampingNone
No damping.
SimGeneralGlobalDampingStructuralAndAcoustic
Applies the damping values to structural and acoustic modes.
SimGeneralGlobalDampingStructural
Applies the damping values to structural modes only.
SimGeneralGlobalDampingAcoustic
Applies the damping values to acoustic modes only.
