SimSpectrumTableColumn (Enumeration)

enum SimSpectrumTableColumn {
  SimSpectrumAccelerationCol,
  SimSpectrumDampingRatioCol,
  SimSpectrumFrequencyCol,
  SimSpectrumDisplacementCol,
  SimSpectrumGravityCol,
  SimSpectrumVelocityCol
}
The columns of the table that determines the spectrum behavior.

The columns in the table will be:
These columns apply to all definitions:
  1. Frequency. Quantity: FREQUENCY, Units: Hz
  2. DampingRatio. Quantity: DIMENSIONLESS, Units: None

For Acceleration definition:
  1. Acceleration. Quantity: ACCELRTN, Units: m_s2

For Displacement definition:
  1. Displacement. Quantity: LENGTH, Units: mm

For Gravity definition:
  1. Gravity. Quantity: ACCELRTN, Units: m_s2

For Velocity definition:
  1. Velocity. Quantity: SPEED, Units: m_s
