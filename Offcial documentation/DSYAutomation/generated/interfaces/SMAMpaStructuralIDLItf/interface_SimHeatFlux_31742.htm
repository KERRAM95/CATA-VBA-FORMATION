SimHeatFlux (Object)

Represents the Heat Flux object.
Example:
Given a SimFeatures object, you can create a SimHeatFlux as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyHeatFlux As SimHeatFlux
 Set MyHeatFlux = MyFeatures.Add("SimHeatFlux")
Given a SimFeatures object, you can retrieve a SimHeatFlux named "Heat Flux.1" as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyHeatFlux As SimHeatFlux
 Set MyHeatFlux = MyFeatures.Item("Heat Flux.1")
Example in Python:
Given a SimFeatures object myFeatures, you can create a SimHeatFlux as following: 
 ...
 myHeatFlux = myFeatures.Add("SimHeatFlux")
Given a SimFeatures object myFeatures, you can retrieve a SimHeatFlux named "Heat Flux.1" as following: 
 ...
 myHeatFlux = myFeatures.Item("Heat Flux.1")
