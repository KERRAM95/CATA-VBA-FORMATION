SimAppliedVelocity (Object)

Represents the Applied Velocity object.
Example:
Given a SimFeatures object, you can create a SimAppliedVelocity as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyAppliedVelocity As SimAppliedVelocity
 Set MyAppliedVelocity = MyFeatures.Add("SimAppliedVelocity")
Given a SimFeatures object, you can retrieve a SimAppliedVelocity named "Applied Velocity.1" as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyAppliedVelocity As SimAppliedVelocity
 Set MyAppliedVelocity = MyFeatures.Item("Applied Velocity.1")
Example in Python:
Given a SimFeatures object myFeatures, you can create a SimAppliedVelocity as following: 
 ...
 myAppliedVelocity = myFeatures.Add("SimAppliedVelocity")
Given a SimFeatures object myFeatures, you can retrieve a SimAppliedVelocity named "Applied Velocity.1" as following: 
 ...
 myAppliedVelocity = myFeatures.Item("Applied Velocity.1")

Property XAngularVelocity() As
Returns or sets the X angular velocity magnitude. Quantity: ANGULAR_VELOCITY, units: turn_mn.
Setting the magnitude will also set the corresponding flag to TRUE. Retrieving the magnitude without setting it first will result in an error.
Property XAngularVelocitySet() As
Returns or sets the flag that determines if an angular velocity in the X direction is specified.
TRUE: velocity is specified.
FALSE: velocity is not specified.
Set the flag to FALSE to unset the velocity. Setting to TRUE is not supported and will result in an error. The flag will automatically be set to TRUE when you set a velocity.
