SimConnectorDisplacement (Object)

Represents the Connector Displacement object.
Example:
Given a SimFeatures object, you can create a SimConnectorDisplacement as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyConnectorDisplacement As SimConnectorDisplacement
 Set MyConnectorDisplacement = MyFeatures.Add("SimConnectorDisplacement")
Given a SimFeatures object, you can retrieve a SimConnectorDisplacement named "Connector Displacement.1" as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyConnectorDisplacement As SimConnectorDisplacement
 Set MyConnectorDisplacement = MyFeatures.Item("Connector Displacement.1")
Example in Python:
Given a SimFeatures object myFeatures, you can create a SimConnectorDisplacement as following: 
 ...
 myConnectorDisplacement = myFeatures.Add("SimConnectorDisplacement")
Given a SimFeatures object myFeatures, you can retrieve a SimConnectorDisplacement named "Connector Displacement.1" as following: 
 ...
 myConnectorDisplacement = myFeatures.Item("Connector Displacement.1")
