SimRadiationAmbient (Object)

Represents the Radiation Ambient object.
Example:
Given a SimFeatures object, you can create a SimRadiationAmbient as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyRadiationAmbient As SimRadiationAmbient
 Set MyRadiationAmbient = MyFeatures.Add("SimRadiationAmbient")
Given a SimFeatures object, you can retrieve a SimRadiationAmbient named "Radiation Ambient.1" as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyRadiationAmbient As SimRadiationAmbient
 Set MyRadiationAmbient = MyFeatures.Item("Radiation Ambient.1")
Example in Python:
Given a SimFeatures object myFeatures, you can create a SimRadiationAmbient as following: 
 ...
 myRadiationAmbient = myFeatures.Add("SimRadiationAmbient")
Given a SimFeatures object myFeatures, you can retrieve a SimRadiationAmbient named "Radiation Ambient.1" as following: 
 ...
 myRadiationAmbient = myFeatures.Item("Radiation Ambient.1")
