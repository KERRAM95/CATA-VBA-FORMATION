SimConnectorLoad (Object)

Represents the Connector Load object.
Example:
Given a SimFeatures object, you can create a SimConnectorLoad as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyConnectorLoad As SimConnectorLoad
 Set MyConnectorLoad = MyFeatures.Add("SimConnectorLoad")
Given a SimFeatures object, you can retrieve a SimConnectorLoad named "Connector Load.1" as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyConnectorLoad As SimConnectorLoad
 Set MyConnectorLoad = MyFeatures.Item("Connector Load.1")
Example in Python:
Given a SimFeatures object myFeatures, you can create a SimConnectorLoad as following: 
 ...
 myConnectorLoad = myFeatures.Add("SimConnectorLoad")
Given a SimFeatures object myFeatures, you can retrieve a SimConnectorLoad named "Connector Load.1" as following: 
 ...
 myConnectorLoad = myFeatures.Item("Connector Load.1")

Property XForceAvailable() As (Read Only)
Returns the flag that determines if the connector force component in the X direction is available.
TRUE: component is available.
FALSE: component is not available.
Property XForceSet() As
Returns or sets the flag that determines if a force in the X direction is specified.
TRUE: force is specified.
FALSE: force is not specified.
Set the flag to FALSE to unset the force. Setting to TRUE is not supported and will result in an error. The flag will automatically be set to TRUE when you set a force. Setting an unavailable component to FALSE will also result in an error.
