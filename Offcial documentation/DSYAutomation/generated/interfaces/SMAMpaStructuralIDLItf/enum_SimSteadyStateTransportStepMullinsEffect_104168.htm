SimSteadyStateTransportStepMullinsEffect (Enumeration)

enum SimSteadyStateTransportStepMullinsEffect {
  SimSteadyStateTransportStepRampMullins,
  SimSteadyStateTransportStepImmediateMullins
}
Mullins effect application.
Values:
SimSteadyStateTransportStepRampMullins
Increase the Mullins effect over the duration of the current step gradually.
SimSteadyStateTransportStepImmediateMullins
Apply the Mullins effect at the start of the current step instantaneously.
