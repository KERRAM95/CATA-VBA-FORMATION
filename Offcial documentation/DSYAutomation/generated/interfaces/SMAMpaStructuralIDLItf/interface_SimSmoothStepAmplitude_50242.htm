SimSmoothStepAmplitude (Object)

Represents the Smooth Step Amplitude object.

The creation of SimSmoothStepAmplitude directly from SimFeatures is deprecated R425. The new procedure is to create SimAmplitude from SimFeatures and retrieve the SimSmoothStepAmplitude. The following example demonstrates this.
Example:
Given a SimFeatures object, you can create a SimAmplitude and retrieve the SimSmoothStepAmplitude as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyAmplitude As SimAmplitude
 Set MyAmplitude = MyFeatures.Add("SimAmplitude")
 MyAmplitude.DefinitionType = SimAmplitudeSmoothStepDefinition
 Dim MySmoothStepAmplitude As SimSmoothStepAmplitude
 Set MySmoothStepAmplitude = MyAmplitude.SmoothStepAmplitude
Example in Python:
Given a SimFeatures object, you can create a SimAmplitude and retrieve the SimSmoothStepAmplitude as following: 
 ...
 myAmplitude = myFeatures.Add("SimAmplitude")
 myAmplitude.DefinitionType = SimAmplitudeSmoothStepDefinition
 mySmoothStepAmplitude = myAmplitude.SmoothStepAmplitude
