SimPrescribedTemperature (Object)

Represents the Prescribed Temperature object.
Example:
Given a SimFeatures object, you can create a SimPrescribedTemperature as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyPrescribedTemperature As SimPrescribedTemperature
 Set MyPrescribedTemperature = MyFeatures.Add("SimPrescribedTemperature")
Given a SimFeatures object, you can retrieve a SimPrescribedTemperature named "Prescribed Temperature.1" as following: 
 Dim MyFeatures As SimFeatures
 ...
 Dim MyPrescribedTemperature As SimPrescribedTemperature
 Set MyPrescribedTemperature = MyFeatures.Item("Prescribed Temperature.1")
Example in Python:
Given a SimFeatures object myFeatures, you can create a SimPrescribedTemperature as following: 
 ...
 myPrescribedTemperature = myFeatures.Add("SimPrescribedTemperature")
Given a SimFeatures object myFeatures, you can retrieve a SimPrescribedTemperature named "Prescribed Temperature.1" as following: 
 ...
 myPrescribedTemperature = myFeatures.Item("Prescribed Temperature.1")

Properties


Property AbsoluteExteriorTolerance() As
Returns or sets the Absolute Exterior Tolerance. This is the value that nodes of the structural model can be outside the region of the elements of the thermal model. If this value is zero, the value for the relative exterior tolerance is used.
Property Activated() As
Returns or sets the activation status.
Property AnalysisCase() As
Returns or sets the Thermal Analysis Case.
Property FeatureHistory() As (Read Only)
Returns the feature history.
Property FromStepFlag() As
Returns or sets the flag for an initial temperature from a SimThermalAnalysisCase and SimStep. This example demonstrates how to use the SimPrescribedTemperature object with a temperature field. 
 Dim MyPrescribedTemperature As SimPrescribedTemperature
 Dim MyThermalAnalysisCase As SimThermalAnalysisCase
 Dim MySteadyStateHeatTransferStep As SimSteadyStateHeatTransferStep
 Dim MyStructuralFeatureStateExtension As SimFeatureStateExtension
 ...
 Dim MyExtension As SimPrescribedTemperatureExtension
 MyPrescribedTemperature.FromStepFlag = True
 MyPrescribedTemperature.AnalysisCase = MyThermalAnalysisCase
 MyPrescribedTemperature.AbsoluteExteriorTolerance = 0.0
 MyPrescribedTemperature.RelativeExteriorTolerance = 0.1
 MyStructuralFeatureStateExtension.BeginningStep = MySteadyStateHeatTransferStep
 MyStructuralFeatureStateExtension.BeginningIncrement = 1
 MyStructuralFeatureStateExtension.EndingStep = MySteadyStateHeatTransferStep
 MyStructuralFeatureStateExtension.EndingIncrement = 202399
Property RelativeExteriorTolerance() As
Returns or sets the Relative Exterior Tolerance. This is the fraction of the average element size that nodes of the structural model can be outside the region of the elements of the thermal model.
Property ScalarField() As (Read Only)
Returns the scalar field.
Property SpecTreeCategory() As (Read Only)
Returns a string representing the specification tree category of the feature. See for usage.
Property UniformMagnitude() As
Temperature value. Quantity: TEMPRTRE, units: Kdeg
Property UniformMagnitudeFlag() As (Read Only)
Returns or sets the flag that determines if the magnitude is uniform.
TRUE: the magnitude is uniform.
FALSE: the magnitude is not uniform.
