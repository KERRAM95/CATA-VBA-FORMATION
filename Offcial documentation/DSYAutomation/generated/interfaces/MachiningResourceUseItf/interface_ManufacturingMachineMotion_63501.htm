ManufacturingMachineMotion (Object)

Interface dedicated to machine positioning.
Role: This interface offers services to move the machine at applicative positions.
Method Index
MoveTo Move the current machine at given position.
MoveToManufacturingFastener Move the current machine at position of given manufacturing fastener.
RestorePosition Restore the machine at its original position.

Methods


Func MoveTo( iPositionCoordinates, iDirectionCoordinates) As
Move the current machine at given position.
Parameters:
iPositionCoordinates
The point where to move the machine, as an array of 3 values.
iDirectionCoordinates
The axial direction where to move the machine, as an array of 3 values.
obIsReachable
Get whether the position is reachable.
Returns:
Return code.
Legal values:
  • S_OK: move is successful
  • E_FAIL: otherwise
Func MoveToManufacturingFastener( iManufacturingFastener) As
Move the current machine at position of given manufacturing fastener.
Parameters:
iManufacturingFastener
The manufacturing fastener where to move the machine.
obIsReachable
Get whether the position is reachable.
Sub RestorePosition()
Restore the machine at its original position.
