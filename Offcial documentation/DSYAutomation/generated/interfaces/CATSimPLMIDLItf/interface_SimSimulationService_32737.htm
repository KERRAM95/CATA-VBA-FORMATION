SimSimulationService (Object)

Represents the Service to return the root occurrence of the product referred by the simulation.
Example:
Given a CATIA application object you can retrieve a SimSimulationService object as following: 
 Dim MySimulationService As SimSimulationService
 Set MySimulationService = CATIA.GetSessionService("SimSimulationService")
Method Index
GetProductRootOccurrence Method to return the root occurrence of the product referred by the simulation.
GetSimulationReferenceFromObject Retrieves the Simulation object that aggregates a simulation feature.
