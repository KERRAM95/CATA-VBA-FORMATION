AGTRoot (Object)

Object to AGTRoot.
To retrieve collection from part object.
Property Index
Connectors Returns the collection of Connector.
Coverings Returns the collection of Covering.
DraughtStops Returns the collection of DraughtStop.
FireBridges Returns the collection of FireBridge.
Insulations Returns the collection of insulation.
Sills Returns the collection of Sill.
Walls Returns the collection of Wall.

Properties


Property Connectors() As (Read Only)
Returns the collection of Connector.
Example:
This example retrieves in Connector Collection from part object. 
 Dim myPart As CATIAPart
 Set myPart = CATIA.ActiveEditor.ActiveObject
 Dim MyRoot As AGTRoot
 Set MyRoot = MyRoot.GetItem("CATAGTRoot")
 Dim ConnectorCollection As AGTConnectors
 Set ConnectorCollection = MyRoot.Connectors
