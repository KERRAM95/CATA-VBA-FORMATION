PLMOccurrence (Object)

Represents a PLM Product Occurrence object.
Enables to retrieve a PLM Entity (Instance/Reference) and a PLM Occurrences collection.
Property Index
PLMEntity Returns the Instance or Reference Product Model object.
PLMOccurrences Returns the PLM Product Occurrences collection.

Properties


Property PLMEntity() As (Read Only)
Returns the Instance or Reference Product Model object.
Role: This method retrieves the corresponding PLM Product Instance if calling object is an instance or the PLM Product Reference if it is a reference.
Property PLMOccurrences() As (Read Only)
Returns the PLM Product Occurrences collection.
Role: This method returns the Product Occurrences collection directly aggregated within this object.
