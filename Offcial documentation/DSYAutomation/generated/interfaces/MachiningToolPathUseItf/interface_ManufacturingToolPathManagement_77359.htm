ManufacturingToolPathManagement (Object)

Interface on the operation to manage links between it and its tool path.
Role: DELIMfgToolPathManagement has methods to set, retrieve and remove tool path on the operation
Property Index
HasAToolPath Indicates if the operation owns a tool path.
Method Index
GetToolPathLimitZ Get the min and max Z of tool path.

Properties


Property HasAToolPath() As (Read Only)
Indicates if the operation owns a tool path.

Methods


Sub GetToolPathLimitZ( oZmin, oZmax)
Get the min and max Z of tool path.
Parameters:
oZmin
return the mininum of Z in tool path
oZmax
return the maxinum of Z in tool path
Returns:
E_FAIL if no tool path found and toolpath don't contain machining motion
