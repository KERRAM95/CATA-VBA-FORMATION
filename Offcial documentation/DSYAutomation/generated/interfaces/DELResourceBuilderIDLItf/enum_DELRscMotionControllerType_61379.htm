DELRscMotionControllerType (Enumeration)

enum DELRscMotionControllerType {
  DELRscMotionControllerType_Default,
  DELRscMotionControllerType_Rail,
  DELRscMotionControllerType_EndOfArm,
  DELRscMotionControllerType_WorkpiecePositioner,
  DELRscMotionControllerType_Arm,
  DELRscMotionControllerType_FixedTool
}
Indicates the type of control on a given by the motion controller.
Role: This enum is used in the interface. It aims at identifying how a given resource is used by another resource.
Values:
DELRscMotionControllerType_Default
The resource is controlled by default by the motion controller (reference case when a resource like a robot is opened as a stand-alone).
DELRscMotionControllerType_Rail
The resource is controlled as a rail by a given motion controller.
DELRscMotionControllerType_EndOfArm
The resource is controlled as a end-effector by a given motion controller.
DELRscMotionControllerType_WorkpiecePositioner
The resource is controlled as a positioner by a given motion controller.
DELRscMotionControllerType_FixedTool
The resource is controlled as a fixed tool by a given motion controller.
