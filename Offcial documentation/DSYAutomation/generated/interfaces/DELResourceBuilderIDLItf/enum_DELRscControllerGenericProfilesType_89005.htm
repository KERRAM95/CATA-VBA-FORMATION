DELRscControllerGenericProfilesType (Enumeration)

enum DELRscControllerGenericProfilesType {
  DELRscControllerGenericProfilesType_Tool,
  DELRscControllerGenericProfilesType_Motion,
  DELRscControllerGenericProfilesType_Accuracy,
  DELRscControllerGenericProfilesType_ObjectFrame
}
Indicates the type of generic profile.
Role: This enum is used in the interface.It is used to identify the type of profiles.
Values:
DELRscControllerGenericProfilesType_Tool
Tool profile. Used to indicate TCP offset and TCP mode.
DELRscControllerGenericProfilesType_Motion
Motion profile. Indicates the type of motion speed to support for robot motions.
DELRscControllerGenericProfilesType_Accuracy
Accuracy profile. Controls the rounding(fly-by).
DELRscControllerGenericProfilesType_ObjectFrame
Object frame profile. Indicates the local referential used for coordinates positioning.
