DELRscJointType (Enumeration)

enum DELRscJointType {
  DELRscJointType_Linear,
  DELRscJointType_Angular
}
Indicates the type of joint.
Role: This enum is used in the interface.It is used to identify the type of joint and their respective dimension.
Values:
DELRscJointType_Linear
Linear joint. Values are expressed in meters.
DELRscJointType_Angular
Angular joint. Values are expressed in radians.
