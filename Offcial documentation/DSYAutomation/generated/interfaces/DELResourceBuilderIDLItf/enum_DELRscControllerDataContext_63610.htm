DELRscControllerDataContext (Enumeration)

enum DELRscControllerDataContext {
  DELRscControllerDataContext_None,
  DELRscControllerDataContext_Reference,
  DELRscControllerDataContext_Instance
}
Indicates the data context for the resource motion controller.
Role: This enum is used in the interface. A resource motion controller can be defined on a given resource (Robot for example): it is called as a reference. That same resource can be used multiple times inside a given context (cell for example). Each usage requires a specific motion controller data to manage other data in that same given context.
Values:
DELRscControllerDataContext_None
There is no motion controller currently defined.
DELRscControllerDataContext_Reference
The data is defined as part of the resource itself.
DELRscControllerDataContext_Instance
The data is defined part of the context where the resource is being used.
