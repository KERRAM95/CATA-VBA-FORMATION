ArcWeldProfile (Object)

Represents an object that is Arc Welding profile Role: Arc Welding profile contains description of arc welding parameters associated with a particular arc welding trajectory.
Each arc weld profile consists of three weld sections (Start Weld, Weld and End Weld) and of a flag that defines the Flying Start status.
