SimNonstructuralMassApplicationMethod (Enumeration)

enum SimNonstructuralMassApplicationMethod {
  SimNonstructuralMassMass,
  SimNonstructuralMassTotalMass
}
The method used to apply the mass.
Values:
SimNonstructuralMassMass
The nonstructural mass is specified in the unit of mass per unit volume, area or length.
SimNonstructuralMassTotalMass
The nonstructural mass is specified in the unit of mass.
