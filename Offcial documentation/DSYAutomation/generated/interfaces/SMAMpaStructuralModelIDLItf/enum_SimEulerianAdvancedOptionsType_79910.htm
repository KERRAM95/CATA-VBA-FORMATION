SimEulerianAdvancedOptionsType (Enumeration)

enum SimEulerianAdvancedOptionsType {
  SimDeactivated,
  SimAdaptiveMesh,
  SimEnhancedContact
}
Method for specifying the Advanced Options Type used for the Eulerian Property Adaptive Mesh Refinement
Values:
SimDeactivated
Indicates that the advanced options are not used
SimAdaptiveMesh
Allows the local refinement of the mesh by subdividing elements identified by user-defined criteria
SimEnhancedContact
Activates the enhanced contact formulation that acts to prevent the potential leakage of liquid or gas through the Lagrangian surfaces
