SimStiffnessBondTableColumn (Enumeration)

enum SimStiffnessBondTableColumn {
  SimStiffnessBondKnn,
  SimStiffnessBondKss,
  SimStiffnessBondKtt,
  SimStiffnessBondKns,
  SimStiffnessBondKnt,
  SimStiffnessBondKst,
  SimStiffnessBondTemp
}

The columns in the stiffness table will be:
For Uncoupled type:
  1. SimStiffnessBondKnn.
    Quantity: FORCE_OVER_DISPLACEMENT, Units: N_m
  2. SimStiffnessBondKss. Quantity: FORCE_OVER_DISPLACEMENT, Units: N_m
  3. SimStiffnessBondKtt. Quantity: FORCE_OVER_DISPLACEMENT, units: N_m
  4. SimStiffnessBondTemp (optional). Quantity: TEMPRTRE, Units: Kdeg
  5. Additional field (optional). Quantity: Real, units: None

For Coupled type:
  1. SimStiffnessBondKnn. Quantity: FORCE_OVER_DISPLACEMENT, Units: N_m
  2. SimStiffnessBondKss. Quantity: FORCE_OVER_DISPLACEMENT, Units: N_m
  3. SimStiffnessBondKtt. Quantity: FORCE_OVER_DISPLACEMENT, units: N_m
  4. SimStiffnessBondKns. Quantity: FORCE_OVER_DISPLACEMENT, Units: N_m
  5. SimStiffnessBondKnt. Quantity: FORCE_OVER_DISPLACEMENT, Units: N_m
  6. SimStiffnessBondKst. Quantity: FORCE_OVER_DISPLACEMENT, units: N_m
  7. SimStiffnessBondTemp (optional). Quantity: TEMPRTRE, Units: Kdeg
  8. Additional field (optional). Quantity: Real, units: None
