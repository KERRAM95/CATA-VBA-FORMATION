SimEulerianContactCriterionZone (Enumeration)

enum SimEulerianContactCriterionZone {
  SimAutomatic,
  SimMaterialElements,
  SimAllElements
}
Method for specifying Contact Criterion Application Zone used for the Eulerian Property
Values:
SimAutomatic
SimMaterialElements option will be used except for materials with Mie-Grüneisen equations of state
SimMaterialElements
Refines only elements containing materials that are in contact with the Lagrangian surfaces
SimAllElements
Refines all elements intersecting the Lagrangian surfaces even if contact has not occurred
