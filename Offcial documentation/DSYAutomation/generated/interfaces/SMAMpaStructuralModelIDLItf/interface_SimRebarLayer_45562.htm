SimRebarLayer (Object)

Represents the Rebar Layer object.
Example:
Given a SimShellSection object, you can create a SimRebarLayer as following: 
 Dim MyShellSection As SimShellSection
 ...
 Dim MyRebarLayer As SimRebarLayer
 Set MyRebarLayer = MyShellSection.Add("SimRebarLayer")
Given a SimShellSection object, you can retrieve a SimRebarLayer named "Rebar Layer.1" as following: 
 Dim MyShellSection As SimShellSection
 ...
 Dim MyRebarLayer As SimRebarLayer
 Set MyRebarLayer = MyShellSection.Item("Rebar Layer.1")
Example in Python:
Given a SimShellSection object myShellSection, you can create a SimRebarLayer as following: 
 ...
 myRebarLayer = myShellSection.Add("SimRebarLayer")
Given a SimShellSection object myShellSection, you can retrieve a SimRebarLayer named "Rebar Layer.1" as following: 
 ...
 myRebarLayer = myShellSection.Item("Rebar Layer.1")
