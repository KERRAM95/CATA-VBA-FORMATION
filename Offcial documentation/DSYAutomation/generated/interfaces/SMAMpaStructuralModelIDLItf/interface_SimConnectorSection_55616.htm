SimConnectorSection (Object)

Represents the Connector Section object.
Property Index

ConnectorBehavior Returns the connector behavior.
FirstAxisSystem Returns the first axis system.
RotationalConnectorType Returns or sets the type of the Rotational connector.
SecondAxisSystem Returns the second axis system.
TranslationalConnectorType Returns or sets the type of the Translational connector.

Properties


Property ActiveDOFs() As (Read Only)
Returns the list of active DOFs. The list can contains the following values: 1 : the first translation DOF is active, 2 : the second translation DOF is active, 3 : the third translation DOF is active, 4 : the first rotation DOF is active, 5 : the second rotation DOF is active, 6 : the third rotation DOF is active.
