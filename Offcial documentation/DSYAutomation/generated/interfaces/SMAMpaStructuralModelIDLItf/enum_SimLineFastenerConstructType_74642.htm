SimLineFastenerConstructType (Enumeration)

enum SimLineFastenerConstructType {
  SimLineFastenerSolidHex,
  SimLineFastenerShell,
  SimLineFastenerWedge
}
The type of construct.
Values:
SimLineFastenerSolidHex
The fastener will be defined as couplings and a contiguous row of hexahedral elements along the placement.
SimLineFastenerShell
The fastener will be defined as couplings and a contiguous row of quadrilateral shell elements along the placement.
SimLineFastenerWedge
The fastener will be defined as couplings and a contiguous row of wedge elements along the placement.
