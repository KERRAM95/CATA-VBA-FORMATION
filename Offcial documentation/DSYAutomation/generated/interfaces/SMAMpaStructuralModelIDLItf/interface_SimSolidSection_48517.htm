SimSolidSection (Object)

Represents the Solid Section object.
Example:
Given a SimProperties object, you can create a SimSolidSection as following: 
 Dim MyProperties As SimProperties
 ...
 Dim MySolidSection As SimSolidSection
 Set MySolidSection = MyProperties.Add("SimSolidSection")
Given a SimProperties object, you can retrieve a SimSolidSection named "Solid Section.1" as following: 
 Dim MyProperties As SimProperties
 ...
 Dim MySolidSection As SimSolidSection
 Set MySolidSection = MyProperties.Item("Solid Section.1")
Example in Python:
Given a SimProperties object myProperties, you can create a SimSolidSection as following: 
 ...
 mySolidSection = myProperties.Add("SimSolidSection")
Given a SimProperties object myProperties, you can retrieve a SimSolidSection named "Solid Section.1" as following: 
 ...
 mySolidSection = myProperties.Item("Solid Section.1")
