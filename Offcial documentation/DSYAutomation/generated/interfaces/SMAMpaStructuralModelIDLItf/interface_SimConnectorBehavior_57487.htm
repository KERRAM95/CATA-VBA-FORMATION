SimConnectorBehavior (Object)

Represents the Connector Behavior object.
Property Index
ConnectorDampings Returns the list of connector dampings.
ConnectorElasticities Returns the connector elasticities.
Method Index
GetReferenceLength Retrieves the reference length for a specific degree of freedom.
GetReferenceLengthFlag Retrieves the flag which determines if a reference length is active on a particular degree of freedom.
SetReferenceLength Sets the reference length for a specific degree of freedom.
SetReferenceLengthFlag Set the flag which determines if a reference length is active on a particular degree of freedom.

Properties


Property ConnectorDampings() As (Read Only)
Returns the list of connector dampings. Each member of the list will adhere to SMAMpaConnectorDamping interface.
Property ConnectorElasticities() As (Read Only)
Returns the connector elasticities. Each member of the list will adhere to SMAIMpaConnectorElasticity interface.
