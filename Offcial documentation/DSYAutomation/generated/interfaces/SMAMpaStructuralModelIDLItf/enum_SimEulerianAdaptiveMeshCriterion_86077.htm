SimEulerianAdaptiveMeshCriterion (Enumeration)

enum SimEulerianAdaptiveMeshCriterion {
  SimMaterialInterfacesCrit,
  SimContactCrit,
  SimPlasticDeformationCrit,
  SimDensityCrit,
  SimPressureCrit
}
Method for specifying Adaptive Mesh Refinement Criteria used for the Eulerian Property
Values:
SimMaterialInterfacesCrit
Refine elements containing material interfaces
SimContactCrit
Refine elements that are in contact with Lagrangian bodies
SimPlasticDeformationCrit
Refine elements in which significant plastic deformation occurs
SimDensityCrit
Refine elements near a sharp density gradient
SimPressureCrit
Refine elements near a sharp pressure gradient
