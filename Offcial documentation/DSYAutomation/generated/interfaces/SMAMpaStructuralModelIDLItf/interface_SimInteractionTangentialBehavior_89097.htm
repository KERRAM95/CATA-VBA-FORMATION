Properties


Property FrictionPressureDependentFlag() As
Returns or sets the flag that determines if friction coefficient value depends on the contact pressure.
The friction coefficient value depends on the contact pressure if this flag is set to TRUE.
Property FrictionSlipRateDependentFlag() As
Returns or sets the flag that determines if friction coefficient value depends on the slip rate.
The friction coefficient value depends on the slip rate if this flag is set to TRUE.
Property FrictionTable() As (Read Only)
Returns the table that determines the friction coefficient with dependencies.
The columns in the friction coefficient table will be:
For Coulomb type:
  1. Friction. Quantity: Real, Units: None
  2. Slip Rate (optional). Quantity: SPEED, Units: m_s
  3. Contact Pressure (optional). Quantity: PRESSURE, units: N_m2
  4. Temperature (optional). Quantity: TEMPRTRE, Units: Kdeg
Property FrictionTemperatureDependentFlag() As
Returns or sets the flag that determines if friction coefficient value depends on the temperature.
The friction coefficient value depends on the temperature if this flag is set to TRUE.
Property FrictionType() As
Returns or sets the friction type.
The friction type determines the type of friction model used.
Property TauMax() As
Returns the equivalent shear limit.
This method is not applicable for interactions in explicit dynamic steps.
Property TauMaxSpecified() As
Returns the flag that determines the equivalent shear limit specified.
The equivalent shear limit is specified if this flag is set to TRUE.

Methods


Func GetTableColumn( iTableColumnName) As
Returns a specific column of the table.
The table column contains the pressure-overclosure relationship.
