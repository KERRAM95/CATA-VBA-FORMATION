Properties


Property Cohere() As
Retrieves or Sets the cohesive enforcement type.
Property StiffnessCoupledCoefficientFlag() As
Retrieves or Sets the flag that determines the stiffness coupled coefficients for cohesive interfaces. TRUE: stiffness coupled coefficients of cohesive interfaces are defined. FALSE: stiffness coupled coefficients of cohesive interfaces are not defined.
Property StiffnessTable() As (Read Only)
Retrieves the table that determines the stiffness of the cohesive interface.
The columns in the stiffness table will be:
For Uncoupled type:
  1. Knn. Quantity: FORCE_OVER_DISPLACEMENT, Units: N_m
  2. Kss. Quantity: FORCE_OVER_DISPLACEMENT, Units: N_m
  3. Ktt. Quantity: FORCE_OVER_DISPLACEMENT, units: N_m
  4. Temperature (optional). Quantity: TEMPRTRE, Units: Kdeg
  5. Additional field (optional). Quantity: Real, units: None

For Coupled type:
  1. Knn. Quantity: FORCE_OVER_DISPLACEMENT, Units: N_m
  2. Kss. Quantity: FORCE_OVER_DISPLACEMENT, Units: N_m
  3. Ktt. Quantity: FORCE_OVER_DISPLACEMENT, units: N_m
  4. Kns. Quantity: FORCE_OVER_DISPLACEMENT, Units: N_m
  5. Knt. Quantity: FORCE_OVER_DISPLACEMENT, Units: N_m
  6. Kst. Quantity: FORCE_OVER_DISPLACEMENT, units: N_m
  7. Temperature (optional). Quantity: TEMPRTRE, Units: Kdeg
  8. Additional field (optional). Quantity: Real, units: None
Parameters:
ospTable
[out] The table that determines the stiffness coefficients of the cohesive interface.
Property StiffnessTemperatureDependentFlag() As
Retrieves or Sets the flag that determines the dependency of stiffness from temperature for cohesive interfaces. TRUE: stiffness dependency from temperature of cohesive interfaces is defined. FALSE: stiffness dependency from temperature of cohesive interfaces is not defined.
Property UserDefinedFlag() As
Retrieves or Sets the flag that determines the options between "Analysis default" and "User defined". TRUE: user defined values will be used for the defining the stiffness bond FALSE: default definition will be used for the stiffness bond
