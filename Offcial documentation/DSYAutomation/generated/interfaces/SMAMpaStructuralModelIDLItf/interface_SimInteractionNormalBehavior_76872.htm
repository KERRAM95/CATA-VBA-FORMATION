Properties


Property AllowSeparationFlag() As
Returns or sets the allow separation flag.
TRUE: separation between the surfaces is allowed once contact has been established.

FALSE: separation between the surfaces is not allowed once contact has been established.
Property ClearanceAtZeroContactPressure() As
Returns or sets the clearance at zero contact pressure.
The clearance at zero contact pressure determines the clearance at which the contact pressure is zero.
Property PressureAtZeroClearance() As
Returns or sets the pressure at zero clearance.
The pressure at zero clearance determines the pressure at which the clearance is zero.
Property PressureOverclosureTable() As (Read Only)
Returns or sets the pressure-overclosure table.
The table contains the pressure-overclosure relationship.
Property PressureOverclosureType() As
Returns or sets the pressure-overclosure type.
Hard: a hard contact will be defined.
Exponential: an exponential contact will be defined.
Tabular: a piecewise linear contact in tabular form will be defined.
Property UserDefinedFlag() As
Returns or sets the flag that determines if user-defined flag.
TRUE: a user-defined flag is constrainted to define the coupling behavior.

FALSE: a user-defined flag is not constrainted to define the coupling behavior.

Methods


Func GetTableColumn( iTableColumnName) As
Returns or sets the table column.
The table column contains the pressure-overclosure relationship.
