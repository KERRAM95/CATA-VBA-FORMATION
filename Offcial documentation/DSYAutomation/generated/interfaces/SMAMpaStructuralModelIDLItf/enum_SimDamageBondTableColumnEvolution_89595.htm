SimDamageBondTableColumnEvolution (Enumeration)

enum SimDamageBondTableColumnEvolution {
  SimDamageBondCohesiveDisplacement,
  SimDamageBondCohesiveEffectiveDisplacement,
  SimDamageBondCohesiveModeMixRatio1,
  SimDamageBondCohesiveModeMixRatio2,
  SimDamageBondEvoTemperature,
  SimDamageBondCohesiveExponentialParam,
  SimDamageBondCohesiveDamageVar,
  SimDamageBondCohesiveFractureEnergy,
  SimDamageBondCohesiveNormalFractureEnergy,
  SimDamageBondCohesiveShearFractureEnergy1,
  SimDamageBondCohesiveShearFractureEnergy2
}
The columns of the table that determines the damage evolution behavior.
The columns in the damage evolution table will be enabled based on the following conditions:
For displacement as evolution type:
  1. SimDamageBondCohesiveDisplacement
    • Enabled when mixed type behavior is specified and softening type is linear Quantity:LENGTH , Units:mm .
    • Enabled when mixed type behavior is specified and softening type is exponential Quantity:LENGTH , Units:mm .
    • Enabled when mixed type behavior is not specified and softening type is linear Quantity:LENGTH , Units:mm .
    • Enabled when mixed type behavior is not specified and softening type is exponential Quantity:LENGTH , Units:mm .
    • Enabled when mixed type behavior is not specified and softening type is tabular Quantity:LENGTH , Units:mm .
  2. SimDamageBondCohesiveEffectiveDisplacement
    • Enabled when mixed type behavior is specified and softening type is tabular Quantity:DIMENSIONLESS , Units:None .
  3. SimDamageBondCohesiveModeMixRatio1
    • Enabled when mixed type behavior is specified and softening type is linear Quantity:DIMENSIONLESS , Units:None .
    • Enabled when mixed type behavior is specified and softening type is exponential Quantity:DIMENSIONLESS , Units:None .
    • Enabled when mixed type behavior is specified and softening type is tabular Quantity:DIMENSIONLESS , Units:None .
  4. SimDamageBondCohesiveModeMixRatio2
    • Enabled when mixed type behavior is specified and softening type is linear Quantity:DIMENSIONLESS , Units:None .
    • Enabled when mixed type behavior is specified and softening type is exponential Quantity:DIMENSIONLESS , Units:None .
    • Enabled when mixed type behavior is specified and softening type is tabular Quantity:DIMENSIONLESS , Units:None .
  5. SimDamageBondEvoTemperature (optional). Quantity: TEMPRTRE, Units: Kdeg
  6. SimDamageBondCohesiveExponentialParam
    • Enabled when mixed type behavior is specified and softening type is exponential Quantity:DIMENSIONLESS , Units:None .
    • Enabled when mixed type behavior is not specified and softening type is exponential Quantity:DIMENSIONLESS , Units:None .
  7. SimDamageBondCohesiveDamageVar
    • Enabled when mixed type behavior is specified and softening type is tabular Quantity:DIMENSIONLESS , Units:None .
    • Enabled when mixed type behavior is not specified and softening type is tabular Quantity:DIMENSIONLESS , Units:None .
