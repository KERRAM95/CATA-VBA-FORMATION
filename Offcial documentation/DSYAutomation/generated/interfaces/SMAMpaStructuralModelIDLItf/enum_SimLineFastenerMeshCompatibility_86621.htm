SimLineFastenerMeshCompatibility (Enumeration)

enum SimLineFastenerMeshCompatibility {
  SimLineFastenerCompatible,
  SimLineFastenerNonCompatible
}
The mesh compatibility.
Values:
SimLineFastenerCompatible
The fastener placements must have been already projected on the mesh parts associated with the selected parts and the mesh parts will have nodes present exactly at the projected locations. Corner nodes of the shell elements will be these nodes at projected location.
Non_Compatible
The fastener placement will be projected on the selected supports and corner nodes of the shell elements will be created at the projected locations.
