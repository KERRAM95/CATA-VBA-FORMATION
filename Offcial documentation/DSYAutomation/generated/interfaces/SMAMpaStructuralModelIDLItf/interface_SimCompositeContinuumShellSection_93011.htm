SimCompositeContinuumShellSection (Object)

Represents the Composite Continnum Shell Section object.
Example:
Given a SimProperties object, you can create a SimCompositeContinuumShellSection as following: 
 Dim MyProperties As SimProperties
 ...
 Dim MyCompositeContinuumShellSection As SimCompositeContinuumShellSection
 Set MyCompositeContinuumShellSection = MyProperties.Add("SimCompositeContinuumShellSection")
Given a SimProperties object, you can retrieve a SimCompositeContinuumShellSection named "Composite Continnum Shell Section.1" as following: 
 Dim MyProperties As SimProperties
 ...
 Dim MyCompositeContinuumShellSection As SimCompositeContinuumShellSection
 Set MyCompositeContinuumShellSection = MyProperties.Item("Composite Continnum Shell Section.1")
Example in Python:
Given a SimProperties object myProperties, you can create a SimCompositeContinuumShellSection as following: 
 ...
 MyCompositeContinuumShellSection = myProperties.Add("SimCompositeContinuumShellSection")
Given a SimProperties object myProperties, you can retrieve a SimCompositeContinuumShellSection named "Composite Continnum Shell Section.1" as following:
