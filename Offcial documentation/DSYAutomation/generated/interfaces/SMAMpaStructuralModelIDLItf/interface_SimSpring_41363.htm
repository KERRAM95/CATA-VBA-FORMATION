SimSpring (Object)

Represents the Spring object.
Example:
Given a SimMCXProperties object, you can create a SimSpring as following: 
 Dim MyMCXProperties As SimMCXProperties
 ...
 Dim MySpring As SimSpring
 Set MySpring = MyMCXProperties.Add("SimSpring")
Given a SimMCXProperties object, you can retrieve a SimSpring named "Spring.1" as following: 
 Dim MyMCXProperties As SimMCXProperties
 ...
 Dim MySpring As SimSpring
 Set MySpring = MyMCXProperties.Item("Spring.1")
Example in Python:
Given a SimMCXProperties object myMCXProperties, you can create a SimSpring as following: 
 ...
 mySpring = myMCXProperties.Add("SimSpring")
Given a SimMCXProperties object myMCXProperties, you can retrieve a SimSpring named "Spring.1" as following: 
 ...
 mySpring = myMCXProperties.Item("Spring.1")
