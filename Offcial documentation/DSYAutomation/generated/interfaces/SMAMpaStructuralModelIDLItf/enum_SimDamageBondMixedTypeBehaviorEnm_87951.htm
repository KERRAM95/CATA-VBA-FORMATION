SimDamageBondMixedTypeBehaviorEnm (Enumeration)

enum SimDamageBondMixedTypeBehaviorEnm {
  SimDamageBondTABULAR2,
  SimDamageBondPOWER_LAW,
  SimDamageBondBENZEGGAGH_KENANE
}
The mixed mode behavior type.
Values:
SimDamageBondTABULAR2
Specify the fracture energy or displacement (total or plastic) directly as a function of the shear-normal mode mix for cohesive elements. This method must be used to specify the mixed-mode behavior for cohesive elements when TYPE=DISPLACEMENT. This option can be used with any choice for the MODE MIX RATIO parameter.
SimDamageBondPOWER_LAW
Specify the fracture energy as a function of the mode mix by means of a power law mixed mode fracture criterion. This option must be used in conjunction with either the MODE MIX RATIO=ACCUMULATED ENERGY option or the MODE MIX RATIO=ENERGY option.
SimDamageBondBENZEGGAGH_KENANE
Specify the fracture energy as a function of the mode mix by means of the Benzeggagh-Kenane mixed mode fracture criterion. This option must be used in conjunction with either the MODE MIX RATIO=ACCUMULATED ENERGY option or the MODE MIX RATIO=ENERGY option.
