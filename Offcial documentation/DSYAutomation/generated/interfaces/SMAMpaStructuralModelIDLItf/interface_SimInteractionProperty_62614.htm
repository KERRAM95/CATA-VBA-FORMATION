SimInteractionProperty (Object)

Represents Interaction Property in Contact.
Example: 
 This example shows how to create the Interactions Property:
 
 Dim MyBehaviors As SimBehaviors
 Set MyBehaviors = MyFEMRoot.GetSet("SimBehaviors")
 ...
 Dim MyInteractionproperty As SimInteractionProperty
 Set MyInteractionproperty = MyBehaviors.Add("SimInteractionProperty")
