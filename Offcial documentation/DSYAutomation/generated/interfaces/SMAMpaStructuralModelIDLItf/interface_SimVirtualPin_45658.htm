SimVirtualPin (Object)

Represents the Virtual Pin object.
Example:
Given a SimMCXProperties object, you can create a SimVirtualPin as following: 
 Dim MyMCXProperties As SimMCXProperties
 ...
 Dim MyVirtualPin As SimVirtualPin
 Set MyVirtualPin = MyMCXProperties.Add("SimVirtualPin")
Given a SimMCXProperties object, you can retrieve a SimVirtualPin named "Virtual Pin.1" as following: 
 Dim MyMCXProperties As SimMCXProperties
 ...
 Dim MyVirtualPin As SimVirtualPin
 Set MyVirtualPin = MyMCXProperties.Item("Virtual Pin.1")
Example in Python:
Given a SimMCXProperties object myMCXProperties, you can create a SimVirtualPin as following: 
 ...
 myVirtualPin = myMCXProperties.Add("SimVirtualPin")
Given a SimMCXProperties object myMCXProperties, you can retrieve a SimVirtualPin named "Virtual Pin.1" as following: 
 ...
 myVirtualPin = myMCXProperties.Item("Virtual Pin.1")
