SimFunctionOrder (Enumeration)

enum SimFunctionOrder {
  SimSecond,
  SimThird,
  SimFifth
}
The kernel method type for element conversion.
Values:
Quadratic
Element conversion based on Kernel method = Second.
Cubic
Element conversion based on Kernel method = Third.
Quintic
Element conversion based on Kernel method = Fifth.
