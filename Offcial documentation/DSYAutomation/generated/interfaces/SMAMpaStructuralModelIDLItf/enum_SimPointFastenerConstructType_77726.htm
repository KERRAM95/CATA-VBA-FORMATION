SimPointFastenerConstructType (Enumeration)

enum SimPointFastenerConstructType {
  SimPointFastenerRigid,
  SimPointFastenerSpring,
  SimPointFastenerSolidHex,
  SimPointFastenerBeam,
  SimPointFastenerCoupling,
  SimPointFastenerAssembled,
  SimPointFastenerSolidNugget
}
The type of construct.
Values:
SimPointFastenerRigid
The fastener will be defined as couplings and a rigid connector.
SimPointFastenerSpring
The fastener will be defined as two rigid beam connectors between surfaces. The rigid beam connectors are joined by a zero-length spring.
SimPointFastenerSolidHex
The fastener will be defined as couplings and a hexahedral element.
SimPointFastenerBeam
The fastener will be defined as couplings and a beam element.
SimPointFastenerCoupling
The fastener will be defined as couplings.
SimPointFastenerAssembled
The fastener will be defined as couplings and multiple connectors.
SimPointFastenerSolidNugget
The fastener will be defined as couplings and multiple hexahedral elements.
