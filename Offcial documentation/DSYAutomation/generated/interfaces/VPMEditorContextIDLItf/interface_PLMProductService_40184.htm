PLMProductService (Object)

Interface representing service to manage VPM Editor content.

object can be retrieved from object thanks to the method with the string "PLMProductService" as input.
The get_EditedContent method implementation returns the root Product of the editor.

Property Index
EditedContent Returns Root entities of active Editor, first of which is the root Product.
RootOccurrence Returns the Root Product Occurrence being edited in current VPM Editor.
Method Index
ComposeLink Creates a Link memory object to be used later for persistent Link.

Properties


Property EditedContent() As (Read Only)
Returns Root entities of active Editor, first of which is the root Product.
Property RootOccurrence() As (Read Only)
Returns the Root Product Occurrence being edited in current VPM Editor.

Methods


Func ComposeLink( iPLMOccurrence, iVPMRepInstance, iCATIAReference) As
Creates a Link memory object to be used later for persistent Link.
Parameters:
iPLMOccurrence
The path to reach the target. It is optional, and must be a CATIAVPMOccurrence ou CATIAVPMRootOccurrence.
iVPMRepInstance
The Product Representation Reference containing the target. It is optional.
iCATIAReference
The target object. It is optional.
Returns:
The CATIABase to be used to create a link.
