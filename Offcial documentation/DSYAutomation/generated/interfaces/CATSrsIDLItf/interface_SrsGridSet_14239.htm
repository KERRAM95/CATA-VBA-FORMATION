SrsGridSet (Object)

Role: Allows accessing of Grid Set's data.
Method Index
GetDirection Gets direction of the grid set.
GetExtremeFaces Gets the extreme faces in the grid set.
GetFace Gets the grid face at the given offset in the grid set.
GetFaces Returns all the faces in the grid set.
GetFacesOfGroup Returns the faces belonging to a particular group in the grid set.
GetFacesSorted Gets all the faces in the grid set.
GetGroupFaces Gets group faces before and after the origin face in the grid set.
GetNamingPrefix Returns the prefix of face's name.
GetNbFaces Returns number of faces present in the grid set.
GetNbFacesSorted Gets number of faces present before and after the origin face in the grid set.
GetOrientation Returns orientation of the Grid Set.
IsSymmetric Returns whether the grid set is symmetric.

Methods


Sub GetDirection( oDirection)
Gets direction of the grid set.
Parameters:
oDirection
Direction of grid set.
Example: 
 

  This example retrieves the direction of the grid set.
  
   Dim GridSetDirection(2) as Variant
   Set ObjHybridShapeDirection = ObjSrsGridSet.GetDirection GridSetDirection
Sub GetExtremeFaces( oGridFaceBfr, oGridFaceAfr)
Gets the extreme faces in the grid set.
Parameters:
oGridFaceBfr
Extreme Grid face present before the origin face in the grid set.
oGridFaceAfr
Extreme Grid face present after the origin face in the grid set.
Example: 
 

  This example retrieves the extreme grid faces in the grid set.
  
   Dim ObjSrsGridFaceBfr As SrsGridFace
   Dim ObjSrsGridFaceAfr As SrsGridFace
   ObjSrsGridSet.GetExtremeFaces ObjSrsGridFaceBfr ObjSrsGridFaceBfr
Sub GetFace( iAbsOffset, oGridFace)

Parameters:
