ManufacturingFastenerOffset (Object)

Interface defining fastener offsets.
Method Index
GetOffsetValues Gets the offset value of a given index.
SetOffsetParam Sets the mapping parameter of a given index.
SetOffsetValue Sets the offset value of a given index.

Methods


Sub GetOffsetValues( iIndex, oValue, oParamName, oParamSign)
Gets the offset value of a given index. (from 1 to 6 x.y.z.rx.ry.rz).
Parameters:
iIndex
The index from 1 to 6: x.y.z.rx.ry.rz
oValue
The returned value expressed in meters or radians
oParamName
The mapping parameter
oParamSign
The mapping sign (-1 or +1)
Sub SetOffsetParam( iIndex, iParamName, iParamSign)
Sets the mapping parameter of a given index.
Parameters:
iIndex
The index from 1 to 6: x.y.z.rx.ry.rz
iParamName
The mapping parameter name of manufacturing fasteners
iParamSign
The mapping sign (-1 or +1)
Sub SetOffsetValue( iIndex, iValue)
Sets the offset value of a given index.
Parameters:
iIndex
The index from 1 to 6: x.y.z.rx.ry.rz
iValue
The value expressed in meters or radians
