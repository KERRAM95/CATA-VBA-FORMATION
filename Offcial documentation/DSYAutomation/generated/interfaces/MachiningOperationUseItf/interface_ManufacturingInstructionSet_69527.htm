ManufacturingInstructionSet (Object)

Properties


Property InsertMode() As
Returns or sets insert mode of local instruction set.
  • 0: Replace
  • 1: Insert After
  • 2: Insert Before
Property Usage() As
Returns or sets scope of usage.
  • 0: Any
  • 1: Local
  • 2: Global
