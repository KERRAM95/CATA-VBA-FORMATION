SurfaceTrajectoryFactory (Object)

Interface representing a Surface Trajectory Factory to create, destroy, and retrieve Surface Trajectories.

Role: This interface is used to create, destroy, and retrieve Surface Trajectories..
Method Index
CreateSurfaceTrajectory This method creates a SurfaceTrajectory
DestroySurfaceTrajectory This method destroys a SurfaceTrajectory
GetAllSurfaceTrajectories This method retrieves all Surface trajectories under the organizational resource.

Methods


Func CreateSurfaceTrajectory( iTrajectoryType, iName, iSurfList, iProdList) As
This method creates a SurfaceTrajectory
Parameters:
iTrajectoryType
Input the trajectory type. The type can be either Paint, Sealant, General.
iName,
Name of the trajectory. The default value is "". This means the default trajectory name would be, "Trajectory".. If a name is provided then the tajectory name would be, ..
iSurfList
List of Surfaces to create trajectory.
iProdList
List of Product Occurrences.
Returns:
S_OK on success and E_FAIL on failure.
Sub DestroySurfaceTrajectory( ospSurfaceTraj)
This method destroys a SurfaceTrajectory
Parameters:
ospSurfaceTraj
SurfaceTrajectory to be deleted.
Returns:
An HRESULT.
Legal values:
S_OK
The TagGroup is successfully deleted
E_FAIL
The TagGroup was not deleted succesfully
Sub GetAllSurfaceTrajectories( iSearchChildren, oTrajList)
This method retrieves all Surface trajectories under the organizational resource.
Parameters:
iSearchChildren
Search children for trajectories recursively
oTrajList
List of Surface Trajectories.
Returns:
An HRESULT.
Legal values:
S_OK
The List of Trajectories is successfully returned
E_FAIL
The List of Trajectories could not be retrieved
