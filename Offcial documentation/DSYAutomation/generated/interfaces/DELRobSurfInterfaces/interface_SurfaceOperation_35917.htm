SurfaceOperation (Object)

Interface to access and manipulate the Surface Operation.
Role: This interface provides methods to set/retrieve data related to Surface Operation.
Property Index
Type Gets the type of the Surface Operation
Method Index
GetSurfaceProfile Gets the Surface profile for the Surface operation
GetTarget Gets the Tag Target corresponding to the Surface Operation
SetSurfaceProfile Sets the Surface profile for the Surface operation
SetTarget Sets the Target for the Surface Operation

Properties


Property Type() As (Read Only)
Gets the type of the Surface Operation
Parameters:
oType,
gives the operation type. One of the types listed in DELApprovedOperationTypes.txt + "Weld"
Returns:
S_OK on success and E_FAIL on failure.

Methods


Sub GetSurfaceProfile( oSurfaceProfile)
Gets the Surface profile for the Surface operation
Parameters:
oSurfaceProfile,
SurfaceProfile for the current SurfaceOperation.Applicable only for Servo Gun Surface Operation.
Returns:
S_OK on success and E_FAIL on failure.
Sub GetTarget( opTarget, opTargetOwner)
Gets the Tag Target corresponding to the Surface Operation
Parameters:
opTagTarget,
Tag Target currently set for the Surface Operation
opTargetOwner,
Occurrence owning the Tag Target
Returns:
S_OK on success and E_FAIL on failure.
Sub SetSurfaceProfile( iSurfaceProfile)
Sets the Surface profile for the Surface operation
Parameters:
iSurfaceProfile,
SurfaceProfile to be set.Applicable only for Servo Gun Surface Operation.
Returns:
S_OK on success and E_FAIL on failure.
Sub SetTarget( ipTarget, opTargetOwner)
Sets the Target for the Surface Operation
Parameters:
ipTagTarget,
Tag Target to be set
pTargetOwner,
Occurrence owning the Tag Target
Returns:
S_OK on success and E_FAIL on failure.
