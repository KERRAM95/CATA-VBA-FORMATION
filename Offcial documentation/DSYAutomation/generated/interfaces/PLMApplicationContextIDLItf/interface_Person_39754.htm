Person (Object)

Represents the current connected Person.
Role: Allows getting current session information about the current connected Person.
Example: 
 This example shows how to retrieve the current connected Person from the PnOService service.
 
 Dim zePnOService as PnOService
 Set zePnOService = CATIA.GetSessionService("PnOService") 
 Dim zePerson as Person
 Set zePerson = zePnOService.Person
