API's to create the various frame selection attributes

Frames Selection Interface

interface SMAIAMpaFramesSelection : CATIABase


Following are the Frame Selection API's:

The complete script of Frame selection creation and usage can be found in the CAAScdFeaResultsSensorSource sample script


1) PROPERTY StepList

put_StepList ( in CATSafeArrayVariant iStepsList )

Sets the list of Step D's to the frame selector. It is a list of strings which are the steps ID's. The step ID's can be retrieved from SMAIAMpaResultsStep::GetIdentifier()

@param iStepsList: It is the list of Step ID's

Sample code: The following example shows how to retrieve the step ID from results step 

Dim stepId As String 
Dim stepName As String
Dim stepDescription As String
oResultsStep.GetIdentifier stepId, stepName, stepDescription


2) PROPERTY SymbolicFrameRange

put_SymbolicFrameRange ( in CATVariant iSymbolicFrameRange )

The following values are available for iSymbolicFrameRange. 

  a) SimUndefined                    
  b) SimAllFrames            
  c) SimLastFrameOfEachStep  
  d) SimAllFramesInStep      
  e) SimTimeBasedFrames      
  f) SimFrequencyFrames

3) SetLoadCases( in CATSafeArrayVariant iNbLoadCasesInStep, in CATSafeArrayVariant iLoadCaseIDList )

Sets the load cases to the frame selector. This method must be called if anyone has specified a list of steps (given through StepList() method). No need to call this if there are no load cases in the steps. Suppose if there are two steps and only one out of them has load cases, then for the steps without load cases the user will have to specify the values as zero and the load case ID as NULL string.

@param iNbLoadCasesInStep: It is the number of load cases in the step. The size of list must be same as the steps specified in the StepList() method. Also it must be in the same order as that of the steps.

@param iLoadCaseIDList: List of load case ID's to set. The ID's can be retrieved through SMAIAMpaResultsManager::GetLoadCaseIdentifier() method.


4) SetFrames(in CATSafeArrayVariant iNbFrames, in CATSafeArrayVariant iFrameIndex)

Sets the frames to the frame selector. The array size must be same as that of steps/load cases.

@param iNbFrames: The number of frames in step/load case to set.

@param iFrameIndex: The frame index to set.
