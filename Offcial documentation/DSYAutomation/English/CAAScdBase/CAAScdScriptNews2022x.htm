What's New with 3DEXPERIENCE R2022x Automation
Technical Article

Abstract

This article presents Automation objects, methods, and properties that are added since 3DEXPERIENCE R2022x GA to enhance V6 Automation capabilities, support functional evolutions, and correct defects. Some of those modifications may already exist in 3DEXPERIENCE R2021x maintenance fixes. A few properties and methods might have been modified or removed. VB and VBScripts code using those methods may have to be modified. VB projects using the corresponding objects must be recompiled.

Consult About Microsoft Automation Languages, Debug, and Compatibility for details on manual adjustments that may be required for existing VBA and VSTA projects to be usable on a new release.

Modified Objects, Methods and Properties
The following objects have been moved into new type libraries. This has no impact on vbscript/CATScript macros and should have no impact on VBA project as well, however, in case of problem, apply the procedure described in the Inter Release Compatibility paragrah:
  • Objects moved from SimRepresentationTypeLib (CATSimRepIDLTypeLib.tlb) into SimFmtModelTypeLib (CATFmtModelIDLTypeLib.tlb): CATIAFmtAbstractions, CATIAFmtBehaviors, CATIAFmtFEMFeatureInitialization, CATIAFmtFemRoot, CATIAFmtGroup, CATIAFmtGroups, CATIAFmtMCXProperties, CATIAFmtMeshPart, CATIAFmtMeshParts, CATIAFmtMeshSet, CATIAFmtMeshSpecification, CATIAFmtMeshSpecifications, CATIAFmtNumbering, CATIAFmtProperties, CATIAFmtSeamWeld, CATIAFmtSelectedElement, CATIAFmtSelectedNode, CATIAFmtTopologySpecification, CATIAFmtTopologySpecifications, CATIAFmtVisualization
  • Objects moved from SimFoundationTypeLib (SMAMpaFoundationIDLItfTypeLib.tlb) into SMAMpaStructuralModelIDLItf (SMAMpaStructuralModelIDLItfTypeLib.tlb): SMAIAMpaConnectorDampings, SMAIAMpaConnectorElasticities
  • Objects moved from SimFoundationTypeLib (SMAMpaFoundationIDLItfTypeLib.tlb) into SimBaseTypeLib (SMAMpaBaseIDLItf.tlb): SMAIAMpaMathAxis, SMAIAMpaMathPlane, SMAIAMpaMathPoint, SMAIAMpaMathVector, SMAIAMpaAnalyticRigidSurface, SMAIAMpaAxis, SMAIAMpaAxisSystem
  • Objects moved from SimStructuralTypeLib (SMAMpaStructuralIDLItfTypeLib.tlb) into SMAMpaStructuralModelIDLItf(SMAMpaStructuralModelIDLItfTypeLib.tlb): SMAIAMpaBeamProfile, SMAIAMpaBeamSection, SMAIAMpaCompositeShellSection, SMAIAMpaCompositeSupportDefinition, SMAIAMpaConnectionServices, SMAIAMpaConnectorBehavior, SMAIAMpaConnectorElasticity, SMAIAMpaConnector, SMAIAMpaConnectorSection, SMAIAMpaConnectorTranslation, SMAIAMpaContinuumShellSection, SMAIAMpaCoupling, SMAIAMpaCyclicSymmetry
None.
