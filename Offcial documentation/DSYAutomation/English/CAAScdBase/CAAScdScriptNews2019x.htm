What's New with 3DEXPERIENCE R2019x Automation
Technical Article

Abstract

This article presents Automation objects, methods, and properties that are added since 3DEXPERIENCE R2019x GA to enhance V6 Automation capabilities, support functional evolutions, and correct defects. Some of those modifications may already exist in 3DEXPERIENCE R2018x maintenance fixes. A few properties and methods might have been modified or removed. VB and VBScripts code using those methods may have to be modified. VB projects using the corresponding objects must be recompiled.

Consult About Microsoft Automation Languages, Debug, and Compatibility for details on manual adjustments that may be required for existing VBA and VSTA projects to be usable on a new release.
