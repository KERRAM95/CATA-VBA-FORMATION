What's New with 3DEXPERIENCE R2024x Automation
Technical Article

Abstract

This article presents Automation objects, methods, and properties that are added since 3DEXPERIENCE R2024x GA to enhance V6 Automation capabilities, support functional evolutions, and correct defects. Some of those modifications may already exist in 3DEXPERIENCE R2023x maintenance fixes. A few properties and methods might have been modified or removed. VB and VBScripts code using those methods may have to be modified. VB projects using the corresponding objects must be recompiled.

Consult About Microsoft Automation Languages, Debug, and Compatibility for details on manual adjustments that may be required for existing VBA and VSTA projects to be usable on a new release.

Modified Objects, Methods and Properties
The following object has been removed into new type libraries. This has no impact on vbscript/CATScript macros and should have no impact on VBA project as well, however, in case of problem, apply the procedure described in the Inter Release Compatibility paragrah:
  • SimInitializationService, moved from SMAMpaCAStructuralIDLItfTypeLib (SMAMpaCAStructuralIDLItfTypeLib.tlb.tlb) into SMAMpaFoundationIDLItfTypeLib (SMAMpaFoundationIDLItfTypeLib.tlb).
  • SimSpaceTimeField and SimMassScaling objects have been moved from SMAMpaStructuralIDLItf type library into SMAMpaBaseIDLItf type library.
  • SMAIAMpaMassScaling object has been moved from SMAMpaStructuralModelIDLItf type library into SMAMpaStructuralIDLItfTypeLib type library.
  • Properties SimCompositeShellSection::NonStructuralPlyAngle, SimCompositeShellSection::NonStructuralPlyFlag, SimCompositeShellSection::NonStructuralPlyMaterial,
  • Properties SimShellSectionOptions::InfiniteMaxThicknessIntervalFlag, SimShellSectionOptions::ThicknessGroupingFlag and SimShellSectionOptions::ThicknessGroupingValue, deprecated since 3DEXPERIENCE R2020x, have been replaced by properties IntervalFlag, SplitByThicknessOffsetFlag and Interval. for example: NonstructuralPlyFlag. Macros written in VBA or VBScript languages are not impacted.
  • Property DELMIAOlpExpressionFixerDownload::FixSubscriptedVariables has been moved into base class DELMIAOlpExpressionFixer with not impact on existing macros.
  • Signature of method ManufacturingPartOperationPosition::GetTopPlaneDirectionInversion was non functional and has been fixed.
Removed Objects, Collections, Methods, and Properties
SimSpecsRepManager2 object has been removed, its methods have been moved into SimSpecsRepManager object.
