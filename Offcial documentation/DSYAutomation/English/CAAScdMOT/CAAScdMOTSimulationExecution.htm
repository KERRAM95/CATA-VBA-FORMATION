Executing a Motion simulation
 
Before you begin: Note that:
  • You should first launch CATIA and import the
    JackSimulation.3dxml
     file supplied in folder InstallRootFolder\CAADoc\Doc\CAAScdMOT\samples\ where InstallRootFolder is the folder where the CAA CD-ROM is installed.

Where to find the scripts: SimulationExecutionScripts

This use case can be divided in nine steps:

  1. Retrieve the current Editor
  2. Retrieve the Product Service from the Editor
  3. Retrieve the Entities from the Product Service
  4. Retrieve the simulation object
  5. Retrieve the Motion Execution Services from the Editor
  6. Run one analysis case
  7. Retrieve execution information
  8. Run all the analysis cases
  9. Retrieve execution information

  1. Retrieve the current Editor

    VBA Python   
    ... 
Dim myEditor As Editor
Set myEditor = CATIA.ActiveEditor
...
    ...
import sys, os
from com3dx import *
CATIA = get3dxClient()

myEditor = CATIA.ActiveEditor
...

  2. Retrieve the Product Service from the Editor

    VBA Python   
    ... 
Dim myPrdService As PLMProductService
Set myPrdService = myEditor.GetService("PLMProductService")
...
    ...
myPrdService = myEditor.GetService("PLMProductService")
...

  3. Retrieve the Entities from the Product Service

    VBA Python   
    ...
Dim myEntities As PLMEntities
Set myEntities = myPrdService.EditedContent
...
    ...
myEntities = myPrdService.EditedContent
...

  4. Retrieve the simulation object

    VBA Python   
    ...					
Dim mySimulationObject As SimulationReference
Set mySimulationObject = myEntities.Item(1)
...
    ...
mySimulationObject = myEntities.Item(1)
...

  5. Retrieve the Motion Execution Services from the Editor

    VBA Python   
    ...					
Dim myExecutionServices As MOTExecutionServices
Set myExecutionServices = myEditor.GetService("MOTExecutionServices")
...
    ...
myExecutionServices = myEditor.GetService("MOTExecutionServices")
...

  6. Run one analysis case

    VBA Python   
    ...					
Dim inputAnalysisCase As String
inputAnalysisCase = "Motion Analysis Case.1"
Dim analysisCaseExecutionInfos As MOTAnalysisCaseExecutionInfos

Call myExecutionServices.RunAnalysisCase(mySimulationObject, inputAnalysisCase, analysisCaseExecutionInfos)
...
    ...
myAnalysisCaseExecutionInfos = myExecutionServices.RunAnalysisCase(mySimulationObject, "Motion Analysis Case.1")
...

  7. Retrieve execution information

    - The analysis case name

    - The execution status (1 = success, 2 = failure, 3 = analysis case not found, 4 = analysis case not allowed, 5 = simulation object not allowed)

    - The execution logs

    VBA Python   
    ...			
Dim analysisCaseName As String
Set analysisCaseName = analysisCaseExecutionInfos.AnalysisCaseName

MsgBox " Analysis Case name = " & analysisCaseName

Dim execStatus As MOTExecutionStatus
Set execStatus = analysisCaseExecutionInfos.ExecutionStatus

MsgBox " Execution status = " & execStatus

Dim executionLog As String
Set executionLog = analysisCaseExecutionInfos.ExecutionLog

MsgBox " Execution logs = " & executionLog
...
    ...
analysisCaseName = myAnalysisCaseExecutionInfos.AnalysisCaseName
print(analysisCaseName)

executionStatus = myAnalysisCaseExecutionInfos.ExecutionStatus
print(executionStatus)

executionLog = myAnalysisCaseExecutionInfos.ExecutionLog
print(executionLog)
...

  8. Run all the analysis cases

    VBA Python   
    ...	
Dim listOfAnalysisCaseExecutionInfos()
Call myExecutionServices.RunAllAnalysisCases(mySimulationObject, listOfAnalysisCaseExecutionInfos)
...
    ...
myAnalysisCaseExecutionInfosList = myExecutionServices.RunAllAnalysisCases(mySimulationObject)
...

  9. Retrieve execution information

    - The analysis case name

    - The execution status (1 = success, 2 = failure, 3 = analysis case not found, 4 = analysis case not allowed, 5 = simulation object not allowed)

    - The execution logs

    VBA Python   
    ...
For Each execInfos In listOfAnalysisCaseExecutionInfos
    Dim analysisCaseExecutionInfos As MOTAnalysisCaseExecutionInfos
    Set analysisCaseExecutionInfos = execInfos
	
    Dim analysisCaseName As String
    Set analysisCaseName = analysisCaseExecutionInfos.AnalysisCaseName

    MsgBox " Analysis Case name = " & analysisCaseName

    Dim execStatus As MOTExecutionStatus
    Set execStatus = analysisCaseExecutionInfos.ExecutionStatus

    MsgBox " Execution status = " & execStatus
	
    Dim executionLog As String
    Set executionLog = analysisCaseExecutionInfos.ExecutionLog

    MsgBox " Execution logs = " & executionLog
Next execInfos
...
    ...
for execInfos in myAnalysisCaseExecutionInfosList:

    analysisCaseName = execInfos.AnalysisCaseName
    print(analysisCaseName)
	
    executionStatus = execInfos.ExecutionStatus
    print(executionStatus)

    executionLog = execInfos.ExecutionLog
    print(executionLog)
