Managing Composite Features

This use case primarily focuses on the methodology to create/edit composite features.
Before you begin:
  • Launch CATIA and then import and open the
    CAAScdFeaSampleCompositeLeadingEdgeSlat.3dxml
     file supplied in the folder InstallRootFolder\CAADoc\Doc\English\CAAScdFeaModeling\samples\ where InstallRootFolder is the folder where the API CD-ROM is installed.
  • Note: Creation of new PLM objects via scripting, as in this use case, requires an enabling license in addition to the interactive roles needed to access the functionality.

Where to find the macro: CAAScdFeaCompositeSource.htm

This use case (UC) is divided into following steps:

  1. Setting up the Model
  2. Managing the Composite Parameters
  3. Adding materials to the Composite Parameters
  4. Managing Rosette in the Composite Parameters
  5. Add Orientation in the Composite Parameters
  6. Managing Laminates in the Composite Parameters
  7. Creation of Grid using reference geometries and Rosette
  8. Assigning Laminate to Grid Cells
  9. Renaming the Grid Cells
  10. Exporting the Grid to a file

  1. Setting up the Model

    For setting up the model, the UC searches and opens the materials to be used in the Composite Parameters, followed by opening the product. Refer to the automation documentation on "PLM Services" for searching and opening, "VPMPublications" for retrieving the publications and "SimMeshSet" for meshing the geometries.

  2. Managing the Composite Parameters

    As a first step, the UC creates/retrieves the Composite Parameters from the Composite Shell Section. Refer to the automation documentation on "SimCompositeShellSection" to create/retrieve the Composite Shell Section.

    Dim myCompShellSection As SimCompositeShellSection
... 
Dim myCompositeParameters As SimCompositeParameters
Set myCompositeParameters = myCompShellSection.GetCompositeParameters
...

  3. Adding materials to the Composite Parameters

    In this step, the UC retrieves the materials in session and adds them into Composite Parameters. Only the materials that are active in the material session can be added into composite parameters.

    ... 
Dim MatService As MATPLMService
Dim ListMaterialInSession As ListObject
MatService.GetMaterialsInSession ListMaterialInSession
Dim mat As Material
For i = 1 To ListMaterialInSession.Count
    Set mat = ListMaterialInSession.Item(i)
    myCompositeParameters.AddMaterial mat	
Next
...

  4. Managing Rosette in the Composite Parameters

    In this step, the UC creates a Rosette in the Composite Parameters.

    ...
Dim myCompositeRosette   As SimCompositeRosette
Set myCompositeRosette  = myCompositeParameters.CreateRosette
...

    UC retrieves the Axis System by name "Axis System.1" and assigns the myCompositeRosette with the Axis System.

    ...
Dim myActiveObject As Part
Set myActiveObject = Editor.ActiveObject
Dim myAxisSystem  As AxisSystem
Set myAxisSystem  = myActiveObject.FindObjectByName("Axis System.1")
myCompositeRosette.SetMainAxis myAxisSystem
...

    myCompositeRosette has "Cartesian" transfer type by default. To modify the transfer type to "Guided by Curve":

    ...
myCompositeRosette.SetRosetteTransferType SimCompositeRosetteTypeGuidedCurve
...

    To associate 1D geometry as a guide to the Rosette, 

    ...
Dim myRosetteCurve  As HybridShape
myRosetteCurve.SetCylindricalRosetteTranferCurve myRosetteCurve
...

    Angle of the Rosette by default is 0°(SimCompositeCylindericalZero). To modify the same to 90°:

    ...
myRosetteCurve.SetCylindricalType SimCompositeCylindericalNinety
...

  5. Add Orientation in the Composite Parameters

    The UC creates a new Orientation in the Composite Parameters. To create Orientation, Angle in Degrees and the RGB coefficient is required as input:

    ...
myCompositeParameters.CreateOrientation "Custom", 255,255,255
...

  6. Managing Laminates in the Composite Parameters

    To create Laminate under Composite Parameters, list of materials and the Orientation of the laying the material is needed. In this step, the UC will create a laminate with type Stacking Sequence.

    ...
Dim myMaterials
myMaterials = myCompositeParameters.GetMaterialList
...
Dim myLaminate As SimCompositeLaminate
Set myLaminate = myCompositeParameters.CreateLaminate("Lamina.1")
...
ReDim myMatDirection(UBound(myMaterials))
For i in UBound(myMaterials)
	myMatDirection(i) = 45
Next
myLaminate.SetStackingSequence myMaterials, myMatDirection
...

  7. Creation of Grid using reference geometries and Rosette

    In this step, the UC creates the Composite Grid. To create the Grid, a list of geometries (Geometrical Set) and the Rosette is required as input.

    ...
Dim myGeomRefSet As HybridShapes
...
ReDim myGeomRefSetForGrid(myGeomRefSet.Count)
For i = 0 To size - 1
        Set myGeomRefSetForGrid(i) = myGeomRefSet .Item(i + 1)
Next
Dim myCompositeGrid As SimCompositeGrid
Set myCompositeGrid  = myCompositeParameters.CreateGrid(myGeomRefSetForGrid, myCompositeRosette)
...

  8. Assigning Laminate to Grid Cells

    In this step, the UC assigns the laminate retrieved by name from the Composite Parameters to the Grid Cells.

    ...
Dim myCompositeLaminate As SimCompositeLaminate
Set myCompositeLaminate = myCompositeParameters.GetLaminateByName("Laminate.1")
Dim myCompositeGridCells
myCompositeGrid.GetCells
For Each myCompositeGridCell in myCompositeGridCells
	myCompositeGridCell.SetLaminateOnCell myCompositeLaminate
Next
...

  9. Renaming the Grid Cells

    In this step, the UC renames the Grid Cells in a Composite Grid.

    ...
For Each myCompositeGridCell in myCompositeGridCells
	myCompositeGridCell.SetCellName("CustomCellName")
Next
...

  10. Exporting the Grid to a file

    In this step, the UC exports the Composite Grid information to a file.

    ...
myCompositeGrid.CreateFileFromCells("C:\Export.xls")
...
