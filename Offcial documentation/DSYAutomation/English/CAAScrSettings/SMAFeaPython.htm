SMAFeaPython Setting Repository
Quick Reference

SMAFeaPython

Role: Settings for SIMULIA Python

Detailed role: Preferences to control the Python interpreter. Options usage of a default Python interpeter, or a configured Python interpreter with configuration options.

The SMAFeaPython repository has the following characteristics:

  • Primary Tab Page: /AnalysisSimulation/Python
  • Repository File: SMAFeaPython
  • Version: 2
  • Full Compatibility: 1
  • Upward Compatibility: 1

SMAFeaPython setting repository attributes:

  • Editor
    Role: Full pathname of the Python Editor executable
    Detailed role: Specify a string that defines the full pathname of the Python executable. An empty string represents using the default Python Editor.
    • Type: string
    • Default value:
  • Executable
    Role: Full pathname of the Python executable
    Detailed role: Specify a string that defines the full pathname of the Python executable. Cannot be empty if using configured Python.
    • Type: string
    • Default value:
  • Library
    Role: Python library paths separated by semicolons
    Detailed role: Specify a string that defines the library paths, separated by semicolons. Cannot be empty if using configured Python.
    • Type: string
    • Default value:
  • PythonChoice
    Role: Parameter to set choice of default or configured python
    • Type: boolean
    • Default value: true
  • PythonConfig
    Role: Parameter to set python configuration
    • Type: boolean
    • Default value: true
  • UseDefault
    Role: Choice of default or configured python
    Detailed role: Check to enable automatic usage of the default installed Python interpreter. Uncheck to use the configured Python interpreter.
    • Type: integer
    • Default value: 3
