ASDPrefGeneral Setting Repository
Quick Reference

ASDPrefGeneral

Role: Setting to handle General preferences for ASD Drawings

The ASDPrefGeneral repository has the following characteristics:

  • Primary Tab Page: /Multi Discipline Engineering/Multi-Discipline Drafting/General
  • Repository File: ASDPrefGeneral
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

ASDPrefGeneral setting repository attributes:

  • AnnotDiagnostic
    Role: Activate diagnostics for Smart Annotations
    Detailed role: Displays the various types of diagnostics in different and customizable colors.
    • Type: boolean
    • Default value: 1
  • AnnotNodeColor
    Role: Set the annotation node color.
    Detailed role: Used in Compass, View title, Fluidic and Coordinate Annotations.
    • Type: unsigned integer
    • Default value: 160
    • Min value: 0
    • Max value: 255
  • AnnotNodeSize
    Role: Set the annotation node size.
    Detailed role: Used in Compass, View title, Fluidic and Coordinate Annotations.
    • Type: double
    • Default value: 3.3
  • AnnotUpdateFailureColor
    Role: Set the color for Smart Annoations update failure
    Detailed role: Used to indicate the Smart Annotations whose update has failed.
    • Type: unsigned integer
    • Default value: 128
    • Min value: 0
    • Max value: 255
  • DefaultFarOffsetForBox
    Role: Set the default far side for structural box for structure CreateViewCmd.
    Detailed role: Default structural box is created when the user selects the plane and with panels that has the same support as view plane.
    Default near and far offset for the default structural box wrt to the view plane.
    • Type: double
    • Default value: 100
  • DefaultMasterAngularTol
    Role: Set the default primary angular tolerance for structure CreateViewCmd.
    Detailed role: Angular tolerance with the view plane to consider during computation of primary list. If structural box is defined then all intersected panels within the box are considered as primary else all panels that have the same support of view plane with angular tolerance is considered as primarys.
    • Type: double
    • Default value: 15.0
    • Min value: 0.0
    • Max value: 45.0
  • DefaultNearOffsetForBox
    Role: Set the default near side for structural box for structure CreateViewCmd.
    Detailed role: Default structural box is created when the user selects the plane and with panels that has the same support as view plane.
    Default near and far offset for the default structural box wrt to the view plane.
    • Type: double
    • Default value: 100
  • DefaultSectionPlaneOrientation
    Role: Set the default section plane orientation mode for alignment section view.
    Detailed role: Default mode for section plane orientation for alignment section view creation.
    • Type: integer
    • Default value: 0
  • DefaultSlaveAngularTol
    Role: Set the default secondary angular tolerance for structure CreateViewCmd.
    Detailed role: Secondary orthogonal angle with primary considered for representation of the secondary.
    • Type: double
    • Default value: 0.0
    • Min value: 0.0
    • Max value: 45.0
