3DShapeInfrastructureReview Setting Repository
Quick Reference

3DShapeInfrastructureReview

Role: Setting to manage options for 3D Shape Infrastructure review

The 3DShapeInfrastructureReview repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/3D Shape Infrastructure/3D Shape review
  • Repository File: 3DShapeInfrastructureReview
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

3DShapeInfrastructureReview setting repository attributes:

  • PublishAllChildrenSets
    Role: Publish all children sets
    • Type: boolean
    • Default value: false
  • PublishInsideThisSetOnly
    Role: Publish data inside this set only
    • Type: boolean
    • Default value: false
  • PublishInsideThisSetOnlyName
    Role: Name of the set to publish only
    • Type: string
    • Default value: Final
  • PublishPublicKnowledgeParameters
    Role: Publish public knowledge parameters
    • Type: boolean
    • Default value: false
  • PublishSetsAndGeometricalFeatures
    Role: Publish 3D Shape sets and geometrical features
    • Type: boolean
    • Default value: false
