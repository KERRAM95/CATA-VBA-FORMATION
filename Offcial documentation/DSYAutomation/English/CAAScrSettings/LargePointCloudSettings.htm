LargePointCloudSettings

Role: Settings to manage Large Point Cloud Options

The LargePointCloudSettings repository has the following characteristics:

  • Primary Tab Page: /AEC Plant/Civil Engineering/Large Point Cloud
  • Repository File: LargePointCloudSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

LargePointCloudSettings setting repository attributes:

  • Location
    Role: Defines the location directory where all the data resulting from the conversion will be stored
    Detailed role: Defines the location directory where all the data resulting from the conversion will be stored
    • Type: URN
    • Default value: UNIX: ${CATTemp} Windows: ${CATTemp}
  • Storage
    Role: Defines storage
    Detailed role: Defines storage
    • Type: string
    • Default value: File
    • Values:
      File The storage is done on file
      Database The storage is done on database
