CATPpxOptionData_Variant Setting Repository
Quick Reference

CATPpxOptionData_Variant

Role: Setting to handle PPX parameters

The CATPpxOptionData_Variant repository has the following characteristics:

  • Primary Tab Page: /Systems/Elements
  • Repository File: CATPpxSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CATPpxOptionData_Variant setting repository attributes:

  • Environment
    Role: Parameter to specify the type of the Environment Variant thumbnail
    • Type: string
    • Default value: Capture all 3D + environment
    • Values:
      Capture all 3D + environment The whole 3D view is shown + Environment
      Capture environment only Just the environment is shown
  • Human
    Role: Parameter to specify the content of the Human Variant thumbnail
    • Type: string
    • Default value: Capture all 3D
    • Values:
      Capture all 3D The whole 3D view is selected
      Capture selected Only the selected objects are included
      Capture selected (no reframe) Only the selected objects are included, but no reframe-on
  • HumanEnvironment
    Role: Parameter to specify whether the current environment is included in the Human Variant thumbnail
    • Type: boolean
    • Default value: false
  • Material
    Role: Parameter to specify the content of the Material Variant thumbnail
    • Type: string
    • Default value: Capture all 3D
    • Values:
      Capture all 3D The whole 3D view is selected
      Capture selected Only the selected objects are included
      Capture selected (no reframe) Only the selected objects are included, but no reframe-on
  • MaterialEnvironment
    Role: Parameter to specify whether the current environment is included in the Material Variant thumbnail
    • Type: boolean
    • Default value: false
  • MaterialVariantActivation
    Role: Parameter to control material variant activation
    • Type: boolean
    • Default value: true
  • Position
    Role: Parameter to specify the content of the Position Variant thumbnail
    • Type: string
    • Default value: Capture all 3D
    • Values:
      Capture all 3D The whole 3D view is selected
      Capture selected Only the selected objects are included
      Capture selected (no reframe) Only the selected objects are included, but no reframe-on
  • PositionEnvironment
    Role: Parameter to specify whether the current environment is included in the Position Variant thumbnail
    • Type: boolean
    • Default value: false
  • Product
    Role: Parameter to specify the content of the Product Variant thumbnail
    • Type: string
    • Default value: Capture all 3D
    • Values:
      Capture all 3D The whole 3D view is selected
      Capture selected Only the selected objects are included
      Capture selected (no reframe) Only the selected objects are included, but no reframe-on
  • ProductEnvironment
    Role: Parameter to specify whether the current environment is included in the Product Variant thumbnail
    • Type: boolean
    • Default value: false
  • Simulation
    Role: Parameter to specify the content of the Simulation Variant thumbnail
    • Type: string
    • Default value: Capture all 3D
    • Values:
      Capture all 3D The whole 3D view is selected
      Capture selected Only the selected objects are included
      Capture selected (no reframe) Only the selected objects are included, but no reframe-on
  • SimulationEnvironment
    Role: Parameter to specify whether the current environment is included in the Simulation Variant thumbnail
    • Type: boolean
    • Default value: false
  • StageEnvironment
    Role: Parameter to specify whether the current environment is included in the Stage Variant thumbnail
    • Type: boolean
    • Default value: false
  • ViewpointBloomEffectState
    Role: Parameter to enable the bloom effect
    • Type: boolean
    • Default value: true
  • ViewpointBloomEffectValueF
    Role: Intensity value of the bloom effect
    • Type: float
    • Default value: 0.22
    • Min value: 0
    • Max value: 500
  • ViewpointEnvironment
    Role: Parameter to specify whether the current environment is included in the Viewpoint Variant thumbnail
    • Type: boolean
    • Default value: false
  • ViewpointShadingModes
    Role: Parameter to restore Vieaw shading modes when activated
    • Type: boolean
    • Default value: true
