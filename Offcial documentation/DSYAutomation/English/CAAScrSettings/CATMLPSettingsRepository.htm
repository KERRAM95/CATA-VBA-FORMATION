CATMLPSettingsRepository Setting Repository
Quick Reference

CATMLPSettingsRepository

Role: Layered Product settings

Detailed role: Layered Product settings

The CATMLPSettingsRepository repository has the following characteristics:

  • Primary Tab Page: /Mechanical Design/Layered Product Design
  • Repository File: CATMLPSettingsRepository
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CATMLPSettingsRepository setting repository attributes:

  • MLPAnnotationDataGeneration
    Role: Activate/Deactivate annotation data generation for 3D Annotations web app
    • Type: integer
    • Default value: 0
  • MLPBindingViewThickness
    Role: Determine the default minimum thickness of the layers of the binding view
    • Type: double
    • Default value: 1.0
    • Min value: 0.0
    • Max value: 1000000
  • MLPCoreLayerRatioBuildErrorGeneration
    Role: Activate/Deactivate the generation of a build error if the overall ratio percentage is not equal to 100%
    • Type: integer
    • Default value: 1
  • MLPRawMaterialAutoSync
    Role: Automatically synchronise raw material definition
    • Type: integer
    • Default value: 0
  • MLPSketchLaunchBehavior
    Role: Determines what happens when launching a sketch through Layer Product commands
    • Type: integer
    • Default value: 0
