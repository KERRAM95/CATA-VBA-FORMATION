DELMfgFastenerSettings Setting Repository
Quick Reference

DELMfgFastenerSettings

Role: Settings to handle the parameters of Manufacturing Fastener

Detailed role: This setting allows to customize the appearance and some behavior of Manufacturing Fastener

The DELMfgFastenerSettings repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/DELMIA Infrastructure/Manufacturing Fastener
  • Repository File: DELMfgFastenerSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DELMfgFastenerSettings setting repository attributes:

  • MFTNameDisplay
    Role: Display name of manufacturing fastener
    Detailed role: Display manufacturing fastener name
    • Type: unsigned integer
    • Default value: 0
  • MfgFastenerUserParam
    Role: Visbility of the User Param
    Detailed role: Enable User param
    • Type: unsigned integer
    • Default value: 0
  • MfgFastenerZAxis
    Role: Visbility of the Manufacturing Fastener
    Detailed role: Whether the Manufacturing Fastener is reversed in Z direction
    • Type: unsigned integer
    • Default value: 0
