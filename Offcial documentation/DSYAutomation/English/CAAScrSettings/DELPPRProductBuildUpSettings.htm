DELPPRProductBuildUpSettings Setting Repository
Quick Reference

DELPPRProductBuildUpSettings

Role: Settings to handle the parameters of Product BuildUp

Detailed role: This setting allows to customize the appearance and the behavior of Product Build Up

The DELPPRProductBuildUpSettings repository has the following characteristics:

  • Primary Tab Page: /Digital Process for Manufacturing/System Product BuildUp
  • Repository File: DELPPRProductBuildUpSettings
  • Version: 6
  • Full Compatibility: 6
  • Upward Compatibility: 6
