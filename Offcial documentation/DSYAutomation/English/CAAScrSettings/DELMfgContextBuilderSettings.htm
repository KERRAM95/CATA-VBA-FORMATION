DELMfgContextBuilderSettings Setting Repository
Quick Reference

DELMfgContextBuilderSettings

Role: Setting to handle CAA model parameters

The DELMfgContextBuilderSettings repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/Elements
  • Repository File: DELMfgContextBuilderSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DELMfgContextBuilderSettings setting repository attributes:

  • DisableQuickFilteronExplore
    Role: Do not display Configuration-Evolution dialog box on Explore
    • Type: integer
    • Default value: 0
  • PredConfigRadioChecked
    Role: Parameter to manage Configuration-Evolution panel's radiobutton states
    • Type: integer
    • Default value: 0
  • SpreadsheetLayout
    Role: Parameter to set preference for Spreadsheet layout
    • Type: integer
    • Default value: 2
  • UDOCheckState
    Role: Parameter to manage UDO panel's checkbox states
    • Type: integer
    • Default value: 1
  • UDODefault
    Role: Display Default UDO procedures
    • Type: integer
    • Default value: 1
  • UDOInWork
    Role: Display In-Work procedures
    • Type: integer
    • Default value: 0
  • UDOSample
    Role: Display Sample UDO procedures
    • Type: integer
    • Default value: 0
  • UDRRevisionRadioChecked
    Role: Parameter to manage UDR panel's radiobutton states
    • Type: integer
    • Default value: 1
