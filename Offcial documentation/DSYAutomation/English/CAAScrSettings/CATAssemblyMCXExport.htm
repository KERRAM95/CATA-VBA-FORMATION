Quick Reference

CATAssemblyMCXExport

Role: Setting to handle the Assembly Export options

Detailed role: Setting to handle the Assembly Export options

The CATAssemblyMCXExport repository has the following characteristics:

  • Primary Tab Page: /Mechanical Design/Assembly Design/Export
  • Repository File: CATAssemblyMCXExport
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CATAssemblyMCXExport setting repository attributes:

  • ExportConstraintInfos
    Role: Option to specify if we enrich exported 3dxml for review with a json file containing constraint informations
    Detailed role: When the preference is activated, exported 3dxml for review with a json file containing constraint informations
    • Type: boolean
    • Default value: false
    • Default Value Role: By default the Export for review will not enrich the output 3dxml with PorductCDS.json
