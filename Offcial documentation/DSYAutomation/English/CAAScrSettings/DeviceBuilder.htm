DeviceBuilder Setting Repository
Quick Reference

DeviceBuilder

Role: Setting to Import Device Specific Information from V5 Products

Detailed role: Indicates that products can be migrated as devices.

The DeviceBuilder repository has the following characteristics:

  • Primary Tab Page: /General/FBDI/Device Builder
  • Repository File: FBDIModeForDeviceBuilder
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DeviceBuilder setting repository attributes:

  • FBDIModeAttachOverload
    Role: Overload attachment relative offset from imported primary document
    Detailed role: The relative offset with attachment is coming from the lower assembly V5 document (not necessarily the imported document picked by end-user). This option will use the offset from the selected document imported(CATProcess or CATProduct)
    • Type: boolean
    • Default value: false
  • FBDIModeForDeviceBuilder
    Role: Import Device Specific Information from V5 Products
    Detailed role: Indicates that products can be migrated as devices
    • Type: boolean
    • Default value: true
  • FBDIModeMountOverload
    Role: Overload robot mount relative offset from imported primary document
    Detailed role: The relative offset about robot mount is coming from the lower assembly V5 document (not necessarily the imported document picked by end-user). This option will use the offset from the selected document imported(CATProcess or CATProduct)
    • Type: boolean
    • Default value: false
