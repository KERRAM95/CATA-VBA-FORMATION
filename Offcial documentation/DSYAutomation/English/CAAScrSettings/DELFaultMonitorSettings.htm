DELFaultMonitorSettings Setting Repository
Quick Reference

DELFaultMonitorSettings

Role: Settings to define Fault Monitor profiles

Detailed role: These settings allow users to define profiles used for Fault Monitor

The DELFaultMonitorSettings repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/DELMIA Infrastructure/Fault Monitor
  • Repository File: DELFaultMonitorSettings
  • Version: 2
  • Full Compatibility: 1
  • Upward Compatibility: 1

DELFaultMonitorSettings setting repository attributes:

  • ActiveIndex
    Role: Active Profile Index
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: Active Index
  • ApplyRscClearance
    Role: Apply Resource Clearances
    • Type: unsigned integer
    • Default value: 0
    • Values:
      0 Do not apply resource clearances
      1 Apply resource clearances
  • DefinedProfiles
    Role: Defined Profiles
    Detailed role: List of strings with following format: ProfileName|FaultID|Active|Min|Max|ApplyMinMax|FaultID|Active|Min|Max|ApplyMinMax|FaultID|Active|Min|Max|ApplyMinMax|...
    • Type: string
    • Default value: Default|Collision|1|0|0|0|DOFTravel|0|0|0|0|DOFSoft|0|0|0|0|DOFSpeed|0|0|0|0|DOFAccel|0|0|0|0|DOFCaution|0|0|0|0|AxisSafety|0|0|0|0|TCPLinSpeed|0|0|0|0|TCPRotSpeed|0|0|0|0|TCPLinAccel|0|0|0|0|TCPRotAccel|0|0|0|0|Reachability|1|0|0|0|Singularity|1|0|0|0|WorkingLoc|0|0|0|0
    • Default Value Role: Fault Monitor Profiles
  • DisplayedIndex
    Role: Displayed Profile Index
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: Index
  • ProfileCount
    Role: Count of available Profiles
    • Type: unsigned integer
    • Default value: 1
    • Default Value Role: count
