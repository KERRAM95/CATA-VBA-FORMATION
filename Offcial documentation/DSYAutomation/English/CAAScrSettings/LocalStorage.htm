LocalStorage Setting Repository
Quick Reference

LocalStorage

Role: Setting to handle the local persistency location for PLM local storage stream/unstream operations

Detailed role: This setting allows to customize the location of local storage persistency operations on authoring extended streams

The LocalStorage repository has the following characteristics:

  • Primary Tab Page: /General/LocalStorage
  • Repository File: LocalStorage
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

LocalStorage setting repository attributes:

  • LSManagedPath
    Role: Root path for the Local Storage managed directory
    Detailed role: This option specifies the repository path on a local disk storing locally stored result streams.
    • Type: URN
    • Default value: UNIX: ${HOME} Windows: CSIDL_LOCAL_APPDATA
