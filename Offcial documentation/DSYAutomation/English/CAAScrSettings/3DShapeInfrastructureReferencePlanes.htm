3DShapeInfrastructureReferencePlanes Setting Repository
Quick Reference

3DShapeInfrastructureReferencePlanes

Role: Setting to manage the preferences for the reference planes

The 3DShapeInfrastructureReferencePlanes repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/3D Shape Infrastructure/Display
  • Repository File: 3DShapeInfrastructureReferencePlanes
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

3DShapeInfrastructureReferencePlanes setting repository attributes:

  • Appearance
    Role: The appearance specifies if the reference planes are wireframe or shaded.
    • Type: short integer
    • Default value: 0
    • Values:
      0 Wireframe
      Detailed role: Default wireframe graphic properties.
      1 Shaded
      Detailed role: Colors and transparency are available.
  • AutoSizeInGeometry
    Role: Auto size or fixed size
    Detailed role: Auto size to the boundary box of the 3D shape else fixed sized defined by a value.
    • Type: boolean
    • Default value: false
  • BackFaceColor
    Role: Default color of the back face if reference planes are shaded.
    • Type: double
    • Default value: 2747850000
  • FrontFaceColor
    Role: Default color of the front face if reference planes are shaded.
    • Type: double
    • Default value: 5670400
  • Preset
    Role: Allows to customize the reference planes
    Detailed role: Specify the behavior of the reference planes in the geometry.
    • Type: short integer
    • Default value: 0
    • Values:
      0 Legacy
      Detailed role: The standard behavior: Visible, fixed size and wireframe.
      1 Enhanced
      Detailed role: Zoomable and shaded with fixed color and transparency.
      2 Customized
      Detailed role: Any combination of size, color and transparency.
  • ShowOrientationArrow
    Role: Shows the orientation arrow of the reference planes.
    • Type: boolean
    • Default value: false
  • ShowPlaneLabel
    Role: Shows the label of the reference planes.
    • Type: boolean
    • Default value: false
  • SizeMode
    Role: Determines of the size is fixed or automatic.
    • Type: short integer
    • Default value: 0
    • Values:
      0 Fixed screen size
      Detailed role: The size is fixed by a value.
      1 Automatic size
      Detailed role: Auto size to the boundary box of the 3D shape.
  • Transparency
    Role: Default transparency if reference planes are shaded.
    • Type: double
    • Default value: 236
    • Min value: 0
    • Max value: 255
