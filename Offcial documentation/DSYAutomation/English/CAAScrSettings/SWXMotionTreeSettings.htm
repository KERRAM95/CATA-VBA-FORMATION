SWXMotionTreeSettings

Role: Settings to customize appearance of Motion Tree

The SWXMotionTreeSettings repository has the following characteristics:

  • Primary Tab Page: /Mechanical Design/SolidWorks V6/Motion Tree
  • Repository File: SWXMotionTreeSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

SWXMotionTreeSettings setting repository attributes:

  • DisablePredictiveComputingColumn
    Role: Color Column

    • Default value: false
  • SymbolColorColumn
    Role: Transparency Column
    Detailed role: Controls appearance of Symbol Color column in Motion Tree
    • Type: boolean
    • Default value: false
  • VisibilityColumn
    Role: Visibility Column
    Detailed role: Controls appearance of Visibility column in Motion Tree
    • Type: boolean
    • Default value: false
