Quick Reference

GeneralDisplayLayerFilter

Role: Filter and Document Display mode

Detailed role: This option modifies the impact of layer filters applied to assemblies representations. As far as other types of documents supporting layer filters (such as Parts and Drawings representations) are concerned, filters are stored in the database and this storage location cannot be modified.

The GeneralDisplayLayerFilter repository has the following characteristics:

  • Primary Tab Page: /General/Display/Layer Filter
  • Repository File: GeneralDisplayLayerFilter
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

GeneralDisplayLayerFilter setting repository attributes:

  • ApplyFilterMode
    Role: Filter and Document Display mode
    • Type: integer
    • Default value: 0
    • Default Value Role: By default filter applies to all documents
    • Values:
      0 Current filter on all documents
      1 Current filter of the document
