CATSysSettingsPlotterUI Setting Repository
Quick Reference

CATSysSettingsPlotterUI

Role: Setting to handle the CATIA Systems' plotter parameters

Detailed role: This setting allows to set options specific to CATIA Systems' plotter

The CATSysSettingsPlotterUI repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/System Behavior Design/System Behavior Properties
  • Repository File: CATSysSettingsPlotterUI
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CATSysSettingsPlotterUI setting repository attributes:

  • PlotPanelDocking
    Role: Default plotter panel docking
    Detailed role: Define the default docking position for new plot panels
    • Type: unsigned integer
    • Default value: 3
    • Default Value Role: Stacked by default
    • Values:
      0 Dock left
      1 Dock right
      2 Dock top
      3 Stacked
      4 Free
