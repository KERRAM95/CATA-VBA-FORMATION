CATPLMLinksAndRelations Setting Repository
Quick Reference

CATPLMLinksAndRelations

Role: Setting to handle the Links and Relations Edition options

Detailed role: Setting to handle the Links and Relations Edition options

The CATPLMLinksAndRelations repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/CATPLMLinksAndRelations
  • Repository File: CATPLMLinksAndRelations
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CATPLMLinksAndRelations setting repository attributes:

  • EastCompassInLinksAndRelations
    Role: Attribute to manage East Compass activation when editing Links and Relations
    Detailed role: The East Compass is activated when editing Links and Relations according to this attribute's value
    • Type: integer
    • Default value: 0
    • Default Value Role: By default East Compass is deactivated when editing Links and Relations
    • Values:
      0 The East Compass is deactivated when editing Links and Relations
      1 The East Compass is activated when editing Links and Relations (potential bad performances)
  • EnableAdvancedActions
    Role: Attribute to display Advanced Actions in Links and Relations dialog
    Detailed role: The Advanced Actions are displayed in Links and Relations dialog according to this attribute's value
    • Type: integer
    • Default value: 0
    • Default Value Role: By default the Advanced Actions are hidden in Links and Relations dialog
    • Values:
      0 The Advanced Actions are hidden in Links and Relations dialog
      1 The Advanced Actions are displayed in Links and Relations dialog
  • SetIdentifier
    Role: Attribute to customize which title is displayed for Set nodes
    Detailed role: The title displayed for the Set node in Links and Relations dialog is modified according to this attribute's value
    • Type: integer
    • Default value: 1
    • Default Value Role: By default the identifier of the instance is displayed
    • Values:
      0 The identifier of the reference is displayed for Set nodes
      1 The identifier of the instance is displayed for Set nodes
      2 The complete identifier, same as in the tree, is displayed for Set nodes
