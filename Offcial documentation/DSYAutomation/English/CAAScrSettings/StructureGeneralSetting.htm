StructureGeneralSetting Setting Repository
Quick Reference

StructureGeneralSetting

Role: Setting to manage the options of Structure

Detailed role: Structure setting contains options necessary to the commands of Structure

The StructureGeneralSetting repository has the following characteristics:

  • Primary Tab Page: /MechanicalDesign/CATSldPLMRepConfig/General
  • Repository File: StructureGeneralSetting
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

StructureGeneralSetting setting repository attributes:

  • ClashAnalysisModeWeld_Settings
    Role: Clash Analysis Mode
    Detailed role: Default clash analysis mode for weld management
    • Type: integer
    • Default value: 0
    • Default Value Role: ClashAnalysisModeWeld_Settings
  • CreatePublicationValue
    Role: Create Publication Value
    Detailed role: Create Publication value for structure objects
    • Type: integer
    • Default value: 0
  • DefaultAutoLimitMode
    Role: Default Limit Mode
    Detailed role: Value of automatic limit mode upon profile creation

    • Default value: 0
    • Default Value Role: DefaultAutoLimitMode
  • DefaultAutoNameValue
    Role: Default Automatic Naming Value
    Detailed role: Value of the autonatic naming attribute upon object creation
    • Type: integer
    • Default value: 1
  • DeleteUselessLimitLink
    Role: Delete useless limit link
    Detailed role: Delete useless limit link after delete limit or replace limit
    • Type: integer
    • Default value: 0
  • ForceNamingBRValue
    Role: Force execution of naming business rules
    Detailed role: Specified whether naming business rules should be launched in the Reset Business Rules command
    even if the automatic name attribute has been set to FALSE.

    • Default value: 0
