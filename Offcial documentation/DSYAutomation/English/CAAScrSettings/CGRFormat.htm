CGRFormat Setting Repository
Quick Reference

CGRFormat

Role: CGR Format customization

Detailed role: These settings handles the parameters that provides the ability to customize the output of the CGR format

The CGRFormat repository has the following characteristics:

  • Primary Tab Page: /General/Display/CGR Loading
  • Repository File: CGRFormat
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CGRFormat setting repository attributes:

  • LightSceneGraph
    Role: Enables/disables light scene graph mode
    Detailed role: Opens CGR files in light scene graph mode (lower memory consumption)
    • Type: boolean
    • Default value: false
  • V4V5_FDT
    Role: Enables/disables the V4V5_FDT CGR applicative container
    Detailed role: Allows to save FDT data in the cache CGR file
    • Type: boolean
    • Default value: false
  • V4_Model_CommentPage
    Role: Enables/disables the V4_Model_CommentPage CGR applicative container
    Detailed role: Allows to save comment page in the cache CGR file
    • Type: boolean
    • Default value: false
  • V4_Model_LnF
    Role: Enables/disables the V4_Model_LnF CGR applicative container
    Detailed role: Allows to save V4 Layers and Filters in the cache CGR file
    • Type: boolean
    • Default value: false
  • V5_SPA
    Role: Enables/disables the V5_SPA CGR applicative container
    Detailed role: Allows to save space analysis data in the cache CGR file
    • Type: boolean
    • Default value: false
  • Voxels
    Role: Enables/disables the Voxels CGR applicative container
    Detailed role: Allows to save the voxelized form of the geometry in the cache CGR file
    • Type: boolean
    • Default value: false
  • VoxelsAccuracy
    Role: Voxels accuracy in mm.
    Detailed role: Defines the accuracy of the voxelization generated in the CGR file in mm.
    • Type: integer
    • Default value: 10
