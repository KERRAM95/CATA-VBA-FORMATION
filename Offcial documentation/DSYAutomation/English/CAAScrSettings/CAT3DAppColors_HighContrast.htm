CAT3DAppColors_HighContrast

Role: High Contrast variant of the colors of the CAT3DApp utilities

Detailed role: High Contrast variant of the colors of the CAT3DApp utilities - Attribute names have to match the CAT3DAppColors repository - Only the default values set here are read, modification of the stored values is without effect!

The CAT3DAppColors_HighContrast repository has the following characteristics:

  • Primary Tab Page:  
  • Repository File: CAT3DAppColors_HighContrast
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
