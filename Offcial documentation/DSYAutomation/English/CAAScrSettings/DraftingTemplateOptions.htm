DraftingTemplateOptions Setting Repository
Quick Reference

DraftingTemplateOptions

Role: Manages Drafting Template options

The DraftingTemplateOptions repository has the following characteristics:

  • Primary Tab Page: /Mechanical Design/Drafting Template/Drafting Template
  • Repository File: DraftingTemplateOptions
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DraftingTemplateOptions setting repository attributes:

  • IsStandardPLM
    Role: This key is used to check if selected standard is a PLM standard or not
    Detailed role: This key is used to check if selected standard is a PLM standard or not
    • Type: boolean
    • Default value: 0
  • NumOfStandardPLM
    Role: Number of PLM standards
    • Type: unsigned integer
    • Default value: 0
  • StandardPLM
    Role: List of PLM standards
    • Type: string
    • Default value:
  • StandardPLMAdd
    Role: This key is used to manage lock of standard PLM addition in list
    Detailed role: This key is used to manage lock of standard PLM addition in list
    • Type: boolean
    • Default value: 0
  • StandardPLMID
    Role: List of PLM standards IDs
    • Type: string
    • Default value:
  • StandardPLMRemove
    Role: This key is used to manage lock of standard PLM addition in list
    Detailed role: This key is used to manage lock of standard PLM addition in list
    • Type: boolean
    • Default value: 0
  • standard
    Role: Standard name
    Detailed role: Standard used at creation of a Sheet.
    • Type: string
    • Default value: ISO
