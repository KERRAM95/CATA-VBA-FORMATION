PyScriptsRepo Setting Repository
Quick Reference

PyScriptsRepo

Role: Defines some options for the python scripts

Detailed role: Enable the user to deal with the python script option such as external packages directory

The PyScriptsRepo repository has the following characteristics:

  • Primary Tab Page: /General/Parameters and Measure/External languages
  • Repository File: PyScriptsRepo
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

PyScriptsRepo setting repository attributes:

  • ExtPackagesDirectory
    Role: Selects the output directory that will contain the external python packages
    Detailed role: Enables the user to select the output directory that will contain the external python packages
    • Type: URN
    • Default value: UNIX: ${CATTemp} Windows: ${CATTemp}
  • ExtPackagesUse
    Role: Enables the use of external python packages
    Detailed role: Enables the use of external python packages
    • Type: boolean
    • Default value: false
