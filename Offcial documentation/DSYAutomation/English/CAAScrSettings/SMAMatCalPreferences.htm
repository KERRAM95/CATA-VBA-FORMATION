SMAMatCalPreferences Setting Repository
Quick Reference

SMAMatCalPreferences

Role: Manages general UI preferences for material calibration

The SMAMatCalPreferences repository has the following characteristics:

  • Primary Tab Page: /None/None/None
  • Repository File: SMAMatCalPreferences
  • Version: 2
  • Full Compatibility:
  • Upward Compatibility:

SMAMatCalPreferences setting repository attributes:

  • AlgorithmChoice
    Role: Optimization algorithm
    • Type: integer
    • Default value: 1
  • ErrorSetting
    Role: Sets the error
    • Type: integer
    • Default value: 5
  • FitErrorSetting
    Role: Sets the fit error
    • Type: integer
    • Default value: 5
  • FitNormalizeError
    Role: Sets whether the fit error is normalized
    • Type: integer
    • Default value: 1
  • LengthUnits
    Role: Stores the unit for length for calibration execution
    • Type: string
    • Default value: METER
  • MassUnits
    Role: Stores the unit for mass for calibration execution
    • Type: string
    • Default value: KILOGRAM
  • MaterialImport
    Role: Stores the preference for importing material during mesh import
    • Type: integer
    • Default value:
  • MaterialSource
    Role: Stores the preference for the source of an imported material
    • Type: integer
    • Default value: 1
  • MeshLengthUnits
    Role: Stores the unit for length for mesh calibration execution
    • Type: string
    • Default value: METER
  • MeshMassUnits
    Role: Stores the unit for mass for mesh calibration execution
    • Type: string
    • Default value: KILOGRAM
  • MeshMoleUnits
    Role: Stores the unit for mole for mesh calibration execution
    • Type: string
    • Default value: MOL
  • MeshTempUnits
    Role: Stores the unit for temperature for mesh calibration execution
    • Type: string
    • Default value: KELVIN
  • MeshTimeUnits
    Role: Stores the unit for time for mesh calibration execution
    • Type: string
    • Default value: SECOND
  • ModelSource
    Role: Stores the preference for the source of a meshed model
    • Type: integer
    • Default value: 1
  • MoleUnits
    Role: Stores the unit for mole for calibration execution
    • Type: string
    • Default value: MOL
  • NormalizeError
    Role: Sets whether the error is normalized
    • Type: integer
    • Default value: 1
  • SignificantDigits
    Role: Stores the number of significant digits for the input file extraction
    • Type: integer
    • Default value: 15
  • TempUnits
    Role: Stores the unit for temperature for calibration execution
    • Type: string
    • Default value: KELVIN
  • TimeUnits
    Role: Stores the unit for time for calibration execution
    • Type: string
    • Default value: SECOND
