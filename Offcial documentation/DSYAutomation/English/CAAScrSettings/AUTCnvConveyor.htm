Quick Reference

AUTCnvConveyor

Role: Setting conveyor visualization parameters

Detailed role: Visualization of conveyor analysis parameters.

The AUTCnvConveyor repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/DELMIA Infrastructure/Conveyors
  • Repository File: AUTCnvConveyor
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

AUTCnvConveyor setting repository attributes:

  • PreciseWorkpiece
    Role: Simulation - Workpiece shape
    Detailed role: Simulation - Workpiece shape
    • Type: short integer
    • Default value: 0
    • Values:
      0 Basic Shape (Best Performance)
      1 Precise Shape (Best Precision)
  • RemoveLegacyIO
    Role: Remove Legacy IO data on PLM Update
    Detailed role: Indicates if Legacy IO data needs to be removed on PLM Update
    • Type: unsigned integer
    • Default value: 0
    • Values:
      0 Disable Remove Legacy IO
      1 Enable Remove Legacy IO
  • SimulationPrecision
    Role: Simulation - Workpiece displacement
    Detailed role: Simulation - Workpiece displacement
    • Type: integer
    • Default value: 10
    • Min value: 1
    • Max value: 10
    • Step: 1
