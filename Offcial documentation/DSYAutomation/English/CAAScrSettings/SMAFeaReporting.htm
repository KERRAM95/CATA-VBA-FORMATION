SMAFeaReporting Setting Repository
Quick Reference

SMAFeaReporting

Role: Manage options for reporting

Detailed role: Manage options for reporting

The SMAFeaReporting repository has the following characteristics:

  • Primary Tab Page: /AnalysisSimulation/Reporting
  • Repository File: SpmReportingOptions
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

SMAFeaReporting setting repository attributes:

  • PictureFormat
    Role: Specify picture format. Available formats: JPEG, JPEG_HIGH, PNG, BMP
    • Type: string
    • Default value: JPEG
  • PictureHeight
    Role: Specify picture height
    • Type: integer
    • Default value: 450
    • Min value: 0
    • Max value: 5000
  • PictureSizeMode
    Role: Use defaut picture size or specify size
    • Type: integer
    • Default value: 0
    • Default Value Role: By default default picture size
    • Values:
      0 Defaut picture size
      1 Specify picture size
  • PictureWidth
    Role: Specify picture width
    • Type: integer
    • Default value: 600
    • Min value: 0
    • Max value: 5000
