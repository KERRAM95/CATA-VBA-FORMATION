CATDMUCommon Setting Repository
Quick Reference

CATDMUCommon

Role: Setting to handle the Interference parameters

The CATDMUCommon repository has the following characteristics:

  • Primary Tab Page: /Digital Mockup/CATDMUCommon/General
  • Repository File: CATDMUCommon
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CATDMUCommon setting repository attributes:

  • DataProtection
    Role: Publish personal data
    • Type: integer
    • Default value: 0
  • ReportPath
    Role: Save HTML report on disk
    • Type: URN
    • Default value: UNIX: $HOME/CATTemp/GraphSVG Windows: CSIDL_LOCAL_APPDATA\DassaultSystemes\CATTemp\GraphSVG
  • ReportSave
    Role: Save HTML Report into VPM
    • Type: integer
    • Default value: 1
