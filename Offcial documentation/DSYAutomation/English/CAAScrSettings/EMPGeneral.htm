EMPGeneral Setting Repository
Quick Reference

EMPGeneral

Role: Short Qualification of the usage of the setting

Detailed role: Full explanation of the Setting Role

The EMPGeneral repository has the following characteristics:

  • Primary Tab Page: /Equipments/Electrical Harness Flattening/General
  • Repository File: ElectricalHarnessFlattening
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

EMPGeneral setting repository attributes:

  • BranchOrientationAtJunction
    Role: Option to maintain branch orientations at junctions constrained by support
    Detailed role: Option to Maintain branch orientations at junctions constrained by support
    • Type: integer
    • Default value: 0
    • Default Value Role: By default this option is OFF
    • Values:
      0 Do not maintain the branch tangency at junction
      1 Maintain the branch tangency at junction
  • CopilotVisibility
    Role: Option if to activate or de-activate co pilot behavior for Live Manipulators
    Detailed role: If the option is ON selected co pilot behavior for Live Manipulators is activated
    • Type: integer
    • Default value: 0
    • Default Value Role: By default this option is OFF for Live Manipulators
    • Values:
      0 Do not show copilot
      1 Show copilot
  • EMPLMRotateHavingPivotNearSelection
    Role: Option if to pivot point appear near hover side
    Detailed role: If the option is selected, then behaviour of shift press while rotate action will become default
    • Type: integer
    • Default value: 0
    • Default Value Role: By default this option is OFF
    • Values:
      0 Pivot point identification not by proximity
      1 Pivot point identification by proximity
  • EasyFTADimOffset
    Role: Option to select the offset distance
    Detailed role: Option to select the offset distance for dimension
    • Type: double
    • Default value: 0.020
    • Min value: 0.0
    • Max value: 10e+307
    • Step: 0.001
  • EasyFTADimType
    Role: Type of Dimension that will be created by Easy FTA Commands
    Detailed role: Allows selection of the type of Dimension that will be created by Easy FTA Commands
    • Type: integer
    • Default value: 0
    • Default Value Role: By default the type of Dimension that will be created by Easy FTA Commands is Offset
    • Values:
      0 Offset type FTA Dimension
      1 Parallel type FTA Dimension
      2 Linear type FTA Dimension
      3 User Defined type FTA Dimension during command execution
  • ExtractFTAAnnotation
    Role: Option to enable extraction of annotations in harness
    Detailed role: Option to enable extraction of annotations in harness
    • Type: integer
    • Default value: 0
    • Default Value Role: By default this option is OFF
    • Values:
      0 Disable extraction
      1 Enable extraction
  • HtmlPath
    Role: HTML path storage
    Detailed role: File path where is store the html report generated during the synchronization
    • Type: URN
    • Default value: UNIX: ${CATTemp} Windows: ${CATTemp}
  • NetworkAssistantExtractandSync
    Role: Option if to run Network Assistant during Extract and Synchronize
    Detailed role: If the option is selected Harness data electrical connectivity will be checked through Network Assistant during Extract and Synchronize
    • Type: integer
    • Default value: 0
    • Default Value Role: By default this option is OFF for the Extract and Synchronize Command
    • Values:
      0 Do not Check Harness data electrical connectivity
      1 Check Harness data electrical connectivity
  • NumberingOfSection
    Role: Will decide the numbering of Supports and Sections
    Detailed role: Will decide the numbering of Supports and Sections
    • Type: integer
    • Default value: 0
    • Default Value Role: By default the Option will be Off to allow different numbering of Supports
    • Values:
      0 The Numbering Of Supports will not be Kept in Automatic Generation
      1 The Numbering Of Supports will be Kept in Automatic Generation
  • RotateSplitWarnMsgOption
    Role: Will decide about the show behaviour of the warning message shown during branch split in Rotate command
    Detailed role: Will decide about the show behaviour of the warning message shown during branch split in Rotate command
    • Type: integer
    • Default value: 1
    • Default Value Role: By default the The warning message while branch split in Rotate command will be Shown with option to turn it off next time
    • Values:
      0 The warning message while branch split in Rotate command will be always shown
      1 The warning message while branch split in Rotate command will be shown and can be turned OFF the next time by clicking 'Do not show next time' in the message box
      2 The warning message while branch split in Rotate command will be never shown
  • SegmentNameOption
    Role: Option to decide the display name of segment
    Detailed role: Option to decide the display name of segment
    • Type: integer
    • Default value: 0
    • Default Value Role: By default this option is segment name
    • Values:
      0 Segment name
      1 Branch name
      2 Segment name/branch name
  • SpineIdentificationCriteria
    Role: Option to decide the criteria of spine identification
    Detailed role: Option to decide the criteria of spine identification based on diameter path or longest network weight
    • Type: integer
    • Default value: 0
    • Default Value Role: By default this option is longest network weight
    • Values:
      0 Longest network weight
      1 Diameter path
  • StartElementSpineIdentification
    Role: Option to decide the start element of the spine
    Detailed role: Option to decide the start element of the spine based on diameter or length
    • Type: integer
    • Default value: 1
    • Default Value Role: By default this option is branch with biggest diameter
    • Values:
      0 None
      1 Branch with biggest diameter
      2 Branch with longest length
  • SupportMeasurePoint
    Role: Calculation of Measure Points for Supports
    Detailed role: Allows to set Middle point of Support as the default Measure Point
    • Type: integer
    • Default value: 0
    • Default Value Role: By default the Middle point is not set as default measure point
    • Values:
      0 To Un-Set middle point of support as default Measure Point
      1 To Set middle point of support as default Measure Point
  • Updatedrawing_in_Synchronize
    Role: Option to simultaneously update drawing during synchronize
    Detailed role: Option to simultaneously update drawing during synchronize
    • Type: integer
    • Default value: 0
    • Default Value Role: By default this option is OFF
    • Values:
      0 Disable update
      1 Enable update
  • VariablitySpaceExtract
    Role: Option if to Extract Varibality Space
    Detailed role: If the option is selected Variablity Space is Extracted into Harness Flatten model by the Extract Command
    • Type: integer
    • Default value: 1
    • Default Value Role: By default this option is ON for the Extract Command
    • Values:
      0 Do not Extract Variablity Space
      1 Extract Variablity Space
