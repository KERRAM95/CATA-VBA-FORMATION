CATLiveShapeSettings Setting Repository
Quick Reference

CATLiveShapeSettings

Role: These settings define the user choice for CATIA Live Shape environment

Detailed role: These settings define the user choice of the display the referent who allows you to get a better idea of the scale factor.

The CATLiveShapeSettings repository has the following characteristics:

  • Primary Tab Page: /Mechanical Design/CATIA Live Shape/Display
  • Repository File: CATLiveShapeSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
