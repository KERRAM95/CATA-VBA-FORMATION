DELAdditiveMfgPreferencesRepository Setting Repository
Quick Reference

DELAdditiveMfgPreferencesRepository

Role: Lists the general settings of the Additive Manufacturing applications

Detailed role: The general additive manufacturing settings are useful to optmimize performances or to edit preferences on tree display, color and highlight and replay

The DELAdditiveMfgPreferencesRepository repository has the following characteristics:

  • Primary Tab Page: /Simulation/Additive_Manufacturing
  • Repository File: DELAdditiveMfgPreferencesRepository
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
