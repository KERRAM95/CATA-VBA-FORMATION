WeightSettings Setting Repository
Quick Reference

WeightSettings

Role: Settings to handle the inertias parameters

The WeightSettings repository has the following characteristics:

  • Primary Tab Page: /General/WCG
  • Repository File: WeightSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

WeightSettings setting repository attributes:

  • CharacteristicsPLMUpdate
    Role: User choice to update the characteristics during PLMUpdate or not
    • Tab Page: /General/WCG/General
    • Type: integer
    • Default value: 1
  • CharacteristicsReport
    Role: User choice to generate Characteristics Report or not
    • Tab Page: /General/WCG/General
    • Type: integer
    • Default value: 0
  • ComputedMargin
    Role: The margin automatically applied to the computed weight. It is a purcentage
    • Tab Page: /General/WCG/General
    • Type: double
    • Default value: 5
  • DeclaredMargin
    Role: The margin automatically applied to the declared weight. It is a purcentage
    • Tab Page: /General/WCG/General
    • Type: double
    • Default value: 10
  • DefaultDensity
    Role: Describe the default density (MKS)
    • Tab Page: /General/WCG/General
    • Type: double
    • Default value: 7770
  • ForceExact
    Role: Describe if the exact geometry has to be loaded in order to perform a computation
    • Tab Page: /General/WCG/General
    • Type: integer
    • Default value: 0
  • MaskFile
    Role: Describe the mask attributes for publishing
    • Tab Page: /General/WCG/General
    • Type: string
    • Default value:
  • OnlyMainBody
    Role: Describe if only main body or all the geometry must be taken into account for weight computation
    • Tab Page: /General/WCG/General
    • Type: integer
    • Default value: 0
  • Propagate
    Role: Describe the way weight checking is integrated to the propagate concepts
    • Tab Page: /General/WCG/General
    • Type: integer
    • Default value: 0
  • PublishedDirectory
    Role: Describe the directory hosting the published reports
    • Tab Page: /General/WCG/General
    • Type: string
    • Default value:
  • RetrieveWeightFromBOM
    Role: Describe if the weight declared on the synchronized eBOM Part should be retrieved in Weight Definition
    • Tab Page: /General/WCG/General
    • Type: integer
    • Default value: 0
