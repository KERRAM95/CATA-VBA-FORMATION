SteelConnectionGeneralSetting Setting Repository
Quick Reference

SteelConnectionGeneralSetting

Role: Setting to manage the options of Steel Connection Design application

Detailed role: Setting contains options necessary to the commands of Steel Connection Design application

The SteelConnectionGeneralSetting repository has the following characteristics:

  • Primary Tab Page: /MechanicalDesign/CATStdPLMRefConfig/General
  • Repository File: SteelConnectionGeneralSetting
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

SteelConnectionGeneralSetting setting repository attributes:

  • DefaultVolumeModelValue
    Role: Setting to manage the default design methodology of steel connection design objects in 3D Shape
    Detailed role: Volumic or Solid design.
    • Type: boolean
    • Default value: true
