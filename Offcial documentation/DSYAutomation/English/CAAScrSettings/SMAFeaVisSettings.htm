SMAFeaVisSettings Setting Repository
Quick Reference

SMAFeaVisSettings

Role: Manages visualization settings for Simulia Physics Modeling and Simulation

The SMAFeaVisSettings repository has the following characteristics:

  • Primary Tab Page: /None/None/None
  • Repository File: SMAFeaVisSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

SMAFeaVisSettings setting repository attributes:

  • AxisOneColor
    Role: Sets the color of the first axis
    • Type: integer
    • Default value: 255
  • AxisThreeColor
    Role: Sets the color of the third axis
    • Type: integer
    • Default value: 255
  • AxisTwoColor
    Role: Sets the color of the second axis
    • Type: integer
    • Default value: 255
  • BeamGlyphAxisDisplay
    Role: Show connector glyph axis display
    • Type: boolean
    • Default value: 1
  • BeamRenderingStyle
    Role: Sets the rendering style for beam section glyphs
    • Type: integer
    • Default value: 0
    • Default Value Role: By default render beam section as solid
    • Values:
      0 Render beam section as solid
      1 Render beam section as wireframe sketch
      2 No rendering of beam section
  • ConnectorGlyphAxisDisplay
    Role: Show connector glyph axis display
    • Type: boolean
    • Default value: 1
  • ConnectorGlyphTextDisplay
    Role: Show connector glyph text annotation
    • Type: boolean
    • Default value: 1
  • GlyphScaleFactor
    Role: Sets the scale factor for all feature glyphs
    • Type: float
    • Default value: 1.0
  • HideManipulatorDisplay
    Role: Disable the manipulator associated to the glyph
    • Type: boolean
    • Default value: 0
  • LargeAssemblyMode
    Role: Sets large assembly mode
    • Type: boolean
    • Default value: 0
  • LinkConnectionMeshVisualization
    Role: Links the visualization of the connection mesh to the visualization of the connection property
    • Type: boolean
    • Default value: 1
  • SFGlyphScaling
    Role: Enable scenario features glyph scaling based on magnitude
    • Type: boolean
    • Default value: 1
