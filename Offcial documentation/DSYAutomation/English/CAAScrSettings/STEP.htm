STEP Setting Repository
Quick Reference

STEP

Role:

The STEP repository has the following characteristics:

  • Primary Tab Page: /General/Compatibility/STEP
  • Repository File: STEP
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

STEP setting repository attributes:

  • 3D annotations
    Role: Enables import of Graphic 3D annotations
    • Alias: 3DAnnotationsMode
    • Type: short integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • 3D annotations Export
    Role: 3D Annotations import mode
    • Alias: 3DAnnotationsExportMode
    • Type: short integer
    • Default value: 0
    • Values:
      0 disabled
      1 as graphic
      2 as semantic
  • 3D annotations Semantic
    Role: Enables import of Semantic 3D annotations
    • Alias: 3DAnnotationsSemanticMode
    • Type: short integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • Angle
    Role: Defines the angle (in degree) below which contiguous elements (curves) can be merged.
    • Alias: AngleDefFitingValue
    • Type: float
    • Default value: 2
    • Min value: 0.0
    • Max value: 10
    • Step: 0.5
  • Application Protocol
    Role: Type of Application Protocole
    • Alias: APMode
    • Type: short integer
    • Default value: 5
    • Values:
      0 For use Application Protocol AP203
      1 For use Application Protocol AP203 with extentions
      2 For use Application Protocol AP214
      3 For use Application Protocol AP203 edition 2
      4 For use Application Protocol AP214 edition 3
      5 For use Application Protocol AP242 Ed1
      6 For use Application Protocol AP242 Ed2
      7 For use Application Protocol AP242 Ed3
  • Assemblies
    Role: Type of Assemblies export
    • Alias: ASMMode
    • Type: short integer
    • Default value: 1
    • Values:
      1 Exports structure with geometry in the same STEP file.
      2 Exports structure and associated geometry in separate STEP files, throw AP214 and AP203 edition 2 external references mechanism.
      3 Exports structure and geometry in nested STEP files, thru external references mechanism.
      4 Exports first level structure with references to references to STEP files, thru external references mechanism.
  • Author
    Role: Indicates the author in the header STEP file.
    • Alias: HeaderAuthorStr
    • Type: string
    • Default value:
  • Authorization
    Role: Indicates the authorization in the header STEP file.
    • Alias: HeaderAuthorisationStr
    • Type: string
    • Default value:
  • Centre of gravity max.Deviation(mm)
    Role: Define tolerances for GVP
    • Alias: GVPCdGValue
    • Type: double
    • Default value: 1
    • Min value: 0.01
    • Max value: 5
  • Clouds of points
    Role: Computes Clouds of points with report information
    • Alias: GVPCOPSMode
    • Type: short integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • Clouds of points definition(mm)
    Role: Define SAG for Clouds of points
    • Alias: GVPCOPSSAG
    • Type: double
    • Default value: 0.1
    • Min value: 0.01
    • Max value: 10
  • Clouds of points deviation(mm)
    Role: Define tolerance for Clouds of points
    • Alias: GVPCOPSTol
    • Type: double
    • Default value: 0.01
    • Min value: 0.001
    • Max value: 0.1
  • Composites
    Role: Enables Composites data
    • Alias: CompositesMode
    • Type: short integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • Continuity optimization of curves ans surfaces
    Role: Type of optimization
    • Alias: OptimizeC2Mode
    • Type: short integer
    • Default value: 1
    • Values:
      0 For use No optimization
      1 For use Automatic optimization
      2 For use Advanced optimization
  • Curves and surfaces approximation
    Role: Allows approximation of B-Spline curves and surfaces, controlled by deformation parameter.
    • Alias: FittingMode
    • Type: short integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • Deformation
    Role: Allows maximum deformation (in millimeter) in the optimization of Curves and Surfaces.
    • Alias: TolDefOptFitValue
    • Type: float
    • Default value: 0.003
    • Min value: 0.0005
    • Max value: 0.10
    • Step: 0.0005
  • Detailed report
    Role: Includes the list of transferred entities in the report
    • Alias: ReportMode
    • Type: short integer
    • Default value: 1
    • Min value: 0
    • Max value: 1
  • Enable Quality enhancement
    Role: Enables quality enhancements.
    • Alias: HealingActive
    • Type: short integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • Exact geometry as
    Role: Excat geometry as
    • Alias: ExportRepMode
    • Type: short integer
    • Default value: 0
    • Values:
      0 Export exact geometry on STEP file
      2 Export tessellated geometry on STEP file
      1 Export exact and tessellated geometry on STEP file
  • Export also NoShow entites
    Role: Enables export of the not shown entities.
    • Alias: ShowMode
    • Type: short integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • Export also edge entites in tessellated representation
    Role: Enables export also edge entites in tessellated representation.
    • Alias: EdgeInTess
    • Type: short integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • Export also non-visualized layers
    Role: Enables export of entities belonging to non-visualized layers, that are ignored if unchecked.
    • Alias: LayersFiltersMode
    • Type: short integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • Export more than 3DShape
    Role: Export more than 3DShape representation per reference
    • Alias: MultiRepMode
    • Type: short integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • File description
    Role: Indicates the file description in the header STEP file.
    • Alias: HeaderDescriptionStr
    • Type: string
    • Default value: 3DEXPERIENCE STEP
  • Geometric Sets
    Role: Enables the support of CATIA Geometrical Sets
    • Alias: GeoSet
    • Type: short integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • Geometry Quality Enhancement Mode
    Role: Enhancement mode
    • Alias: GeomQualityMode
    • Type: short integer
    • Default value: 0
    • Values:
      0 Default enhancement without deformation
      1 Advanced enhancement with customized allowed deformation
      Detailed role: To define a maximum tolerance value which may enhance the quality.
  • Groups
    Role: Enables groups handling, STEP groups being mapped with Selection Sets
    • Alias: GroupMode
    • Type: short integer
    • Default value: 1
    • Min value: 0
    • Max value: 1
  • Import as reference
    Role: Enables import as reference.
    • Alias: ImportAsRefMode
    • Type: short integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • Import products and geometries in one 3DPart
    Role: Enables import product and geometries in one 3DPart.
    • Alias: PrdTo3DPartMode
    • Type: short integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • Kinematics
    Role: Enables Kinematics data
    • Alias: KinematicsMode
    • Type: short integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • Maximum Shape Deformation Factor for Geometry Quality Enhancement
    Role: To define the maximum deformation allowed in the Geometry Quality Enhancement. This is a factor of the Resolution.
    • Alias: GeomQualityTolFact
    • Type: short integer
    • Default value: 5
    • Min value: 5
    • Max value: 100
  • Organization
    Role: Indicates the organization in the header STEP file.
    • Alias: HeaderOrganisationStr
    • Type: string
    • Default value:
  • Parameters
    Role: Define parameters for data quality enhancements
    • Alias: ButtonToOpenAdvHealingParams
    • Type: string
    • Default value:
  • Part as
    Role: Import terminal references with geometry as:
    • Alias: PartMode
    • Type: short integer
    • Default value: 0
    • Values:
      0 Import terminal references with geometry as 3D Part
      1 Import terminal references with geometry as Product and Shareable 3D Shape
  • Prefer 3D tessellated geometry

    • Type: short integer
    • Default value: 0

    • Max value: 1
  • STEP Persistence
    Role: Persistence
    • Alias: PersistenceMode
    • Type: short integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • STEP XML Application Protocol

      0 For use Application Protocol AP242 XML Ed1

  • ShapeDeformation
    Role: To define the shape Deformation Value

  • UDAs

    • Default value: 0

  • Volume and area max.Deviation(%)
    Role: Define tolerances for GVP

  • XHealing
    Role: Enables XHealing handling, Geometry will be heal

  • XHealingMode
