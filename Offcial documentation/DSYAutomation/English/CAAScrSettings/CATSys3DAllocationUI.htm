CATSys3DAllocationUI Setting Repository
Quick Reference

CATSys3DAllocationUI

Role: Systems 3D Allocation setting

Detailed role: Systems 3D Allocation setting

The CATSys3DAllocationUI repository has the following characteristics:

  • Primary Tab Page: /3D Modeling/Systems Architecture/Systems 3D Allocation
  • Repository File: CATSys3DAllocationUI
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CATSys3DAllocationUI setting repository attributes:

  • Sys3DEquivalentAreaPacking
    Role: The percentage factor to be added to the computed equivalent area
    Detailed role: The Packing area which can be filled is considered using this factor
    • Type: double
    • Default value: 15.0
    • Default Value Role: By default the factor to be considered is 15%
  • Sys3DLoadNetwork
    Role: Load dependent network to cross highlight
    Detailed role: Load dependent network to cross highlight
    • Type: integer
    • Default value: 0
    • Default Value Role: By default do not load dependent network to cross highlight
    • Values:
      0 Do not load dependent network to cross highlight
      1 Load dependent network to cross highlight
  • Sys3DMaximumFillingRatio
    Role: The Maximum filling ratio percentage to consider for network arc
    Detailed role: The network arc can be filled maximum upto the percentage defined
    • Type: double
    • Default value: 80.0
    • Default Value Role: By default the Maximum filling ratio is set to be 80%
