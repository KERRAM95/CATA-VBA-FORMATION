ElectricalDeviceLibrarian

Role: Electrical Library setting

Detailed role: Electrical Library setting

The ElectricalDeviceLibrarian repository has the following characteristics:

  • Primary Tab Page: /Equipments/Electrical Assembly Design/General
  • Repository File: ElectricalDeviceLibrarian
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

ElectricalDeviceLibrarian setting repository attributes:

  • AllowMultiConfigConnection
    Role: Check Button to allow more than one connection to same connection point
    Detailed role: Check Button to allow more than one connection to same connection point
    • Type: integer
    • Default value: 0
    • Default Value Role: By default only one connection with a connection point
    • Values:
      0 Only one connection with a connection point
      1 Mulitple connections with same connection point
  • CnctPtAxisOrientation
    Role: Setting to manage the default axis for connection points
    Detailed role: Setting to manage the default orientation axis for connection points
    • Type: integer
    • Default value: 0
    • Default Value Role: X
    • Values:
      0 X
      1 Y
      2 Z
  • CnctPtConnectedWarningMsg
    Role: Check Button to display warning if connection point is already connected
    Detailed role: Check Button to display warning message if connection point is already connected
    • Type: integer
    • Default value: 1
    • Default Value Role: By default warning will be displayed if connection point is already connected
    • Values:
      0 Warning will not be displayed if connection point is already connected
      1 Warning will be displayed if connection point is already connected
  • DistantLinkCreation
    Role: Check Button to create only distant link on branch extremity or branch point
    Detailed role: Check Button to create the distant link only on branch point or branch extremitys
    • Type: integer
    • Default value: 0
    • Default Value Role: By default Distant link creation is not restricted
    • Values:
      0 Electrical behavior extension is not added when add connection point
      1 Electrical behavior extension is added automatically when add connection point
  • DistantLinkMode
    Role: Option to choose the default mode when creating a new distant link
    Detailed role: Computes the distance between the point on curve and the branch extremity in Ratio or Lengh mode
    • Type: integer
    • Default value: 0
    • Default Value Role: By default the distance between the point on curve and the branch extremity is computed in Lenght
  • ElectricalBehaviorAutomatically
    Role: Check Button to add electrical behavior extension when add connection point
    Detailed role: Check Button to add electrical behavior extension when add connection point
    • Type: integer
    • Default value: 0
    • Default Value Role: By default electrical behavior extension is not added automatically
    • Values:
      0 Electrical behavior extension is not added when add connection point
      1 Electrical behavior extension is added automatically when add connection point
  • ElectricalGroupingNodeModeActivation
    Role: Check Button to activate working mode with Connector Group
    Detailed role: Check Button to activate working mode with Connector Group
    • Type: integer
    • Default value: 0
    • Default Value Role: Working mode with Connector Group off
    • Values:
      0 Working mode with Connector Group off
      1 Working mode with Connector Group on
  • OrientationAlongParentAxis
    Role: Check Button to allow orientation along parent axis after smart place
    Detailed role: Check Button to allow orientation along parent axis after smart place
    • Type: integer
    • Default value: 0
    • Default Value Role: By default orientation will not be along parent axis
    • Values:
      0 Orientation will not be along parent axis
      1 Orientation will not be along parent axis
  • PlaceCompass
    Role: Place compass on instance after smart move
    Detailed role: check to place compass on instance after smart move to refine movement
    • Type: integer
    • Default value: 1
    • Default Value Role: By default compass is placed
    • Values:
      0 No compass
      1 With compass
