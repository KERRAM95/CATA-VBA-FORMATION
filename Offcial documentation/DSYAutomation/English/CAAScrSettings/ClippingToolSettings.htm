ClippingToolSettings Setting Repository
Quick Reference

ClippingToolSettings

Role: Setting to handle the Clipping Tool parameters

The ClippingToolSettings repository has the following characteristics:

  • Primary Tab Page: /Common_Preferences/Modeling
  • Repository File: ClippingToolSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Setting controller:
    • Framework_CPP: CATSpaceAnalysisSettings
    • Interface_CPP: CATISPAClippingToolSettingAtt
    • LateType: CATSPAClippingToolSettingCtrl

ClippingToolSettings setting repository attributes:

  • ClippingUpdateMode
    Role: Default mode for clipping tool automatic update mode on product structure modifications.
    • Type: integer
    • Default value: 1
    • Values:
      0 Do NOT update clipping tool on product structure modifications.
      1 Update clipping tool on product structure modifications.
  • DefaultNormalDir
    Role: Default clipping tool normal direction.
    • Type: integer
    • Default value: 3
    • Values:
      0 X
      1 Y
      2 Z
      3 Automatic
