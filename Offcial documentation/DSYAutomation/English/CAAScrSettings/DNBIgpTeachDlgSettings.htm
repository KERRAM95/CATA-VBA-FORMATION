DNBIgpTeachDlgSettings Setting Repository
Quick Reference

DNBIgpTeachDlgSettings

Role: Allow user to set teachin parameters

Detailed role: All teaching parameters

The DNBIgpTeachDlgSettings repository has the following characteristics:

  • Primary Tab Page: /Resource Detailing/Teach Dialog Setting
  • Repository File: DNBIgpTeachDlgSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
