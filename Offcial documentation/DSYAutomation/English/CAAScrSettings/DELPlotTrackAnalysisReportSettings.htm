Quick Reference

DELPlotTrackAnalysisReportSettings

Role: Settings to handle the parameters in the Manufacturing Path Optimization App

Detailed role: This setting allows to customize the appearance and the behavior in Manufacturing Path Optimization

The DELPlotTrackAnalysisReportSettings repository has the following characteristics:

  • Primary Tab Page: /Manufacturing Planning/Track Analysis Assistant
  • Repository File: DELPlotTrackAnalysisReportSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DELPlotTrackAnalysisReportSettings setting repository attributes:

  • DisplayGlobalDistance_Plot_Report
    Role: Display Global Distances
    Detailed role: Display Global distances when local distances are activated.
    • Type: integer
    • Default value: 1
