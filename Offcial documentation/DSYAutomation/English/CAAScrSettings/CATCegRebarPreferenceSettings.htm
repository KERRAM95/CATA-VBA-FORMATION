Quick Reference

CATCegRebarPreferenceSettings

Role: Manages Rebar options

The CATCegRebarPreferenceSettings repository has the following characteristics:

  • Primary Tab Page: /App Preferences/3D Modeling/Multi Discipline Engineering/Building and Civil Structures Design
  • Repository File: CATCegRebarPreferenceSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CATCegRebarPreferenceSettings setting repository attributes:

  • CegReb_AllCoveringLinked_ChkBox
    Role: Parameter to link / unlink the limit coverings to the main one
    • Type: boolean
    • Default value: true
  • CegReb_DiameterRatherThanRadius_ChkBox
    Role: Parameter to choose between diameter and radius for rebar creation
    • Type: boolean
    • Default value: false
  • CegReb_Show3DTagAtCreation
    Role: Parameter to show / hide 3D Tag at creation
    • Type: boolean
    • Default value: true
