CATSrsSpaceReferentialPage Setting Repository
Quick Reference

CATSrsSpaceReferentialPage

Role: Setting to handle Space Referential app option

The CATSrsSpaceReferentialPage repository has the following characteristics:

  • Primary Tab Page: /MechanicalDesign/CATSrsPLMRepConfig/SpaceReferential
  • Repository File: CATSrsSpaceReferentialPage
  • Version: 2
  • Full Compatibility: 1
  • Upward Compatibility: 1

CATSrsSpaceReferentialPage setting repository attributes:

  • 3DSsnGridInSktCombo
    Role: 3D Session Grid in Sketcher
    Detailed role: Allow Sketcher to use 3D Session Grid
    • Type: integer
    • Default value: 1
    • Default Value Role: 3DSsnGridInSktCombo
  • WindowSpecificClipping
    Role: Activation state of clipping on the current window
    • Type: boolean
    • Default value: false
