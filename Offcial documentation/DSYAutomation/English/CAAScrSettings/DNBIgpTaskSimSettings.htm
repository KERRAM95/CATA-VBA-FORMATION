DNBIgpTaskSimSettings Setting Repository
Quick Reference

DNBIgpTaskSimSettings

Role: Setting Robot task simulation parameters

Detailed role: Robot taks simulation options

The DNBIgpTaskSimSettings repository has the following characteristics:

  • Primary Tab Page: /Resource Detailing/Robot Task Simulation
  • Repository File: DNBIgpTaskSimSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DNBIgpTaskSimSettings setting repository attributes:

  • RobotTaskCacheSimulation
    Role: option to cache DOF values during simulation run
    • Type: boolean
    • Default value: false
    • Default Value Role: Cache Simulation DOF values
  • RobotTaskCacheSimulationInfoMsg
    Role: option to show information message if not playing cached simulation
    • Type: boolean
    • Default value: false
    • Default Value Role: Show info. msg.
  • RobotTaskSimulateViaCacheValues
    Role: option to simulate robot task(s) using cached DOF values
    • Type: boolean
    • Default value: false
    • Default Value Role: Run simulation via cached DOF values.
