DNBIgpAnalysisDiscSettings Setting Repository
Quick Reference

DNBIgpAnalysisDiscSettings

Role: Settings to handle the Approach Axis, Rotation Axis of Analysis Disc in TSA and Robotics Apps

Detailed role: This setting allows to customize the properties of Analysis Disc

The DNBIgpAnalysisDiscSettings repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/DelmiaInfrastructure/Analysis Disc
  • Repository File: DNBIgpAnalysisDiscSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DNBIgpAnalysisDiscSettings setting repository attributes:

  • ApproachAxis
    Role: The rotation axis
    Detailed role: Which is the rotation axis
    • Type: short integer
    • Default value: 1
    • Default Value Role: By default +ve X-Axis is the Approach axis
    • Values:
      0 -veX-Axis
      1 X-Axis
      2 -veY-Axis
      3 Y-Axis
      4 -veZ-Axis
      5 Z-Axis
  • ClashRegionColour
    Role: Clash Region Colour
    Detailed role: Colour of Clash Region
    • Type: integer
    • Default value: 255
    • Default Value Role: Blue in RGB order
    • Min value: 0
    • Max value: 255
  • DiscSize
    Role: Radius of Analysis Disc
    • Alias: DiscSize
    • Type: double
    • Default value: 0.1
  • DiscThickness
    Role: Thickness of Analysis Disc
    • Alias: DiscThickness
    • Type: double
    • Default value: 0.002
  • DiscTransparency
    Role: Transparency of Analysis Disc
    • Alias: DiscTransparency
    • Type: double
    • Default value: 75
  • GunOCLower
    Role: Gun Open Clearance Lower Value
    • Type: double
    • Default value: 0.0000
  • GunOCUpper
    Role: Gun Open Clearance Upper Value
    • Type: double
    • Default value: 0.0000
  • GunOpenClearance_Check
    Role: Set Gun Open Clearance Option
    • Type: unsigned integer
    • Default value: 0
  • GunTALower
    Role: Gun Tip Abrasion Lower Value
    • Type: double
    • Default value: 0.0000
  • GunTAUpper
    Role: Gun Tip Abrasion Upper Value
    • Type: double
    • Default value: 0.0000
  • GunTipAbrasion_Check
    Role: Set Gun Tip Abrasion Option
    • Type: unsigned integer
    • Default value: 0
  • InterlockRegionColor
    Role: Interlock Region Color
    Detailed role: Color of Interlock Zone on Analysis Disc
    • Type: integer
    • Default value: 255
    • Default Value Role: White in RGB order
    • Min value: 0
    • Max value: 255
  • NotComputedRegionColour
    Role: Not Computed Region Colour
    Detailed role: Colour of Not Computed Region
    • Type: integer
    • Default value: 255
    • Default Value Role: White in RGB order
    • Min value: 0
    • Max value: 255
  • NotReachableRegionColour
    Role: Not Reachable Region Colour
    Detailed role: Colour of Not Reachable Region
    • Type: integer
    • Default value: 94
    • Default Value Role: Red in RGB order
    • Min value: 0
    • Max value: 255
  • ReachableRegionColour
    Role: Reachable Region Colour
    Detailed role: Colour of Reachable Region
    • Type: integer
    • Default value: 0
    • Default Value Role: Green in RGB order
    • Min value: 0
    • Max value: 255
  • RingWidth
    Role: Travel Limits ring width
    • Alias: RingWidth
    • Type: double
    • Default value: 0.005
  • RotationAxis
    Role: The rotation axis
    Detailed role: Which is the rotation axis
    • Type: short integer
    • Default value: 0
    • Default Value Role: By default Z is the rotation axis
    • Values:
      0 Z-Axis
      1 Y-Axis
      2 X-Axis
  • ToolPositionsCheck
    Role: Set Tool Positions in Analysis Disc Panel
    • Alias: ToolPositionsCheck
    • Type: unsigned integer
    • Default value: 0
  • ZbufferCheck
    Role: Set Z buffer of Analysis Disc
    • Alias: ZbufferCheck
    • Type: unsigned integer
    • Default value: 0
