DELRscClearanceSettings Setting Repository
Quick Reference

DELRscClearanceSettings

Role: Settings to define the Resource Clearances

Detailed role: These settings allow users to customize the clearance values between various resource types

The DELRscClearanceSettings repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/DELMIA Infrastructure/Resource Clearances
  • Repository File: DELRscClearanceSettings
  • Version: 5
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Relation;
    • Role: Number of Robot Motion clearance values
    • Detailed role:
    • src: NbRobotMotionVals
    • dest: RobotMotionClearVals
    • kind: dimensionOfArray

    • Min value: 0

    • Min value: 0
    • Max value: 255
  • GunBodyTemplate

  • GunEndColor
