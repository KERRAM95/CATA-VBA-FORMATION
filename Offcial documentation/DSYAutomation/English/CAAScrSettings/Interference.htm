Interference Setting Repository
Quick Reference

Interference

Role: Setting to handle the Interference parameters

The Interference repository has the following characteristics:

  • Primary Tab Page: /Digital Mockup/Interference
  • Repository File: Interference
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

Interference setting repository attributes:

  • AutomaticMetricDisplay
    Role: Automatically display loaded metric
    • Tab Page: /Digital Mockup/Interference
    • Type: integer
    • Default value: 1
  • ClashCurveAndVolume
    Role: Draw clash volume and curve
    • Tab Page: /Digital Mockup/Interference
    • Type: integer
    • Default value: 0
    • Default Value Role: Curve
    • Values:
      0 Curve
      1 Volume
      2 Curve and volume
  • ClearanceAreaRediscretization
    Role: Area accuracy
    • Type: double
    • Default value: 10.00
    • Min value: 0.00001
    • Max value: 10e+307
    • Step: 1.0
  • ClearanceDisplayMode
    Role: Clearance Triangle and Area
    • Tab Page: /Digital Mockup/Interference
    • Type: integer
    • Default value: 1
    • Default Value Role: Triangle
    • Values:
      1 Triangle
      2 Area
  • ColorITFParts
    Role: Select color according to instance
    • Tab Page: /Digital Mockup/Interference
    • Type: integer
    • Default value: 0
    • Values:
      0 Select color according to instance
      1 Select color according to interference
  • ColorInterferrringPart1
    Role: Select color for first interferring part
    • Tab Page: /Digital Mockup/Interference
    • Type: unsigned integer
    • Default value: 255
    • Default Value Role: Blue in RGB order
    • Min value: 0
    • Max value: 255
  • ColorInterferrringPart2
    Role: Select color for second interferring part
    • Tab Page: /Digital Mockup/Interference
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: Yellow in RGB order
    • Min value: 0
    • Max value: 255
  • ColorInterferrringPartClash
    Role: Select color for clashing parts
    • Tab Page: /Digital Mockup/Interference
    • Type: unsigned integer
    • Default value: 47
    • Default Value Role: Red in RGB order
    • Min value: 0
    • Max value: 255
  • ColorInterferrringPartClear
    Role: Select color for parts in clearance
    • Tab Page: /Digital Mockup/Interference
    • Type: unsigned integer
    • Default value: 75
    • Default Value Role: Green in RGB order
    • Min value: 0
    • Max value: 255
  • ColorInterferrringPartContact
    Role: Select color for parts in contact
    • Tab Page: /Digital Mockup/Interference
    • Type: unsigned integer
    • Default value: 46
    • Default Value Role: Orange in RGB order
    • Min value: 0
    • Max value: 255
  • CustomizeDisplay
    Role: Display Customized instance names in multi list
    • Tab Page: /Digital Mockup/Interference
    • Type: integer
    • Default value: 0
  • DisplayItfGeometryClash
    Role: Draw clash volume and curve
    • Tab Page: /Digital Mockup/Interference
    • Type: integer
    • Default value: 1
    • Default Value Role: Curve
    • Values:
      0 None
      1 Curve
      2 Volume
      3 Curve and volume
  • DisplayItfGeometryClearance
    Role: Draw Clearance area and triangles
    • Tab Page: /Digital Mockup/Interference
    • Type: integer
    • Default value: 1
    • Default Value Role: Triangles
    • Values:
      0 None
      1 Triangles
      2 Area
  • DisplayItfGeometryContact
    Role: Draw Contact triangles
    • Tab Page: /Digital Mockup/Interference
    • Type: integer
    • Default value: 1
    • Default Value Role: Triangles
    • Values:
      0 None
      1 Triangles
  • DisplayItfGeometryMode
    Role: Dynamically draw interference geometry
    • Tab Page: /Digital Mockup/Interference
    • Type: integer
    • Default value: 1
    • Values:
      1 Draw only geometries corresponding to interference type
      2 Draw all clashes, contacts and clearances geometries for each interference
  • DisplayItfQuantifierClash
    Role: Draw clash quantifiers
    • Tab Page: /Digital Mockup/Interference
    • Type: integer
    • Default value: 1
    • Default Value Role: Penetration vector
    • Values:
      0 None
      1 Penetration vector
      2 Penetration volume
      3 Vector and volume
  • DisplayItfQuantifierClearance
    Role: Draw Clearance quantifiers
    • Tab Page: /Digital Mockup/Interference
    • Type: integer
    • Default value: 1
    • Default Value Role: Minimum distance

      0 None
      1 Minimum distance
  • DisplayItfStoredResult
    Role: Display Stored Result
    • Tab Page: /Digital Mockup/Interference
    • Type: integer
    • Default value: 0
  • DrawInterferenceGeometry
    Role: Dynamically draw interference geometry
    • Tab Page: /Digital Mockup/Interference

    • Default value: 1
    • Values:

    Role: Display Interference Icon
    • Tab Page: /Digital Mockup/Interference

      0 HTML
