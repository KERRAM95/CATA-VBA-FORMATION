DNBImportD5 Setting Repository
Quick Reference

DNBImportD5

Role: Defines Settings to control conversion of DELMIA D5 files

The DNBImportD5 repository has the following characteristics:

  • Primary Tab Page: /General/Compatibility/DELMIA D5
  • Repository File: DNBImportD5
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DNBImportD5 setting repository attributes:

  • AnnotationsImport
    Role: Activation of annotations and dimensions import.
    • Type: integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • ConfigPaths
    Role: D5 Config Paths
    • Type: URN
    • Default value:
  • CoorsysImport
    Role: Activation of coorsys import.
    • Type: integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • D5Cache
    Role: D5 Cache Path
    • Type: URN
    • Default value:
  • D5ImportClassify
    Role: Imported device library classification in 3DX.
    • Type: integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • D5Root
    Role: D5 Root Path
    • Type: URN
    • Default value:
  • KinCorrectionOption
    Role: Activation of auto correction mode.
    • Type: boolean
    • Default value: true
  • ToolFrameImport
    Role: Activation of tool frame import.
    • Type: integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • UDAImport
    Role: Activation of user defined attributes import.
    • Type: integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • WcellPathsImport
    Role: Activation of workcell paths import.
    • Type: integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
