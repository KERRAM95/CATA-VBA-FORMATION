MultiCAD Setting Repository
Quick Reference

MultiCAD

Role: defines Settings to control conversion of MultiCAD supported files

The MultiCAD repository has the following characteristics:

  • Primary Tab Page: /General/Compatibility/ExternalFormats
  • Repository File: MultiCAD
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

MultiCAD setting repository attributes:

  • 3D annotations
    Role: To convert or not to convert 3D Annotations
    Detailed role: idem
    • Alias: Annotation3D
    • Type: integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • Active layers only
    Role: To specify that only active layers should be taken into account
    • Alias: UGActiveLayersOnly
    • Type: integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • Convert construction surfaces
    Role: To convert or not construction surfaces
    Detailed role: idem
    • Alias: ConstructionSurfaces
    • Type: integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • Enable Quality enhancement at export
    Role: Enables quality enhancements at export.
    • Alias: HealingActive
    • Type: short integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • Geometry Quality Enhancement Mode
    Role: Enhancement mode
    • Alias: GeomQualityMode
    • Type: short integer
    • Default value: 0
    • Values:
      0 Default enhancement without deformation
      1 Advanced enhancement with customized allowed deformation
      Detailed role: To define a maximum tolerance value which may enhance the quality.
  • GroupingNode
    Role: Grouping node
    • Alias: GroupingNode
    • Type: short integer
    • Default value: 2
    • Values:
      0 Site: All content below Site nodes will be grouped inside a 3DShape
      1 Building: All content below Building nodes will be grouped inside a 3DShape
      2 Storey: All content below Storey nodes will be grouped inside a 3DShape
  • ImportOneCGR
    Role: Import part as one CGR
    Detailed role: idem
    • Alias: ImportOneCGR
    • Type: integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • ImportProfile
    Role: Import profile
    • Alias: ImportProfile
    • Type: short integer
    • Default value: 0
    • Values:
      0 For Construction: Create a fully exposed PLM model
      1 For Design: Create a lightweight model while keeping exact geometry and the attributes
      2 For Coordination: Create a lightweight model as a single CGR
  • Input Tessellation
    Role: Define tessellation value for input document
    • Alias: InputTessellationParam
    • Type: float
    • Default value: 0.05
    • Min value: 0.001
    • Max value: 0.10
    • Step: 0.001
  • Link Mode
    Role: Choose if you want to obtain exact or tessellated geometry
    Detailed role: idem
    • Alias: ActiveExactGeometry
    • Type: integer
    • Default value: 1
    • Min value: 0
    • Max value: 1
  • LinkedFiles
    Role: Linked files
    • Alias: LinkedFiles
    • Type: integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • Maximum Shape Deformation Factor for Geometry Quality Enhancement
    Role: To define the maximum deformation allowed in the Geometry Quality Enhancement. This is a factor of the Resolution.
    • Alias: GeomQualityTolFact
    • Type: short integer
    • Default value: 5
    • Min value: 5
    • Max value: 100
  • Mode of the Parts Parameterization
    Role: To define the mode of the Parts Parameterization
    • Alias: PartsParameterMode
    • Type: integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • Mode of the ProE instance name

    • Default value: 0
    • Min value: 0
    • Max value: 1
  • Mode of the ProE simplified representation
