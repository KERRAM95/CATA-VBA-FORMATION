DNBOlpSettings Setting Repository
Quick Reference

DNBOlpSettings

Role: Options for robot offline programming

The DNBOlpSettings repository has the following characteristics:

  • Primary Tab Page: /Robotics/Offline Programming
  • Repository File: DNBOlpSettings
  • Version: 4
  • Full Compatibility: 2
  • Upward Compatibility: 2

DNBOlpSettings setting repository attributes:

  • ControllerInfo
    Role: Controller Information
    Detailed role: A list of available robot controllers and their corresponding content
    • Type: string
    • Default value: UNIVERSAL
  • DownloaderV2
    Role: Downloader Information
    Detailed role: A downloader that is available for a corresponding controller
    • Type: string
    • Default value: FD
  • JavaClassPath
    Role: Java class path
    Detailed role: A list of JAR-files or a java package directory structure which contains CLASS-files to add to the java class path. This class path will be used for download XSLT translators and upload java parsers. The list should most likely contain DNBIgpOlpXSLExtension.jar and xalan.jar.
    • Type: string
    • Default value: UNIX: ${CATInstallPath}/docs/java/DNBIgpOlpXSLExtension.jar:${CATInstallPath}/startup/OLP/Translators/DownloaderTools/EncodingConverter.jar Windows: ${CATInstallPath}\docs\java\DNBIgpOlpXSLExtension.jar;${CATInstallPath}\startup\OLP\Translators\DownloaderTools\EncodingConverter.jar
  • JavaExe
    Role: Java executable directory
    Detailed role: The location where java.exe is located. This is used for XSLT translators on download and Java parsers on upload.
    • Type: URN
    • Default value: UNIX: ${JAVA_HOME}/bin/javaw.exe Windows: ${JAVA_HOME}\bin\javaw.exe
  • ShowPulseCounts
    Role: Show Pulse Counts Configuration
    Detailed role: A setting to enable/disable showing pulse count values in DOF mapping
    • Type: string
    • Default value:
  • ShowRrsAutoDowns
    Role: Show RRS-II Auto Download Configuration
    Detailed role: A setting to enable/disable showing RRS-II automatic download options
    • Type: string
    • Default value:
  • Translator
    Role: Translator Information
    Detailed role: A translator that is available for a corresponding controller
    • Type: string
    • Default value: DELMIA UniversalRobots Translator
  • TranslatorPLMXID
    Role: Translator PLM_ExternalID Information
    Detailed role: A translator PLM_ExternalID that is available for a corresponding controller
    • Type: string
    • Default value:
  • TranslatorRevision
    Role: Translator Revision Information
    Detailed role: A translator revision that is available for a corresponding controller
    • Type: string
    • Default value:
  • UploaderV2
    Role: Uploader Information
    Detailed role: An uploader that is available for a corresponding controller
    • Type: string
    • Default value: FD
