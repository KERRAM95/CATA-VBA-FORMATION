CATPpxOptionData_Panels Setting Repository
Quick Reference

CATPpxOptionData_Panels

Role: Setting to handle PPX parameters

The CATPpxOptionData_Panels repository has the following characteristics:

  • Primary Tab Page: /Systems/Elements
  • Repository File: CATPpxSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CATPpxOptionData_Panels setting repository attributes:

  • NewLineControl
    Role: Parameter to enable/disable the display of the new line control
    • Type: boolean
    • Default value: true
  • Scale
    Role: Parameter to set the size scaling factor of the panels
    • Type: integer
    • Default value: 100
    • Default Value Role: scale
    • Min value: 80
    • Max value: 140
