AmbianceStudioSettings Setting Repository
Quick Reference

AmbianceStudioSettings

Role: Ambiance Studio options

The AmbianceStudioSettings repository has the following characteristics:

  • Primary Tab Page: /Interactive_Experience/Ambiance_Studio/AmbianceGeneral
  • Repository File: AmbianceStudioSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

AmbianceStudioSettings setting repository attributes:

  • ComputationHardware
    Role: Computation hardware
    • Type: string
    • Default value: CPU
  • InteractiveLightsBlur
    Role: Enable interactive lights blur
    • Type: integer
    • Default value: 1
  • InteractiveSourceSize
    Role: Interactive source light size
    • Type: string
    • Default value: 512*256
