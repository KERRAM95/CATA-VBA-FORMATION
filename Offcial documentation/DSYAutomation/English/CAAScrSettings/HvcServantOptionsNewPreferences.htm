HvcServantOptionsNewPreferences Setting Repository
Quick Reference

HvcServantOptionsNewPreferences

Role: Manage options for visualization servant

Detailed role: Manage options for simulation results visualization servant

The HvcServantOptionsNewPreferences repository has the following characteristics:

  • Primary Tab Page: /AnalysisSimulation/Servant Visualization
  • Repository File: HvcServantOptions
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

HvcServantOptionsNewPreferences setting repository attributes:

  • AutoUpdate
    Role: call AutoUpdate for Tabular Export Feature
    • Type: integer
    • Default value: 0
    • Values:
      0 exit
      1 call AutoUpdate for Tabular Export Feature
  • AvoidCache
    Role: Avoid caching geometry from servant to save memory. Performance will suffer.
    • Type: integer
    • Default value: 0
  • DRMModeForCloudBatchExecution
    Role: (enum: Built-in DRM, LSF DRM) Choice of DRM.
    • Type: integer
    • Default value: 0
    • Values:
      0 Built-in
      1 LSF
  • DRMModeForCloudSimulationExecution
    Role: (enum: Built-in DRM, LSF DRM) Choice of DRM.
    • Type: integer
    • Default value: 0
    • Values:
      0 Built-in
      1 LSF
  • Decimation
    Role: Specify whether to use decimation for NavRep generation.
    • Type: integer
    • Default value: 0
  • DecimationValue
    Role: When decimation is specified, specify the value to use for decimation. 0 for low, 1 for medium, 2 for high.
    • Type: integer
    • Default value: 1
    • Min value: 1
    • Max value: 4
    • Step: 1
  • DomainDecomposition
    Role: Specify whether to use parMetis or RCB method to perform domain decomposition.
    • Type: integer
    • Default value: 2
  • DynamicAnimation
    Role: Specify whether to automatically switch to a dynamics display style when playing an animation.
    • Type: integer
    • Default value: 1
  • ElementOrderReduction
    Role: Specify whether to use element order reduction.
    • Type: integer
    • Default value: 0
  • FieldCache
    Role: Specify if fields should be cached over multiple frames.
    • Type: integer
    • Default value: 0
  • HideLabelsViewManip
    Role: Specify whether to hide node and element labels when performing view manipulations.
    • Type: integer
    • Default value: 0
  • LSFGroupForCloudBatchExecution
    Role: LSF Group name where Batch execution is to be performed.
    • Type: string
    • Default value:
  • LSFGroupForCloudSimulationExecution
    Role: LSF Group name where simulation execution is to be performed.
    • Type: string
    • Default value:
  • LSFOptionsForCloudBatchExecution
    Role: Additional LSF options for batch execution.
    • Type: string
    • Default value:
  • LSFOptionsForCloudSimulationExecution
    Role: Additional LSF options for simulation execution.
    • Type: string
    • Default value:
  • LSFQueueForCloudBatchExecution
    Role: LSF Queue name where Batch execution is to be performed.
    • Type: string
    • Default value:
  • LSFQueueForCloudSimulationExecution
    Role: LSF Queue name where simulation execution is to be performed.
    • Type: string
    • Default value:
  • LocalExecutionMaxNbCores
    Role: Use the maximum core of the machine.
    • Type: integer
    • Default value: 1
  • LocalMachine
    Role: Specify whether to use a local machine or a remote machine.
    • Type: integer
    • Default value: 1
  • NormalCompression
    Role: Specify whether to use normal compression.
    • Type: integer
    • Default value: 1
  • RemoteServantCores
    Role: Specify how many cores to utilize on the servant.
    • Type: integer
    • Default value: 4
    • Min value: 1
    • Max value: 8
  • ResultsStorageWarning
    Role: Display results storage warning.
    • Type: integer
    • Default value: 1
  • SIMReaderType
    Role: Specify which kind of SIM Reader to use.
    • Type: integer
    • Default value: 1
  • ServantCores
    Role: Specify how many cores to utilize on the servant.
    • Type: integer
    • Default value: 4
  • ServantGridName
    Role: Specify the name of the submission host to use for the servant.
    • Type: string
    • Default value:
  • ServantName
    Role: Specify the name of the machine to use for the servant.
    • Type: string
    • Default value:
  • ShowVisuSelectionModeUI
    Role: Shows the visualization mode dialog while switching to results.
    • Type: integer
    • Default value: 1
  • StationNameForCloudBatchExecution
    Role: Hostname where Batch execution is to be performed.
    • Type: string
    • Default value:
  • StationNameForCloudSimulationExecution
    Role: Hostname where simulation execution is to be performed.
    • Type: string
    • Default value:
  • UseCloudVizualization
    Role: Specify whether to use Cloud for Visualization or not.
    • Type: integer
    • Default value: 0
  • UseInstancing
    Role: Specify whether to use graphic instancing of Vector/Tensor symbols and DEM nodes.
    • Type: integer
    • Default value: 0
  • UseLgFileSettings
    Role: Specify whether to use PLM server large file management settings or not.
    • Type: integer
    • Default value: 0
  • ValetName
    Role: Specify the name of the machine to use for the valet.
    • Type: string
    • Default value:
  • WireframeViewManip
    Role: Specify whether to render the model in wireframe when performing view manipulations.
    • Type: integer
    • Default value: 0
