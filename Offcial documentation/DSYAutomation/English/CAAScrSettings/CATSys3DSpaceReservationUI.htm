CATSys3DSpaceReservationUI Setting Repository
Quick Reference

CATSys3DSpaceReservationUI

Role: Systems Space Reservation setting

Detailed role: Systems Space Reservation setting

The CATSys3DSpaceReservationUI repository has the following characteristics:

  • Primary Tab Page: /3D Modeling/Systems Architecture/Systems Space Reservation
  • Repository File: CATSys3DSpaceReservationUI
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CATSys3DSpaceReservationUI setting repository attributes:

  • ObjectCompatibilityWithPathway
    Role: Ensure object is compatible with pathway
    Detailed role: Object and pathway should be compatible
    • Type: integer
    • Default value: 0
    • Default Value Role: By default compatibility between object and pathway is not mandatory
    • Values:
      0 Compatibility between object and pathway is not mandatory
      1 Compatibility between object and pathway is mandatory
  • SinglePathwayToPCP
    Role: Allow only one pathway to a pathway connection point
    Detailed role: Only one pathway can be connected to pathway connection point
    • Type: integer
    • Default value: 1
    • Default Value Role: By default only one pathway to a pathway connection point is allowed
    • Values:
      0 Allow multiple pathway to a pathway connection point
      1 Allow only one pathway to a pathway connection point
  • Sys3DArchitectureAttrPropOption
    Role: Will decide if to propogate systems 3D architecture attributes
    Detailed role: Will decide if to propogate systems 3D architecture attributes
    • Type: integer
    • Default value: 0
    • Default Value Role: By default the Option will be Off to not to allow propogation of systems 3D architecture Attributes
    • Values:
      0 The systems 3D architecture atributes will not be considered for propogate
      1 The systems 3D architecture atributes will be considered for propogate
  • Sys3DAutoPCPCreation
    Role: Will decide if to automatically start creation of pathway connection point
    Detailed role: Will decide if to Automatically start creation of pathway connection point
    • Type: integer
    • Default value: 1
    • Default Value Role: By default the Option will be on to display warning to choose creation of pathway connection point
    • Values:
      0 Automatically start creation of pathway connection point
      1 Display warning to choose creation of pathway connection point
  • Sys3DDirectionalLengthCoefficient
    Role: The coefficient used to compute the directional length
    Detailed role: The coefficient multipled with major dimension of pathway section to compute the directional length
    • Type: double
    • Default value: 2.0
    • Default Value Role: By default the directional length will be twice the major dimension
