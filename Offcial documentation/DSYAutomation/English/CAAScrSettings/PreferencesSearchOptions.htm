PreferencesSearchOptions Setting Repository
Quick Reference

PreferencesSearchOptions

Role: Settings for Preferences search

The PreferencesSearchOptions repository has the following characteristics:

  • Primary Tab Page: /General/PreferencesSearch
  • Repository File: PreferencesSearchOptions
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

PreferencesSearchOptions setting repository attributes:

  • IndexGenerationInProgress
    Role: Index generation status
    • Type: boolean
    • Default value: false
  • NumberOfPagesToExpand
    Role: Number of pages to expand in a tree node
    Detailed role: Number of pages to expand in a tree node
    • Type: integer
    • Default value: 5
    • Min value: 1
    • Max value: 50
  • PageExpandOptions
    Role: Page expand options
    • Type: integer
    • Default value: 1
    • Values:
      0 All pages in a node
      Detailed role: Options for expanding all pages in any Preference Tree node
      1 Specific number of pages in a node
      Detailed role: Option for specific number of pages form user input
  • TreeFiltering
    Role: Parameter to set Preferences Tree Filtereing ON or OFF
    • Type: boolean
    • Default value: false
