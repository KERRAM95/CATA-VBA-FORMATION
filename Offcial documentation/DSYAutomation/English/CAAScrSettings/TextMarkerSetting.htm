TextMarkerSetting

Role: Setting to handle the text markers visual properties and parameters

The TextMarkerSetting repository has the following characteristics:

  • Primary Tab Page: /Digital Mockup/General
  • Repository File: TextMarker
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
