SMAModelEditor Setting Repository
Quick Reference

SMAModelEditor

Role: Settings for SIMULIA Model Editor

Detailed role: Automatic file name generation for External Simulation Representations. Options to specify generation with the Representation name or without. Use Representation attribute values at creation time. Specify file name extension.

The SMAModelEditor repository has the following characteristics:

  • Primary Tab Page: /AnalysisSimulation/Model Editor
  • Repository File: SMAModelEditor
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

SMAModelEditor setting repository attributes:

  • EnableGeneration
    Role: Parameter to set automatic file name generation
    Detailed role: Check to enable automatic file name generation. Uncheck to disable automatic file name generation.
    • Type: boolean
    • Default value: true
  • Extension
    Role: File name extension
    Detailed role: Specify the generated file name extension. Must not be empty.
    • Type: string
    • Default value: inp
  • PatternChoice
    Role: Choice of pattern with or without representation name
    Detailed role: Choose one of two options for the file name pattern. The first uses the Representation name, and the second does not use the Representation name.
    • Type: string
    • Default value: with
    • Values:
      with File name generation with representation name
      without File name generation without representation name
  • WithRepNamePrefix
    Role: Prefix string preceding representation name
    Detailed role: Specify the string that will appear before the Representation name in the generated file name. Can be empty.
    • Type: string
    • Default value:
  • WithRepNameSuffix
    Role: Suffix string after representation name
    Detailed role: Specify the string that will appear after the Representation name in the generated file name. Can be empty.
    • Type: string
    • Default value:
  • WithoutRepNameText
    Role: Full pattern string without representation name
    Detailed role: Specify the full string that will appear the generated file name. Must not be empty.
    • Type: string
    • Default value:
