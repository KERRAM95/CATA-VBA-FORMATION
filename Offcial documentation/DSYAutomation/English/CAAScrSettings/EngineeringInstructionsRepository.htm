EngineeringInstructionsRepository Setting Repository
Quick Reference

EngineeringInstructionsRepository

Role: Setting to handle the Instruction Text"

Detailed role: This setting allows to customize the Instruction Text Colors

The EngineeringInstructionsRepository repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/RFLP Editor/RFLP Editor Properties
  • Repository File: EngineeringInstructionsRepository
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

EngineeringInstructionsRepository setting repository attributes:

  • ApplyOnlyReleasedTemplates
    Role: Allow application of released Templates only
    Detailed role: This preference prevents user from applying a non-released Instruction Template
    • Type: boolean
    • Default value: false
  • ForceUppercase
    Role: Force uppercase of text in Engineering Notes Template
    Detailed role: This preference allows user to force uppercase on Note Template Text
    • Type: boolean
    • Default value: true
  • InstructionTextListValues
    Role: Instruction Text List Values Color
    Detailed role: Instruction Text List Values Color
    • Type: unsigned integer
    • Default value: 36
    • Default Value Role: Instruction Text List Values Color
    • Min value: 0
    • Max value: 255
  • InstructionTextModifiable
    Role: Instruction Modifiable Text Color
    Detailed role: Instruction Modifiable Text Color
    • Type: unsigned integer
    • Default value: 124
    • Default Value Role: Instruction Modifiable Text Color
    • Min value: 0
    • Max value: 255
  • InstructionTextOptional
    Role: Instruction Optional Text Color
    Detailed role: Instruction Optional Text Color
    • Type: unsigned integer
    • Default value: 118
    • Default Value Role: Instruction Optional Text Color
    • Min value: 0
    • Max value: 255
  • InstructionTextProcess
    Role: Instruction Process Text Color
    Detailed role: Instruction Process Text Color
    • Type: unsigned integer
    • Default value: 210
    • Default Value Role: Instruction Process Text Color
    • Min value: 0
    • Max value: 255
  • InstructionTextProduct
    Role: Instruction Product Text Color
    Detailed role: Instruction Product Text Color
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: Instruction Product Text Color
    • Min value: 0
    • Max value: 255
  • InstructionTextRawMat
    Role: Instruction Raw Material Text Color
    Detailed role: Instruction Raw Material Text Color
    • Type: unsigned integer
    • Default value: 71
    • Default Value Role: Instruction Raw Material Text Color
    • Min value: 0
    • Max value: 255

    Role: Block the enforced uppercase option in Engineering Notes Template
    Detailed role: This preference prevents user from applying a force uppercase option on Note Template
    • Type: boolean
    • Default value: false
