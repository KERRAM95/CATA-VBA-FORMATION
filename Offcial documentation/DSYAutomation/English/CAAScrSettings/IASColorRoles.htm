IASColorRoles Setting Repository
Quick Reference

IASColorRoles

Role: Colors of the ICEM Design Experience App

Detailed role: Colors of the ICEM Design Experience App

The IASColorRoles repository has the following characteristics:

  • Primary Tab Page:  
  • Repository File: IASColorRoles
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

IASColorRoles setting repository attributes:

  • BarrelArcBodyColorLabel
    Role: Color of the point of the Barrel arc revolution
    • Type: unsigned integer
    • Default value: 255
    • Min value: 0
    • Max value: 255
  • BarrelDefaultColorLabel
    Role: Default color of the point of the Barrel
    • Type: unsigned integer
    • Default value: 255
    • Min value: 0
    • Max value: 255
  • BarrelGuideBodyColorLabel
    Role: Color of the point of the Barrel arc guide
    • Type: unsigned integer
    • Default value: 0
    • Min value: 0
    • Max value: 255
  • CurveBoxModifierColorLabel
    Role: Color of the Curve Box Modifier
    • Type: unsigned integer
    • Default value: 242
    • Min value: 0
    • Max value: 255
  • CurveBoxModifierHighlightColorLabel
    Role: Highlight color of the Curve Box Modifier
    • Type: unsigned integer
    • Default value: 128
    • Min value: 0
    • Max value: 255
  • CurveBoxModifierPivotPointColorLabel
    Role: Color of the pivot point of the Curve Box Modifier
    • Type: unsigned integer
    • Default value: 0
    • Min value: 0
    • Max value: 255
  • CurveBoxModifierPivotPointHighlightColorLabel
    Role: Highlight color of the pivot point of the Curve Box Modifier
    • Type: unsigned integer
    • Default value: 128
    • Min value: 0
    • Max value: 255
  • CurveBoxModifierPivotPointLowlightColorLabel
    Role: Lowlight color of the pivot point of the Curve Box Modifier
    • Type: unsigned integer
    • Default value: 125
    • Min value: 0
    • Max value: 255
  • HighContrastRepositoryExists
    Role: Does a High Contrast version of this repository exist?
    Detailed role: If true, must have the same name + _HighContrast and all color entries must be mirrored there with appropriate values
    • Type: boolean
    • Default value: false
  • PreviewInput
    Role: Color of the input object during Preview
    • Type: unsigned integer
    • Default value: 128
    • Min value: 0
    • Max value: 255
  • PreviewMousePointer
    Role: Color of the mouse pointer during Preview
    • Type: unsigned integer
    • Default value: 0
    • Min value: 0
    • Max value: 255
  • PreviewOutput
    Role: Color of the output object during Preview
    • Type: unsigned integer
    • Default value: 128
    • Min value: 0
    • Max value: 255
  • PreviewProjection
    Role: Color of projections during Preview
    • Type: unsigned integer
    • Default value: 128
    • Min value: 0
    • Max value: 255
  • ResourcesFileExists
    Role: Does a resources file for this repository exist?
    Detailed role: If true, must have a CATNls file with the same name. For each attribute a AttributeName.Title entry is required there.
    • Type: boolean
    • Default value: true
  • SelectionBackgroundLabel
    Role: Color of the Background Label
    • Type: unsigned integer
    • Default value: 255
    • Min value: 0
    • Max value: 255
  • SelectionHighlightSlot0
    Role: Color of the Slot 0
    • Type: unsigned integer
    • Default value: 255
    • Min value: 0
    • Max value: 255
  • SelectionHighlightSlot1
    Role: Color of the Slot 1
    • Type: unsigned integer
    • Default value: 76
    • Min value: 0
    • Max value: 255
  • SelectionHighlightSlot2
    Role: Color of the Slot 2
    • Type: unsigned integer
    • Default value: 64
    • Min value: 0
    • Max value: 255
  • SelectionHighlightSlot3
    Role: Color of the Slot 3
    • Type: unsigned integer
    • Default value: 255
    • Min value: 0
    • Max value: 255
  • SelectionTableHighlight
    Role: Color used to mark the mouse over'ed element in the selection table
    • Type: unsigned integer
    • Default value: 255
    • Min value: 0
    • Max value: 255
