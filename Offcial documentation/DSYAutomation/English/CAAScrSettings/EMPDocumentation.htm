EMPDocumentation Setting Repository
Quick Reference

EMPDocumentation

Role: Short Qualification of the usage of the setting

Detailed role: Full explanation of the Setting Role

The EMPDocumentation repository has the following characteristics:

  • Primary Tab Page: /Equipments/Electrical Harness Flattening/Documentation
  • Repository File: EMPDocumentation
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

EMPDocumentation setting repository attributes:

  • DimensionCheckComponents
    Role: Option to select whether to check dimension on components
    Detailed role: Option to select whether to check dimension on components
    • Type: integer
    • Default value: 1
    • Default Value Role: By default this option is check for components
    • Values:
      0 No Check for components
      1 Check for components
  • DimensionDevice
    Role: Option to select how to dimension on device
    Detailed role: Option to select whether to dimension on device from segment connection point, bounding box point or cutting line point
    • Type: integer
    • Default value: 0
    • Default Value Role: By default this option is device segment connection point
    • Values:
      0 Dimension from device segment connection point
      1 Dimension from device bounding box point
      2 Dimension from device cutting line point
  • DimensionLineUp
    Role: Option to select whether to line up dimensions
    Detailed role: Option to select whether to line up dimensions
    • Type: integer
    • Default value: 1
    • Default Value Role: By default this option is check for dimensions
    • Values:
      0 No line up for dimensions
      1 Line up for dimensions
  • DimensionMechPart
    Role: Option to select whether to dimension on publish point or not
    Detailed role: Option to select whether to dimension on publish point or not
    • Type: integer
    • Default value: 1
    • Default Value Role: By default this option is dimension on publish point
    • Values:
      0 Not to dimension on publish point
      1 Dimension on publish point
  • DimensionProtection
    Role: Option to select whether to dimension on protection start or center
    Detailed role: Option to select whether to dimension on protection start or center
    • Type: integer
    • Default value: 1
    • Default Value Role: By default this option is dimension on protection start
    • Values:
      0 Not to dimension protection
      1 Dimension on protection start
      2 Dimension on protection center
  • DimensionRounding
    Role: Option to define roundup value
    • Type: string
    • Default value: 10 mm
  • DimensionSplice
    Role: Option to select whether to dimension splice on position point or not
    Detailed role: Option to select whether to dimension splice on position point or not
    • Type: integer
    • Default value: 1
    • Default Value Role: By default this option is dimension on position point
    • Values:
      0 Not to dimension on position point
      1 Dimension on position point
  • DimensionSupport
    Role: Option to select whether to dimension on support start or center
    Detailed role: Option to select whether to dimension on support start or center
    • Type: integer
    • Default value: 1
    • Default Value Role: By default this option is dimension on support start
    • Values:
      0 Not to dimension support
      1 Dimension on support start
      2 Dimension on support center
  • DocumentationProcessType
    Role: Option to select type of documentation process
    Detailed role: Option to select type of documentation process whether to run standard or advanced documentation process
    • Type: integer
    • Default value: 0
    • Default Value Role: By default documentation process will run with standard behaviour
    • Values:
      0 Standard
      1 Advanced
  • GVSSelection
    Role: Option to select GVS from a discrete list
    Detailed role: Option to select GVS from a discrete list
    • Type: URN
    • Default value: DefaultGenerativeStyle.xml
    • Default Value Role: By default GVS
  • LengthLimitsMax
    Role: Option to select the maximum length
    Detailed role: Option to select maximum length
    • Type: integer
    • Default value: 1
    • Default Value Role: Select Max
    • Values:
      0 Do not Select Max
      1 Select Max
  • LengthLimitsMaxSpinner
    Role: Option to select the maximum length
    Detailed role: Option to select maximum length
    • Type: double
    • Default value: 0.01
    • Min value: 0.01
    • Max value: 10e+307
    • Step: 0.001
  • LengthLimitsMin
    Role: Option to select the minimum length
    Detailed role: Option to select minimum length
    • Type: integer
    • Default value: 0
    • Default Value Role: Select min
    • Values:
      0 Do not select min
      1 Select min
  • LengthLimitsMinSpinner
    Role: Option to select the minimum length
    Detailed role: Option to select minimum length
    • Type: double
    • Default value: 0.01
    • Min value: 0.01
    • Max value: 10e+307
    • Step: 0.001
  • ScaleOption
    Role: Option to decide whether to scale or not
    Detailed role: Option to decide whether to scale based on some formula or not
    • Type: integer
    • Default value: 1
    • Default Value Role: By default this option is to scale
    • Values:
      0 Not to scale
      1 Scale
  • ScaleSegments
    Role: Option to decide whether to scale or not
    Detailed role: Option to decide whether to scale based on some formula or not
    • Type: integer
    • Default value: 1
    • Default Value Role: By default this option is to scale
    • Values:
      0 Not to scale
      1 Scale
  • ScalingFormula
    Role: Option to decide the scaling formula
    Detailed role: Option to decide the scaling formula
    • Type: integer
    • Default value: 0
    • Default Value Role: By default this option is logarithmic
    • Values:
      0 Logarithmic
      1 Linear
      2 Equal
