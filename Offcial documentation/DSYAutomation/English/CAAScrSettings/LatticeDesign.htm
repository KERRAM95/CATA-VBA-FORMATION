LatticeDesign Setting Repository
Quick Reference

LatticeDesign

Role: Setting to manage the options of LatticeDesign

Detailed role: LatticeDesign setting contains options necessary to the commands of Lattice Design

The LatticeDesign repository has the following characteristics:

  • Primary Tab Page: /Lattice Design/Lattice Design/General
  • Repository File: Lattice
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

LatticeDesign setting repository attributes:

  • Deactivate_GPGPU_Computation
    Role: Setting to disable the TPMS Volume button
    Detailed role: Hide the TPMS Volume button
    • Type: boolean
    • Default value: false
  • Enable_Standard_Values
    Role: Setting to enable Standard Definition values
    Detailed role: Displays standard values of beam length and beam radius
    • Type: boolean
    • Default value: false
