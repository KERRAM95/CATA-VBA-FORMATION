CATPpxOptionData_PlayMode Setting Repository
Quick Reference

CATPpxOptionData_PlayMode

Role: Setting to handle PPX parameters

The CATPpxOptionData_PlayMode repository has the following characteristics:

  • Primary Tab Page: /Systems/Elements
  • Repository File: CATPpxSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CATPpxOptionData_PlayMode setting repository attributes:

  • AnimationLengthMS
    Role: The duration of the animated transition between Variants
    • Type: float
    • Default value: 5.0
    • Min value: 0.0
    • Max value: 3600.0
  • AnimationSmoothing
    Role: Enable smoothing of the animation at the beginning and end
    • Type: boolean
    • Default value: true
  • FasterTriggerDetection
    Role: Enable faster trigger detection
    • Type: boolean
    • Default value: false
  • SlideshowDelayMS
    Role: The delay before activating the next Variant in the Line
    • Type: float
    • Default value: 1.0
    • Min value: 0.0
    • Max value: 3600.0
  • StageCaptureViewpoint
    Role: Enable the capturing of the viewpoint when capturing the stage variant
    • Type: boolean
    • Default value: true
  • VRHMDFocalLength
    Role: HMD focal length
    • Type: string
    • Default value: 16
    • Values:
      16 HTCVive
      18 WMR
      11 XTAL
      50 Default
      0 User
  • VRHMDUserFocalLength
    Role: HMD user-defined focal length
    • Type: integer
    • Default value: 50
    • Min value: 1
    • Max value: 500
