LPCommonEditor Setting Repository
Quick Reference

LPCommonEditor

Role: Settings for Product Visualization

Detailed role: Settings variables for Product Viewer and Editor

The LPCommonEditor repository has the following characteristics:

  • Primary Tab Page: /Infrastructure
  • Repository File: LPCommonEditor
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

LPCommonEditor setting repository attributes:

  • MaximumObjectLoadingLimit
    Role: Max number of elements to load without check
    Detailed role: This option saves the maximum number of elements to load without warning the user.
    • Type: integer
    • Default value: 1000
    • Min value: 0
    • Max value: 10000
