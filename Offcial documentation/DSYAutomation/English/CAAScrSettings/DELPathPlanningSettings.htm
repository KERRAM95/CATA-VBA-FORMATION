DELPathPlanningSettings Setting Repository
Quick Reference

DELPathPlanningSettings

Role: Settings to handle the parameters in the Manufacturing Path Optimization App

Detailed role: This setting allows to customize the appearance and the behavior in Manufacturing Path Optimization

The DELPathPlanningSettings repository has the following characteristics:

  • Primary Tab Page: /Manufacturing Planning/Path Finder
  • Repository File: DELPathPlanningSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DELPathPlanningSettings setting repository attributes:

  • AllowFindTrackBatch
    Role: Allow Find Track in Batch Mode.
    Detailed role: Allows interactive batch for Find Track in Path planning command
    • Type: integer
    • Default value: 0
  • AutomaticRenaming
    Role: Automatically rename TPoints after Track computation
    Detailed role: Determines whether TPoints are automatically renamed after Track computation with Find Track or Optimize commands.
    • Type: integer
    • Default value: 1
  • ComputeMinimumDistance
    Role: Minimum distance computation mode.
    Detailed role: Determines the mode of minimum distance computation for TPoints in Path Planning: automatic or on demand.
    • Type: integer
    • Default value: 1
  • ComputeTrajectory
    Role: Track is checked until end point with target area
    Detailed role: Track is checked until end point with target area
    • Type: integer
    • Default value: 0
  • ExtractMaxRadius
    Role: Maximum radius for extract
    Detailed role: Determines user defined value for maximum radius to extract tpoint.
    • Type: double
    • Default value: 10.00
  • ExtractMaxRadiusLock
    Role: Extract options lock
    Detailed role: Admin lock for Extract options
    • Type: integer
    • Default value: 0
  • ExtractTPointsInCollision
    Role: Extract TPoints in collision before finding track
    Detailed role: TPoints that are in collision are extracted during track computation
    • Type: integer
    • Default value: 1
  • FindTrackComplexityDisplay
    Role: Display solver complexity level during computation
    Detailed role: Determines whether solver complexity level is displayed during computation
    • Type: integer
    • Default value: 0
  • FindTrackInAssistance
    Role: Find Track After Manual Assiatnce
    Detailed role: Automatically find Collision Free Track after Manual Assistance
    • Type: integer
    • Default value: 0
  • MaxComputationTimeChkBtn
    Role: Set maximum computation time to find a track
    Detailed role: Allows user to set maximum computation time to find a track
    • Type: integer
    • Default value: 0
  • MaxComputationTimeVal
    Role: Set maximum computation time to find a track
    Detailed role: Set maximum computation time to find a track
    • Type: integer
    • Default value: 10
  • MinimumDistanceCheckButton
    Role: Show Minimum Distance while simulating
    Detailed role: Allows user to visualize minimum distance or penetration of the simulating moving object with the neighbouring objects
    • Type: integer
    • Default value: 0
  • OptimizationQuality
    Role: Defines optimization quality
    Detailed role: Determines optimization quality
    • Type: integer
    • Default value: 2
  • OptimizeTrack
    Role: Optimize after finding track
    Detailed role: Track is optimized after computation
    • Type: integer
    • Default value: 0
  • QualityOfComputedTrack
    Role: Defines quality of track computed
    Detailed role: Determines quality of track computed
    • Type: integer
    • Default value: 2
  • ShowComputationTime
    Role: Show elapsed time during computation.
    Detailed role: Show elapsed time in a panel while Find Track command is computing a collision free Track.
    • Type: integer
    • Default value: 1
  • TargetArea
    Role: Define target area at end point
    Detailed role: Define target area at end point
    • Type: integer
    • Default value: 0
  • TransientTPointVisuDisplay
    Role: Visualize transient TPoint during computation
    Detailed role: Determines whether transient TPoints are displayed during computation
    • Type: integer
    • Default value: 0
  • UseConstantRandomSeed
    Role: Use random sampling during track computation
    Detailed role: Random sampling seed is used during track computation
    • Type: integer
    • Default value: 1
  • UserDefFindQuality
    Role: User defined quality for finding track
    Detailed role: Determines User defined quality for finding track.
    • Type: double
    • Default value: 0.2
  • UserDefOptiQuality
    Role: User defined quality for Optimization
    Detailed role: Determines User defined quality for Optimization.
    • Type: double
    • Default value: 0.2
  • VisualFeedbackMode
    Role: Defines visual feedback display mode.
    Detailed role: Determines what is displayed when visual feedback option is active.
    • Type: integer
    • Default value: 0
