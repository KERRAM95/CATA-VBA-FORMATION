SIMAnimationSettings Setting Repository
Quick Reference

SIMAnimationSettings

Role: Settings to define the capture behavior of Compile Animation

Detailed role: These settings allow users to customize the capture behavior of Compile Animation

The SIMAnimationSettings repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/DELMIA Infrastructure/Compile Animation
  • Repository File: SIMAnimationSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

SIMAnimationSettings setting repository attributes:

  • ColorInitState
    Role: Capture Initial State for color and opacity during animation
    Detailed role: Indicates if initial states for color and opacity should be captured during animation
    • Type: unsigned integer
    • Default value: 0
    • Values:
      0 Disable Initial State for Color/Opacity
      1 Enable Initial State for Color/Opacity
  • PositionInitState
    Role: Capture Initial State for positions during animation
    Detailed role: Indicates if initial states for position should be captured during animation
    • Type: unsigned integer
    • Default value: 0
    • Values:
      0 Disable Initial State for Position
      1 Enable Initial State for Position
  • ProductHighlight
    Role: Capture Product Highlight during animation
    Detailed role: Indicates if product highlighting should be captured during animation
    • Type: unsigned integer
    • Default value: 1
    • Values:
      0 Disable Product Highlight
      1 Enable Product Highlight
  • VisInitState
    Role: Capture Initial State for visibility during animation
    Detailed role: Indicates if initial states for color and opacity should be captured during animation
    • Type: unsigned integer
    • Default value: 0
    • Values:
      0 Disable Initial State for Visibility
      1 Enable Initial State for Visibility
