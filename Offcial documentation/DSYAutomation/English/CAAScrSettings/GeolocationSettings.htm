GeolocationSettings Setting Repository
Quick Reference

GeolocationSettings

Role: Manages Geolocation options

The GeolocationSettings repository has the following characteristics:

  • Primary Tab Page: /AEC Plant/Civil Engineering/Geolocation
  • Repository File: GeolocationSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

GeolocationSettings setting repository attributes:

  • Ceg_Geolocation_UrlLocate
    Role: URL Locator
    Detailed role: Uniform Resource Locator (URL) preference to locate objects on a web map.
    • Type: string
    • Default value: https://maps.google.com/maps?q=##Longitude##,##Latitude##&z=15
