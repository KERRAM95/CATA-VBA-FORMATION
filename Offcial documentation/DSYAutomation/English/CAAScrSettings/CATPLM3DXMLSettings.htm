CATPLM3DXMLSettings Setting Repository
Quick Reference

CATPLM3DXMLSettings

Role: Setting to handle CAA model parameters

The CATPLM3DXMLSettings repository has the following characteristics:

  • Primary Tab Page: /General/3DXML
  • Repository File: CATPLM3DXMLSettings
  • Version: 2
  • Full Compatibility: 2
  • Upward Compatibility: 2

CATPLM3DXMLSettings setting repository attributes:

  • ConflictManagementDecision
    Role: Role Visibility
    • Type: integer
    • Default value: 1
    • Values:
      0 The local data are kept and the new revision is ignored
      1 The local data are overwritten by the new revision
  • ConflictManagementOption
    Role: Manage conflict error at import
    • Type: boolean
    • Default value: false
  • DownwardCompatibilityDecision
    Role: Role Visibility
    • Type: integer
    • Default value: 0
    • Values:
      0 Accept weakened data
      1 Refuse weakened data
  • DownwardCompatibilityOption
    Role: Manage behavior for weakened data
    • Type: boolean
    • Default value: false
  • EditionTokenManagementOption
    Role: Edition token operation specification through BL
    • Type: boolean
    • Default value: false
  • ExportEnterpriseItemNumberOption
    Role: Export Enterprise Item Number
    • Type: boolean
    • Default value: true
  • ExportOfChangeActionOption
    Role: Export Of Change Action at export of Effectivities
    • Type: boolean
    • Default value: false
  • ExtendedBatchDelegationScopeOption
    Role: Include Representations in scope of Delegation BL
    • Type: boolean
    • Default value: false
  • GeometricFormats
    Role: Role Visibility
    • Type: integer
    • Default value: 1
    • Values:
      0 The geometric format used is XML tesselation
      1 The geometric format used is Static tesselation
  • ReidentificationOption
    Role: Reidentification at import as New
    • Type: boolean
    • Default value: false
  • RevisionStringOption
    Role: Revision String Management while 3DXML Import
    • Type: boolean
    • Default value: false
  • SplitSize
    Role: Split Size option
    • Type: integer
    • Default value: 1000
    • Min value: 1
    • Max value: 9999
  • WaiveConflictManagementDecision
    Role: Role Visibility
    • Type: integer
    • Default value: 1
    • Values:
      0 The local data are kept and the new revision is ignored
      1 The local data are overwritten by the new revision
  • WaiveConflictManagementOption
    Role: Manage conflict error at import
    • Type: boolean
    • Default value: false
