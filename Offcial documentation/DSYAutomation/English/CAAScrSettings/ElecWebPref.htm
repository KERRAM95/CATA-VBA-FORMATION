ElecWebPref Setting Repository
Quick Reference

ElecWebPref

Role: Electrical Harness Installation setting

Detailed role: Electrical Harness Installation setting

The ElecWebPref repository has the following characteristics:

  • Primary Tab Page: /Equipments/Electrical Harness Installation/Harness Management
  • Repository File: ElectricalHarnessInstallation
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

ElecWebPref setting repository attributes:

  • OpenInRFLPCheck
    Role: Defines option for opening physical data
    Detailed role: Decides whether to open physical data from Web-top to win-top session in RFLP editor or HVAC 3D design
    • Type: integer
    • Default value: 0
    • Default Value Role: By default open in Electrical 3D design
    • Values:
      0 Open in Electrical 3D design
      1 Open in RFLP editor
