Marker3DHyperlinkSetting Setting Repository
Quick Reference

Marker3DHyperlinkSetting

Role: Setting to handle the Marker visual properties and parameters

The Marker3DHyperlinkSetting repository has the following characteristics:

  • Primary Tab Page: /Design Review/Marker
  • Repository File: Marker3DHyperlink
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

Marker3DHyperlinkSetting setting repository attributes:

  • BorderColor
    Role: Default border color of a 3DHyperlink Marker
    • Tab Page: /Design Review/Marker
    • Type: unsigned integer
    • Default value: 61
    • Default Value Role: Grey in RGB order
    • Min value: 0
    • Max value: 255
  • BorderMode
    Role: Default flag for border of a 3DHyperlink Marker
    • Tab Page: /Design Review/Marker
    • Type: boolean
    • Default value: true
  • BorderThickness
    Role: Default border thickness of a 3DHyperlink Marker.
    • Tab Page: /Design Review/Marker
    • Type: unsigned integer
    • Default value: 1
    • Default Value Role: 0.7 mm
    • Min value: 1
    • Max value: 55
  • BorderType
    Role: Default border type of a 3DHyperlink Marker.
    • Tab Page: /Design Review/Marker
    • Type: unsigned integer
    • Default value: 1
    • Min value: 1
    • Max value: 7
  • FillColor
    Role: Default color to fill 3DHyperlink Marker
    • Tab Page: /Design Review/Marker
    • Type: unsigned integer
    • Default value: 255
    • Default Value Role: >White in RGB order
    • Min value: 0
    • Max value: 255
  • FillMode
    Role: Default fill flag for 3DHyperlink Marker
    • Tab Page: /Design Review/Marker
    • Type: boolean
    • Default value: true
  • FillTransparency
    Role: Default transparency value of 3DHyperlink Marker
    • Tab Page: /Design Review/Marker
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: Filling is transparent(0%)
    • Min value: 0
    • Max value: 100
  • LeaderEndType
    Role: Default leader end type of a 3DHyperlink Marker.
    • Tab Page: /Design Review/Marker
    • Type: unsigned integer
    • Default value: 0
    • Min value: 0
    • Max value: 4
  • LeaderMode
    Role: Default Leader flag for 3DHyperlink Marker
    • Tab Page: /Design Review/Marker
    • Type: boolean
    • Default value: true
  • Marker3DHyperlinkDisplay
    Role: Default flag for display of a 3DHyperlink Marker
    • Tab Page: /Design Review/Marker
    • Type: boolean
    • Default value: false
