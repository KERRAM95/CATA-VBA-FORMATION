KPISettings Setting Repository
Quick Reference

KPISettings

Role: Settings to handle the KPIs parameters

The KPISettings repository has the following characteristics:

  • Primary Tab Page: /General/KPI
  • Repository File: KPISettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

KPISettings setting repository attributes:

  • CurrencyName
    Role: Currency name
    • Tab Page: /General/KPI/General
    • Type: string
    • Default value: EUR
  • DefaultAnalytics
    Role: Default Analytics
    • Tab Page: /General/KPI/General
    • Type: string
    • Default value: KPILifeCycle
  • DefaultMarginPercentageOnDeclared
    Role: Default Margin percentage On Declared (%)
    • Tab Page: /General/KPI/General
    • Type: double
    • Default value: 10
  • DefaultPurcentageOnDeclared
    Role: Default Confidence percentage On Declared (%)
    • Tab Page: /General/KPI/General
    • Type: double
    • Default value: 10
  • DefaultSelectionMode
    Role: Default melection mode
    • Tab Page: /General/KPI/General
    • Type: integer
    • Default value: 0
  • ExchangeRate
    Role: Exchange rate
    • Tab Page: /General/KPI/General
    • Type: string
    • Default value: DefaultRate
  • LockSelection
    Role: Lock selection
    • Tab Page: /General/KPI/General

    • Default value: false
