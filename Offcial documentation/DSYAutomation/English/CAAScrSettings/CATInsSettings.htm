CATInsSettings Setting Repository
Quick Reference

CATInsSettings

Role: These settings define the user choice for CATIA Insulation environment

Detailed role: These settings define the user choice of the display the referent who allows you to get a better idea of the scale factor.

The CATInsSettings repository has the following characteristics:

  • Primary Tab Page: /Marine and Offshore/Accommodation Design/Accommodation
  • Repository File: CATInsSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CATInsSettings setting repository attributes:

  • InsEdgeExtenderHeight
    Role: Insulation edge extender height
    Detailed role: Define the height of insulation edge extender
    • Type: double
    • Default value: 400
  • InsFeatureExtensionCopy
    Role: To copy or not name of feature extension to value of Categories Attribute of Insulation PLM object
    • Type: integer
    • Default value: 0
    • Values:
      0 Do not copy feature extension
      1 Copy when the Category attribute is empty
      2 Copy value and delete the feature extension
  • InsSpecifierVisibility
    Role: To show or hide the visibility of insulation specifier
    • Type: boolean
    • Default value: 1
  • OpeningInsulationArea
    Role: Openings Insulation Area
    Detailed role: Defines the Area of Opening to consider for insulation
    • Type: double
    • Default value: 0
