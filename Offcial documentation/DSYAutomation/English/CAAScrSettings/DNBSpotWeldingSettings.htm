DNBSpotWeldingSettings

Role: Settings for Spot Welding

Detailed role: Settings for Spot Welding

The DNBSpotWeldingSettings repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/Resource Detailing/Spot Welding
  • Repository File: DNBSpotWeldingSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Setting controller:
    • Framework_CPP: System
    • Framework_IDL: System
    • Interface_CPP: CATISysCacheSettingAtt
    • Interface_IDL: CATIACacheSttingAtt
    • LateType: CATSysCacheSettingCtrl

DNBSpotWeldingSettings setting repository attributes:

  • AppPointColor
    Role: Color for Approach point
    • Type: integer
    • Default value: 255
    • Default Value Role: White in RGB order
    • Min value: 0
    • Max value: 255
  • AppPointDefaultOffset
    Role: Default offset for Apporach point
    • Type: double
    • Default value: 10.000
  • AppPointNameDisplayMode
    Role: Decides if the name of Approach point has to be displayed
    • Type: integer
    • Default value: 0
  • ApplicationType
    Role: Options for Application Type
    Detailed role: This option allows user to select the type of application to create
    • Type: integer
    • Default value: 0
    • Min value: 0
    • Max value: 255
  • DatabaseFileName
    Role: Arc Weld Table path from the database
    Detailed role: Stores Arc Weld Table database path.
    • Type: string
    • Default value:
    • Default Value Role: By default, the path will be empty
  • DepPointColor
    Role: Color for Departure point
    • Type: integer
    • Default value: 255
    • Default Value Role: White in RGB order
    • Min value: 0
    • Max value: 255
  • DepPointDefaultOffset
    Role: Default offset for Departure point
    • Type: double
    • Default value: 10.000
  • DepPointNameDisplayMode
    Role: Decides if the name of Departure point has to be displayed
    • Type: integer
    • Default value: 0
  • DisplayMotionDirectionMode
    Role: Decides if the direction of motion has to be displayed for the trajectory
    • Type: integer
    • Default value: 0
  • PointFasteningMethod
    Role: Decides Point Fastening Methodology used
    • Type: integer
    • Default value: 1
  • SpotOperationNamingFromTarget
    Role: Whether Spot Operation Naming is based on Spot Target
    • Type: integer
    • Default value: 0
  • SpotParamSpecsFileSystemPath
    Role: Arc Weld Table path from the file system
    Detailed role: Stores an Arc Weld table name from the file system.
    • Type: string
    • Default value:
  • SpotParamSpecsRadioButtons
    Role: Spot Param Specs radio buttons
    Detailed role: Options for defining Spot Param Specs.
    • Type: unsigned integer
    • Default value: 0
    • Values:
      0 NoneOption

      2 Database Selection
  • XMLStyleSheetPath
    Role: XML Style sheet path
    Detailed role: Stores XML Style sheet path
    • Type: string
    • Default value:
    • Default Value Role: By default, the path will be empty
