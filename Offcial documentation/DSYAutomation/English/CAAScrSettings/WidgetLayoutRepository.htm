WidgetLayoutRepository Setting Repository
Quick Reference

WidgetLayoutRepository

Role: Setting to handle widget's layout parameters

The WidgetLayoutRepository repository has the following characteristics:

  • Primary Tab Page: /General
  • Repository File: WidgetLayoutRepository
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

WidgetLayoutRepository setting repository attributes:

  • CreateInMaxState
    Role: Parameter to create a new tab page in maximized state
    • Type: integer
    • Default value: 1
  • DisplayGlobalHomePage
    Role: Parameter to display the Welcome Page
    • Type: integer
    • Default value: 1
  • ExtractedAB
    Role: Parameter to create a new tab page with action bar extracted
    • Type: integer
    • Default value: 0
  • GlobalHomePageRecentsDoubleClick
    Role: Parameter to specify the action when double clicking a recent content from the Welcome Page
    • Type: integer
    • Default value: 0
  • OpenInNewTP
    Role: Parameter to open a widget in a new tab page
    • Type: integer
    • Default value: 1
