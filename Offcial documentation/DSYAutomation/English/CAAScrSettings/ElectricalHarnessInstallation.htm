ElectricalHarnessInstallation

Role: Electrical Harness Installation setting

Detailed role: Electrical Harness Installation setting

The ElectricalHarnessInstallation repository has the following characteristics:

  • Primary Tab Page: /Equipments/Electrical Harness Installation/Harness Management
  • Repository File: ElectricalHarnessInstallation
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

ElectricalHarnessInstallation setting repository attributes:

    Role: Setting to manage the balance between performance and accuracy

    • Default value: 1
    • Min value: 1

    • Values:
      1 Bend Radius Set By Resource is Editable
      0 Bend Radius Set By Resource is Editable

    Detailed role: Setting to allow System to use a Min Bend Radius for Curve Computation
    • Type: integer
    • Default value: 0
    • Default Value Role: By default The Check Button is UNCHECKED
    • Values:
      0 If Unchecked the User set Bend Radius will be respected
      1 If Checked the System may override the User set Bend Radius
