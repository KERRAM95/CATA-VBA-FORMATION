DNBTSASpotWeldTrajectorySettings Setting Repository
Quick Reference

DNBTSASpotWeldTrajectorySettings

Role: Settings to handle the parameters of the TSA Spot Weld Trajectory Settings

Detailed role: This setting allows to customize the appearance and the behavior of TSA Spot Weld Trajectory Settings

The DNBTSASpotWeldTrajectorySettings repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/Resource Detailing/Tool Selection Assistant
  • Repository File: DNBTSASpotWeldTrajectorySettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DNBTSASpotWeldTrajectorySettings setting repository attributes:

  • ApproachAxis
    Role: The rotation axis
    Detailed role: Which is the rotation axis
    • Type: short integer
    • Default value: 1
    • Default Value Role: By default +ve X-Axis is the Approach axis
    • Values:
      0 -veX-Axis
      1 X-Axis
      2 -veY-Axis
      3 Y-Axis
      4 -veZ-Axis
      5 Z-Axis
  • GunOCLower
    Role: Gun Open Clearance Lower Value
    • Type: double
    • Default value: 0.0000
  • GunOCUpper
    Role: Gun Open Clearance Upper Value
    • Type: double
    • Default value: 0.0000
  • GunTALower
    Role: Gun Tip Abrasion Lower Value
    • Type: double
    • Default value: 0.0000
  • GunTAUpper
    Role: Gun Tip Abrasion Upper Value
    • Type: double
    • Default value: 0.0000
  • IgnoreHiddenTrajectory
    Role: Ignore the processing of trajectories that are hidden
    Detailed role: Indicates if end-user is filtering hidden trajectory when using TSA scenarios
    • Type: unsigned integer
    • Default value: 1
  • PointColor
    Role: The Trajectory Point Color
    Detailed role: The points in the trajectory that are set
    • Type: integer
    • Default value: 255
    • Default Value Role: White in RGB order
    • Min value: 0
    • Max value: 255
  • RotationAxis
    Role: The rotation axis
    Detailed role: Which is the rotation axis
    • Type: short integer
    • Default value: 0
    • Default Value Role: By default Z is the rotation axis
    • Values:
      0 Z-Axis
      1 Y-Axis
      2 X-Axis
  • TSADefaultHomeName
    Role: Indicate the default home position name to use on toolswhen creating TSA scenario.
    Detailed role: This option allows user to specify what is the name of the home position to automatically pick when defining a TSA(Tool selection assistant) scenario.
    • Type: string
    • Default value: CLOSE
  • ViewAnimationSimu
    Role: View animation when TSA scenarios are simulating
    Detailed role: Indicates if end-user wants to view the tools moving when the computation is running
    • Type: unsigned integer
    • Default value: 0
  • XMLStyleSheetPath
    Role: XML Style sheet path
    Detailed role: Stores XML Style sheet path
    • Type: string
    • Default value:
    • Default Value Role: By default, the path will be empty
