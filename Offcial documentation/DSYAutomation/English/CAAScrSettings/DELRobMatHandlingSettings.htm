DELRobMatHandlingSettings Setting Repository
Quick Reference

DELRobMatHandlingSettings

Role: Options for the Robot Material Handling app

Detailed role: This option allows user to configure the behavior of the Robot Material Handling app.

The DELRobMatHandlingSettings repository has the following characteristics:

  • Primary Tab Page: /Robotics/Material Handling
  • Repository File: DELRobMatHandlingSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DELRobMatHandlingSettings setting repository attributes:

  • ProcessTagPointColor
    Role: Options for Material Handling Process Tag Point Color
    Detailed role: This option allows the user to configure the color of Process Tags
    • Type: integer
    • Default value: 255
    • Default Value Role: Blue in RGB order
    • Min value: 0
    • Max value: 255
  • SlipSheetColor
    Role: Options for Material Handling Slip Sheet Color
    Detailed role: This option allows the user to configure the color of Slip Sheets
    • Type: integer
    • Default value: 230
    • Default Value Role: Light Blue in RGB order
    • Min value: 0
    • Max value: 255
