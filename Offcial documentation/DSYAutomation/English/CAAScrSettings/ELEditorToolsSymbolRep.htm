ELEditorToolsSymbolRep setting repository attributes:

  • DefaultEquipmentRepSymbolOption
    Role: Default Equipment symbol
    • Type: integer
    • Default value: 1
    • Values:
      0 Full symbol
      1 Partial symbol
  • DefaultRepSymbolOption
    Role: Default symbol
    • Type: integer
    • Default value: 0
    • Values:
      0 Automatic symbol
      1 Full symbol
      2 Partial symbol
  • DefaultRepresentationSymbolOption
    Role: Default representation
    • Type: integer
    • Default value: 0
    • Values:
      0 Wiring representation
      1 IE representation
      2 Contact grid representation
  • DisplayAllPinsChkBoxOption
    Role: Activate/Desactivate the display of all pins
    Detailed role: Display all pins desactivated by default
    • Type: boolean
    • Default value: false
  • OffsetGridValueOption
    Role: The value of the Grid Offset
    • Type: double
    • Default value: 20.0
  • PinSingleRepresentation
    Role: Pin uniq representation
    Detailed role: Prevent pins to be globaly multirepresented
    • Type: boolean
    • Default value: false
  • StepGridValueOption
    Role: The value of the Grid Step
    • Type: double
    • Default value: 10.0
  • StepValueOption
    Role: The value of the Step
    • Type: double
    • Default value: 5.0
