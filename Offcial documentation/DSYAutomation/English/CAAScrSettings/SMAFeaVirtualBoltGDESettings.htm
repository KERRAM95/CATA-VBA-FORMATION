Quick Reference

SMAFeaVirtualBoltGDESettings

Role: Manages the virtual bolt settings for GDE

The SMAFeaVirtualBoltGDESettings repository has the following characteristics:

  • Primary Tab Page: /None/None/None
  • Repository File: SMAFeaVirtualBoltGDESettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

SMAFeaVirtualBoltGDESettings setting repository attributes:

  • BoltStrengthExpanderStatus
    Role: Set bolt strength data expander status in SRD
    • Type: integer
    • Default value: 1
  • BoltType
    Role: Sets coupling type for support 1
    • Type: integer
    • Default value: 0
  • Ground
    Role: Sets coupling type for support 2
    • Type: boolean
    • Default value: 1
  • RotationalStiffnessValue
    Role: Sets rotational stiffness value
    • Type: double
    • Default value: 0
  • TranslationalStiffnessValue
    Role: Sets Translational stiffness value
    • Type: double
    • Default value: 0
