GeneralHelp Setting Repository
Quick Reference

GeneralHelp

Role: Option to handle documentation parameters

The GeneralHelp repository has the following characteristics:

  • Primary Tab Page: /General/Help
  • Repository File: GeneralHelp
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

GeneralHelp setting repository attributes:

  • DocLanguage
    Role: Language of the technical documentation
    • Type: integer
    • Default value: 0
  • DocPath
    Role: Path to access technical documentation
    • Type: string
    • Default value:
  • OnlineHelp
    Role: Parameter to set whether 3ds.com online help can be used instead of installed documentation
    • Type: integer
    • Default value: 1
    • Values:
      0 The installed documentation defined in DocPath attribute is accessed
      1 The 3DS.com online help is accessed
  • OnlineLanguagePolicy
    Role: Parameter to set whether the current Online English documentation must be accessed if there is no current Online documentation in the choosen language
    • Type: integer
    • Default value: 1
    • Values:
      0 The latest Online documentation in the choosen language is accessed
      1 The current English documentation is accessed
