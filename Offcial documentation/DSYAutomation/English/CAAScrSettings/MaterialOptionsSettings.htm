MaterialOptionsSettings Setting Repository
Quick Reference

MaterialOptionsSettings

Role: Setting to handle the Materials Palette parameters

Detailed role: Materials Palette is used to assign materials

The MaterialOptionsSettings repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/Material Editor/Materials Palette
  • Repository File: MaterialOptionsSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

MaterialOptionsSettings setting repository attributes:

  • CheckButtonAutoUpdateCnx
    Role: Select to allow the automatic synchronization of the applied-material
    • Type: integer
    • Default value: 0
    • Values:
      0 Automatic synchronization deactivated
      1 Automatic synchronization activated
  • CheckButtonSearchAll
    Role: Enable the search all of the material browser
    • Type: integer
    • Default value: 1
    • Values:
      0 Search all deactivated
      1 Search all activated
  • CheckButtonShowBrokenCnx
    Role: Enable the display of broken connections in Tree
    • Type: integer
    • Default value: 0
    • Values:
      0 Broken connections are not shown in Tree
      1 Broken connections are shown in Tree
  • EditAppliedTipsPanel
    Role: Display a help panel about edit applied material command
    • Type: integer
    • Default value: 0
  • EnvImagePath
    Role: Path towards the default environment image
    • Type: URN
    • Default value:
  • KeepMaterialInSession
    Role: Keep opened materials in session without Editor
    • Type: integer
    • Default value: 1
    • Values:
      0 Materials are not kept in session
      1 Materials are kept in session
  • LayerInApplyCmd
    Role: Select to allow the layer creation with apply material command
    • Type: integer
    • Default value: 0
    • Values:
      0 Layer creation deactivated
      1 Layer creation activated
  • MaterialSettingsChanged
    Role: Settings changed by user
    • Type: integer
    • Default value: 0
    • Values:
      0 Settings not changed by user
      1 Settings changed by user
  • MsgInfoPalette
    Role: Display a message about palette
    • Type: integer
    • Default value: 0
    • Values:
      0 Information about palette not displayed
      1 Information about palette displayed
  • RadioButtonEROLD
    Role: Enable search on Unified Materials in the Material Browser
    • Type: integer
    • Default value: 0
    • Values:
      0 Search on Unified Materials is deactivated
      1 Search on Unified Materials is activated
  • RadioButtonIRPCOLD
    Role: Enable search on Materials in the Material Browser
    • Type: integer
    • Default value: 1
    • Values:
      0 Search on Materials is deactivated
      1 Search on Materials is activated
  • RadioButtonSessionOLD
    Role: Enable search on session in the Material Browser
    • Type: integer
    • Default value: 1
    • Values:
      0 Search in session is deactivated
      1 Search in session is activated
  • SearchFamilly
    Role: Select to specify that the search is to be done on V_Group attribute
    • Type: integer
    • Default value: 1
    • Values:
      0 Search on V_Group deactivated
      1 Search on V_Group activated
  • SearchKeyWordEditor
    Role: The predicate used in the query for the search in the Material Browser
    • Type: string
    • Default value:
  • SearchKeyWordEditorOLD
    Role: The predicate used in the query for the search in the Material Browser
    • Type: string
    • Default value: DS
  • SearchName
    Role: Select to specify that the search is to be done on PLM_ExternalID attribute
    • Type: integer
    • Default value: 0
    • Values:
      0 Search on PLMExternalID deactivated
      1 Search on PLMExternalID activated
  • SearchPaletteDefault
    Role: Select to specify that the search is to be done from Query or Session
    • Type: integer
    • Default value: 0
    • Values:
      0 Fill immersive material palette from Search
      1 Fill immersive material palette from Session
  • SearchString
    Role: The predicate used in the query to fill the Materials palette
    • Type: string
    • Default value: DS-Standard
  • SearchVName
    Role: Select to specify that the search is to be done on V_Name attribute
    • Type: integer
    • Default value: 0
    • Values:
      0 Search on V_Name deactivated
      1 Search on V_Name activated
  • SimplifyApplyCmd
    Role: Choose the interface for apply material command
    • Type: integer
    • Default value: 0
    • Values:
      0 Default
      1 Simplified
      2 Legacy
  • VectorInApplyCmd
    Role: Select to allow the vector creation with apply material command
    • Type: integer
    • Default value: 0
    • Values:
      0 Vector creation deactivated
      1 Vector creation activated
