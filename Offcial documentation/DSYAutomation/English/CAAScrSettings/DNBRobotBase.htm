DNBRobotBase Setting Repository
Quick Reference

DNBRobotBase

Role: Setting Robot visualization parameters

Detailed role: Visualization of Robot analysis parameters. They are Linear speed limit, Rotational Speed Limit, Rotational Acceleration limit, Reachability and Sigularity.

The DNBRobotBase repository has the following characteristics:

  • Primary Tab Page: /Delmia Infrastructure/Robot Analysis
  • Repository File: DNBRobotBase
  • Version: 3
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Root tab page /x reused as /x
