CATOIC Setting Repository
Quick Reference

CATOIC

Role: Online Information Collection Settings

Detailed role: Online Information Collection Settings allow to trigger the uploading of information onto a server.

The CATOIC repository has the following characteristics:

  • Primary Tab Page: /General/Online Information Collection
  • Repository File: CATOIC
  • Version: 1
  • Full Compatibility: 0
  • Upward Compatibility: 0

CATOIC setting repository attributes:

  • ActivateIRManager
    Role: If Activation is set to true the V6 client will run the incident report manager automatically considering the following command line.
    • Type: boolean
    • Default value: true
  • IRManagerCmdLine
    Role: Incident Report Manager command line
    • Type: string
    • Default value: DSYSysIRManager -sc
  • MTBFActivation
    Role: If Activation is set to true the V6 client will upload MTBF reports
    • Type: boolean
    • Default value: true
  • MTBFServerURL
    Role: Http URL where the MTBF infrastructure uploads its file
    • Type: string
    • Default value: https://ds3dexp-eu-listencat-ext.extranet.3ds.com/upload
  • MTBF_Keep_Local_Report
    Role: Keep files in DSYSysIRManager/[Session|Incident]
    • Type: boolean
    • Default value: true
  • OICHttpMode
    Role: Enables the OIC to use a POST request.
    Detailed role: By default the OIC process uploads its data with a PUT request. If a customer does not want to allow these request on his server, this setting enables to use a POST request in place of the PUT, but he has to develop an applet.
    • Type: boolean
    • Default value: false
  • UsageActivation
    Role: If Activation is set to true the V6 client will upload USAGE metrics information
    • Type: boolean
    • Default value: false
  • UsageServerURL
    Role: Http URL where the OIC infrastructure uploads its file
    • Type: string
    • Default value:
  • WD_DumpType
    Role: Generate mini or full dump (full dump disable upload)
    • Type: unsigned integer
    • Default value: 0
  • WD_Mode
    Role: Select WD mode. Legacy or WER (rely on Windows Error Reporting, capture kills) (keep as int in case more mode are available later).
    • Type: unsigned integer
    • Default value: 42
