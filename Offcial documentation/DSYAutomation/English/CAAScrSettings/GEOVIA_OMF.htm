GEOVIA_OMF Setting Repository
Quick Reference

GEOVIA_OMF

Role:

The GEOVIA_OMF repository has the following characteristics:

  • Primary Tab Page: /General/Compatibility/GEOVIA_OMF
  • Repository File: GEOVIA_OMF
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

GEOVIA_OMF setting repository attributes:

  • importOptionHealing
    Role: Toggle healing
    • Type: boolean
    • Default value: false
  • importOptionHealingMode
    Role: Healing surface mode
    • Type: integer
    • Default value: 0
    • Default Value Role: By default, the healing surface mode is Closed surface
    • Values:
      0 Closed surface
      1 Open surface
  • importOptionOutputGeometry
    Role: Output geometry
    • Type: integer
    • Default value: 0
    • Default Value Role: By default output geometry is tessellated
    • Values:
      0 Tessellated
      1 Polyhedral
  • importOptionTargetOrigin
    Role: Target Origin
    • Type: boolean
    • Default value: false
  • importProfileMode
    Role: Profile prefered:
    • Type: integer
    • Default value: 1
    • Values:
      0 Import the entire structure grouped under a CGR
      1 Create a full PLM 3DExperience Model
      2 Create a design PLM 3DExperience Model
