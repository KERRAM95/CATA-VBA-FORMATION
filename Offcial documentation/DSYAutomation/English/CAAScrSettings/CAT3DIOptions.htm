CAT3DIOptions Setting Repository
Quick Reference

CAT3DIOptions

Role: Manages 3D Geometry Insight options

The CAT3DIOptions repository has the following characteristics:

  • Primary Tab Page: /Digital Mockup/3D Geometry Insight/3D Geometry Analysis
  • Repository File: CAT3DIOptions
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CAT3DIOptions setting repository attributes:

  • Prevent3DIContextModificationSetting
    Role: Prevent 3D Product modification
    Detailed role: Automatically switches the 3D Products opened in Read Only mode (forbids the edition of the 3D products)
    • Tab Page: /Digital Mockup/3D Geometry Insight/3D Geometry Analysis
    • Type: boolean
    • Default value: 1
