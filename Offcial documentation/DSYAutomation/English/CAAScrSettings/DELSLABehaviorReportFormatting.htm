DELSLABehaviorReportFormatting

Role: The format of Factory Flow Simulation reports

Detailed role: Select the directory that contains the XSLT files for reports formatting

The DELSLABehaviorReportFormatting repository has the following characteristics:

  • Primary Tab Page: /Factory/ReportFormatting
  • Repository File: DELSLABehaviorReportFormatting
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
