L3dHVACDesignCriteria Setting Repository
Quick Reference

L3dHVACDesignCriteria

Role: Short Qualification of the usage of the option

Detailed role: Full explanation of the option Role

The L3dHVACDesignCriteria repository has the following characteristics:

  • Primary Tab Page: /3D Modeling/Fluid Discipline/HVAC 3D Design
  • Repository File: HVACDesignCriteria
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

L3dHVACDesignCriteria setting repository attributes:

  • L3dAutoSetContextCheck
    Role: Force context set to first child under UI Active
    Detailed role: Set Context command will automatically set the current context to first child under UI active
    • Type: integer
    • Default value: 0
    • Default Value Role: By default the option will be OFF for not to force context set to first child under UI active
    • Values:
      0 Do not force context set to first children under UI active
      1 Force context set to first children under UI active
  • L3dCompatibilityCheck
    Role: Need to perform compatibility checking during part placement
    Detailed role: Defines whether or not to perform compatibility checking when placing part
    • Type: integer
    • Default value: 0
    • Default Value Role: By default the option will be OFF for not to perform compatibility checking during part placement
    • Values:
      0 Need to perform compatibility checking during part placement
      1 Do not perform compatibility checking during part placement
  • L3dConsiderInnerThicknessOfDuctCheck
    Role: Defines option for considering Inner Thickness of Duct
    Detailed role: Defines whether or not to consider Inner Thickness of Duct
    • Type: integer
    • Default value: 0
    • Default Value Role: By default this option is OFF for not to consider Inner Thickness of Duct i.e HVAC Duct is created with outward thickness
    • Values:
      0 Do not consider Inner Thickness of Duct
      1 Consider Inner Thickness of Duct
  • L3dCopyColorFromRouteRefOptionCheck
    Role: Copy color from route reference
    Detailed role: Defines whether or not to Copy color from route reference
    • Type: integer
    • Default value: 0
    • Default Value Role: By default this option is OFF i.e. Copy color from route reference is allowed
    • Values:
      0 Copy color from route reference is not allowed.
      1 Copy color from route reference is allowed.
  • L3dCreate3DShapeComboBoxWithOptionCheck
    Role: Custom 3DShape
    Detailed role: Create custom 3DShape
    • Type: string
    • Default value: DICTest3DShape3
  • L3dCreate3DShapeWithOptionCheck
    Role: Defines option for 3DShape creation
    Detailed role: Create 3D shape depends on user selection
    • Type: integer
    • Default value: 3
    • Default Value Role: By default User Defined
    • Values:
      0 Create 3D Part
      1 Create Product with 3D Shape
      2 Create Product with Custom Shape
      3 User Defined
  • L3dCreateSingleSpoolCheck
    Role: Defines option for spool creation
    Detailed role: Defines whether or not to create spool with single element automatically
    • Type: integer
    • Default value: 0
    • Default Value Role: By default this option is OFF for not to Create Spool with single element automatically
    • Values:
      0 Do not Create Spool with single element automatically
      1 Create Spool with single element automatically
  • L3dDataSourceCheck
    Role: Data source for part selection
    Detailed role: If the option is selected Variablity Space is Extracted into Harness Flatten model by the Extract Command
    • Type: integer
    • Default value: 0
    • Default Value Role: By default this option is search part from catalog
    • Values:
      0 Search new part from catalog
      4 Search new part from database
  • L3dDisplayRouteTableNeededWhenMultipleChoiceCheck
    Role: Setting to display the Chooser of Route and the tables if there is multiple choice
    Detailed role: Defines whether or not to allow user choosing precisely the route when creating it
    • Type: integer
    • Default value: 0
    • Default Value Role: By default this option is OFF i.e. first valid route is taken.
    • Values:
      0 First valid route will be taken, the properties can then be edited in DIP
      1 The Route Chooser and Table Browser will be displayed if there is multiple choices
  • L3dDontApplyNSFilterCheck
    Role: Allow search for all Sizes while part placement
    Detailed role: Don't apply size filter while part search
    • Type: integer
    • Default value: 0
    • Default Value Role: By default the option will be OFF for not to apply size filter while part search
    • Values:
      0 Apply size filter while part search
      1 Don't apply size filter while part search | Search for all sizes available
  • L3dFBDIObjectTypeCreation
    Role:
    • Type: integer
    • Default value: 0
    • Values:
      0 Use SubPart Type Table for object type creation
      1 Create Miscellaneous type object
  • L3dFBDIOfRunToCheck
    Role: Defines option to FBDI the run to no geometry
    Detailed role: Defines option to FBDI the run to no geometry
    • Type: integer
    • Default value: 0
    • Default Value Role: FBDI of run results in no Geometry
    • Values:
      0 FBDI of run results in Geometry
      1 FBDI of run results results in Solid rep
      2 FBDI of run results results in Rigid Route
  • L3dFlexStraightRouteCheck
    Role: Maintain straight slice from port for flexible route
    Detailed role: Maintain straight slice from port for flexible route
    • Type: integer
    • Default value: 1
    • Default Value Role: By default the option will be OFF for not to maintain straight slice from port for flexible route
    • Values:
      0 Do not maintain straight slice from port for flexible route
      1 Maintain straight slice from port for flexible route
  • L3dFlexTableCheck
    Role: Force usage of Flexible Pipe Reference Definition Table
    Detailed role: Defines whether or not the user has to use the Flexible Pipe Reference Definition Table
    • Type: integer
    • Default value: 0
    • Default Value Role: By default the User will not forced to use the Flexible Pipe Reference Definition Table
    • Values:
      0 User will not forced to use the Flexible Pipe Reference Definition Table
      1 ser will be forced to use the Flexible Pipe Reference Definition Table
  • L3dIgnoreSpoolMaturityForRevisionCreationCheck
    Role: Defines option for ignore maturity state of spool for revision creation
    Detailed role: Defines whether ignore or not ignore maturity state for revision creation
    • Type: integer
    • Default value: 0
    • Default Value Role: By default this option is OFF for for not to Do not igonre maturity state of spool
    • Values:
      0 Ignore spool maturity state and allow to create revision
      1 Create revision with any maturity state
  • L3dInstantiationForbiddenForCfgCheck
    Role: Forbid instantiation of components
    Detailed role: Forbid instantiation of components under an object carrying the configuration model
    • Type: integer
    • Default value: 0
    • Default Value Role: By default this option is OFF for forbidding instantiation of components
    • Values:
      0 Instantiation of components is allowed
      1 Instantiation of components is not allowed
  • L3dLineIDCheck
    Role: Force usage of Line ID
    Detailed role: Defines whether or not to force usage of Line ID for part placement and routing
    • Type: integer
    • Default value: 0
    • Default Value Role: By default this option is OFF for Force usage of Line ID
    • Values:
      0 Do not force the usage of the Line ID
      1 Force the usage of the Line ID
  • L3dLocalbendRadiusForceUsageFromTblOptionCheck
    Role: Force usage from table
    Detailed role: Defines whether or not force usage from table to allow user to modify local bend radius of route.
    • Type: integer
    • Default value: 0
    • Default Value Role: By default this option is OFF i.e. Do not force usage from table during local bend radius modification
    • Values:
      0 Do not force usage from table during local bend radius modification.
      1 Force usage from table during local bend radius modification.
  • L3dLocalbendRadiusModifOptionCheck
    Role: Allow local bend radius modification
    Detailed role: Defines whether or not to allow user to modify local bend radius of route
    • Type: integer
    • Default value: 1
    • Default Value Role: By default this option is ON i.e. Local bend radius modification is allowed
    • Values:
      0 Local bend radius modification is not allowed.
      1 Local bend radius modification is allowed.
  • L3dMultiRouteOnPortCheck
    Role: Setting to Allow multiple route on port
    Detailed role: Defines whether or not allow multiple route on same port
    • Type: integer
    • Default value: 0
    • Default Value Role: By default this option is OFF i.e. No multiple route allowed on same port.
    • Values:
      0 Do not allow multiple route on port, port will be selected automatically
      1 Allow multiple route on port, port will not be selected automatically
  • L3dOpenInRFLPCheck
    Role: Defines option for opening physical data
    Detailed role: Decides whether to open physical data from Web-top to win-top session in RFLP editor or HVAC 3D design
    • Type: integer
    • Default value: 0
    • Default Value Role: By default open in HVAC 3D design
    • Values:
      0 Open in HVAC 3D design
      1 Open in RFLP editor
  • L3dPartOriginCheck
    Role: Place at component's origin when placing in free space
    Detailed role: Place at component's origin when placing in free space except place in basic shape
    • Type: integer
    • Default value: 0
    • Default Value Role: By default the option will be OFF for not to place at component's origin when placing in free space
    • Values:
      0 Do not restrict to component origin placing in free space
      1 Place at component's origin when placing in free space
  • L3dPartPlacementWithOptionCheck
    Role: Defines option for Part Placement
    Detailed role: Place Part depends on user selection
    • Type: integer
    • Default value: 0
    • Default Value Role: By default Create New Instance
    • Values:
      0 Create New Instance
      1 Move Selected Instance
  • L3dPopUpFlexPipeTablesCheck
    Role: Defines option for Pop Up Tables
    Detailed role: Defines whether or not to Pop Up Tables for Flexible Routing
    • Type: integer
    • Default value: 0
    • Default Value Role: By default this option is OFF i.e. no Pop up will be done for flexible pipe creation if attributes are valuated for previous flexible pipe
    • Values:
      0 Do not Pop Up Tables for flexible pipe if attributes are valuated for previous flexible pipe
      1 Pop Up tables for every single flexible pipe creation and ignore attribute values set for previous flexible pipe
  • L3dPopUpPipeTablesCheck
    Role: Defines option for Pop Up Tables
    Detailed role: Defines whether or not to Pop Up Tables for Routing
    • Type: integer
    • Default value: 0
    • Default Value Role: By default this option is OFF i.e. no Pop up will be done for pipe creation
    • Values:
      0 Do not Pop Up Tables
      1 Pop Up tables for every single pipe creation
  • L3dResourceTablesCheck
    Role: Force usage of resource tables for routing
    Detailed role: Force usage of Turn rules and Minimum Tangent length rule Tables for pipe reference selection
    • Type: integer
    • Default value: 0
    • Default Value Role: By default the user will not forced to use the resource tables for routing
    • Values:
      0 User will not forced to use the resource tables for routing
      1 User will be forced to use the resource tables for routing
  • L3dSearchRouteNewRefCheck
    Role: Setting to Allow search for NewReference route
    Detailed role: Defines whether or not allow Search for route with Instatiation Rule = New Reference
    • Type: integer
    • Default value: 0

      0 Do not allow Search for route with Instatiation Rule = NewReference
      1 Allow Search for route with Instatiation Rule = New Reference for NonStandard Port
