DELFITMinimumDistanceSettings Setting Repository
Quick Reference

DELFITMinimumDistanceSettings

Role: Settings to handle the parameters in the Manufacturing Path Optimization App

Detailed role: This setting allows to customize the appearance and the behavior in Manufacturing Path Optimization

The DELFITMinimumDistanceSettings repository has the following characteristics:

  • Primary Tab Page: /Manufacturing Planning/Multiple Minimum Distances
  • Repository File: DELFITMinimumDistanceSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DELFITMinimumDistanceSettings setting repository attributes:

  • MinimumDistanceContextPoint
    Role: ContextPoint
    Detailed role: The trajectory mode that can be run
    • Type: integer
    • Default value: 0
  • MinimumDistanceMovingPoint
    Role: Moving Point
    Detailed role: The trajectory mode that can be run
    • Type: integer
    • Default value: 0
  • MinimumDistanceObjectName
    Role: Object Name
    Detailed role: The trajectory mode that can be run
    • Type: integer
    • Default value: 0
