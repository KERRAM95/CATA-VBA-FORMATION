NavigationNext Setting Repository
Quick Reference

NavigationNext

Role: Navigation Next Settings

Detailed role: Allows customization of navigation functionalities

The NavigationNext repository has the following characteristics:

  • Primary Tab Page: /General/VirtualReality/NavigationNext
  • Repository File: NavigationNext
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

NavigationNext setting repository attributes:

  • MaxNavSpeed
    Role: Navigation speed
    Detailed role: Controls the navigation speed
    • Type: float
    • Default value: 1.0
    • Min value: 0.1
    • Max value: 99999999
    • Step: 0.1
  • RotMode
    Role: Determines the rotation type
    • Type: integer
    • Default value: 1
    • Values:
      1 Step-Wise
      2 Continuous
  • RotSpeed
    Role: Rotation speed
    Detailed role: Controls the speed for the continuous rotation mode
    • Type: float
    • Default value: 5
    • Min value: 0.1
    • Max value: 99999999
    • Step: 0.1
  • RotStepAngle
    Role: Size of the step angle
    Detailed role: Controls the rotation angle for step-wise rotation mode
    • Type: float
    • Default value: 20.0
    • Min value: 1
    • Max value: 180
    • Step: 1
