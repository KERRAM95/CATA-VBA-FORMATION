GeometricModeler Setting Repository
Quick Reference

GeometricModeler

Role: Setting to handle design limits for new parts creation

Detailed role: Setting defines the design limits for new parts and products creation, in order to choose tolerances and maximum design size.

The GeometricModeler repository has the following characteristics:

  • Primary Tab Page: /General/Parameters And Measures/Design Limits
  • Repository File: GeometricModeler
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

GeometricModeler setting repository attributes:

  • GeometricScale
    Role: Contains the ratio of tolerances for the current scale compared to standard normal scale tolerances.
    • Type: double
    • Default value: 1.0
    • Values:
      0.0001 Extra small Range
      0.01 Small Range
      1.0 Normal Range
      100.0 Large Range
      10000.0 Extra large Range
