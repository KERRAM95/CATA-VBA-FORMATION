FbdiResourceSettingsRep Setting Repository
Quick Reference

FbdiResourceSettingsRep

Role: Settings for the FBDI Resource command.

Detailed role: This setting allows to apply occurence position to first instance during File-based Design Import.

The FbdiResourceSettingsRep repository has the following characteristics:

  • Primary Tab Page: /General/FBDI
  • Repository File: FbdiResourceSettingsRep
  • Version: 4
  • Full Compatibility: 1
  • Upward Compatibility: 1

FbdiResourceSettingsRep setting repository attributes:

  • FBDI_FlexStateLoc
    Role: Controls the location for Flexible Simulation States creation.
    • Type: integer
    • Default value: 1
    • Values:
      0 Any object, stop at physical resources
      1 MfgCell only
  • FBDI_Resource_checkBox
    Role: allows to apply occurence position to first instance in FBDI.
    • Type: integer
    • Default value: 0
    • Values:
      0 NoAction
      1 PropagatePosition
      2 ConvertFlexibility
