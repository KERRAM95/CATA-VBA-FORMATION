CATSsmPreferenceRepository Setting Repository
Quick Reference

CATSsmPreferenceRepository

Role: Setting to handle Space Allocation app preference setting

Detailed role: Space Allocation app preference setting

The CATSsmPreferenceRepository repository has the following characteristics:

  • Primary Tab Page: /MechanicalDesign/CATSsmPLMRepConfig/SpaceAllocation
  • Repository File: CATSsmPreferenceRepository
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CATSsmPreferenceRepository setting repository attributes:

  • SpaceHierarchyType
    Role: Defines the option for the type of space hierarchy.
    Detailed role: Choose whether to use the Space Hierarchy with Space Concept Node or without Space Concept Node.
    • Type: integer
    • Default value: 0
