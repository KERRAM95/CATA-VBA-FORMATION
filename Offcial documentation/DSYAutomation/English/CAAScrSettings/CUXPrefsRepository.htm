Quick Reference

CUXPrefsRepository

Role: CATIA User Experience Preferences

Detailed role: CATIA User Experience Preferences in various CATIA Workbenches

The CUXPrefsRepository repository has the following characteristics:

  • Primary Tab Page:  
  • Repository File: CUXPrefsRepository
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CUXPrefsRepository setting repository attributes:

  • DisplayNavBar
    Role: Navigation Bar display flag
    • Type: boolean
    • Default value: 1
  • MHC_feature
    Role: MultiHighlight color for features.
    • Type: integer
    • Default value: 144
