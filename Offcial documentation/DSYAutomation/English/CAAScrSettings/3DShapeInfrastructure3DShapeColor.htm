3DShapeInfrastructure3DShapeColor

Role: Setting to manage some 3D Shape Color Infrastructure options

The 3DShapeInfrastructure3DShapeColor repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/3D Shape Infrastructure/Colors
  • Repository File: 3DShapeColor
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

3DShapeInfrastructure3DShapeColor setting repository attributes:

  • ColorInheritanceMode
    Role: Specifies how the color / opacity are propagated.
    Detailed role: Specifies how the color / opacity are propagated on surface based features
    • Alias: ColorInheritanceMode
    • Type: short integer
    • Default value: 1
    • Values:
      0 Inheritance mode
      Detailed role: The Inheritance mode has predefined values.
      1 Inherit sublement and all inputs.
      Detailed role: Inherit all inputs and sub element colors.
      2 Inherit sublement and main input.
      Detailed role: Inherit main input and sub element colors.
  • ColorSynchronizationEditability
    Role: Color synchronization property editability
