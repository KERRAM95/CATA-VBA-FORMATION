EV5VPMNOpen Setting Repository
Quick Reference

EV5VPMNOpen

Role: Setting to handle the parameters of the VPM Open Options user interface for the Open command

Detailed role: This setting allows to customize the behavior of the Open command through the user interface

The EV5VPMNOpen repository has the following characteristics:

  • Primary Tab Page: /ENOVIA V5 VPM/Open Options
  • Repository File: EV5VPMNOpen
  • Version: 2
  • Full Compatibility: 1
  • Upward Compatibility: 1

EV5VPMNOpen setting repository attributes:

  • ExploreBehavior
    Role: Indicate the default behavior for Explore
    Detailed role: This option allows to choose the default behavior of the Explore command.
    • Type: integer
    • Default value: 0
  • MoveWebInWinMode
    Role: Move Web in Win Command
    • Type: boolean
    • Default value: false
  • VPMNGeometryQuality
    Role: To control quality of geometry for objects loaded in Lightweight mode
    • Type: string
    • Default value: Optimized
    • Values:
      Optimized Simplified geometry will be displayed
      High High Geometry exactly the same as in standard open will be displayed
  • VPMNOpenDMUMode
    Role: Open the elements in the DMU mode
    Detailed role: This option allows to open the generated DMU cgr of the selected and founded element. In this mode, many authoring commands are forbidden.
    • Type: boolean
    • Default value: false
  • VPMNOpenDesignSpecsMode
    Role: Retrieve all the relational design specifications
    Detailed role: This option allows to retrieve all the relational design specifications.
    • Type: boolean
    • Default value: true
  • VPMNOpenDlgMaxNbElemBeforeWarningMode
    Role: Check number of elements to open before open transaction
    Detailed role: This option allows to check the maximum of elements to open without warning the user before the open transaction.
    • Type: boolean
    • Default value: false
  • VPMNOpenEditabilityMode
    Role: Control privileges at open
    Detailed role: This option allows to control privileges of the element to open.
    • Type: boolean
    • Default value: false
  • VPMNOpenForbidEditionMode
    Role: Forbid edition of read-only data
    Detailed role: This option allows to forbid edition of the element to open.
    • Type: boolean
    • Default value: false
  • VPMNOpenLightweight
    Role: Enable Open in Lightweight mode
    Detailed role: This option enables opening product structure in Lightweight mode
    • Type: boolean
    • Default value: false
  • VPMNOpenLockMode
    Role: Lock all the selected element and their reference
    Detailed role: This option allows to lock all the selected element to open and their reference.
    • Type: boolean
    • Default value: false
  • VPMNOpenMaxNbElemBeforeWarningMode
    Role: Max number of elements to open before open transaction without check
    Detailed role: This option saves the maximum number of elements to open without warning the user before the open transaction.
    • Type: integer
    • Default value: 1000
    • Min value: 0
    • Max value: 100000
  • VPMNOpenMultiRoot
    Role: Enable multi-root for Open
    Detailed role: This option controls the availability of multi-root for Open.
    • Type: boolean
    • Default value: false
  • VPMNOpenNavAsmExpMode
    Role: Retrieve the expanded children from database
    Detailed role: This option allows to retrieve to the expanded children from database of the current selected element to open.(for VPM Navigator application only )
    • Type: boolean
    • Default value: false
  • VPMNOpenNavAsmMode
    Role: Retrieve the visible expanded children
    Detailed role: This option allows to retrieve to the visible expanded children of the current selected element to open.(for VPM Navigator application only)
    • Type: boolean
    • Default value: true
  • VPMNOpenNone3DSRepInSeparateEditorMode
    Role: Edit in separate editor all none 3D Shape representation
    Detailed role: This option allows to edit separatly a selected none 3D shape representation in its own editor and its context.
    • Type: boolean
    • Default value: true
  • VPMNOpenRepMode
    Role: Retrieve the representation mode
    Detailed role: This option allows to retrieve the wanted representation of the reference depending on its type.
    • Type: integer
    • Default value: 1
    • Default Value Role: By default 3D shape representation is retrieved
    • Values:
      1 Retrieve the 3D Shape representation
      Detailed role: This option allows to retrieve the 3D Shape representation of the reference.
      0 Do not retrieve any kind of representation
      Detailed role: This option allows to not retrieve any kind of representation for all references.
      2 Retrieve all kind of representation
      Detailed role: This option allows to retrieve all representations for all references.
  • VPMNOpenRepsWithout3D
    Role: Open the representations without 3D
    Detailed role: This option allows to open the representations without their geometry.
    • Type: boolean
    • Default value: false
  • VPMNOpenSearchMode
    Role: Retrieve the expanded children from database
    Detailed role: This option allows to retrieve to the expanded children from database of the current selected element to open.(for VPM Search application only)
    • Type: boolean
    • Default value: true
  • VPMNOpenWarnOpportEditionMode
    Role: Display a warning if read-only data are edited and auto-lock the data to modification
    Detailed role: This option allows to warn the user if read only data are edited. This option auto-lock the data if the switch is a success (done during first modification)
    • Type: boolean
    • Default value: false
  • VPMNOpenWarnPessEditionMode
    Role: Display a warning if read-only data are edited and auto-lock the data
    Detailed role: This option allows to warn the user if read only data are edited. This option auto-lock the data if the switch is a success
    • Type: boolean
    • Default value: false
  • VPMNOpenWarningEditionMode
    Role: Display a warning if read-only data are edited
    Detailed role: This option allows to warn the user if read only data are edited.
    • Type: boolean
    • Default value: false
