LayerNameDisplay Setting Repository
Quick Reference

LayerNameDisplay

Role: Layer Display Format

Detailed role: This option controls the display format of Layers.

The LayerNameDisplay repository has the following characteristics:

  • Primary Tab Page: /General/Display/Layer Filter
  • Repository File: LayerNameDisplay
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

LayerNameDisplay setting repository attributes:

  • ApplyFilterMode
    Role: Filter and Document Display mode
    • Type: integer
    • Default value: 0
    • Default Value Role: By default display layer number
    • Values:
      0 Display layer number
      1 Display layer name
      2 Display layer number and name
