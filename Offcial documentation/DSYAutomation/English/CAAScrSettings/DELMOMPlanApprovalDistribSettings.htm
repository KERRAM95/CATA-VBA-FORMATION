DELMOMPlanApprovalDistribSettings Setting Repository
Quick Reference

DELMOMPlanApprovalDistribSettings

Role: Settings for Plan Approval Distribution

Detailed role: This setting allows to customize the activity parameters in plan approvval distribution.

The DELMOMPlanApprovalDistribSettings repository has the following characteristics:

  • Primary Tab Page: /WSPROC/DPManufacturing/Plan Approval Distribution
  • Repository File: DELMOMPlanApprovalDistribSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DELMOMPlanApprovalDistribSettings setting repository attributes:

  • ChangeMaturity
    Role: Change Maturity
    Detailed role: Change Maturity After Publish option
    • Type: boolean
    • Default value: false
  • ChangeMaturityTo
    Role: To Maturity - to be changed to after publish is done
    • Type: string
    • Default value: FROZEN
    • Default Value Role: By default FROZEN
    • Default Value Detailed Role Workplan or System maturity to which System or Workplan Maturity will be changed after publish
  • MOMAprisoServer
    Role: Update path to Apriso drop file location
    • Type: string
    • Default value:
    • Default Value Role: By default blank
    • Default Value Detailed Role Blank indicates that Appriso is not being used
  • MOMAutoSave
    Role: Auto Save
    Detailed role: Automatically save after generating graphics and views
    • Type: boolean
    • Default value: false
  • MOMCheckLatestRevision
    Role: Check All Operations At Latest Revision
    Detailed role: Check to make sure that all operations are at the latest revision before generating or pubishing.
    • Type: boolean
    • Default value: false
  • MOMExportToExecutionDropFolder
    Role: Update path to Apriso drop file location
    • Type: string
    • Default value:
    • Default Value Role: By default blank
    • Default Value Detailed Role Blank indicates that Appriso is not being used
  • MOMOperationReleasedMaturity
    Role: Operation Maturity that allows publishing of the system
    • Type: string
    • Default value: RELEASED
    • Default Value Role: By default RELEASED
    • Default Value Detailed Role Operation Maturity that allows publishing of the system
  • MOMSingleMaturity
    Role: Single Maturity Enabled (requires autosave)
    Detailed role: Allow single maturity for generate and publish generating or pubishing. Requires Auto Save.
    • Type: boolean
    • Default value: false
  • MOMSystemReleasedMaturity
    Role: System Maturity that allows publishing of the system
    • Type: string
    • Default value: RELEASED
    • Default Value Role: By default RELEASED
    • Default Value Detailed Role System Maturity that allows updates to the system
  • MOMSystemUpdatableMaturity
    Role: System Maturity That allows updates to the system
    • Type: string
    • Default value: IN_WORK
    • Default Value Role: By default IN_WORK
    • Default Value Detailed Role System Maturity that allows updates to the system
  • MOMxPDMSiteId
    Role: Update path to Apriso drop file location
    • Type: string
    • Default value:
    • Default Value Role: By default blank
    • Default Value Detailed Role Blank indicates that Appriso is not being used
  • Publish2DViews
    Role: Publish 2D Views
    Detailed role: Publish 2D images corresponding to Views
    • Type: boolean
    • Default value: false
  • Publish3DBuildup
    Role: Publish 3D Buildup
    Detailed role: Publish 3D Buildup corresponding to the Operations
    • Type: boolean
    • Default value: false
  • Publish3DViews
    Role: Publish 3D Views
    Detailed role: Publish 3D Views along with 3D Buildup corresponding to the Operations
    • Type: boolean
    • Default value: true
  • PublishGenDoc
    Role: Generation of Documentation after Publish
    Detailed role: Generate Documents after Publish Option
    • Type: boolean
    • Default value: false
  • UnpublishedViewsOnly
    Role: Publish Only Unpublished 2D Views
    Detailed role: Publish only unpublished 2D images corresponding to Views
    • Type: boolean
    • Default value: false
  • WorkInstructionAprisoProtocol
    Role: Protocol to use for web service calls
    • Type: string
    • Default value: https
    • Default Value Role: By default set to 'https'
    • Default Value Detailed Role We use https (SSL) by default
  • WorkInstructionEnablePublishToExecution
    Role: Enable Publish To Execution
    • Type: integer
    • Default value: 0
    • Default Value Role: By default disable
    • Default Value Detailed Role The value can be 0(disable) or 1(enable)
  • WorkInstructionGenerate3D
    Role: Generate 3D Graphics
    • Type: integer
    • Default value: 1
    • Default Value Role: By default enable
    • Default Value Detailed Role The value can be 0(disable) or 1(enable)
  • WorkInstructionPublishFormat
    Role: Published Data Format
    • Type: integer
    • Default value: 0
    • Default Value Role: File based is default
    • Default Value Detailed Role The value can be 0(File) or 1(Connected)
