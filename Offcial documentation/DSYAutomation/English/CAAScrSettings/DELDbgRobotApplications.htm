DELDbgRobotApplications Setting Repository
Quick Reference

DELDbgRobotApplications

Role: Setting industrial robot applications

Detailed role: Industrial robots are used for different types of applications. These options defines what types of applications are available.

The DELDbgRobotApplications repository has the following characteristics:

  • Primary Tab Page: /Delmia Infrastructure/Robot Applications
  • Repository File: DELDbgRobotApplications
  • Version: 2
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Root tab page /x reused as /x

DELDbgRobotApplications setting repository attributes:

  • RobotApplicationsType
    Role: Industrial robot applications type
    • Type: string
    • Default value: Polishing
