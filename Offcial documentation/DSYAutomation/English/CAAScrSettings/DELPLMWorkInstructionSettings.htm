DELPLMWorkInstructionSettings Setting Repository
Quick Reference

DELPLMWorkInstructionSettings

Role: Settings for Work Instructions

Detailed role: This setting allows to customize the activity parameters in work instruction planning workbench

The DELPLMWorkInstructionSettings repository has the following characteristics:

  • Primary Tab Page: /WSPROC/DPManufacturing/Work Instructions
  • Repository File: DELPLMWorkInstructionSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
