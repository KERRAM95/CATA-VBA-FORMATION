WDGGeneral Setting Repository
Quick Reference

WDGGeneral

Role: These setting Parameters define the general Weld related Parameters

Detailed role: Enable the user to specify the weld related parameters

The WDGGeneral repository has the following characteristics:

  • Primary Tab Page: /Mechanical/WeldDesign/General
  • Repository File: WDGGeneral
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

WDGGeneral setting repository attributes:

  • WdgCreationMode
    Role: Indicates which approach will be used for the creation of welds
    • Type: integer
    • Default value: 1
    • Values:
      1 AdvancedSketchApproach: Weld creation with advanced user swept approach
      2 SimpleSketchApproach: Weld creation with simple approach
