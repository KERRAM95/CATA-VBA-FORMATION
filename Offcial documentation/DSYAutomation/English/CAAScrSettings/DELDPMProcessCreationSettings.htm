DELDPMProcessCreationSettings Setting Repository
Quick Reference

DELDPMProcessCreationSettings

Role: Process Creation settings

Detailed role: Setting to handle Process creation options

The DELDPMProcessCreationSettings repository has the following characteristics:

  • Primary Tab Page: /Digital Process for Manufacturing//Assembly Process Planner
  • Repository File: DELDPMProcessCreationSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DELDPMProcessCreationSettings setting repository attributes:

  • ALDiameter
    Role: Check Same Diameter for Fasteners
    • Type: boolean
    • Default value: false
  • ALJoinedParts
    Role: Check Same Joined Parts for Fasteners
    • Type: boolean
    • Default value: true
  • ALType
    Role: Check Same Type for Fasteners
    • Type: boolean
    • Default value: false
  • AssignPrdToProcessLPE
    Role: Current product should be assigned to the process structure if Live Process Editor is opened on VPMProduct
    • Type: boolean
    • Default value: true
  • AssignmentFilterMode
    Role: Assignment Filter Mode
    • Type: boolean
    • Default value: true
  • AutoFastenerIndus
    Role: Generate the complete process structure of a Create Part process
    • Type: boolean
    • Default value: true
  • ComputeResultingProducts
    Role: Compute resulting products
    • Type: boolean
    • Default value: true
  • ControlReassignmentInLPE
    Role: Control Reassignment of parts if exposed port exists.
    • Type: boolean
    • Default value: false
  • CopyModelFromFather
    Role: Copy configuration model from father
    • Type: boolean
    • Default value: false
  • CopyModelFromProduct
    Role: Copy configuration model from product
    • Type: boolean
    • Default value: false
  • CopyModelVersion
    Role: Copy model version from product
    • Type: boolean
    • Default value: false
  • CreateScopeOnProvide
    Role: Create scope on provided part
    • Type: boolean
    • Default value: true
  • DirectDetailedFastenWithStrategy
    Role: Direct Detailed Fastens
    • Type: boolean
    • Default value: false
  • DirectFastenDetailing
    Role: Direct Fasten Detailing
    • Type: boolean
    • Default value: false
  • DisableCompute
    Role: Disable the Compute and Update resulting product commands
    • Type: boolean
    • Default value: false
  • FastenReference
    Role: Use Same Fasten Reference For Fasteners
    • Type: boolean
    • Default value: true
  • FstAutoAssign
    Role: Fasteners Automatic Assignment with products
    • Type: boolean
    • Default value: false
  • HoleDepth
    Role: Parameter Depth of the material removel processes created when generating a Create Part process
    • Type: double
    • Default value: 0.0010
  • HoleDiameter
    Role: Parameter Diameter of the material removel processes created when generating a Create Part process
    • Type: double
    • Default value: 0.0025
  • IndustryMode
    Role: Enum identifying the Industry mode for process creation. 0=PMS,1=Aero,2=Admin,3=BiW,4=FA,5=Advanced,6=UserDefined
    • Type: unsigned integer
    • Default value: 5
  • InsertSpareItemOnAssignment
    Role: Inserts a Spare Item when a product is assigned to a Service Assembly/Kit/Part/Attaching Item
    • Type: boolean
    • Default value: false
  • InsertionModeForInstallOnInstall
    Role: Enum needed for inserting an Install on Install 0 - As Child, 1 - As Input
    • Type: unsigned integer
    • Default value: 1
  • LoadResultingProducts
    Role: Load resulting products
    • Type: boolean
    • Default value: true
  • LockAssistant
    Role: Locks the Create Part Assistant
    • Type: boolean
    • Default value: false
  • MfgFastenerStrategy
    Role: Define and use Manufacturing Strategy
    • Type: boolean
    • Default value: false
  • MountFastenerComponents
    Role: Mount the fastener components when generating a Create Partprocess
    • Type: boolean
    • Default value: false
  • Overload3DValue
    Role: Default value for Overload column
    • Type: unsigned integer
    • Default value: 1
  • ReconnectOperations
    Role: Reconnect operations
    • Type: boolean
    • Default value: true
  • RemoveMaterialType
    Role: Type a material removal created when generating a Create Part process. 0=PreDrill, 1=Drill, 2=NoDrill
    • Type: integer
    • Default value: 1
  • ReuseSearchMode
    Role: Search Mode for Use Same Process Reference for multi-instantiated Parts
    • Type: unsigned integer
    • Default value: 1
  • ReuseServiceItemSearchMode
    Role: Search Mode for Use Same Service Item Reference for multi-instantiated Parts
    • Type: unsigned integer
    • Default value: 1
  • SearchAssignableParent
    Role: Search for an assignable parent
    • Type: boolean
    • Default value: false
  • ShowOutputOnTile
    Role: Show output on tile
    • Type: boolean
    • Default value: true
  • SpotAssemblyFstModeALDiameter
    Role: Check Same Diameter for Fasteners
    • Type: boolean
    • Default value: false
  • SpotAssemblyFstModeALJoinedParts
    Role: Check Same Joined Parts for Fasteners
    • Type: boolean
    • Default value: true
  • SpotAssemblyFstModeALType
    Role: Check Same Type for Fasteners
    • Type: boolean
    • Default value: false
  • SpotAssemblyFstModeFastenReference
    Role: Use Same Fasten Reference For Fasteners
    • Type: boolean
    • Default value: true
  • UpdateStatusMode
    Role: To identify if process output update status is done from design (0) or from process (1)
    • Type: integer
    • Default value: 0
  • UseSameProcessReference
    Role: Use Same Process Reference for multi-instantiated Parts
    • Type: boolean
    • Default value: true
  • UseSameServiceItemReference
    Role: Use Same Service Item Reference for multi-instantiated Parts
    • Type: boolean
    • Default value: true
  • WorkingUnderAction
    Role: Working Under Action
    • Type: boolean
    • Default value: false
