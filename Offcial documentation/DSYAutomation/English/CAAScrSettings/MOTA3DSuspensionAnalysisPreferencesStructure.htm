Quick Reference

MOTA3DSuspensionAnalysisPreferencesStructure

Role: Setting to manage Suspension Analysis options

Detailed role: Suspension Analysis setting contains all options necessary to Suspension Analysis commands

The MOTA3DSuspensionAnalysisPreferencesStructure repository has the following characteristics:

  • Primary Tab Page: /Physics Simulation and Simulation for Designers/Motion/Joint Solve
  • Repository File: MOTA3DSuspensionAnalysisPreferencesStructure
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Root tab page /Physics Simulation and Simulation for Designers/Motion/Joint Solve reused as /Physics Simulation and Simulation for Designers/Motion/Joint Solve

MOTA3DSuspensionAnalysisPreferencesStructure setting repository attributes:

  • MechanismSolveManagementForMotion
    Role: Attribute to activate/deactivate animation during joint solve.
    Detailed role: Animate body positioning during joint definition.
    • Type: boolean
    • Default value: true
    • Default Value Role: By default, animate the joint solve.
