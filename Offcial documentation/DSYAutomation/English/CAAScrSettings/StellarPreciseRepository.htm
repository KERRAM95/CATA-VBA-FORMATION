StellarPreciseRepository Setting Repository
Quick Reference

StellarPreciseRepository

Role: Setting to handle the stellar precise settings

The StellarPreciseRepository repository has the following characteristics:

  • Primary Tab Page: /General/Display/RayTracing View Mode
  • Repository File: StellarPreciseRepository
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

StellarPreciseRepository setting repository attributes:

  • CompressionRatio
    Role: Compression Ratio
    • Type: unsigned integer
    • Default value: 60
  • DStellarHardwareMode
    Role: Selection of CPU or GPU
    • Type: unsigned integer
    • Default value: 0
    • Values:
      0 CPU
      1 GPU
  • DisableNoShowTranslation
    Role: Disable NoShow Translation
    • Type: boolean
    • Default value: false
  • DistributedRendering
    Role: Enable distributed rendering
    • Type: boolean
    • Default value: false
  • DynamicMode
    Role: Parameter to set the dynamic mode
    • Type: unsigned integer
    • Default value: 0
    • Values:
      0 Rasterizer
      1 Global Illumination
  • EnableCompression
    Role: Enable Compression
    • Type: boolean
    • Default value: false
  • HardwareMode
    Role: Selection of CPU or GPU
    • Type: unsigned integer
    • Default value: 1
    • Values:
      0 CPU
      1 GPU
  • HostName
    Role: Cluster host name
    • Type: string
    • Default value:
  • LocalHostName
    Role: Local host name
    • Type: string
    • Default value:
  • PerformanceQualityMode
    Role: Performance / Quality
    • Type: unsigned integer
    • Default value: 4
    • Values:
      0 1/16 Resolution
      1 1/8 Resolution
      2 1/4 Resolution
      3 1/2 Resolution
      4 Full Resolution
  • Port
    Role: Cluster port
    • Type: unsigned integer
    • Default value: 8080
  • TransitionFrames
    Role: Number of transition frames
    • Type: unsigned integer
    • Default value: 0
