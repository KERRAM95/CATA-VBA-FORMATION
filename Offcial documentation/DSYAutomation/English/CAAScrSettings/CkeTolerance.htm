CkeTolerance Setting Repository
Quick Reference

CkeTolerance

Role: Defines some options for the Report generation

Detailed role: Enable the user to deal with these categories of options: Tolerance, Length and Angle

The CkeTolerance repository has the following characteristics:

  • Primary Tab Page: /General/Parameters and Measure/Report Generation
  • Repository File: CkeTolerance
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CkeTolerance setting repository attributes:

  • defaultTolerance
    Role: Enables the user to specify default values for length and angle if set to true
    Detailed role: Enables the user to specify default values for length and angle if set to true
    • Type: boolean
    • Default value: false
  • maxAngleTolerance
    Role: Enables you to assign a maximum tolerance angle
    Detailed role: Enables you to assign a maximum tolerance angle
    • Type: double
    • Default value: 0.0
  • maxLengthTolerance
    Role: Enables you to assign a maximum tolerance length
    Detailed role: Enables you to assign a maximum tolerance length
    • Type: double
    • Default value: 0.0
  • minAngleTolerance
    Role: Enables you to assign a minimum tolerance angle
    Detailed role: Enables you to assign a minimum tolerance angle
    • Type: double
    • Default value: 0.0
  • minLengthTolerance
    Role: Enables you to assign a minimum tolerance length
    Detailed role: Enables you to assign a minimum tolerance length
    • Type: double
    • Default value: 0.0
