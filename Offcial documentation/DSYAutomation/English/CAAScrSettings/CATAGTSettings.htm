CATAGTSettings Setting Repository
Quick Reference

CATAGTSettings

Role: These settings define the user choice for CATIA Accommodation Design environment

Detailed role: These settings define the user choice of the various parameters related to Accommodation Design.

The CATAGTSettings repository has the following characteristics:

  • Primary Tab Page: /Marine and Offshore/Accommodation Design/Accommodation
  • Repository File: CATAGTSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CATAGTSettings setting repository attributes:

  • CeilingsPanelGap
    Role: Ceiling Panel Gap
    Detailed role: Defines Panel Gap in the ceiling
    • Type: double
    • Default value: 0.0
  • CoveringsPanelGap
    Role: Covering Panel Gap
    Detailed role: Defines Panel Gap in the covering
    • Type: double
    • Default value: 0.0
  • WallsPanelGap
    Role: Wall Panel Gap
    Detailed role: Defines Panel Gap in the Wall
    • Type: double
    • Default value: 0
