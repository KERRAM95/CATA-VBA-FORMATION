CATAssemblySymmetry Setting Repository
Quick Reference

CATAssemblySymmetry

Role: Setting to handle the Assembly Symmetry options

Detailed role: Setting to handle the Assembly Symmetry options

The CATAssemblySymmetry repository has the following characteristics:

  • Primary Tab Page: /Mechanical Design/Assembly Design/Symmetry
  • Repository File: CATAssemblySymmetry
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CATAssemblySymmetry setting repository attributes:

  • DefaultGenerationPlane
    Role: Default generation plane for the symmetric structure
    • Type: integer
    • Default value: 1
    • Default Value Role: By default, the symmetry generation plane is YZ-Plane
    • Values:
      0 XY-Plane
      1 YZ-Plane
      2 ZX-Plane
  • ValidationClickOnAir
    Role: Option to specify clicking anywhere in 3D area validate the command
    Detailed role: Depending on this option, clicking anywhere in 3D area validate the symmetry command
    • Type: integer
    • Default value: 1
    • Default Value Role: By default the click on air validate the symmetry command
    • Values:
      0 Click on air doesn't validate the symmetry command
      1 Click on air validate the symmetry command
