EngineeringTemplate Setting Repository
Quick Reference

EngineeringTemplate

Role: Settings to manage default Business Logic setting

The EngineeringTemplate repository has the following characteristics:

  • Primary Tab Page: /MechanicalDesign/EngineeringTemplate
  • Repository File: EngineeringTemplate
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

EngineeringTemplate setting repository attributes:

  • CATBLNewTemplate
    Role: Defines if the specific option 'New object identification Business Logic' to be activated as default selection
    • Type: boolean
    • Default value: false
