VOCSettings Setting Repository
Quick Reference

VOCSettings

Role: Setting to handle the VOC parameters

The VOCSettings repository has the following characteristics:

  • Primary Tab Page: /Digital Mockup/VOC Settings
  • Repository File: VOCSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

VOCSettings setting repository attributes:

  • DisplayMode
    Role: Default choice for Display mode.
    • Type: integer
    • Default value: 0
    • Values:
      0 Smooth
      1 Flat
  • UseGeometryColor
    Role: Use geometry color for Thickness and Offset volumes
    • Type: integer
    • Default value: 0
    • Values:
      0 Use geometry color for Thickness and Offset volumes Unchecked
      1 Use geometry color for Thickness and Offset volumes Checked
  • UsePartColor
    Role: Use Part Color Checkbox Status
    • Type: integer
    • Default value: 0
    • Values:
      0 Use Part Color Unchecked
      1 Use Part Color Checked
  • VOCImprovedPerformance
    Role: Improved Performance Checkbox Status
    • Type: integer
    • Default value: 0
    • Values:
      0 Use Convex Envelope
      1 Use Simple Envelope
  • VOCResultColor
    Role: Default color of the VOC Result
    • Type: unsigned integer
    • Default value: 105
    • Default Value Role: Brown in RGB order
    • Min value: 0
    • Max value: 255
