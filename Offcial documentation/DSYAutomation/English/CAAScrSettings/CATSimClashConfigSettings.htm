CATSimClashConfigSettings Setting Repository
Quick Reference

CATSimClashConfigSettings

Role: Settings to handle clash management

Detailed role: Default settings that will be used when performing some clash analysis.

The CATSimClashConfigSettings repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/Delmia Infrastructure/Clash
  • Repository File: CATSimClashConfigSettings
  • Version: 6
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Root tab page /x reused as /x

CATSimClashConfigSettings setting repository attributes:

  • ClashColor
    Role: Color used for Clash
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: Red in RGB order
    • Min value: 0
    • Max value: 255
  • ClashTransp
    Role: Clash transparency
    • Type: unsigned integer
    • Default value: 255
    • Min value: 0
    • Max value: 255
  • ClearThresholdVal
    Role: Default value for clearance threshold warnings [in mm].
    • Type: double
    • Default value: 0
  • ClearanceColor
    Role: Color used for Clearance
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: Yellow in RGB order
    • Min value: 0
    • Max value: 255
  • ClearanceTransp
    Role: Clearance transparency
    • Type: unsigned integer
    • Default value: 255
    • Min value: 0
    • Max value: 255
  • ContactColor
    Role: Color used for Contact
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: Orange in RGB order
    • Min value: 0
    • Max value: 255
  • ContactTransp
    Role: Contact transparency
    • Type: unsigned integer
    • Default value: 255
    • Min value: 0
    • Max value: 255
  • MovingColor
    Role: Color used for Moving objects
    • Type: unsigned integer
    • Default value: 255
    • Default Value Role: Cyan in RGB order
    • Min value: 0
    • Max value: 255
  • MovingTransp
    Role: Moving object transparency
    • Type: unsigned integer
    • Default value: 255
    • Min value: 0
    • Max value: 255
  • PtCldFilterThreshold
    Role: Default value for Point Cloud collision filtering threshold.
    • Type: unsigned integer
    • Default value: 5
  • PtCldSphereRadius
    Role: Default value for point Cloud collision sphere radius [in mm].
    • Type: unsigned integer
    • Default value: 10
    • Min value: 1
    • Max value: 50
  • SimClashConfigClashReporting
    Role: Activate the warning reporting.
    Detailed role: Indicates if warnings coming from the solver should be reported or not.
    • Type: unsigned integer
    • Default value: 0
    • Values:
      0 Warnings are not reported.
      1 Warnings are reported.
  • SimClashConfigDetectionClashState
    Role: Detect and report any clash between parts.
    Detailed role: Indicates if parts that are in collision should be detected and reported. It is only valid for dynamic clash computation.
    • Type: unsigned integer
    • Default value: 1
    • Values:
      0 Parts in collision are ignored
      1 Parts in collision are taken into account
  • SimClashConfigDetectionContactState
    Role: Detect and report any contact between parts.
    Detailed role: Indicates if parts that are in contact should be detected and reported. It is only valid for dynamic clash computation.
    • Type: unsigned integer
    • Default value: 0
    • Values:
      0 Parts in contact are ignored
      1 Parts in contact are taken into account
  • SimClashConfigDetectionInclusionState
    Role: Activate inclusion detection.
    Detailed role: Indicates if parts included into another should be reported as clash or not.
    • Type: unsigned integer
    • Default value: 1
    • Values:
      0 Parts inside other parts are not reported as clash
      1 Parts inside other parts are reported as clash
  • SimClashConfigDetectionPointCloudState
    Role: Activates Point Cloud support.
    Detailed role: Indicates if Point Cloud support is activated or not.
    • Type: unsigned integer
    • Default value: 0
    • Values:
      0 Point Cloud support is not active
      1 Point Cloud support is activated
  • SimClashConfigFeedbackColorState
    Role: Activates(deactivates) the graphical feedback using color.
    Detailed role: This option controls the display of parts in clash/contact/clearance using color.
    • Type: unsigned integer
    • Default value: 0
    • Values:
      0 Deactivates the display of parts in clash/contact/clearance using color.
      1 Activates the display of parts in clash/contact/clearance using color.
  • SimClashConfigFeedbackCurveState
    Role: Activates(deactivates) the graphical feedback as curve.
    Detailed role: This option controls the display of parts in clash using curves.
    • Type: unsigned integer
    • Default value: 0
    • Values:
      0 Deactivates the display of parts in clash using curves
      1 Activates the display of parts in clash using curves
  • SimClashConfigFeedbackHighlightState
    Role: Activates(deactivates) the graphical feedback by highlighting the part in clash.
    Detailed role: This option controls the display of parts in clash by highlighting them.
    • Type: unsigned integer
    • Default value: 1
    • Values:
      0 Deactivates the display of parts in clash by highlighting them
      1 Activates the display of parts in clash by highlighting them
  • SimClashConfigFeedbackMovingColorState
    Role: Activates(deactivates) the graphical feedback using customized color for moving object.
    Detailed role: This option controls the display of moving object using customized color.
    • Type: unsigned integer
    • Default value: 0
    • Values:
      0 Deactivates the display of moving object using customized color.
      1 Activates the display of moving object using customized color.
  • SimClashConfigFeedbackPointCloudLocalState
    Role: Activates local color application on Point Cloud.
    Detailed role: Indicates if local color is applied or not on Point Cloud.
    • Type: unsigned integer
    • Default value: 0
    • Values:
      0 Local color is not applied on Point Cloud
      1 Local color is applied on Point Cloud
  • SimClashConfigUseSAGValue
    Role: Activate usage of SAG Value for Clash computation.
    Detailed role: Indicate whether Clash computation should apply SAG Value.
    • Type: unsigned integer
    • Default value: 0
    • Values:
      0 SAG Value is not used.
      1 SAG Value is used.
  • StopComputeSteps
    Role: Stop on clash computation steps
    • Type: unsigned integer
    • Default value: 5
    • Min value: 0
    • Max value: 10
  • TabularDisplay
    Role: Activates(deactivates) the tabular display for dynamic clash results.
    Detailed role: This option controls the display of dynamic clash results in a tabular format.
    • Type: unsigned integer
    • Default value: 0
    • Values:
      0 Deactivates the tabular display.
      1 Activates the tabular display.
