DELSweptVolSettings Setting Repository
Quick Reference

DELSweptVolSettings

Role: Settings to define the options used for Swept Volume computation

Detailed role: These settings allow users to customize the options used for Swept Volume computation

The DELSweptVolSettings repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/DELMIA Infrastructure/Swept Volume
  • Repository File: DELSweptVolSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DELSweptVolSettings setting repository attributes:

  • AccuracyStat
    Role: Accuracy ON/OFF
    Detailed role: Indicates if Accuracy is turned ON or OFF
    • Type: unsigned integer
    • Default value: 0
    • Values:
      0 OFF
      Detailed role: Accuracy is OFF
      1 ON
      Detailed role: Accuracy is ON
  • AccuracyVal
    Role: Accuracy Value
    Detailed role: Indicates default value for Accuracy (in mm)
    • Type: double
    • Default value: 5
  • ExecStepSize
    Role: Execution Step Size
    Detailed role: Indicates default value for execution step size (in s)
    • Type: double
    • Default value: 0.1
  • ExecSweepColor
    Role: Sweep color
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: Red color
    • Min value: 0
    • Max value: 255
  • ExecSweepTransp
    Role: Sweep transparency
    • Type: unsigned integer
    • Default value: 125
    • Min value: 0
    • Max value: 255
  • OperSweepsStat
    Role: Generate Operation Sweeps ON/OFF
    Detailed role: Indicates if Generate Operation Sweeps is turned ON or OFF
    • Type: unsigned integer
    • Default value: 0
    • Values:
      0 OFF
      Detailed role: Generate Operation Sweeps is OFF
      1 ON
      Detailed role: Generate Operation Sweeps is ON
  • PosPrecisionStat
    Role: Position Precision ON/OFF
    Detailed role: Indicates if Position Precision is turned ON or OFF
    • Type: unsigned integer
    • Default value: 1
    • Values:
      0 OFF
      Detailed role: Position Precision is OFF
      1 ON
      Detailed role: Position Precision is ON
  • PosPrecisionVal
    Role: Position Precision Value
    Detailed role: Indicates default value for position precision (in mm)
    • Type: double
    • Default value: 0.2
  • SingleSweepStat
    Role: Wrap into single sweep ON/OFF
    Detailed role: Indicates if Wrap into single sweep is turned ON or OFF
    • Type: unsigned integer
    • Default value: 0
    • Values:
      0 OFF
      Detailed role: Wrap into single sweep is OFF
      1 ON
      Detailed role: Wrap into single sweep is ON
  • SweepMemoryThreshold
    Role: Memory threshold for swept volume computation
    • Type: double
    • Default value: 5
  • SweepStartOperIndexStat
    Role: Sweep from 1st step of task ON/OFF
    Detailed role: Indicates if Sweep from 1st step of task is turned ON (1) or OFF (0)
    • Type: unsigned integer
    • Default value: 0
  • WrapGrainStat
    Role: Wrapping Grain ON/OFF
    Detailed role: Indicates if Wrapping Grain is turned ON or OFF
    • Type: unsigned integer
    • Default value: 1
    • Values:
      0 OFF
      Detailed role: Wrapping Grain is OFF
      1 ON
      Detailed role: Wrapping Grain is ON
  • WrapGrainVal
    Role: Wrapping Grain Value
    Detailed role: Indicates default value for wrapping grain (in mm)
    • Type: double
    • Default value: 50
