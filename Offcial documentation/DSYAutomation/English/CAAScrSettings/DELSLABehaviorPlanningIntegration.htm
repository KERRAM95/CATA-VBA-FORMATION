DELSLABehaviorPlanningIntegration Setting Repository
Quick Reference

DELSLABehaviorPlanningIntegration

Role: Default values settings of the Factory Flow Simulation application Planning Integration

Detailed role: The default values settings allows default values for Planning Integration modification by the user

The DELSLABehaviorPlanningIntegration repository has the following characteristics:

  • Primary Tab Page: /Factory/PlanningIntegration
  • Repository File: DELSLABehaviorPlanningIntegration
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DELSLABehaviorPlanningIntegration setting repository attributes:

  • AssignResourcePoolForWorker
    Role: Assign Resource Pool For Worker
    • Type: unsigned integer
    • Default value: 1
  • ClearActivitiesOnResourceOnGenOfModelFromPlanning
    Role: Clear Activities on Resources when genrating factory model from Planning
    • Type: unsigned integer
    • Default value: 0
  • ConditionalFlowIntegrationFromPP
    Role: Planning Integration for Conditional Flows
    • Type: unsigned integer
    • Default value: 0
  • CreateAPoolForAllWorkers
    Role: Create a ResourcePool for all Workers
    • Type: unsigned integer
    • Default value: 0
  • CreatePoolForEachWorker
    Role: Create A ResourcePool for each Worker
    • Type: unsigned integer
    • Default value: 1
  • EnableReplaceProductForPlanningLinkedActivity
    Role: Enable add\replace of product for planning linked activity
    • Type: unsigned integer
    • Default value: 0
  • RemoveTimeDistributionWhileSync
    Role: Remove Time Distribution on Activity during Sync if not defined on Operation
    • Type: unsigned integer
    • Default value: 0
