RenderingLiveSettings Setting Repository
Quick Reference

RenderingLiveSettings

Role: Live Rendering options

The RenderingLiveSettings repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/CATIA Live Rendering/General
  • Repository File: RenderingLiveSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

RenderingLiveSettings setting repository attributes:

  • DefaultEngine
    Role: Default engine
    • Tab Page: /Infrastructure/CATIA Live Rendering/General
    • Type: integer
    • Default value: 1
    • Values:
      0 Iray
      1 Stellar
  • EnvironmentSnap
    Role: Snap environment to geometry when created
    • Type: boolean
    • Default value: false
  • FileSaveAllowed
    Role: Allows image save on disk
    • Tab Page: /Infrastructure/CATIA Live Rendering/General
    • Type: integer
    • Default value: 1
  • ForceMaterialSelector
    Role: Force advanced material selector mode when entering Live Rendering app
    • Tab Page: /Infrastructure/CATIA Live Rendering/General
    • Type: integer
    • Default value: 1
  • ForceMaterialShading
    Role: Force material shading mode when entering Live Rendering app
    • Tab Page: /Infrastructure/CATIA Live Rendering/General
    • Type: integer
    • Default value: 1
  • GpuAssignment
    Role: GPU assignment
    • Tab Page: /Infrastructure/CATIA Live Rendering/General
    • Type: string
    • Default value:
  • InsertLogo
    Role: Insert logo in saved images
    • Type: integer
    • Default value: 0
  • IrtPSF
    Role: Path space filtering activation
    • Tab Page: /Infrastructure/CATIA Live Rendering/General
    • Type: integer
    • Default value: 0
  • LREViewpointMode
    Role: Configure Viewpoint Mode setting from Preferences
    • Tab Page: /Infrastructure/CATIA Live Rendering/General
    • Type: integer
    • Default value: 0
    • Values:
      0 Force Perspective mode when entering Live Rendering app
      1 Inherit mode from Rendering Scene Design app
  • Materials3dxmlImporterPopUp
    Role: We open importer dialog at start up
    • Tab Page: /Infrastructure/CATIA Live Rendering/General
    • Type: integer
    • Default value: 1
  • MaterialsPerFace
    Role: Materials applied on faces are raytraced
    • Tab Page: /Infrastructure/CATIA Live Rendering/General
    • Type: integer
    • Default value: 1
  • MaxPathLength
    Role: Live rendering maximum path length
    • Tab Page: /Infrastructure/CATIA Live Rendering/General
    • Type: integer
    • Default value: 23
    • Min value: 1
    • Max value: 1000000
  • NetworkStatus
    Role: Network status
    • Tab Page: /Infrastructure/CATIA Live Rendering/Network
    • Type: integer
    • Default value: 0
  • RenderEngine
    Role: Render engine
    • Tab Page: /Infrastructure/CATIA Live Rendering/General
    • Type: integer
    • Default value: 0
    • Values:
      0 iray
      1 stellar
      2 irt
      3 iray_cloud
      4 nitro_cloud
      5 irt_cloud
      6 stellar_cloud
  • RenderMode
    Role: Render mode
    • Tab Page: /Infrastructure/CATIA Live Rendering/General
    • Type: string
    • Default value: iray:mixed; irt:cuda
  • SaveDirectory
    Role: Automatic save directory
    • Tab Page: /Infrastructure/CATIA Live Rendering/General
    • Type: URN
    • Default value:
  • SaveExtension
    Role: Automatic save file extension
    • Tab Page: /Infrastructure/CATIA Live Rendering/General
    • Type: string
    • Default value: jpg
  • SaveFrameLimit
    Role: Frame number for automatic save
    • Tab Page: /Infrastructure/CATIA Live Rendering/General
    • Type: integer
    • Default value: 100
    • Min value: 1
    • Max value: 100000000
  • SaveLimit
    Role: Automatic save limit type
    • Tab Page: /Infrastructure/CATIA Live Rendering/General
    • Type: integer
    • Default value: 0
  • SaveName
    Role: Automatic save file name
    • Tab Page: /Infrastructure/CATIA Live Rendering/General
    • Type: string
    • Default value: Image
  • SaveStatus
    Role: Enable automatic save
    • Tab Page: /Infrastructure/CATIA Live Rendering/General
    • Type: integer
    • Default value: 0
  • SaveTimeLimit
    Role: Time limit for automatic save (in seconds)
    • Tab Page: /Infrastructure/CATIA Live Rendering/General
    • Type: integer
    • Default value: 60
    • Min value: 1
    • Max value: 100000000
  • SaveType
    Role: Automatic save file format
    • Tab Page: /Infrastructure/CATIA Live Rendering/General
    • Type: integer
    • Default value: 8
  • UseIRT
    Role: Use Iray Interactive instead of Photoreal
    • Type: boolean
    • Default value: false
  • VideoFrameRate
    Role: Cloud render maximum frame rate
    • Tab Page: /Infrastructure/CATIA Live Rendering/General
    • Type: integer
    • Default value: 15
    • Min value: 1
    • Max value: 120
  • VideoMaxLag
    Role: Streaming maximum frame lag
    • Tab Page: /Infrastructure/CATIA Live Rendering/General
    • Type: integer
    • Default value: 3
    • Min value: 1
    • Max value: 99
