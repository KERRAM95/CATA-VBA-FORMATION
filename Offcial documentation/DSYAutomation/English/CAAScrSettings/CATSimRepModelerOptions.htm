CATSimRepModelerOptions Setting Repository
Quick Reference

CATSimRepModelerOptions

Role: Manage options for simulation

Detailed role: Manage options for simulation

The CATSimRepModelerOptions repository has the following characteristics:

  • Primary Tab Page: /AnalysisSimulation/Simulation
  • Repository File: CATSimRepModelerOptions
  • Version: 2
  • Full Compatibility: 1
  • Upward Compatibility: 1

CATSimRepModelerOptions setting repository attributes:

  • DynLoadingMode
    Role: (enum: Enable always, Enable when parts are many, Disable always) Choice of behavior for dynamic loading of parts.
    • Type: integer
    • Default value: 0
    • Values:
      0 Enable always (default)
      1 Enable when parts are many
      2 Disable always
  • DynLoadingNbPartsLimit
    Role: (quantity) Threshold value for conditionally enabling/disabling the dynamic loading of parts.
    Detailed role: Dynamic loading will be automatically enabled whenever the simulation has more than the specified number of parts. Otherwise, dynamic loading will be disabled. (This attribute is applicable only when DynLoadingMode == Enable when parts are many.)
    • Type: integer
    • Default value: 50
    • Min value: 0
    • Max value: 1000000
