AssemblyDesignImport Setting Repository
Quick Reference

AssemblyDesignImport

Role: Setting to manage Assembly Design Import options

Detailed role: Assembly Design Import setting contains all options necessary to Assembly Design Import

The AssemblyDesignImport repository has the following characteristics:

  • Primary Tab Page: /3DExperience Platform Common Preferences/Import
  • Repository File: AssemblyDesignImport
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

AssemblyDesignImport setting repository attributes:

  • AsmSymFbdiActivateStructAssocMode
    Role: Activate Structure Associativity
    Detailed role: Activate the Structure Associativity for the imported Assembly Symmetries.
    • Type: boolean
    • Default value: false
  • AsmSymFbdiAutoSolveStructDiffMode
    Role: Automatically solve structure differences
    Detailed role: Automatically solve structure differences of the Assembly Symmetries. Products that do not have counterparts are commited.
    • Type: boolean
    • Default value: false
  • AssemblyFeatureFbdiMode
    Role: File based Design Import of Assembly Features mode
    Detailed role: According to this attribute's value, the file based design import of assembly features can be managed in three differents ways
    • Type: integer
    • Default value: 1
    • Default Value Role: By default Import as Assembly feature with functional specification and functional impacts. V5 impacts are imported but deactivated.
    • Values:
      0 Import only V5 impacts
      1 Import as Assembly feature with functional specification and functional impacts. V5 impacts are imported but deactivated.
      2 Import as Assembly feature with functional specification and functional impacts. V5 impacts are not imported
  • BrokenCstsFbdiMode
    Role: File based design import of broken Constraints mode
    Detailed role: According to this attribute's value, the file based design import of broken Constraints can be managed in three differents ways
    • Type: integer
    • Default value: 0
    • Default Value Role: By default Broken Constraints are imported and deactivated
    • Values:
      0 Broken Constraints are imported and deactivated
      1 Broken Constraints are imported
      2 Broken Constraints are not imported
