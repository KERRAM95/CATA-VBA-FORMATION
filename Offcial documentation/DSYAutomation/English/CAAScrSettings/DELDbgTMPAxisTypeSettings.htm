DELDbgTMPAxisTypeSettings Setting Repository
Quick Reference

DELDbgTMPAxisTypeSettings

Role: Axis visualization settings for resource applications

Detailed role: Visualization parameters for different types of axis encountered in resource applications. These axis are used to represent different types of frames on the resources.

The DELDbgTMPAxisTypeSettings repository has the following characteristics:

  • Primary Tab Page: /Delmia Infrastructure/Resource axis
  • Repository File: DELDbgTMPAxisTypeSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Root tab page /x reused as /x
