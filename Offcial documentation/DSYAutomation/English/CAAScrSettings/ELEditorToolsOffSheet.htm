ELEditorToolsOffSheet

Role: Setting to handle CAA model parameters

The ELEditorToolsOffSheet repository has the following characteristics:

  • Primary Tab Page:  
  • Repository File: ELEditorToolsOffSheet
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

ELEditorToolsOffSheet setting repository attributes:

  • OffSheetWarningMessage
    Role: Warning message display
    Detailed role: Display a warning message display when attempt to break a cross reference
    • Type: integer
    • Default value: 1
    • Values:
      0 Not displayed
      1 Displayed
  • OffSheetXRefBreak
    Role: Allow or deny cross reference break
    • Type: integer
    • Default value: 1
    • Default Value Role: Allow
    • Values:
      1 Allow
      2 Deny
