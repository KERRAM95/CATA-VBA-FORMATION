SectioningSettings Setting Repository
Quick Reference

SectioningSettings

Role: Setting to handle the Sectioning parameters

The SectioningSettings repository has the following characteristics:

  • Primary Tab Page: /Digital Mockup/DMU Space Analysis/DMU Sectioning
  • Repository File: SectioningRepository
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

SectioningSettings setting repository attributes:

  • ComputeWireIntersectionsMode
    Role: Default mode to compute wireframe intersections results or not.
    • Type: integer
    • Default value: 0
    • Values:
      0 Do NOT compute wireframe intersections results for section during creation.
      1 Compute wireframe intersections results for section during creation.
  • OnlyPlaneDynMode
    Role: Default mode for only plane dynamic
    • Type: integer
    • Default value: 0
    • Values:
      0 Do NOT use only plane dynamic mode
      1 Use only plane dynamic mode
  • ResizeInAllDirMode
    Role: Default mode for resize behavior during manipulation using single face.
    • Type: integer
    • Default value: 0
    • Values:
      0 Resize clipping box size only in manipulation direction.
      1 Resize clipping box in all directions during manipulation.
  • SectionContourColor
    Role: Default color of the section contour
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: No Color by default.
    • Min value: 0
    • Max value: 255
  • SectionContourDominant
    Role: Default mode for section contour dominant behavior
    • Type: integer
    • Default value: 1
    • Values:
      0 Section contour will not be seen dominant in viewer when behing solid.
      1 Section contour will be seen dominant in viewer when behing solid.
  • SectionInfoDispMode
    Role: Default mode for Display Section Information
    • Type: integer
    • Default value: 1
    • Values:
      0 Do NOT display section information in 3D
      1 Display Section information in 3D
  • SectionMode
    Role: Default choice for the main viewer of section result
    • Type: integer
    • Default value: 4
    • Values:
      0 Borders
      1 Fill
      2 Clash2D
      3 Cross View
      4 Cut View
  • SectionPlaneVisuMode
    Role: Default choice for the plane visu mode.
    • Type: integer
    • Default value: 0
    • Values:
      0 Contour with plane
      3 Contour with grid plane
      1 Only contour
  • SectionPositionalLinkMode
    Role: Create positional link for section
    • Type: integer
    • Default value: 0
    • Values:
      0 Do NOT use pre-selection to create positional link for section during creation.
      1 Use pre-selection to create positional link for section during creation.
  • SectionUpdateMode
    Role: Default mode for section automatic update mode on product structure modifications.
    • Type: integer
    • Default value: 1
    • Values:
      0 Do NOT update section on product structure modifications.
      1 Update section on product structure modifications.
  • SnapOnGeomMode
    Role: Default mode for snap on geometry mode during manipulation using single face.
    • Type: integer
    • Default value: 1
    • Values:
      0 Do not snap on geometry under mouse during manipulation.
      1 Snap on geometry under mouse during manipulation.
