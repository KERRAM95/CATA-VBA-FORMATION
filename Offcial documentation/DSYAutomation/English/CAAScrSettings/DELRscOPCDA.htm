DELRscOPCDA Setting Repository
Quick Reference

DELRscOPCDA

Role: OPC classic options

Detailed role: These options allow to specify default values for OPC classic options

The DELRscOPCDA repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/Robotics/RRS
  • Repository File: DELRscOPCDA
  • Version: 2
  • Full Compatibility: 1
  • Upward Compatibility: 1

DELRscOPCDA setting repository attributes:

  • DELRscOPCDASignalDefaultDirection
    Role: Indicates the default direction of the signal for read-write data.
    Detailed role: This settings allows to specify what is the default direction to apply when creating a signal for OPC DA when the data is read-write.
    • Type: unsigned integer
    • Default value: 0
    • Values:
      0 None
      1 Input
      2 Output
