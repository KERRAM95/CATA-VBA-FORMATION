MarkerSetting Setting Repository
Quick Reference

MarkerSetting

Role: Setting to handle the markers visual properties and parameters

The MarkerSetting repository has the following characteristics:

  • Primary Tab Page: /Digital Mockup/General
  • Repository File: Marker
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

MarkerSetting setting repository attributes:

  • Associativity
    Role: To enable or disable the associativity of markers
    • Tab Page: /Digital Mockup/Markers
    • Type: boolean
    • Default value: true
  • GridSpace
    Role: Size of the Grid
    • Type: boolean
    • Default value: true
  • GridSpaceValue
    Role: Setting the value of grid spacing
    • Type: unsigned integer
    • Default value: 1
    • Min value: 1
    • Max value: 10000000
  • IconRatioToTextSize
    Role: Icon ratio to text size
    • Type: unsigned integer
    • Default value: 2
    • Min value: 1
    • Max value: 10
  • LabelMarkerGraphicProps
    Role: Default value for graphic properties edition of a Label Marker.
    • Tab Page: /Digital Mockup/Markers
    • Type: boolean
    • Default value: false
  • Marker2DName
    Role: Default marker name selection
    • Tab Page: /Digital Mockup/Markers
    • Type: unsigned integer
    • Default value: 0
    • Min value: 0
    • Max value: 15
  • Marker3DName
    Role: Default marker name selection
    • Tab Page: /Digital Mockup/Markers
    • Type: unsigned integer
    • Default value: 0
    • Min value: 0
    • Max value: 8
  • MarkerAutoCustomiseMode
    Role: To enable or disable the auto customization of markers visual properties
    • Tab Page: /Digital Mockup/Markers
    • Type: boolean
    • Default value: false
  • MarkerDefaultColor
    Role: Default color of a marker
    • Tab Page: /Digital Mockup/Markers
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: Red in RGB order
    • Min value: 0
    • Max value: 255
  • MarkerDefaultColorNew
    Role: Default color of a marker
    • Tab Page: /Digital Mockup/Markers
    • Type: unsigned integer
    • Default value: 47
    • Default Value Role: Red in RGB order
    • Min value: 0
    • Max value: 255
  • MarkerDefaultDashed
    Role: Default dashed value of a marker
    • Tab Page: /Digital Mockup/Markers
    • Type: unsigned integer
    • Default value: 1
    • Min value: 1
    • Max value: 7
  • MarkerDefaultWeight
    Role: Default weight value of a marker
    • Tab Page: /Digital Mockup/Markers
    • Type: unsigned integer
    • Default value: 2
    • Default Value Role: 1 mm
    • Min value: 1
    • Max value: 55
  • MarkerDocMode
    Role: Document Management
    • Tab Page: /Digital Mockup/Markers
    • Type: unsigned integer
    • Default value: 0
    • Min value: 0
    • Max value: 2
  • MarkerFillColor
    Role: Default color to fill
    • Tab Page: /Digital Mockup/Markers

    • Default value: 255
    • Default Value Role: >White in RGB order
    • Min value: 0
    • Max value: 255

    Role: Default color to fill
