The EMPFormboard repository has the following characteristics:

  • Primary Tab Page: /Equipments/Electrical Harness Flattening/Formboard
  • Repository File: EMPFormboard
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

EMPFormboard setting repository attributes:

  • BackboneVectorPitch
    Role: Option to select the backbone vector field pitch
    Detailed role: Option to select the backbone vector field pitch
    • Type: double
    • Default value: 0.005
    • Min value: 0.0
    • Max value: 10e+307
    • Step: 0.001
  • BoundingBoxLength
    Role: Option to select the maximum length
    Detailed role: Option to select maximum length
    • Type: double
    • Default value: 2
    • Min value: 0.0
    • Max value: 10e+307
    • Step: 0.001
  • BoundingBoxWidth
    Role: Option to select the maximum length
    Detailed role: Option to select maximum length
    • Type: double
    • Default value: 1
    • Min value: 0.0
    • Max value: 10e+307
    • Step: 0.001
  • ClashDetection
    Role: Option if to show Clash Detection
    Detailed role: If the option is selected then the clashes will be detected in Prepare Layout
    • Type: integer
    • Default value: 0
    • Default Value Role: By default this option is OFF for the Clash Detection
    • Values:
      0 No Clash detection in Prepare Layout
      1 Clash detection in Prepare Layout
  • ComponentOrientation
    Role: Option to decide how to maintain orientation of device
    Detailed role: Option to decide how to maintain orientation of device
    • Type: integer
    • Default value: 0
    • Default Value Role: By default this option is component placed in optimized plane
    • Values:
      0 Component placed in optimized plane
      1 Component torsion free
  • ComponentOrientationDevices
    Role: Option to decide the orientation of devices
    Detailed role: Option to decide the orientation of devices
    • Type: integer
    • Default value: 0
    • Default Value Role: By default this option is closest preferred plane
    • Values:
      0 Closest preferred plane
      1 Torsion free
  • ComponentOrientationSupport
    Role: Option to decide the orientation of support
    Detailed role: Option to decide the orientation of support
    • Type: integer
    • Default value: 0
    • Default Value Role: By default this option is rounding orientation angle to closest to 45 degree
    • Values:
      0 Rounding orientation angle to closest to 45 degree
      1 Rounding orientation angle to closest to 90 degree
      2 Rounding orientation angle to closest to 180 degree
      3 Rounding orientation angle to torsion free
  • OffsetAmbiguityZone
    Role: Option to select the offset distance for ambiguity zone
    Detailed role: Option to select the offset distance on the secondary backbone segment of the junction for ambiguity zone
    • Type: double
    • Default value: 0.005
    • Min value: 0.0
    • Max value: 10e+307
    • Step: 0.001
  • OffsetFromGlobalXYPlane
    Role: Option to select the offset distance
    Detailed role: Option to select the offset distance from xy plane
    • Type: double
    • Default value: 0.2
    • Min value: 0.0
    • Max value: 10e+307
    • Step: 0.001
  • OffsetValue
    Role: Option to select the offset distance
    Detailed role: Option to select the offset distance from xy plane
    • Type: double
    • Default value: 200
    • Min value: 0.0
    • Max value: 10e+307
    • Step: 1
  • SolveTorsion
    Role: Option to select whether to solve torsion or not
    Detailed role: Option to select whether to solve torsion or not
    • Type: integer
    • Default value: 0
    • Default Value Role: By default this option is do not solve torsion
    • Values:
      0 Do not solve torsion
      1 solve torsion
  • TShapeAmbiguityAngle
    Role: Option to select the T-shape ambiguity zone identification angle
    Detailed role: Option to select the T-shape ambiguity zone identification angle
    • Type: double
    • Default value: 0.174533
    • Min value: 0
    • Max value: 6.28319
    • Step: 0.0174533
  • TableManagement
    Role: Option if to make manage table dimension
    Detailed role: Option if to make manage table dimension
    • Type: integer
    • Default value: 0
    • Default Value Role: By default this option is manage table
    • Values:
      0 Do not manage table
      1 Manage table
  • TangencyAtDeviceJunction
    Role: Option to select whether to keep branch definition
    Detailed role: Option to select whether to keep branch definition at device junction
    • Type: integer
    • Default value: 0
    • Default Value Role: By default this option is Do not keep branch definition
    • Values:
      0 Do not keep branch definition
      1 Keep branch definition
  • TangencyAtJunction
    Role: Option to decide whether to maintain tangency or not
    Detailed role: Option to decide whether to maintain tangency or not
    • Type: integer
    • Default value: 0
    • Default Value Role: By default this option is do not maintain tangency
    • Values:
      0 Do not maintain tangency
      1 Maintain tangency
  • TorsionManageSupportNonConstraint
    Role: Option if to make support non-constraining during torsion
    Detailed role: If the option is selected then support will not be considered as a constraining element for torsion analysis
    • Type: integer
    • Default value: 0
    • Default Value Role: By default this option is OFF for the Torsion Command
    • Values:
      0 Do not make support Non-constraining during torsion
      1 make support Non-constraining during torsion
  • TshapeJunctionIdentificationAngle
    Role: Option to select the T-shape junction identification angle
    Detailed role: Option to select the T-shape junction identification angle
    • Type: double
    • Default value: 0.174533
    • Min value: 0
    • Max value: 6.28319
    • Step: 0.0174533
