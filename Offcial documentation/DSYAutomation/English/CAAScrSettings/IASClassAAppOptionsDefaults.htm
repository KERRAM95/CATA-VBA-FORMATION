IASClassAAppOptionsDefaults Setting Repository
Quick Reference

IASClassAAppOptionsDefaults

Role: Setting to handle the ICEM Design Experience App Attributes

Detailed role: Setting to handle the default settings inside the ICEM Design Experience App

The IASClassAAppOptionsDefaults repository has the following characteristics:

  • Primary Tab Page:  
  • Repository File: IASClassAAppOptionsDefaults
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

IASClassAAppOptionsDefaults setting repository attributes:

  • ActivateIntermediatePresets
    Role: Activate intermediate presets in Omni-features dialog panels
    • Type: boolean
    • Default value: false
  • DefaultPlayMode
    Role: Default play mode for new frames in Organize
    • Type: integer
    • Default value: 0
  • DevelopmentOnly_ApproximationLayoutTest
    Role: For internal use only
    • Type: integer
    • Default value: 1
  • DevelopmentOnly_ApproximationSurfaceG0
    Role: For internal use only - G0 tolerance for surface approximation
    • Type: double
    • Default value: 0.001
  • DevelopmentOnly_ApproximationSurfaceG1
    Role: For internal use only - G1 tolerance for surface approximation
    • Type: double
    • Default value: 0.0043633231
  • DevelopmentOnly_FixOrderFactoryDefault
    Role: Specify factory default for Approximation Fix Order
    • Type: integer
    • Default value: 6
  • DevelopmentOnly_FixOrderMax
    Role: Specify max value for Approximation Fix Order
    • Type: integer
    • Default value: 8
  • KeyboardLayout
    Role: Keyboard layout: automatic detection, US, UK, FR, DE or JA
    • Type: integer
    • Default value: 0
  • KeyboardLayoutLanguage
    Role: Keyboard layout: automatic detection, US, UK, FR, DE or JA
    • Type: string
    • Default value: "en-US"
  • MultiSelectionActivationWithCTRL
    Role: Use CTRL key for multi selection activation.
    • Type: integer
    • Default value: 1
  • ProjectionOnSupport
    Role: Project curve(s) to support surface(s)
    • Type: boolean
    • Default value: true
  • PtBasedCmdsAutoCloseBehaviour
    Role: Auto ok for point and patch
    • Type: boolean
    • Default value: false
  • SelectionBehaviour
    Role: The selection behaviour: depth, all till shaded, all
    • Type: integer
    • Default value: 2
  • SelectionFilterLastDefaults
    Role: Use last defaults for selection filter settings.
    • Type: integer
    • Default value: 0
  • SelectionFilterListVisibility
    Role: Visibility of selection filter list widget.
    • Type: integer
    • Default value: 0
  • SelectionPickThroughWireframe
    Role: Enable pick through wireframe surfaces.
    • Type: integer
    • Default value: 0
  • SelectionSetsGuided
    Role: Is selection set guided enabled or not
    • Type: boolean
    • Default value: true
  • SelectionTableAuxiliaryViewerButtonState
    Role: Default state of check button that enables / disables the auxiliary viewer in Selection Table
    • Type: integer
    • Default value: 1
  • SelectionTableBehaviour
    Role: The selection table behaviour: automatic or on shortcut
    • Type: integer
    • Default value: 0
  • SelectionTableFilterByDimension
    Role: The Selection Table filters the elements by the lowest dimension appearing in the list of objects.
    • Type: integer
    • Default value: 1
  • SelectionTableListPathNameLength
    Role: The length of elements in list: Short, medium, long
    • Type: integer
    • Default value: 0
  • SelectionTableShowAllControls
    Role: Show all types of controls in selection table list.
    • Type: integer
    • Default value: 0
  • SelectionTraceTrapButtonConfig
    Role: Mouse button configuration for trace/trap selection.
    • Type: integer
    • Default value: 1
  • SurfLikeSelection
    Role: Is selection on filter button press enabled or not
    • Type: boolean
    • Default value: true
  • TrapSelectionDepth
    Role: The selection depth of when using trap selection: nearest only or full depth
    • Type: integer
    • Default value: 1
  • TrapSelectionSubElements
    Role: Select also the sub-elements when doing a trap selection or not
    • Type: integer
    • Default value: 1
