SIMAnimCamViewerSettings Setting Repository
Quick Reference

SIMAnimCamViewerSettings

Role: Settings to define the options for Camera Viewers

Detailed role: These settings allow users to customize the options for Camera Viewers

The SIMAnimCamViewerSettings repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/DELMIA Infrastructure/Camera Viewers
  • Repository File: SIMAnimCamViewerSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

SIMAnimCamViewerSettings setting repository attributes:

  • CamViewExportImageFormat
    Role: Camera Viewer Export Image Format
    • Type: integer
    • Default value: 1
  • CamViewExportImageHeight
    Role: Camera Viewer Export Image Height
    • Type: integer
    • Default value: 600
    • Min value: 1
    • Max value: 10000
  • CamViewExportImagePath
    Role: Camera Viewer Export Image Path
    • Type: URN
    • Default value: UNIX: ${CATTemp} Windows: ${CATTemp}
  • CamViewExportImageWidth
    Role: Camera Viewer Export Image Width
    • Type: integer
    • Default value: 800
    • Min value: 1
    • Max value: 10000
