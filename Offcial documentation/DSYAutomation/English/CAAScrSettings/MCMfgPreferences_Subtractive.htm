MCMfgPreferences_Subtractive

Role: Lists the settings of the Manufacturability Check application for subtractive process

Detailed role: The Manufacturability Check application settings are useful to customize the checks that will be performed whithin the app for subtractive process

The MCMfgPreferences_Subtractive repository has the following characteristics:

  • Primary Tab Page: /Simulation/Additive_Manufacturing
  • Repository File: MCMfgPreferences_Subtractive
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
