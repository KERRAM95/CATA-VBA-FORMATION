IFC Setting Repository
Quick Reference

IFC

Role:

The IFC repository has the following characteristics:

  • Primary Tab Page: /General/Compatibility/IFC
  • Repository File: IFC
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

IFC setting repository attributes:

  • Exact geometry as
    Role: Excat geometry as
    • Alias: ExportDTVMode
    • Type: short integer
    • Default value: 0
    • Values:
      0 Export exact geometry in IFC file
      1 Export Model View Definition geometry in IFC file
  • Export also NoShow entites
    Role: Enables export of the not shown entities.
    • Alias: ShowMode
    • Type: short integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • FullyExposed:
    Role: Expose or not the entire structure:
    • Alias: FullyExposed
    • Type: short integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • IFC Geometry as:
    Role: IFC Geometry as:
    • Alias: ImportRepPreferedMode
    • Type: short integer
    • Default value: 1
    • Values:
      0 Import geometry as Exact on IFC file
      1 Import geometry as Tessellated on IFC file
      2 Import geometry as optimized Brep on IFC file
  • IFCReleaseVersionDep
    Role: IFC Deprecated Schema
    • Alias: IFCReleaseVersionDep
    • Type: short integer
    • Default value: 0
    • Values:
      0 IFC2X3
      1 IFC4
      2 IFC4X3_ADD2
      3 IFC4X1
      4 IFC4X2
  • IFCReleaseVersionIso
    Role: IFC ISO Schema
    • Alias: IFCReleaseVersionIso
    • Type: short integer
    • Default value: 0
    • Values:
      0 IFC2X3
      1 IFC4
      2 IFC4X3_ADD2
  • ImportAttributes
    Role: Import or not attributes of the IFC file
    • Alias: ImportAttributes
    • Type: short integer
    • Default value: 1
    • Min value: 0
    • Max value: 1
  • Profile prefered:
    Role: Profile prefered:
    • Alias: ImportProfileMode
    • Type: short integer
    • Default value: 0
    • Values:
      0 Import the entire structure grouped under a CGR from deepest spatial product
      1 Create a full PLM 3DExperience Model
      2 Create a lightweight model while keeping exact geometry and the IFC attributes
