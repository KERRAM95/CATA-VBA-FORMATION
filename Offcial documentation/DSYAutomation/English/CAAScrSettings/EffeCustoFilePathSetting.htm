EffeCustoFilePathSetting Setting Repository
Quick Reference

EffeCustoFilePathSetting

Role: To select the file for customisation of effectivity expression.

Detailed role: The file is selcted from hard drive to customise the effectivity expression.

The EffeCustoFilePathSetting repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/Configuration
  • Repository File: EffeCustoFilePathSetting
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

EffeCustoFilePathSetting setting repository attributes:

  • ChkDefaultMode
    Role: To set the default Filtering mode.
    Detailed role: User can customise the default filtering mode as 150%.
    • Type: integer
    • Default value: 0
    • Default Value Role: By default Filtering Mode will be Standerd.
  • EffectivityView
    Role: To set the old view of effectivity tabs
    Detailed role: Allow to remember the old selection of the user for effectivity tabs of properties panel
    • Type: integer
    • Default value: 0
    • Default Value Role: By default official view will be selected.
  • ExpandedInheritedEff
    Role: To set the old view of effectivity tabs
    Detailed role: Allow to remember the old selection of the user for effectivity tabs of properties panel
    • Type: integer
    • Default value: 0
    • Default Value Role: By default official view will be selected.
  • ShowDetailsEff
    Role: To set the old view of effectivity tabs
    Detailed role: Allow to remember the old selection of the user for effectivity tabs of properties panel
    • Type: integer
    • Default value: 0
    • Default Value Role: By default official view will be selected.
