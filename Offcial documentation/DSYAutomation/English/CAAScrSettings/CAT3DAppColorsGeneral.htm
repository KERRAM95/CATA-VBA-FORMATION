CAT3DAppColorsGeneral Setting Repository
Quick Reference

CAT3DAppColorsGeneral

Role: General setting of the selected color template connected for CAT3DApp utilities

Detailed role: The global option (in MePreferences) to select the color template for CAT3DApp utilities

The CAT3DAppColorsGeneral repository has the following characteristics:

  • Primary Tab Page:  
  • Repository File: CAT3DAppColorsGeneral
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CAT3DAppColorsGeneral setting repository attributes:

  • SelectedTemplate
    Role: Basic type of the selected color template
    Detailed role: Basic type of the selected color template - custom templates are part of User Defined
    • Type: integer
    • Default value: 0
    • Values:
      0 Default
      1 High Contrast
      2 User Defined
  • SelectedTemplateName
    Role: Name of the selected custom color template
    Detailed role: Name of the selected custom color template - Do not expose!
    • Type: string
    • Default value: "Default"
