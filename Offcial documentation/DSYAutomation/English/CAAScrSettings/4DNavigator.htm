4DNavigator Setting Repository
Quick Reference

4DNavigator

Role: Setting to handle the DMU parameters

The 4DNavigator repository has the following characteristics:

  • Primary Tab Page: /Digital Mockup/General
  • Repository File: N4DNavigator
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Setting controller:
    • Framework_CPP: NavigatorInterfaces
    • Framework_IDL: NavigatorInterfaces
    • Interface_CPP: CATIDMUN4DNavigatorSettingAtt
    • Interface_IDL: CATIAN4DNavigatorSettingAtt
    • LateType: CATDMUN4DNavigatorAutoSettingCtrl

4DNavigator setting repository attributes:

  • DMUGroundColorOfCurvesLink
    Role: Link between both curves distances
    • Tab Page: /Digital Mockup/DMU 3D Ground
    • Type: unsigned integer
    • Default value: 1
    • Min value: 0
    • Max value: 1
  • DMUGroundColorOfFirstDirectionCurves
    Role: Color of the curves in the first direction
    • Tab Page: /Digital Mockup/DMU 3D Ground
    • Type: unsigned integer
    • Default value: 255
    • Default Value Role: Red in RGB order
    • Min value: 0
    • Max value: 255
  • DMUGroundColorOfSecondDirectionCurves
    Role: Color of the curves in the second direction
    • Tab Page: /Digital Mockup/DMU 3D Ground
    • Type: unsigned integer
    • Default value: 255
    • Default Value Role: Red in RGB order
    • Min value: 0
    • Max value: 255
  • DMUGroundDisplayLabels
    Role: The DMU Ground can have some labels
    • Tab Page: /Digital Mockup/DMU 3D Ground
    • Type: unsigned integer
    • Default value: 1
    • Min value: 0
    • Max value: 1
  • DMUGroundDisplayMode
    Role: The display of the DMU Ground can be choosen between lines or crosses
    • Tab Page: /Digital Mockup/DMU 3D Ground
    • Type: unsigned integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • DMUGroundDisplayShadingBottomColor
    Role: Color of the shading of the bottom of the DMU Ground
    • Tab Page: /Digital Mockup/DMU 3D Ground
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: Red in RGB order
    • Min value: 0
    • Max value: 255
  • DMUGroundDisplayShadingBottomTransparency
    Role: Transparency of the shading of the bottom of the DMU Ground
    • Tab Page: /Digital Mockup/DMU 3D Ground
    • Type: unsigned integer
    • Default value: 2
    • Min value: 0
    • Max value: 4
  • DMUGroundDisplayShadingStatus
    Role: The DMU Ground can be shaded
    • Tab Page: /Digital Mockup/DMU 3D Ground
    • Type: unsigned integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • DMUGroundDisplayShadingTopColor
    Role: Color of the shading of the top of the DMU Ground
