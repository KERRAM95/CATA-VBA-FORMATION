RenderingBatchSettings Setting Repository
Quick Reference

RenderingBatchSettings

Role: Live Rendering Batch options

The RenderingBatchSettings repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/CATIA Live Rendering/Batch
  • Repository File: RenderingBatchSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Relation;
    • Role: Number of render passes
    • Detailed role:
    • src: RenderPassCount
    • dest: CanvasArray
    • kind: dimensionOfArray
