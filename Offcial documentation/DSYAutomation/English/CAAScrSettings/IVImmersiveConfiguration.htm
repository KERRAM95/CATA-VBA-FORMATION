IVImmersiveConfiguration

Role: Setting to handle the immersive configuration parameters

Detailed role: Setting to handle the immersive configuration parameters

The IVImmersiveConfiguration repository has the following characteristics:

  • Primary Tab Page: /General/VirtualReality
  • Repository File: IVImmersiveConfiguration
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Relation;
    • Role: Size of multipipe environment files
    • Detailed role:
    • src: ImmersiveEnvironmentFilesCount
    • dest: ImmersiveEnvironmentFiles
    • kind: dimensionOfArray

IVImmersiveConfiguration setting repository attributes:

  • ImmersiveEnvironmentFiles
    Role: Multipipe environment files
    • Type: URN
    • Default value:
  • ImmersiveEnvironmentFilesCount
    Role: Number of multipipe environment files
    • Type: unsigned integer
    • Default value: 0
  • QuadView
    Role: Enables quad-view
    • Type: boolean
    • Default value: false
  • Stereoscopy
    Role: Enables support of stereoscopic display
    • Type: boolean
    • Default value: false
  • Synchronization
    Role: Enables synchronization between several displays
    • Type: boolean
    • Default value: false
