CATPublishingSettings Setting Repository
Quick Reference

CATPublishingSettings

Role: Publishing Settings

Detailed role: Publishing settings allow to configure the publishing parameters

The CATPublishingSettings repository has the following characteristics:

  • Primary Tab Page: /General/Options/GraphicsFormat
  • Repository File: CATPublishingSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
