FbdiSplitRep Setting Repository
Quick Reference

FbdiSplitRep

Role: Settings for the split import.

Detailed role: It allows to define the migration in split import mode.

The FbdiSplitRep repository has the following characteristics:

  • Primary Tab Page: /General/FBDI/Split and Import
  • Repository File: FbdiSplitRep
  • Version: 2
  • Full Compatibility: 1
  • Upward Compatibility: 1

FbdiSplitRep setting repository attributes:

  • FBDI_ComputeMode
    Role: Switch between manual and automatic split.
    • Type: integer
    • Default value: 1
  • FBDI_EnableMaxDocs
    Role: Set a maximum number of documents.
    • Type: boolean
    • Default value: 1
  • FBDI_EnableMaxSize
    Role: Set a maximum documents size.
    • Type: boolean
    • Default value: 1
  • FBDI_HostCPUs
    Role: This option allows to control the number of CPUs for each host.
    • Type: integer
    • Default value: 1
  • FBDI_HostNames
    Role: This option allows to control the hostnames to use.
    • Type: string
    • Default value:
  • FBDI_MaxNumberOfDocuments
    Role: Maximum number of documents inside a package.
    • Type: integer
    • Default value: 100
  • FBDI_MaxSizeOfDocuments
    Role: Maximum pacakge size in megabytes.
    • Type: double
    • Default value: 100
  • FBDI_MultiProMode
    Role: Switch between local and multi-processing mode.
    • Type: integer
    • Default value: 0
