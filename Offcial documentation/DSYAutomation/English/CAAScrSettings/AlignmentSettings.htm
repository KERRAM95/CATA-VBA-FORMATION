AlignmentSettings Setting Repository
Quick Reference

AlignmentSettings

Role: Settings to manage Alignment options

The AlignmentSettings repository has the following characteristics:

  • Primary Tab Page: /App Preferences/3D Modeling/Multi Discipline Engineering/Civil Engineering 3D Design/Alignment
  • Repository File: AlignmentSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

AlignmentSettings setting repository attributes:

  • AlignmentSolvingTime
    Role: Solving Time
    Detailed role: Parameter to set the maximum alignment solving time.
    • Type: double
    • Default value: 2
    • Min value: 0.001
    • Max value: 3600
