MOTA3DMotionAnalysisPreferencesStructure Setting Repository
Quick Reference

MOTA3DMotionAnalysisPreferencesStructure

Role: Setting to manage Motion Analysis options

Detailed role: Motion Analysis setting contains all options necessary to Motion Analysis commands

The MOTA3DMotionAnalysisPreferencesStructure repository has the following characteristics:

  • Primary Tab Page: /Physics Simulation and Simulation for Designers/Motion/General
  • Repository File: MOTA3DMotionAnalysisPreferencesStructure
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Root tab page /Physics Simulation and Simulation for Designers/Motion/Joint Solve reused as /Physics Simulation and Simulation for Designers/Motion/Joint Solve

MOTA3DMotionAnalysisPreferencesStructure setting repository attributes:

  • CopyPasteOfMarkerForMotion
    Role: Attribute to define behavior of copied marker.
    Detailed role: According to the attribute's value, copy of marker across body can keep absolute position or relative position.
    • Tab Page: /Physics Simulation and Simulation for Designers/Motion/Markers
    • Type: integer
    • Default value: 0
    • Default Value Role: By default use the absolute position and orientation of the copied marker
    • Values:
      0 Use the absolute position and orientation of the copied marker
      1 Use the relative position and orientation of the copied marker
  • ElementContextOption
    Role: Creation context of new probes and external loads.
    Detailed role: Defines whether to create functions, drivers, applied loads, and probes in the mechanism or analysis case.
    • Tab Page: /Physics Simulation and Simulation for Designers/Motion/General Modeling
    • Type: integer
    • Default value: 2
    • Default Value Role: Defines whether to create functions, drivers, applied loads, and probes in the mechanism or analysis case
    • Values:
      0 Always create in current mechanism
      1 Always create in current analysis case
      2 Always choose context upon creation
  • MechanismSolveManagementForMotion
    Role: Attribute to activate/deactivate animation during joint solve.
    Detailed role: Animate body positioning during joint definition.
    • Tab Page: /Physics Simulation and Simulation for Designers/Motion/Joint Solve
    • Type: boolean
    • Default value: true
    • Default Value Role: By default, animate the joint solve.
