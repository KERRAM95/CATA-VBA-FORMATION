CATSrsVisuAssistantPage

Role: Setting to handle SRS Visu Assistant option

The CATSrsVisuAssistantPage repository has the following characteristics:

  • Primary Tab Page: /MechanicalDesign/CATSrsPLMRepConfig/DesignAssistant
  • Repository File: CATSrsVisuAssistantPage
  • Version: 3
  • Full Compatibility: 1
  • Upward Compatibility: 2

CATSrsVisuAssistantPage setting repository attributes:

  • ApplySettingAutomatically
    Role: Setting to apply contract settings Automatically
    • Type: boolean
    • Default value: false
  • DeepSearch
    Role: Activation state of Deep Search
    • Type: boolean
    • Default value: false
  • NbOfNodeOnDummy
    Role: Activation state of displaying number of node on dummy
    • Type: boolean
    • Default value: false
  • SketchOrientation
    Role: Activation state of sketch orientation
    • Type: boolean
    • Default value: false
