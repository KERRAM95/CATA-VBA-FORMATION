IHUGeneralSettings Setting Repository
Quick Reference

IHUGeneralSettings

Role: Setting to handle human general settings parameters

The IHUGeneralSettings repository has the following characteristics:

  • Primary Tab Page: /Shape/Human Design/General
  • Repository File: IHUGeneralSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

IHUGeneralSettings setting repository attributes:

  • CustomNPresetDefaultAction
    Role:
    • Type: string
    • Default value: ComboItem0
    • Values:
      ComboItem0
      ComboItem1
  • CustomVisuProjectionType
    Role:
    • Type: string
    • Default value: ComboItem1
    • Values:
      ComboItem0
      ComboItem1
  • EnableCustomVisu
    Role: Enable custom visu
    • Type: boolean
    • Default value: true
  • EnableCustomVisuAO
    Role: Enable ambient occlusion
    • Type: boolean
    • Default value: true
  • EnableCustomVisuReflections
    Role: Enable reflections
    • Type: boolean
    • Default value: true
  • FirstPersonViewDefaultAction
    Role:
    • Type: string
    • Default value: ComboItem2
    • Values:
      ComboItem0
      ComboItem1
      ComboItem2
  • MotionRootOnFloorAfterCreate
    Role: Apply motion root on floor after creation
    • Type: boolean
    • Default value: true
  • ShowCustomVisuStartupDialog
    Role: Show startup dialog
    • Type: boolean
    • Default value: true
  • ShowTopBannerIK
    Role: Show IK top banner
    • Type: boolean
    • Default value: true
  • ShowTopBannerLookAt
    Role: Show LookAt top banner
    • Type: boolean
    • Default value: true
  • SkeletonViewMode
    Role:
    • Type: string
    • Default value: ComboItem2
    • Values:
      ComboItem0
      ComboItem1
      ComboItem2
  • ViewAngleDefaultAction
    Role: View Angle
    Detailed role: View angle to used in first person view
    • Type: integer
    • Default value: 16
    • Min value: 1
    • Max value: 89
