CAT3DPPreferencesRepository Setting Repository
Quick Reference

CAT3DPPreferencesRepository

Role: Settings customizing Functional Generative Design workbench and commands

Detailed role: CATIA Functional Generative Design Workbench Settings

The CAT3DPPreferencesRepository repository has the following characteristics:

  • Primary Tab Page: /Mechanical/CAT3DPrintingDesignWbk/General
  • Repository File: CAT3DPPreferencesRepository
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CAT3DPPreferencesRepository setting repository attributes:

  • AdvancedMode
    Role: Display the UI for advanced users.
    • Alias: AdvancedMode
    • Type: boolean
    • Default value: 0
  • AllowFaceSelection
    Role: Allow selection of faces in the functional regions command.
    • Alias: AllowFaceSelection
    • Type: boolean
    • Default value: 0
  • AnalysisSetupInActionBar
    Role: Add new Action Bar section with Analysis Setup commands.
    • Alias: AnalysisSetupInActionBar
    • Type: boolean
    • Default value: 0
  • DisplayMeshTypeOption
    Role: Display the mesh type option.
    • Alias: DisplayMeshTypeOption
    • Type: boolean
    • Default value: 0
  • KeepBeadToscaIterations
    Role: Keep all intermediate iterations in Compute Shape command for Bead Optimization.
    • Alias: KeepBeadToscaIterations
    • Type: boolean
    • Default value: 1
  • KeepDOFToscaIterations
    Role: Keep all intermediate iterations in Compute Shape command for Shape Optimization.
    • Alias: KeepDOFToscaIterations
    • Type: boolean
    • Default value: 1
  • KeepGDEToscaIterations
    Role: Keep all intermediate iterations in Compute Shape command for Topology Optimization.
    • Alias: KeepGDEToscaIterations
    • Type: boolean
    • Default value: 0
  • KeepSizingToscaIterations
    Role: Keep all intermediate iterations in Compute Shape command for Sizing Optimization.
    • Alias: KeepSizingToscaIterations
    • Type: boolean
    • Default value: 1
  • MaximumViewerInTradeOffStudy
    Role: Maximum viewer available in the trade off study.
    • Alias: MaximumViewerInTradeOffStudy
    • Type: integer
    • Default value: 3
    • Min value: 0
    • Max value: 255
  • PartitionBodyCopyWithLink
    Role: Use copy with link for reference design partition.
    • Alias: PartitionBodyCopyWithLink
    • Type: boolean
    • Default value: 1
  • UseOGSUnderHB
    Role: Create Ordered Geometrical Set under Hybrid Body instead of Geometrical Set.
    • Alias: UseOGSUnderHB
    • Type: boolean
    • Default value: 0
  • VolumeActivation
    Role: Switch from solid or both.
    • Alias: VolumeActivation
    • Type: integer
    • Default value: 0
    • Values:
      0 Solid
      1 Both
