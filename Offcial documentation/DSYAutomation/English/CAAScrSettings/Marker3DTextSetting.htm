Marker3DTextSetting

Role: Setting to handle the Marker visual properties and parameters

The Marker3DTextSetting repository has the following characteristics:

  • Primary Tab Page: /Design Review/Marker
  • Repository File: Marker3DText

  • Full Compatibility: 1
  • Upward Compatibility: 1
