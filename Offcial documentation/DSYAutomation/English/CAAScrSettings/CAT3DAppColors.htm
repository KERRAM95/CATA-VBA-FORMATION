CAT3DAppColors Setting Repository
Quick Reference

CAT3DAppColors

Role: Detailed settings to handle colors of the CAT3DApp utilities

Detailed role: Detailed setting to handle colors of the CAT3DApp utilities - Do not modify directly! - Use the CAT3DAppColorSettings!

The CAT3DAppColors repository has the following characteristics:

  • Primary Tab Page:  
  • Repository File: CAT3DAppColors
  • Version: 2
  • Full Compatibility: 1
  • Upward Compatibility: 1
