HvcLegendPlotOptions Setting Repository
Quick Reference

HvcLegendPlotOptions

Role: Manage options for visualization servant

Detailed role: Manage options for simulation results visualization servant

The HvcLegendPlotOptions repository has the following characteristics:

  • Primary Tab Page: /AnalysisSimulation/Servant Visualization
  • Repository File: HvcLegendPlotOptions
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

HvcLegendPlotOptions setting repository attributes:

  • 2DElementDepthEnd
    Role: Specify an end depth value relative to the plane.
    • Type: double
    • Default value: 10.0
  • 2DElementDepthStart
    Role: Specify a start depth value relative to the plane.
    • Type: double
    • Default value: 0
  • AboveMaximumColor
    Role: Pick above maximum color.
    • Type: integer
    • Default value: 98
    • Min value: 0
    • Max value: 255
  • BeamScale
    Role: For visualization enhancement, the scale factor multiplies the dimensions of the beam in the two directions perpendicular to the length of the beam.
    • Type: double
    • Default value: 1.00
  • BeamStyle
    Role: Specify the beam style.
    • Type: integer
    • Default value: 0
  • BelowMinimumColor
    Role: Pick below minimum color.
    • Type: integer
    • Default value: 98
    • Min value: 0
    • Max value: 255
  • EdgeAngle
    Role: Specify edge angle.
    • Type: double
    • Default value: 44.0
  • ExtrudePlanar
    Role: Select this option to extrude planar elements to improve their display in your results.
    • Type: integer
    • Default value: 1
  • Format
    Role: Specify format for the legend.
    • Type: integer
    • Default value: 0
  • NumDigits
    Role: Modify the number of significant digits shown for result values.
    • Type: integer
    • Default value: 3
  • NumIntervals
    Role: Specify the number of colors to be used in the legend.
    • Type: integer
    • Default value: 10
  • Orientation
    Role: Specify orientation for the legend.Default in preferences means legend orientation by default Horizontal for designer App and Vertical for Expert App.
    • Type: integer
    • Default value: 0
  • OutOfRangeColors
    Role: To control the color of values that are above or below the plot limits.
    • Type: integer
    • Default value: 1
  • OutputDirPath
    Role: Update path to Template file location
    • Type: URN
    • Default value: UNIX: ${HOME}/DSYUserTemplates Windows: ${HOMEDRIVE}${HOMEPATH}\DSYUserTemplates
  • OutputFileName
    Role: Filename for the template file
    • Type: string
    • Default value: SMAUserTemplateDefinition.xml
  • Position
    Role: Specify position for the legend.
    • Type: integer
    • Default value: 10
  • ReverseColors
    Role: Reverse the color order in the legend.
    • Type: integer
    • Default value: 0
  • ShellStyle
    Role: Specify the shell style.
    • Type: integer
    • Default value: 0
  • ShowDeformation
    Role: Display deformed phase of the plot.
    • Type: integer
    • Default value: 1
  • SmoothColors
    Role: Modify the distribution of values across the color range.
    • Type: integer
    • Default value: 1
  • SweepAxisymmetric
    Role: Select this option to sweep axisymmetric elements to improve their display in your results.
    • Type: integer
    • Default value: 0
  • SweepAxisymmetricAngleEnd
    Role: Specify an end angle value.
    • Type: double
    • Default value: 90
  • SweepAxisymmetricAngleStart
    Role: Specify a start angle value.
    • Type: double
    • Default value: 0
  • SweepAxisymmetricSegsCount
    Role: Specify the number of segments used to construct the angle.
    • Type: integer
    • Default value: 10
  • ThicknessData
    Role: Specify the thickness data source for the rendering thickness, either Section or Field.
    • Type: integer
    • Default value: 0
  • ThicknessScale
    Role: Specify the thickness scale factor, expressed as a multiplier, to enhance visibility.
    • Type: double
    • Default value: 1.00
  • VisibleEdges
    Role: Specify edge type for the visualization.
    • Type: integer
    • Default value: 0
