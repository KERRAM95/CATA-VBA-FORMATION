DNBFittingSettings Setting Repository
Quick Reference

DNBFittingSettings

Role: Settings to handle the parameters in the Assembly Path Definition workbench

Detailed role: This setting allows to customize the appearance and the behavior in Assembly Path Definition

The DNBFittingSettings repository has the following characteristics:

  • Primary Tab Page: /Digital Process for Manufacturing/Assembly Path Definition
  • Repository File: DNBFittingSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DNBFittingSettings setting repository attributes:

  • AutoReverseTrack
    Role: Automatically Reverse Track after creation
    Detailed role: When option is checked, the track direction gets reversed automatically after authoring
    • Type: integer
    • Default value: 0
  • AutomaticTPointInsertionAngleSpinner
    Role: Define the Orientation Angle of T-Point Insertion.
    Detailed role: Define the Orientation Angle of T-Point Insertion.
    • Type: double
    • Default value: 0.087266463
  • AutomaticTPointInsertionDistanceSpinner
    Role: Define the Distance of T-Point Insertion.
    Detailed role: Define the Distance of T-Point Insertion.
    • Type: double
    • Default value: 0.02
  • AutomaticTPointInsertionWhileMoving
    Role: Automatic TPoint Insertion While mouse moving.
    Detailed role: Automatic TPoint Insertion While mouse moving.
    • Type: integer
    • Default value: 0
  • CreateFuncProcTrack
    Role: Create Functional Process Track when object is selected in 3D
    Detailed role: When option is checked, Functional Process Track is created when an object is selected in 3D
    • Type: integer
    • Default value: 0
  • DisplayDistanceForTPoint
    Role: Display distance for each TPoint
    Detailed role: Display distance for each TPoint in TPoint Reorder tab page
    • Type: integer
    • Default value: 0
  • HighlightMovObjOnSel
    Role: Highlights moving objects when track is selected
    Detailed role: When option is checked, the moving objects of the selected track are highlighted
    • Type: integer
    • Default value: 0
  • HighlightMovObjOnSimu
    Role: Highlights moving objects when track is simulated
    Detailed role: When option is checked, the moving objects of the simulated track are highlighted
    • Type: integer
    • Default value: 0
  • PreviewWindowAvailability
    Role: Show the Preview Window in Track create or edit
    Detailed role: Determines whether the Preview Window is shown in Track create or edit
    • Type: integer
    • Default value: 0
  • SequenceGanttChoreoDisplay
    Role: Displays Choreography at bottom of the Assembly Sequence Gantt chart
    Detailed role: When option is checked, Choreographies are displayed at the bottom of the Assembly Sequence Gantt chart
    • Type: integer
    • Default value: 0
  • SimulateWithDistance
    Role: The track is simuated in the distance Mode
    Detailed role: When option is checked, the track is simuated in the distance Mode
    • Type: integer
    • Default value: 0
  • StepDistance
    Role: Step distance of the track
    Detailed role: Step distance of the track
    • Type: double
    • Default value: 0.0
  • SweptVolumeAngularAccuracySpinner
    Role: Define Angular accuracy value for Swept volume
    Detailed role: Define Angular accuracy value for Swept volume
    • Type: double
    • Default value: 0.087266463
  • SweptVolumeLinearAccuracySpinner
    Role: Define Linear accuracy value for Swept volume
    Detailed role: Define Linear accuracy value for Swept volume
    • Type: double
    • Default value: 0.0002
  • SweptVolumeOptionsLock
    Role: Swept volume options lock
    Detailed role: Admin lock for Swept volume options
    • Type: integer
    • Default value: 0
  • TPointAxisPatternDisplay
    Role: Displays T-Point pattern as Axis in 3D
    Detailed role: When option is checked, T-Points display pattern will be axis in 3D visu
    • Type: integer
    • Default value: 0
  • TPointCoordDisplay
    Role: Displays T-Point co-ordinates in 3D
    Detailed role: When option is checked, T-Points co-ordinates will be displayed in 3D visu
    • Type: integer
    • Default value: 0
  • TPointNameDisplay
    Role: Displays T-Point name in 3D
    Detailed role: When option is checked, T-Points name will be displayed in 3D visu
    • Type: integer
    • Default value: 0
  • TiltRestriction
    Role: Option to restrict object orientation to an angle w.r.to an axis.
    Detailed role: Option to restrict object orientation to an angle w.r.to an axis.
    • Type: integer
    • Default value: 0
  • TiltRestrictionAngleSpinner
    Role: Define an angle for tilt restriction.
    Detailed role: Define an angle for tilt restriction.
    • Type: double
    • Default value: 0.087266463
  • TiltRestrictionVectorCombo
    Role: Define the axis along which tilt is restricted.
    Detailed role: Define the axis along which tilt is restricted.
    • Type: integer
    • Default value: 0
    • Default Value Role: By default axis is Z
    • Values:
      0 ZVector
      1 YVector
      2 XVector
